# 资源与产业链

## 设计目标

> 对标《纪元1800》的产业链设计：原料→中间品→成品的每一环都有经济意义。控制关键资源就是控制战争命脉。

## 系统概述

战国时代的核心资源分为战略资源（决定军事上限）、基础资源（维持民生）和奢侈资源（提升声望/满意度）。每种资源有固定产地，产地争夺是战争的重要动机。产业链系统允许玩家将原料加工为高价值成品。

## 核心机制

### 3.1 资源分类

#### 战略资源（军事命脉）

| 资源 | 军事用途 | 主要产地 | 控制者优势 |
|------|---------|---------|-----------|
| **铁矿石** | 制造铁兵器/铁甲/农具 | 宜阳(韩)、宛城(楚)、邯郸(赵) | 装备等级+1级 |

| **海盐** | 军粮腌制/保存 | 临淄(齐)、会稽(楚) | 军粮消耗-15% |
| **池盐** | 军粮腌制/保存 | 河东(魏) | 军粮消耗-10% |
| **皮革** | 皮甲/马鞍/箭囊 | 代郡(赵)、陇西(秦) | 轻装部队防御+10% |
| **铜** | 铜兵器/铜甲/铸钱 | 宛城(楚)、中条山(魏) | 招募成本-10% |

### 3.2 产业链

#### 武器产业链

```
铁矿石(原料)  ──→  冶铁(中间品)  ──→  兵器(成品)
  25金/石          50金/石             60-80金/把
  宜阳/宛城/邯郸    需冶铁坊             需兵器坊
  
  利润梯度：原料(25) → 精炼(+100%) → 成品(+20-60%)
  
  经营属性加成：每10点经营 → 加工效率+5%
```

#### 防具产业链

```
皮革(原料)  ──→  鞣革(中间品)  ──→  皮甲/马具(成品)
  15金/张          30金/张             80-200金/套
  代郡/陇西         需制革坊             需制甲坊
  
  精铁(原料)  ──→  炼钢(中间品)  ──→  铁甲(成品)
  50金/石          120金/石             300-800金/套
  宜阳/邯郸         需炼钢坊(高级)        需制甲坊
```

#### 建筑工程产业链

```
木材(原料) ──→ 加工木材(中间品) ──→ 攻城器械/战车(成品)
 8金/石        20金/石              200-500金/台
 楚地/巴蜀     需锯木厂              需木工坊

石材(原料) ──→ 加工石料(中间品) ──→ 城墙/宫殿(成品)
 6金/石        15金/石              按工程量计算
 各地          需采石场              需石匠坊
```

#### 民生产业链

```
粮食(原料) ──→ 面粉/酿酒(加工品)
 10金/石       面粉30金/石 | 酒80-150金/坛
               需磨坊         需酿酒坊

海盐(原料) ──→ 精制盐(加工品) ──→ 腌制军粮(成品)
 15金/石       35金/石              军粮保存期+200%
 临淄/会稽     需晒盐场              需军粮坊
```

#### 奢侈品产业链

```
丝绸 → 刺绣(加工) → 锦袍(成品)
 80金/匹     200金/匹         500-2000金/件

漆器(原料) → 精漆(加工) → 漆画屏风(成品)
 45金/件     120金/件          300-1000金/件

珍珠(原料) → 珍珠饰品(成品)
 200金/颗     500-3000金/件
```

### 3.5 资源争夺

#### 资源点控制

地图上的资源点可被各势力争夺：

```
资源点类型：
├── 矿场类（铁/铜/金/银）：占领后每月产出资源
├── 农田区：占领后增加周边城池粮食产量
├── 马场：占领后获得战马产出+特殊马种
├── 盐场：占领后获得盐产出
├── 渔场：占领后增加食物供给
└── 猎场：占领后获得皮革+肉

占领方式：
├── 军事占领：派军团驻守→资源归玩家所属势力
├── 经济购买：向控制国购买开采权(每年续费)
└── 外交获取：条约/联姻获得资源点
```

## 与其他系统的交互

| 关联系统 | 交互方式 | 影响 |
|---------|---------|------|
| 封地经济 | 资源采集/加工建筑建在封地中 | 封地是生产的基础 |
| 贸易系统 | 资源可买卖/运输 | 贸易是资源流通的渠道 |
| 战斗系统 | 战略资源决定装备等级 | 控制资源=控制军力 |
| 科技系统 | 科技解锁新加工建筑/提升效率 | 铁犁→农业+30%，冶铁术→冶铁效率+50% |
| 国家策略 | 盐铁专营/资源管制政策 | 策略决定资源的归属和价格 |

## 数值范围

| 参数 | 最小值 | 默认值 | 最大值 | 说明 |
|------|--------|--------|--------|------|
