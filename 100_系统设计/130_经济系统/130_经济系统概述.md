# 经济系统概述

## 设计目标

> 让每一枚铜钱都有重量，让每一次交易都有风险，让经济成为与军事同等重要的国力支柱。

经济系统不只是"赚钱花錢"的数值循环，而是驱动世界运转的底层引擎——物价随战争波动、商路因战乱中断、货币随国力涨跌、封地经济决定军事实力上限。

## 系统架构

```
                        经济系统
                            │
    ┌───────┬───────┬───────┼───────┬───────┬───────┐
    ▼       ▼       ▼       ▼       ▼       ▼       ▼
  贸易系统  货币系统  资源与   封地经济  商队系统  市场模拟  国家财政
  (跨城    (七国   产业链   (领地    (护卫/   (供需    (税收/
  物价差)  币制)   链条)    建设)    路线)   动态)    国库)
```

## 核心循环

```
生产(采集/制造) → 运输(商队/船运) → 交易(市场/商铺) → 消费(军民需求) → 税收 → 再生产
        ↑                                                              │
        └────────────────── 战争/灾荒/政策打断 ─────────────────────────┘
```

## 设计支柱

| 支柱 | 含义 | 参考对标 |
|------|------|---------|
| **跨城物价差** | 每城独立物价，低买高卖是核心贸易玩法 | 大航海时代4 |
| **七国货币** | 不同国家使用不同货币，汇率随国力浮动 | 历史考据 |
| **产业链深度** | 原料→加工→成品，每一环都产生利润 | 纪元系列 (Anno) |
| **供需动态** | 物价根据战争/季节/灾荒/囤货实时变化 | EVE Online |
| **商路风险** | 运输途中有被劫/被征/沉船的风险 | 大航海时代4 |
| **封地经济** | 玩家领地可建设/经营/收税 | 凯撒大帝/太阁5 |

## 经济指标

### 个人经济指标

| 指标 | 说明 | 范围 |
|------|------|------|
| 持有金 | 玩家随身携带的现金 | 0 ~ 无上限 |
| 资产总值 | 现金 + 货物价值 + 房产 + 封地估值 | 0 ~ 无上限 |
| 月收入 | 封地税收 + 商铺分红 + 商队利润 - 支出 | 可为负 |
| 信用等级 | 影响借贷额度和利率 | 1-10级 |
| 经营属性 | 六维属性中的"经营"，影响贸易利润和还价能力 | 0-120 |

### 国家经济指标

| 指标 | 说明 | 影响 |
|------|------|------|
| 国库 | 国家可用资金 | 征兵/建设/外交资金上限 |
| 年税收 | 农业税 + 商业税 + 关税 + 专营收入 | 国库主要来源 |
| 年支出 | 军费 + 官员俸禄 + 工程 + 外交贡金 | — |
| 通胀率 | 货币发行量 vs 实物产出 | 物价全面波动 |
| 国力评级 | 综合经济实力 | 影响货币汇率 |

## 经济与各系统交互

| 关联系统 | 交互方式 | 影响 |
|---------|---------|------|
| 战斗系统 | 战争消耗军费/粮草，战胜获得赔款/战利品 | 经济支撑军事，军事掠夺经济 |
| 角色系统 | 经营属性影响贸易利润，商人出身利润+25% | 出身决定经济起点 |
| 国家策略 | 税收政策/专营制度/贸易禁令由策略层决定 | 策略决定经济规则 |
| 城邑建设 | 封地建筑决定产出类型和效率 | 建设提升经济产能 |
| 科技系统 | 经济科技解锁新产业链/降低交易成本 | 科技驱动经济升级 |
| 声望系统 | 信义声望影响借贷利率和贸易伙伴信任 | 声望=经济信用 |

## 经济系统六大模块

### 一、贸易系统 [→131]

- 跨城物价差（大航海式）
- 特产系统
- 市场供需动态
- 讨价还价机制
- 商铺投资

### 二、货币系统 [→132]

- 七国货币（秦半两/楚蚁鼻钱/齐刀币等）
- 货币兑换与汇率
- 通货膨胀/紧缩
- 借贷与利息

### 三、资源与产业链 [→133]

- 战略资源分布
- 原料→加工→成品链条
- 季节性产出
- 资源争夺

### 四、封地经济 [→134]

- 封地建设（凯撒式规划）
- 税收管理
- 人口与劳动力
- 满意度与民忠

### 五、商队系统

- 商队组建与护卫
- 商路规划
- 运输风险（流寇/军队/天灾）
- 海上贸易（齐楚越沿海）

### 六、国家财政

- 国库收支
- 税收政策
- 专营制度（盐铁专营等）
- 战争赔款/进贡

## 数值范围

| 参数 | 最小值 | 默认值 | 最大值 | 说明 |
|------|--------|--------|--------|------|
| 玩家初始金(农民) | — | 100 | — | 出身决定 |
| 玩家初始金(商人) | — | 5000 | — | 出身决定 |
| 单笔交易利润上限 | -100% | — | +300% | 超过300%标记为异常 |
| 商队规模 | 1辆马车 | 5辆 | 50辆 | 随等级/资金增长 |
| 封地建筑数量 | 3 | 10 | 50 | 根据爵位/封地大小 |
| 市场物价波动范围 | -50% | 基准价 | +200% | 极端情况(战争/灾荒) |
| 年利率 | 5% | 12% | 36% | 信用等级决定 |
