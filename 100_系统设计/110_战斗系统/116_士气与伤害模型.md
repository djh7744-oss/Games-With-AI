# 士气与伤害模型

## 设计目标

> 士气是战斗的隐形胜负手——一支士气高昂的弱旅可以击退士气崩溃的强军。伤害模型确保数值透明、可预测，同时保留足够的深度让玩家通过操作和策略改变结果。

## 第一部分：士气系统

### 1.1 士气基础

```
士气值范围：0-100

每支部队（编队）有独立的士气值。
全局士气 = 各编队士气的加权平均（按人数加权）。

士气状态：
  90-100：狂热 —— 攻击+15%，防御+10%，不会溃逃
  70-89：高昂 —— 攻击+5%
  50-69：正常 —— 无修正
  30-49：动摇 —— 攻击-10%，防御-10%，响应延迟+50%
  10-29：恐慌 —— 攻击-25%，防御-25%，有溃逃风险
  0-9：崩溃 —— 立即溃逃
```

### 1.3 溃逃系统

#### 溃逃触发

```
编队士气归零 → 该编队溃逃

溃逃过程：
  阶段1(0-5秒)：混乱——士兵开始溃散，编队阵型消失
  阶段2(5-30秒)：溃逃——士兵各自逃跑(向地图边缘)
  阶段3(30秒后)：彻底溃散——该编队从战场上消失

溃逃中：
  移速+30%（逃命速度）
  防御-50%（背后挨打）
  无法接收任何指令
  可被敌方追击→俘虏
```

#### 连锁溃逃

```
一队溃逃 → 临近编队判定：

连锁溃逃概率 = 20% + (溃逃队距离/50) × 5% - 守方士气/5

修正：
  守方编队士气>70：概率-30%
  有名将镇守：概率-25%
  守方是精锐部队(训练度>80)：概率-20%
  守方也是动摇状态(士气<40)：概率+35%

成功阻止连锁：全军士气-5（震慑效应仍在）
连锁溃逃发生：第二队溃逃→可能引发第三队→全军崩溃

历史上：长平之战赵军被围→一队溃→全军溃→40万降
```

#### 溃逃后恢复

```
溃逃部队恢复条件：
  ├── 溃逃出战场边界后，需要时间重新集结
  ├── 战斗结束后溃逃部队部分回归（70-90%）
  ├── 可被敌方追击拦截（俘虏/击杀）
  └── 可被统帅亲自拦截（需要统帅在附近，统率判定）

恢复后的部队：
  士气重置为40（低迷）
  需要休整才能重新投入战斗
  本场战斗内无法再次参战（除非技能"重整溃兵"）
```

### 1.4 激励与镇压

```
统帅可使用的激励/镇压手段：

激励（提升士气）：
  ├── 演讲（战前一次性，魅力判定，+5-20士气）
  ├── 赏赐（消耗军费，每1000金+10士气，有边际递减）
  ├── 身先士卒（统帅亲自冲锋，+10士气，但有阵亡风险）
  └── 技能（统帅系/魅力系技能节点）

镇压（阻止溃逃）：
  ├── 督战队（在溃逃路线放督战队→逃跑者被斩杀→威慑+15士气但残忍-10仁义）
  ├── 杀一儆百（处决逃兵首领→该队士气+20，其他队士气-5）
  └── 禁止撤退（号令"退后一步者斩"→溃逃阈值降低到士气<5才触发）
```

## 第二部分：伤害模型

### 2.1 完整伤害公式

```
最终伤害 = [武器基础伤害 × (1 + 武勇修正) × 速度加成 × 部位倍率 
           × 兵种相克修正 × 阵型修正 × 地形修正 - 护甲减伤] 
           × 士气修正 × 技能修正

详细展开：

武器基础伤害：见武器数据表（剑35 / 刀50 / 戟65 / 矛45 / 弓30 / 弩55 / etc.）

武勇修正 = 武勇 × 1.5%（每点武勇+1.5%）

速度加成：
  静止：1.0x | 步行冲锋：1.2x | 骑马慢走：1.3x
  骑马快走：1.6x | 骑马疾驰：2.0x | 极限冲刺：2.5x-3.5x

部位倍率：
  头部2.0x → 眩晕概率 = (伤害/目标生命) × 50%
  颈部2.5x → 致命一击概率 = (伤害/目标生命) × 80%
  躯干1.0x → 正常
  手臂0.7x → 缴械概率 = (伤害/目标生命) × 30%
  腿部0.6x → 减速(移速-30%持续5秒)

兵种相克修正：
  优势对位 ×1.3 | 均势 ×1.0 | 劣势 ×0.8 | 严重劣势 ×0.6

阵型修正：
  方阵(守) ×0.85(受伤害) | 锥阵(攻) ×1.15(造成伤害)
  圆阵(守) ×0.70(受伤害) | 长蛇 ×1.40(受伤害)

地形修正：
  平地 ×1.0 | 山地守方 ×0.8(受伤害) | 河岸攻方 ×0.7(造成伤害)
  高地攻方(自上而下) ×1.2 | 高地守方(自下而上) ×0.8

护甲减伤 = 护甲值 × (1 - 破甲率)
  布甲 5% | 皮甲 15% | 铜甲 30% | 铁甲 45% | 精铁甲 60%
  弩自带破甲30%，技能/名剑可增加破甲率

士气修正：
  狂热(90-100)：造成伤害×1.15，受到伤害×0.90
  高昂(70-89)：造成伤害×1.05
  正常(50-69)：×1.0
  动摇(30-49)：造成伤害×0.90，受到伤害×1.10
  恐慌(10-29)：造成伤害×0.75，受到伤害×1.25
  崩溃(0-9)：造成伤害×0.50，受到伤害×1.50
```

### 2.2 伤害类型

| 伤害类型 | 来源 | 特点 | 护甲互动 |
|---------|------|------|---------|
| **砍伤** | 刀/剑砍击 | 中等伤害，可连击 | 铁甲有效减伤 |
| **刺伤** | 矛/枪/剑刺/箭 | 高单体伤害，精准 | 可穿透铠甲缝隙 |
| **钝击** | 戟/锤/棍 | 低伤害，高破甲 | 无视30%护甲 |
| **远程** | 弓/弩 | 距离决定伤害衰减 | 弩穿甲30% |
| **冲锋** | 骑兵/战车 | 速度加成极高 | 架枪可反制 |
| **攻城** | 投石/弩炮 | 对建筑加成200%，对人群溅射 | 无差别伤害 |
| **火焰** | 火攻/火箭 | 持续伤害+士气打击 | 布甲/皮甲额外+30%伤害 |
| **毒** | 毒箭/毒药 | 持续伤害+debuff | 无视护甲（毒渗透） |

### 2.3 暴击与致命一击

```
暴击判定（个人战斗）：
  基础暴击率 = 5% + 武勇/200（最高+60%=65%总暴击率）
  暴击伤害 = 150% 普通伤害
  
  暴击条件：
  ├── 命中头部 → 自动暴击
  ├── 完美时机（攻击到达前0.15秒内）→ 暴击率+20%
  └── 对方硬直中 → 暴击率+15%

致命一击（处决）：
  触发条件：目标生命<15% + 重击 + 命中头部/颈部/躯干
  效果：无视剩余生命→直接击杀
  仅在个人战斗/单挑中生效，军团普通士兵不触发处决动画
```

### 2.4 军团兵伤害简化

```
士兵间战斗使用简化版伤害计算（性能优化）：

军团士兵伤害 = 攻击力(A) × 兵种相克 × 士气修正 × 随机(0.85-1.15) - 防御力(D)

无部位判定，无速度加成（士兵不骑马或默认骑马）
暴击仅个人战斗中计算
每0.5-1.5秒判定一次（取决于武器速度和动画）

士兵生命归零 → 倒地死亡/受伤（战后部分恢复）
```

### 2.5 战后统计

#### 伤亡计算

```
真正死亡 = 倒地士兵 × 阵亡率

阵亡率：
  胜利方：10-20%
  失败方：30-50%
  被追击中：额外+20-30%
  有优秀医官(药材+医术)：阵亡率-15%

伤兵恢复：
  轻伤：战后3-7天恢复
  重伤：战后15-30天恢复（部分可能终身残疾→属性-）
  需要：药材+医馆/医官
```

#### 俘虏

```
俘虏来源：
  ├── 溃逃时被骑兵追击拦截
  ├── 围城投降
  ├── 单挑失败被擒
  └── 战后清点战场

俘虏处置选项（同攻城战）：
  释放 → 仁义+，可能被重新招募
  招募 → 花费金+魅力判定，扩大己方兵力
  索赎 → 送回敌国换取赎金
  处决 → 武名+，仁义-，对方势力仇恨+
  劳役 → 用于工程建设
```
