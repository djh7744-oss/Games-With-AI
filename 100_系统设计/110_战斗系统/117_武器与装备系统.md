# 武器与装备系统

## 设计目标

> 每种武器都应该有自己的"性格"——剑有灵巧、刀有厚重、戟有霸道、矛有精准。装备不仅是数值，更是战斗风格的延伸。十大名剑不只是更强的武器，每把都有独特的获取故事和专属效果。

## 系统概述

武器系统分为常规武器和特殊武器（名剑/神兵）。装备系统包含武器、防具（头盔/身甲/护臂/护腿/盾牌）、坐骑（战马/马甲）。所有装备有品质分级（白/绿/蓝/紫/金），可通过锻造、战利品、任务获得。

## 第一部分：武器系统

### 1.1 武器分类

#### 武器类型总览

| 类型 | 攻击方向 | 速度 | 基础伤害范围 | 破甲率 | 特点 |
|------|---------|------|------------|--------|------|
| **剑** | 四方向全 | 快(0.35s) | 30-80 | 10% | 最灵活，连招多，刺击精准 |
| **刀** | 左/右砍+上劈 | 中(0.45s) | 40-100 | 5% | 砍击威力大，骑兵首选 |
| **戟** | 上劈+下刺 | 慢(0.60s) | 50-120 | 20% | 长杆+钩拉，克制骑兵 |
| **戈** | 上劈+钩拉 | 慢(0.55s) | 40-90 | 15% | 钩拉步兵盾牌/骑兵下马 |
| **矛/枪** | 仅下刺 | 快(0.30s刺) | 35-70 | 25% | 刺击穿透高，骑兵冲锋×3 |
| **弓** | 远程瞄准 | 中(1.2s拉弓) | 20-50 | 0% | 高射速远程，精度中等 |
| **弩** | 远程瞄准 | 慢(2.5s装填) | 40-80 | 30% | 破甲远程，高精度，装填慢 |

| **匕首** | 四方向（短距） | 极快(0.25s) | 10-25 | 20% | 近身/暗杀/副武器 |
| **盾** | 无攻击(盾击) | 盾击0.5s | 5-15 | 0% | 防御装备，盾击可打断 |

#### 武器属性详解

| 属性 | 说明 |
|------|------|
| 基础伤害 | 武器面板伤害值 |
| 攻击速度 | 轻击的动画时长（秒） |
| 破甲率 | 无视护甲减伤的百分比 |
| 武器长度 | 影响攻击距离和狭窄空间适应性 |
| 耐久度 | 武器耐久，归零→攻击力减半→需修理 |
| 重量 | 影响体力消耗和携带数量 |
| 品质 | 白(普通)/绿(精良)/蓝(稀有)/紫(史诗)/金(传说) |
| 材质 | 铜/铁/精铁/百炼钢/特殊(名剑) |
| 特殊效果 | 品质蓝以上可能附加效果 |

### 1.5 武器锻造系统

```
锻造条件：
  ├── 需要工艺属性（影响成品品质上限）
  ├── 需要对应科技（冶铁术/炼钢术等）
  ├── 需要材料（铁/精铁/百炼钢+辅料）
  ├── 需要锻造台/兵器坊建筑
  └── 需要配方（基础配方自动拥有，高级配方需获取）

锻造流程：
  1. 选择武器类型+品质目标
  2. 投入材料（材料品质决定品质上限）
  3. 锻造小游戏（类似M&B2的锻造UI）
     ├── 加热(控制温度曲线)
     ├── 锤炼(鼠标/手柄节奏敲击)
     ├── 淬火(选择淬火液体：水/油/尿液/血)
     ├── 打磨(精度控制)
     └── 装配(装柄/缠绳/配鞘)
  4. 工艺属性判定→成品品质

品质概率（工艺属性80+百炼钢）：
  普通30% / 精良35% / 稀有20% / 史诗10% / 传说5%

命名：自动生成/玩家命名
```

### 2.2 护甲减伤计算

```
身体部位护甲叠加：

头部防具：头盔防御 + 身甲防御×0（头部独立）
躯干防具：身甲防御（主要）
手臂防具：护臂防御（如果装备）
腿部防具：护腿防御（如果装备）

最终部位护甲 = 该部位装备防御值 × (1 + 品质修正) × (1 + 工艺属性修正)

品质修正：
  白0% / 绿5% / 蓝12% / 紫20% / 金30%

工艺属性修正：工艺/200（最高+60%）

最终减伤率 = 装备减伤率 × (1 - 攻击方破甲率)
```

## 第四部分：特殊装备

### 4.1 虎符（调兵权）

```
虎符等级：
  青铜虎符：可指挥1000人以下
  银虎符：可指挥5000人以下
  金虎符：可指挥10000人以下
  玉虎符：全国调兵权（君主/上将军专属）

获得方式：爵位提升/君主授予/夺取
```

### 4.2 节杖（外交信物）

```
持有节杖 → 外交成功率+15-50%（按等级）
丢失/被夺 → 无法进行正式外交
```

### 4.3 玉佩/绶带（身份标识）

```
不同等级的玉佩标识身份：
  士：无佩玉（铜带钩）
  大夫：玉环
  卿：玉璧
  侯：玉组佩
  王/帝：传国玉玺

效果：NPC对话态度改变，某些城池/区域需要相应身份才能进入
```

| 关联系统 | 交互方式 | 影响 |

| 个人战斗 | 武器决定攻击方向和连招 | 装备=战斗风格 |
