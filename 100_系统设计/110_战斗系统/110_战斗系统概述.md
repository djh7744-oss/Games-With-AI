# 战斗系统概述

## 系统架构

```
                战斗系统
                    │
    ┌───────┬───────┼───────┬───────┐
    ▼       ▼       ▼       ▼       ▼
 个人战斗  军团指挥  攻城战   单挑    水战
(即时动作)(即时战术)(综合)  (1v1)  (即时战术)
```

## 核心循环

```
战略地图行军 → 遭遇敌军 → 战场加载 → 部署阶段 → 战斗阶段 → 结算阶段
     ↑                                                              │
     └──────────────── 返回战略地图 ←────────────────────────────────┘
```

## 战斗进入

| 触发方式 | 说明 |
|---------|------|
| 战略地图遭遇 | 双方军团接触→加载战场实例（根据当前位置地形） |
| 主动进攻城池 | 进攻方选择攻城方式→加载攻城战场 |
| 遭遇伏击 | 被伏击方处于不利阵型+地形 |
| 剧情战斗 | DLC历史战役，固定战场+初始部署 |

## 战场规模

| 规模 | 每方人数 | 玩家配置要求 |
|------|---------|-------------|
| 小型 | 50-100 | 最低(GTX 1060) |
| 中型 | 200-500 | 推荐(RTX 3060) |
| 大型 | 500-1000 | 高端(RTX 4070+) |
| 史诗 | 1000-3000 | 旗舰 |

玩家可在设置中选择战场规模。

## 战斗阶段

### 阶段一：部署 (30秒)
- 玩家在战场地图上布置各编队初始位置
- 选择初始阵型（方阵/圆阵/锥阵/鹤翼等）
- 下达初始指令（前进/驻守/自由行动）

### 阶段二：战斗 (无限时，直到一方溃退或投降)
- 即时制，无暂停（可选慢动作模式）
- 玩家可亲自战斗 or 俯视指挥
- 委任副将接管指定编队

### 阶段三：结算
- 统计伤亡/俘虏/战利品
- 计算将领经验+声望变化
- 确定俘虏处置（释放/招募/处决/索赎）
- 返回战略地图，时间推进

## 伤害模型

```
最终伤害 = (武器基础伤害 + 武勇修正) × 速度加成 × 部位倍率 - 护甲减伤

武器基础伤害：剑35 / 刀50 / 戟65 / 矛45 / 弓30 / 弩55
武勇修正：每点武勇+1.5%
速度加成：静止1.0x → 步行1.2x → 骑马2.0-3.5x(取决于马速)
部位倍率：头部2.0x / 躯干1.0x / 四肢0.7x
护甲减伤：布甲5% → 皮甲15% → 铜甲30% → 铁甲45% → 精铁甲60%
```

## 士气系统

```
每支部队独立士气值(0-100)

士气增益：
+ 击杀敌将 +15
+ 占领高地 +10
+ 友军鼓舞 +5~10
+ 处于优势兵力 +5

士气损失：
- 被侧翼攻击 -10
- 友军溃逃 -15
- 被包围 -20/回合
- 粮草断绝 -5/回合
- 主将阵亡 -40(全局)

士气归零 → 该队溃逃 → 可能引发连锁溃逃
```
