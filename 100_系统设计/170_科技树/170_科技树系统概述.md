# 科技树系统

## 设计目标

> 对标《文明》系列的科技树 + 《群星》的科技卡牌系统。科技树不只是"点一项→等时间→获得加成"的线性流水线。它应该是一种有策略深度的资源分配博弈——每一笔研发投入都是对未来的押注。科技树有七大分支、四层时代、各国专属路线、隐藏科技和协同突破。

## 系统架构

```
                        科技树系统
                             │
    ┌──────┬──────┬──────┬──┴───┬──────┬──────┬──────┐
    ▼      ▼      ▼      ▼      ▼      ▼      ▼      ▼
  军事   农业   经济   政治   文化   工艺   航海   医学
  (10项) (8项)  (8项)  (8项)  (8项)  (8项)  (6项)  (6项)
                             │
                    各国专属科技(每国3项)
                    隐藏科技(条件触发)
                    协同突破(科技组合)
```

### 四层时代体系

```
时代一：青铜与铁器时代 (起始)
  可用：七大分支各T1-T2 (14项基础科技)
  特征：战国初期技术水平
  时代BUFF：研发速度×1.0
  进阶条件：完成≥10项基础科技

时代二：争鸣与扩张时代
  可用：七大分支各T3-T4 (14项中级科技)
  特征：百家争鸣驱动技术爆发
  时代BUFF：研发速度×1.2 (学术繁荣)
  进阶条件：完成≥10项中级科技

时代三：帝国与统合时代
  可用：七大分支各T5-T6 (14项高级科技)
  特征：大一统帝国的超级工程
  时代BUFF：已完成科技效果+15% (集成优势)
  进阶条件：完成≥10项高级科技

时代四：革命与超越时代
  可用：七大分支各T7-T8 (终极科技，部分架空)
  特征：超时代发明，改变历史进程
  时代BUFF：所有科技研发速度×1.5 (加速突破)
  进入条件：完成≥8项高级科技 + 至少3项T7
```

## 核心机制

### 3.1 科技研发模型

```
每日研发进度 = (基础研发力 + 人才加成 + 投入加成) × 时代修正 × 灵感修正 × 传播修正

基础研发力:
  古典时代: 1.0/天
  每完成1项该分支科技: +0.1/天 (知识积累)
  每座学宫: +0.2/天
  国家科技值平均/100: +加成

人才加成:
  有相关领域专家(对应属性≥80): +0.5/天
  有学派宗师: +0.3/天
  该分支有"科学家"职位(需要任命): +0.2~1.0/天

投入加成:
  每投入1000金/天: +0.1/天 (上限+2.0/天)
  每投入竹简/书籍: +0.05/天/卷 (上限+0.5/天，即10卷)

时代修正:
  时代一: ×1.0
  时代二: ×0.85 (技术更难)
  时代三: ×0.70
  时代四: ×0.55 (最高难度)

灵感修正:
  触发灵感: ×1.5 (持续30天)
  无灵感: ×1.0

传播修正:
  邻国已完成此科技: ×1.0 + (邻国科技值/500)
  该科技已有多国完成: 每个额外国家+0.05 (最多到×1.3)
```

### 3.2 科技卡牌与科研队列

```
科技卡牌系统（类似《群星》的科技选择）：

每次开始新研究时，系统从"当前可研究"的科技池中随机抽取3-5张
科技卡牌供玩家选择。这模拟了"科研有意外发现"的现实。

卡牌出现概率 = 基础权重 × 时代修正 × 分支偏好修正

基础权重: 每项科技=10
时代修正: 跳时代研究×0.3 (不鼓励跳级)
分支偏好: 当前科学家擅长的分支×1.5

玩家可在抽取的卡牌中选择1项开始研究。
未被选择的卡牌放回池中，下次可能再次出现。

科研队列:
  可排队最多5项科技 (基础3 + 学宫+1 + 百家争鸣+1)
  每项科技独立计算进度
  但资源投入是共享的 → 同时研究越多 → 每项越慢
  最佳策略: 同时研究2-3项 (投入/产出平衡点)
```

### 3.3 灵感与突破系统

```
灵感触发条件（每项科技对应1-2种触发方式）：

军事类灵感:
  铁兵器: 在战场上缴获敌方铁制武器 → 灵感触发
  弩机: 守城战中使用弓弩击退3波敌军
  骑兵革新: 用骑兵完成侧翼冲锋击溃敌军
  筑城术: 被攻破一座城池后 → "必须建更坚固的城墙!"

农业类灵感:
  水利灌溉: 经历1次旱灾 → "必须兴修水利!"
  良种选育: 连续3年农业丰收 → 有余力做改良
  梯田技术: 在丘陵地形建设农田时发现

经济类灵感:
  货币统一: 在一次跨国交易中因汇率损失>20%利润
  钱庄体系: 闲置资金超过20000金时
  海贸: 控制第一座沿海城池

政治类灵感:
  察举制: 连续3次在野招募失败
  密折制度: 发现官员贪污团伙(>3人)
  车同轨: 统一后第一次跨区域大规模调兵时

灵感效果:
  触发后: 该科技研发速度×1.5 (持续30天)
  灵感可主动追求: 玩家可通过特定行为触发灵感
  但灵感是一次性的: 每项科技最多触发1次灵感
  未触发灵感也可正常研发 (只是慢一点)
```

### 3.4 科技传播网络

```
科技传播地图:

科技像水一样从高向低流动——
  高科技国家 → 低科技邻国 (自然传播)
  盟国之间 → 传播加速 (×2速度)
  敌对国之间 → 传播受阻 (需要间谍窃取)

传播可视化:
  在大地图上可切换"科技传播视图"
  高科技区域: 亮色 (绿/蓝)
  低科技区域: 暗色 (灰/红)
  传播箭头: 显示科技流动方向

传播事件:
  "楚国商人带来了先进的冶铁技术" → 解锁铁兵器10%进度
  "秦国逃兵透露了弩机制造方法" → 解锁弩机15%进度
  "齐国学者在我们学宫讲学" → 文化科技传播+30%

人为加速传播:
  派遣学者出访 (消耗金+时间 → 特定科技传播+30%)
  举办学术会议 (稷下学宫式 → 文化科技全面传播)
  招募外国人才 (直接获得其母国的部分科技进度)
```

### 4.2 科技树交互

```
点击任意科技节点:
  ├── 已解锁(绿色): 显示效果回顾
  ├── 正在研究(金色边框+旋转光效): 显示进度+投入调节
  ├── 可研究(黄色): 加入科研队列
  └── 锁定(灰色): 显示前置条件(哪些科技不够)

右键点击: 查看科技详情弹窗
  ├── 完整效果描述
  ├── 解锁内容(兵种/建筑/政策/指令)
  ├── 前置科技树路径
  ├── 后继可解锁科技
  ├── 灵感触发条件(灰色=未触发/金色=已触发)
  ├── 各国此科技的特殊版本(如有)
  └── 历史典故(小知识卡片)

拖拽: 将可研究科技拖入科研队列
Ctrl+点击: 将此科技设为最高优先级(顶到队列最前)
```
