# 科技时代与进阶系统

## 设计目标

> 四层时代体系不只是"名字变了"。每个时代有独特的视觉风格、研发节奏、可用的科技类型和世界事件。跨越时代是游戏进程的关键里程碑——就像《文明》中从古典进入中世纪的震撼一样，在战国背景下，从青铜兵器到火药武器的跨越更为戏剧性。

---

## 时代一：青铜与铁器时代（起始时代）

```
时代定位：战国初期（约公元前475-前350年）
技术水平：铁器开始取代青铜，郡县制推行初期，百家争鸣萌芽

视觉风格：
  科技树底色：青铜色调（绿锈色）
  科技节点：粗糙铸造感
  解锁动画：锤击锻造效果

可用科技（各分支T1-T2）：
  军事: T1铁兵器 / T2弩机 / T3骑兵革新
  农业: T1铁犁牛耕 / T2轮作休耕
  经济: T1市集 / T2商路开拓
  政治: T1郡县制 / T2集权
  文化: T1学宫 / T2著书 / T3乐府
  工艺: T1冶铁术 / T2炼钢 / T3铸造
  航海: T1造船基础 / T2近海航行
  医学: T1草药学 / T2解剖基础

时代BUFF：
  研发速度: ×1.0 (基准)
  科技传播: 基础速度
  灵感概率: 正常

时代特有事件：
  ├── "铁器革命"：第一个完成铁兵器+冶铁术的国家→触发全局事件
  │   效果：该国武器伤害额外+5%，持续20年（先驱者优势）
  ├── "学宫初建"：建造第一座学宫→吸引周边学者
  └── "变法之秋"：进入本时代后→AI国家更倾向于尝试变法

进阶到下一时代条件：
  完成≥10项本时代科技 (共有14项可选)
  拥有至少1座T3+城池
  至少与2个其他国家有外交关系
  完成任意1项变法(可选，加速进阶)
```

## 时代二：争鸣与扩张时代

```
时代定位：战国中期（约公元前350-前260年）
技术水平：百家争鸣鼎盛、大规模战争技术成熟、商业网络形成

视觉风格：
  科技树底色：黄金色调（百家争鸣的辉煌感）
  科技节点：精雕细琢感
  解锁动画：书页翻开+光芒四射

可用科技（各分支T3-T4）：
  军事: T4连弩 / T5攻城器械 / T6精铁甲
  农业: T3水利灌溉 / T6良种选育
  经济: T3盐铁专营 / T4均输平准
  政治: T3察举制 / T4俸禄制
  文化: T4历法 / T5医学 / T6百家争鸣
  工艺: T4纺织 / T5造船术 / T6陶瓷
  航海: T3远洋探索 / T4制海权
  医学: T3外科手术 / T4防疫体系

时代BUFF：
  研发速度: ×1.2 (学术繁荣加速)
  科技传播: +30%速度 (学派交流频繁)
  灵感概率: +20%

时代特有事件：
  ├── "百家争鸣大会"：定期在稷下/学宫举办学术辩论
  │   获胜学派→影响力+20% → 该学派关联科技研发+25%
  ├── "技术军备竞赛"：战争频发→军事科技研发互相追赶
  │   如果邻国军事科技领先你2项+ → 你获得军事科技灵感概率+50%
  ├── "商业网络形成"：完成经济T4+商路≥5条→触发
  │   所有国家商业收入+10% (全球化萌芽)
  └── "筑城竞赛"：强国边境城池自动触发城防升级需求

进阶到下一时代条件：
  完成≥10项本时代科技 (共有14项可选)
  拥有至少1座T4+城池
  国家总人口≥50000
  完成任意1项"超级工程"(都江堰/郑国渠/万里长城...)
```

## 时代三：帝国与统合时代

```
时代定位：战国晚期（约公元前260-前221年）
技术水平：帝国级工程、统一标准化、远洋航行萌芽

视觉风格：
  科技树底色：深蓝+金色（帝国的威严感）
  科技节点：精密齿轮+篆书纹样
  解锁动画：齿轮咬合+标准度量衡展开

可用科技（各分支T5-T6）：
  军事: T7筑城术
  农业: T4都江堰模式 / T5郑国渠 / T7梯田技术
  经济: T5货币统一 / T6钱庄体系
  政治: T5考课制 / T6密折制度 / T7文字统一
  文化: T7史学
  工艺: T7织造精进
  航海: T5旗舰建造 / T6环球航行（架空）
  医学: T5手术精进 / T6公共卫生

时代BUFF：
  研发速度: ×0.85 (技术更复杂)
  已完成科技效果: +15% (集成优势)
  新征服领土同化速度: +30% (帝国整合力)

时代特有事件：
  ├── "统一标准化"：完成货币统一+文字统一→触发
  │   全国行政效率+30%，持续50年 (历史级BUFF)
  ├── "超级工程竞赛"：各国竞相建造超级工程
  │   都江堰/郑国渠/万里长城/阿房宫/驰道→先完成者获得永久加成
  ├── "帝国宣言"：第一个进入本时代的国家→可宣布"天命"
  │   霸业声望获取+50%，小国臣服概率+30%
  └── "技术奇点"：如果某国领先其他国家>10项科技→触发
      该国获得"科技碾压"BUFF → 所有军事科技效果+25%

进阶到下一时代条件：
  完成≥10项本时代科技
  国家统一度≥70% (控制领土占战国总领土的百分比)
  建造≥2项超级工程
  霸业声望≥70
```

## 时代四：革命与超越时代（架空/隐藏）

```
时代定位：架空未来（超越历史时间线）
技术水平：火药武器、蒸汽萌芽、远洋帝国——如果战国没有终结于秦统一

视觉风格：
  科技树底色：暗紫+金色裂痕（突破极限的震撼感）
  科技节点：齿轮+火药+星图混合
  解锁动画：火药爆炸→星图展开→光芒穿透

可用科技（各分支T7-T8，多为架空）：
  军事: T8火器初探 (火药武器)
  农业: T8农政全书
  经济: T7海贸 / T8天下市集
  政治: T8车同轨书同文
  文化: T8经学大成
  工艺: T8终极工艺
  航海: T7全球探索 / T8制海权
  医学: T7药物化学 / T8公共卫生体系

时代BUFF：
  研发速度: ×1.5 (加速突破——技术奇点)
  跨时代研究惩罚取消 (可以安全地研究之前跳过的科技)
  所有科技效果上限+25% (突破历史限制)

时代特有事件：
  ├── "火药革命"：第一个研发火器初探→触发世界事件
  │   火药武器使所有T3以下城防过时→城防优势大幅下降
  │   "城墙时代终结"——攻守平衡被永久改变
  ├── "远洋时代"：完成海贸+全球探索→触发
  │   世界地图扩大→新的海外领土和资源可探索
  ├── "医学革命"：完成公共卫生+药物化学→触发
  │   瘟疫从此免疫→人口进入快速增长期
  │   平均寿命+10岁→角色可能活到80-90岁
  ├── "工业萌芽"：完成终极工艺+火器初探→触发
  │   解锁"蒸汽动力"概念（架空，战国蒸汽机!）
  └── "超越时代"事件链：进入本时代后→每5年触发一次
      随机获得一项"未来科技"灵感(跨时代的发明)
      但过度依赖未来科技→可能引发"科技灾难"(伦理/环境后果)

本时代为"无限延续"——没有下一个时代
你已走到科技树的尽头
但世界的可能性才刚刚展开...
```

## 跨时代机制

### 时代进阶仪式

```
当满足进阶条件后，触发"时代进阶仪式"：

仪式流程（特殊事件界面）:
  1. 时代评估报告 → 系统自动检查条件
  2. 满足条件 → 弹出"新时代召唤"事件
  3. 君主宣言 → 选择宣言内容(影响时代BUFF偏向)
     选项A: "我们将以武力征服新时代" → 军事科技+15%
     选项B: "我们将以智慧引领新时代" → 文化/科技研发+15%
     选项C: "我们将以财富建设新时代" → 经济建筑效率+15%
  4. 新时代开启 → 全服公告(如果是多人)或世界事件(单人)
     其他国家反应: 警惕/追赶/漠视
  5. 科技树视觉切换 → 新的时代配色和解锁动画

仪式花费:
  金: 10000 × 时代数
  文化影响力: 需要≥该时代基础要求
  不能处于战争中(除非选择"武力宣言")
```

### 时代落差惩罚

```
科技落后的国家面临"时代落差"：

时代落差 = 最高时代 - 本国时代

时代落差=1: 
  ├── 邻国科技传播速度+30% (追赶红利)
  ├── 可从先进国购买/窃取科技更容易
  └── 本国科技灵感概率+15%

时代落差=2:
  ├── 传播速度+50%
  ├── 先进国的军事科技对你+15%伤害 (装备代差)
  ├── 外交中被轻视 ("野蛮人")
  └── 人才流失→每年有概率智略高的NPC移民到先进国

时代落差≥3:
  ├── 几乎必然被吞并或成为附庸
  ├── 唯一的希望: 专注于1-2项颠覆性科技(弯道超车)
  └── 历史上有"落后国家突然崛起"的先例吗? 有——秦国!
      秦国在商鞅变法前也是落后国家 → 通过制度创新而非科技追赶

弯道超车:
  落后国家可选择"跳跃发展"策略:
  跳过某些科技(不研究)→节省时间→直接研究更高级科技
  但跳跃跨度越大→研究惩罚越重(×1.5/跳1级 → ×3/跳2级)
  历史上的"跳跃发展"几乎都是军事领域(秦国的标准化兵器)
```

## 时代统计与成就

```
时代成就（Steam/游戏内成就）：

青铜与铁器时代:
  "铁器先驱": 第一个完成铁兵器+冶铁术的国家
  "变法先锋": 在本时代完成任意变法

争鸣与扩张时代:
  "百家争鸣": 同时拥有≥5个学派成员
  "技术军备": 军事科技领先所有邻国

帝国与统合时代:
  "超级工程师": 完成3项超级工程
  "统一标准化": 同时完成货币统一+文字统一
  "始皇帝": 第一个完成车同轨书同文

革命与超越时代:
  "火药革命": 第一个研发火器初探
  "跨越千年的发明": 研发了历史时间线之外的科技
  "科技巅峰": 完成全部91项科技 (通用+专属+隐藏)
```
