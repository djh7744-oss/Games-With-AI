# 科技协同与组合系统

## 设计目标

> 科技不是孤岛。炼钢+冶铁=更好的兵器；航海+天文=远洋航行；火药+攻城器=摧毁城墙。科技之间存在着设计好的"协同效应"——完成特定科技组合后，解锁超越单项科技之和的强大加成。这是科技树的"隐藏深度"。

---

## 第一部分：科技协同效应

### 1.1 双科技协同 (12组)

```
协同1: 铁与火
  铁兵器(军事T1) + 冶铁术(工艺T1)
  效果: 所有近战武器附带微弱火焰伤害(+5%)→ "灼热之刃"
  
协同2: 弩与城
  弩机(军事T2) + 筑城术(军事T7)
  效果: 守城时弩兵射程+30% → "城防弩阵"

协同3: 马与铁
  骑兵革新(军事T3) + 冶铁术(工艺T1)
  效果: 解锁"铁甲骑兵"(重骑兵装备+1级)

协同4: 商与海
  商路开拓(经济T2) + 造船术(工艺T5)
  效果: 海上商路利润+50% → "海上丝绸之路"

协同5: 盐与铁
  盐铁专营(经济T3) + 冶铁术(工艺T1)
  效果: 盐铁专营收入+50% → "国家垄断强化"

协同6: 钱与政
  钱庄体系(经济T6) + 俸禄制(政治T4)
  效果: 官员可用俸禄投资钱庄→官员满意度+20%

协同7: 学与书
  学宫(文化T1) + 著书(文化T2)
  效果: 科技传播速度+50% → "知识爆炸"

协同8: 医与农
  医学(文化T5) + 良种选育(农业T6)
  效果: 人口死亡率-25% → "健康人口红利"

协同9: 历与农
  历法(文化T4) + 水利灌溉(农业T3)
  效果: 农业产出额外+10% → "精准农时"

协同10: 铸与钱
   铸造(工艺T3) + 货币统一(经济T5)
   效果: 铸币成本-50% → "高效铸币"

协同11: 织与商
   纺织(工艺T4) + 商路开拓(经济T2)
   效果: 纺织品贸易利润+40% → "丝绸溢价"

协同12: 陶与文
   陶瓷(工艺T6) + 乐府(文化T3)
   效果: 文化影响力+25% → "陶瓷文化输出"
```

## 第二部分：隐藏科技

### 2.1 隐藏科技触发条件

```
隐藏科技在科技树UI中不可见，需要在游戏中满足特定条件才会出现：

隐藏1: 铁甲战舰 (航海，T5级别)
  触发条件: 冶铁术 + 造船术 + 在水战中击沉10艘敌舰
  效果: 战舰防御+50%，耐久+100%
  
隐藏2: 毒药学 (医学，T3级别)
  触发条件: 医学 + 草药学 + 遭遇刺客下毒事件
  效果: 解锁毒药制作，刺客任务成功率+30%
        但: 仁义声望-10 (不光彩的学问)

隐藏3: 密码术 (文化/军事，T4级别)
  触发条件: 文字统一 + 密折制度 + 被敌方截获情报≥3次
  效果: 你的外交/军事通信无法被敌方截获
        敌方情报精度对你-50%

隐藏4: 长生术 (医学/文化，T6级别，架空)
  触发条件: 医学 + 历法 + 炼丹(多次尝试制作药物失败后)
         + 君主年龄≥50岁(对死亡的恐惧)
  效果: 君主寿命+10年(仅一次)
        但: 可能中毒(20%概率→寿命反减15年)
        历史上秦始皇求长生→这是对"永生"梦想的映射

隐藏5: 地动仪 (工艺/文化，T5级别)
  触发条件: 铸造 + 历法 + 经历地震灾害
  效果: 可提前3天预警地震(减少损失)，所有城池地震损失-50%

隐藏6: 水力机械 (工艺，T6级别，架空)
  触发条件: 水利灌溉 + 铸造 + 炼钢
  效果: 解锁水力驱动的自动锻造锤→工艺产出+40%
        解锁水力磨坊→粮食加工效率+50%

隐藏7: 飞行器 (工艺，T8级别，纯粹架空)
  触发条件: 终极工艺 + 纺织(风筝原理) + 骑兵革新(空气动力学概念)
         + 进入革命时代
  效果: 解锁"载人风筝"(侦查用，非战斗)→战场视野+100%
        解锁"热气球"(极晚期)→战略侦察范围+300%

隐藏8: 和平主义 (文化，T8级别)
  触发条件: 百家争鸣 + 经学大成 + 仁义≥95
         + 已经连续20年没有发动任何战争
  效果: 解锁"非攻"终极形态 → 其他国家主动攻击你的概率-90%
        你的文化影响力→所有非敌对国人自发传播
        解锁"和平统一"胜利条件(无需战争统一天下)
```

### 2.2 隐藏科技的设计理念

```
隐藏科技服务于两个目的:

1. 惊喜发现
   玩家在正常游戏中偶然触发隐藏科技→惊喜
   增加探索感和"我的世界是活的"的沉浸感
   模拟历史上"偶然发现"的科学进展

2. 奖励深度
   某些隐藏科技奖励特定玩法(如和平主义奖励和平路线)
   给非主流玩法提供独特的优势
   鼓励玩家尝试不同的策略

隐藏科技不公布在官方攻略中——
让玩家社区自己发现和分享触发条件。
```

## 第三部分：科技组合策略

### 3.1 推荐科技发展路径

```
路径1: 军事征服流
  优先: 铁兵器 → 骑兵革新 → 冶铁术 → 精铁甲
  追求协同: 军事铁三角(+10%全属性)
  跳过: 初期跳过文化/航海科技
  国家: 适合秦国/赵国

路径2: 经济繁荣流
  优先: 市集 → 商路 → 盐铁 → 货币统一
  追求协同: 商业循环(+20%商业收入)
  跳过: 初期跳过军事(靠钱买和平)
  国家: 适合齐国

路径3: 科技超越流
  优先: 学宫 → 著书 → 百家争鸣 → 经学大成
  追求协同: 知识就是力量(免费政策槽)
  跳过: 初期军事/经济都不强→靠科技代差弥补
  国家: 适合韩国(防御科技优势)

路径4: 海上帝国流
  优先: 造船 → 商路 → 海贸 → 远洋探索
  追求协同: 海上帝国(水军+30%，贸易翻倍)
  跳过: 初期内陆科技
  国家: 适合楚国/齐国

路径5: 全面均衡流
  每分支均衡点到T3-T4 → 触发多个小协同
  追求: 百工革新+农业革命+商业循环(三大协同同时在线)
  优势: 没有短板
  劣势: 没有特别突出的领域
```

### 3.2 协同效应可视化

```
科技协同面板:

已完成协同: 绿色高亮 + 连线动画
部分完成(差1-2项): 黄色虚线 + 提示"还差X科技"
未触发: 灰色

鼠标悬停→显示:
  ├── 协同名称
  ├── 需要科技列表(已完成打勾/未完成打叉)
  ├── 协同效果预览
  └── 推荐优先研发缺失的科技

协同触发特效:
  完成最后一项触发科技→全屏短暂特效
  ├── 军事协同: 刀剑交击+火花
  ├── 经济协同: 金币洒落
  ├── 文化协同: 书页翻飞
  └── 大协同: 史诗级特效(天降光芒+音效)
```
