# 科技研发与传播系统

## 设计目标

> 科技不会凭空产生——它需要人才、资金、时间、灵感和交流。科技也不是孤立的——它会通过贸易、战争、外交、移民等各种途径在国际间传播。本系统定义科技从"想法"到"成果"的完整生命周期。

---

## 第一部分：科技研发

### 1.1 科学家系统

```
科学家（Technology Specialist）：

科学家是推动科技研发的核心人才。
每个科技分支可任命1位"首席科学家"。

科学家来源:
  ├── 诸子百家学者 (儒家→文化/政治，墨家→工艺/农业)
  ├── 在野发明家 (随机刷新)
  ├── 工坊名匠 (工艺分支专家)
  ├── 外国招聘 (从科技先进国挖人)
  └── 自己培养 (派遣门生学习→成长为科学家)

科学家属性影响:
  智略: 影响基础研发速度 (每点+0.5%/天)
  工艺: 影响工艺/农业/军事科技研发
  经营: 影响经济科技研发
  魅力: 影响文化/政治科技研发
  
  专精度: 该科学家擅长分支内的研发速度×1.5
  博学度: 可同时参与多个分支(但每多一个分支效率-20%)

科学家等级:
  Lv1 学徒 (智略<60): 基础研发贡献 0.3/天
  Lv2 学者 (智略60-75): 基础研发贡献 0.5/天
  Lv3 专家 (智略76-88): 基础研发贡献 0.8/天，特定分支×1.3
  Lv4 宗师 (智略89-95): 基础研发贡献 1.2/天，可触发"重大突破"
  Lv5 巨匠 (智略96+): 基础研发贡献 1.5/天，可跨时代研发无惩罚

科学家维护:
  月俸: 按等级 (500-5000金/月)
  实验经费: 可选额外投入(加速研发但花费大)
  科学家可能跳槽 (被更高薪挖走)
  科学家可能衰老/死亡 → 需要培养接班人
```

### 1.2 实验与失败

```
科技研发不是100%成功的线性过程：

实验机制:
  每项科技研发中有3-5个"关键实验节点"
  进度到达节点时→触发实验判定

  实验成功率 = 50% + 科学家智略/3% + 实验室等级×5% + 投入×2%
              - 科技难度修正

  实验成功 → 继续研发(进度+10%)
  实验失败 → 三种可能:
    1. 轻微挫折(60%概率): 进度回退5%，第二天继续
    2. 重大失败(30%概率): 进度回退20%，需要重新设计实验
    3. 意外发现(10%概率): 虽然原目标失败→但解锁了另一项科技的灵感!

意外发现(Serendipity):
  "研究冶铁术时意外发现了更好的淬火方法" → 炼钢灵感触发
  "尝试制造更强弩机时发现了弹簧原理" → 后续科技加速
  这是科技史上真实存在的现象(青霉素/X光/微波炉都是意外发现)
```

### 1.3 实验室与科研设施

```
科研设施等级:

Lv1 基础工坊 (初始)
  效果: 研发速度+5%
  花费: 0 (初始自带)

Lv2 专业实验室
  效果: 特定分支研发速度+15%
  花费: 5000金 + 30天建造
  每个分支需独立建造

Lv3 学宫附属研究院
  效果: 所有分支研发速度+10%，科学家容量+2
  花费: 15000金 + 60天建造
  需要: 学宫(文化T1) + 至少1位Lv3科学家

Lv4 帝国科学院 (仅帝国时代+)
  效果: 全国研发速度+25%，可同时研发5项科技
  花费: 50000金 + 120天建造
  需要: 完成≥30项科技 + 至少3位Lv4科学家

Lv5 未来实验室 (仅革命时代+)
  效果: 跨时代研究无惩罚，意外发现概率+30%
  花费: 100000金 + 240天建造
  需要: 进入革命时代 + 至少1位Lv5科学家
```

## 第二部分：科技传播

### 2.1 自然传播

```
科技如流水，从高向低：

基础传播公式:
  每日传播进度 = (源国科技值 - 目标国科技值) / 1000 
                × 传播因子 × 关系修正

  传播因子:
    邻国(有陆地边界): ×1.0
    近国(距离<500里): ×0.7
    远国(500-1000里): ×0.4
    极远(>1000里): ×0.1
    有商路连接: ×1.5
    有海上航线: ×1.3

  关系修正:
    血盟: ×3.0
    坚定盟友: ×2.0
    友好: ×1.5
    善意/中立: ×1.0
    紧张: ×0.5
    敌对: ×0.2 (基本不传播)
    战争: ×0.05 (几乎完全阻断)

传播可视化(科技传播地图):
  在大地图切换"科技传播视图"
  每个国家显示其"科技总指数" (所有科技完成度的加权平均)
  国家间有"传播流"动画箭头(粗=快/细=慢/红=阻断)
  鼠标悬停→显示具体传播的科技名称和速度
```

### 2.2 人为传播

```
主动加速科技传播的方式:

1. 派遣学者出访
   花费: 2000金 + 学者30天时间
   效果: 目标国的某特定科技→你获得20-40%进度
   限制: 每年最多3次出访
   风险: 学者可能被目标国扣留/招募

2. 举办学术会议
   花费: 10000金 + 60天准备
   效果: 邀请各国学者→所有参与国共享科技进度10-25%
   需要: 学宫建筑 + 至少1位Lv3科学家
   冷却: 2年

3. 招募外国人才
   花费: 高额安家费(5000-20000金) + 高月俸
   效果: 该人才携带其母国部分科技进度加入
   携带进度 = 该人才在原国的研究领域 × 人才等级
   例: 招募韩国Lv3冶铁专家 → 获得韩国冶铁/炼钢科技30-50%进度

4. 留学生制度 (需政治T3察举制)
   派遣年轻学生到科技先进国学习
   花费: 1000金/年 + 学生3-5年时间
   效果: 学生回国时携带目标国科技进度(最多3项)
   回报周期长但成本低，适合长期投资

5. 军事缴获
   击败科技先进国的军队 → 有概率缴获装备
   缴获装备→专家研究→获得该装备相关科技进度
   例: 击败韩国弩兵→缴获劲弩→研究→弩机科技+30%
```

### 2.3 科技贸易

```
外交中的科技转让:

直接购买科技:
  在外交谈判中要求对方转让科技
  对方接受率 = 关系/2 + 该科技战略价值(对方评估) + 你提供的条件
  
  代价选项:
    一次性金支付: 该科技研发花费×2-5
    领土交换: 割让部分领土
    联姻/质子: 长期绑定
    军事同盟: 用军事保护换取科技
    科技互换: 用你的先进科技换对方的先进科技(最公平)

科技转让后:
  获得该科技100%进度(立即完成)
  但: 转让的科技不触发灵感(没有自己的研发过程)
  转让的科技无法触发协同效应(需要"自己研发"才算)
  转让科技的国家→你对其有"科技债务"(外交中处于弱势)

科技禁运:
  你可以对某国实施科技禁运(外交选项)
  效果: 该国从你这里获得科技传播速度-80%
  代价: 该国关系-30，其他国家看在眼里
  常用于: 阻止潜在对手获取你的核心技术
```

### 2.4 科技同化

```
征服后的科技整合:

占领科技先进国的城池:
  自动获得该城池所在国的部分科技进度
  获得量 = 该城技术值/200 + 占领城池等级×5%
  例: 占领韩国宜阳(T4城池,技术值80) → 获得韩国科技树40%进度

完全吞并一个国家:
  自动完成该国已完成的所有科技(100%继承)
  但: 需要时间消化(每项科技需30-60天"整合期")
  整合期间: 该科技效果减半(你的工匠/官员还在学习)

多种文明的科技融合:
  吞并不同科技传统的国家 → 可能触发"科技融合"事件
  例: 吞并韩国(冶铁强)+吞并楚国(造船强) → 解锁"铁甲战舰"隐藏科技
```

## 第三部分：科技生命周期

```
每项科技从"前沿"到"普及"经历四个阶段:

阶段1: 独占期 (0-5年)
  研发完成后 → 仅你的国家拥有此科技
  效果: 100%
  优势: 你独享该科技的全部收益
  策略: 利用独占期建立军事/经济优势

阶段2: 扩散期 (5-15年)
  邻近国家通过传播逐渐获得
  效果: 100% (但你不再独占)
  你的科技开始向外传播(不可阻止的自然过程)
  策略: 在独占期内最大化利用优势

阶段3: 普及期 (15-30年)
  大多数国家已掌握此科技
  效果: 变为"基础设施"级别(所有国家都有基础加成)
  此科技不再有竞争优势 → 但也不再有追赶压力

阶段4: 过时期 (30年+)
  更新更好的科技出现 → 此科技变为"历史遗产"
  但仍保留基础效果(后人享用你的发明)
  你的国家在科技史上留名("冶铁术的发明者: 秦国")
```
