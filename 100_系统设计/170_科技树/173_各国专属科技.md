# 各国专属科技

## 设计目标

> 七国各有独特的科技传统和优势领域。秦国的标准化、楚国的水军、齐国的商业、赵国的骑兵——这些不只是"初始BUFF"，而是贯穿整个游戏进程的专属科技路线。每国3项专属科技提供其他国家无法复制的独特优势。

---

## 秦国专属科技

```
秦国科技哲学: 标准化 + 集权 + 军事效率

专属科技1: 标准化兵器 (军事，替代T2弩机分支)
  研发: 90天 / 6000金
  前置: 铁兵器
  效果:
    ├── 所有兵器伤害+10% (标准化生产的质量控制)
    ├── 兵器制造成本-25% (流水线生产)
    ├── 兵器维修消耗-30% (零件可互换)
    └── 解锁"兵器标准化工坊"建筑
  历史: 秦始皇兵马俑的兵器展现了高度标准化的生产体系
        箭镞有三棱形→标准化模具铸造→每个箭镞几乎一模一样
        这是当时世界上最先进的制造体系
  
专属科技2: 远交近攻策 (政治，T4级别)
  研发: 120天 / 10000金
  前置: 集权(政治T2)
  效果:
    ├── 远距离国家外交成功率+30%
    ├── 邻国对你的军事行动预警时间-50%
    ├── 解锁"远交近攻"战争策略(对邻国宣战时→远方国家不加干涉)
    └── 与远国结盟时→自动获得军事通行权(无需额外谈判)
  历史: 范雎向秦昭襄王提出的战略→奠定了秦国统一的外交基础

专属科技3: 大一统 (政治T8替代，仅秦国可研发)
  研发: 360天 / 60000金
  前置: 文字统一(政治T7) + 车同轨(政治T8) + 完成统一
  效果:
    ├── 取代"车同轨书同文"→效果+50%升级版
    ├── 所有被征服领土的文化同化速度×3
    ├── 解锁"郡县直隶"→中央直接管理所有城池(无中间层级)
    ├── 解锁"驰道网络"→全国移速+50%
    └── 永久BUFF: "书同文车同轨统一度量衡"→国家永不分裂
  历史: 秦始皇的统一措施是人类历史上最大规模标准化运动之一
```

## 楚国专属科技

```
楚国科技哲学: 水军 + 南方开发 + 文化独特性

专属科技1: 水军优势 (航海，T2级别)

  前置: 造船基础
  效果:
    ├── 所有战舰建造时间-30%
    ├── 水军士兵经验获取+50%

  历史: 楚国拥有战国最强的水军，控制长江中下游

专属科技2: 漆器精工 (工艺，T3级别)

  前置: 纺织(工艺T4) 或 市集(经济T1)

    ├── 漆器价值+60% (奢侈品收入暴增)

  前置: 水利灌溉(农业T3) + 良种选育(农业T6)
  效果:
    ├── 长江以南所有城池农业产出+50%
    ├── 解锁南方新可耕地(沼泽排水→稻田)
    ├── 南方人口承载力翻倍
    └── 楚国领土自动向南扩张(每5年自动获得南方新领土)
  历史: 楚国持续向南扩张，将长江以南大片地区纳入华夏文化圈
```

```

  研发: 150天 / 15000金
  前置: 货币统一(经济T5)
  效果:
    ├── 你的货币成为"国际结算货币"(其他国家自愿使用)

    └── 解锁"经济附庸"外交选项(小国在经济上依附于你)
  历史: 齐国刀币在战国是最受欢迎的"硬通货"之一

专属科技3: 海盐垄断 (经济T8替代)
  研发: 240天 / 35000金
  前置: 盐铁专营(经济T3) + 海贸(经济T7)
  效果:
    ├── 海盐产量+100%，海盐贸易利润+200%
    ├── 所有内陆国家对你产生"盐依赖"(外交优势)
    ├── "盐禁运"→目标国军粮保存能力-80%(致命打击)
    └── 齐国GDP可超过其他六国总和
  历史: 齐国海盐贸易是其财富根基，"齐盐"天下闻名
```

## 专属科技平衡

```
专属科技的设计原则:

1. 不可转让
   专属科技无法通过外交/间谍/传播获得
   其他国家只能羡慕(或征服韩国来获取其工匠)

2. 各有定位
   每国的3项专属科技覆盖不同阶段:
     第一项(T2级别): 早期优势，让开局就有特色
     第二项(T3-T5级): 中期特色，深化优势
     第三项(T8级): 终极科技，定义国家终极形态

3. 征服继承
   完全吞并一国→获得该国专属科技50%效果
   (不是100%→因为你没有该国的文化和历史积累)
   灭韩国→获得冶铁精工50%效果(仍然很强)

4. 不孤立
   专属科技仍需要通用科技作为前置
   韩国的精密制造需要先有冶铁+炼钢(通用科技)
   确保专属科技是"锦上添花"而非"另起炉灶"
```
