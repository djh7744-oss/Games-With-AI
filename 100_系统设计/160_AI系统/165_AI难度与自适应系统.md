# AI难度与自适应系统

## 设计目标

> 不同玩家需要不同挑战。新手应该感到"我能赢"，老手应该感到"我需要动脑子"。AI难度不是通过简单粗暴的数值作弊实现——而是通过AI的决策质量和策略深度来区分。同时，动态难度系统确保游戏始终有挑战性而不会劝退。

## 系统架构

```
                    AI难度系统
                         │
    ┌────────┬──────┬────┼────┬──────┬──────┐
    ▼        ▼      ▼    ▼    ▼      ▼      ▼
  四级     AI    资源  信息  决策  动态   反玩家
  难度    作弊   加成  优势  质量  难度  策略
```

## 核心机制

### 3.1 四级难度设定

| 难度 | 名称 | 目标玩家 | AI决策质量 | AI作弊程度 | 玩家资源修正 | 敌人侵略性 |
|------|------|---------|-----------|-----------|------------|-----------|
| ★ | **简单** | 新手/休闲 | 低(基础策略) | 无作弊 | 玩家收入+30% | 低(不主动进攻) |
| ★★ | **普通** | 标准玩家 | 中(标准策略) | 轻微(合理范围) | ±0 | 中 |
| ★★★ | **困难** | 策略老手 | 高(精通策略) | 中等 | 玩家收入-10% | 高(积极扩张) |
| ★★★★ | **极难** | 骨灰级挑战 | 极高(接近人类高手) | 较高 | 玩家收入-25% | 极高(最优战略) |

### 3.2 AI作弊程度详解

```
AI作弊不是"让AI资源凭空产生"，而是给AI合理的"隐性优势"：

简单难度(AI劣势)：
  AI征兵速度-30%
  AI外交容易被拒绝
  AI战场响应慢(3秒延迟)
  AI不会使用计谋
  AI不会伏击
  AI士兵士气-10(基础值)
  玩家受伤恢复速度+50%

普通难度(公平)：
  AI无任何作弊(与玩家同等规则)
  玩家也无加成
  AI偶尔使用简单计谋
  AI偶尔尝试伏击(30%概率)

困难难度(AI轻微优势)：
  AI征兵成本-15%（治国效率高）
  AI外交成功率+10%（AI善于谈判）
  AI科技研发速度+15%
  AI战场响应快(无延迟)
  AI积极使用计谋(70%概率)
  AI善于利用地形和阵型相克
  AI士兵基础士气+5

极难难度(AI显著优势)：
  AI征兵成本-25%
  AI外交成功率+20%
  AI科技研发速度+25%
  AI在远距离也能做出精准的战术判断
  AI预判玩家战术(记录玩家习惯→针对性应对)
  AI合纵反玩家(当玩家过于强大时)
  AI士兵基础士气+10
  AI初始资源+50%（开局即有更强基础）
```

### 3.3 AI决策质量差异

```
同一场景，不同难度下AI的不同表现：

场景：AI国家在面对玩家军队入侵时

简单难度AI：
  "敌人来了→我有兵就去打→不管地形→不管阵型"
  表现：把所有兵力集结→直线冲向玩家→正面决战
  结果：容易被利用地形伏击/被少量精锐击败

普通难度AI：
  "敌人来了→评估数量对比→选择防御位置→占据高地"
  表现：主力在有利地形布防→派小队骚扰玩家补给线
  结果：需要玩家正面攻坚或寻找侧翼突破口

困难难度AI：
  "敌人来了→数量劣势→不正面决战→坚壁清野
   →放弃边境小城→主力退守核心城池→等待援军
   →同时派使者向盟友求援→可能形成多国夹击"
  表现：撤退+焦土战术+外交求援→让玩家陷入消耗战
  结果：玩家需要速战速决或承受持续消耗

极难难度AI：
  "不仅做了困难难度的所有事，还：
   →派间谍散布玩家后方即将发生叛乱的假消息→玩家分兵
   →在玩家退路布置伏兵→即使打不赢也要让你损失惨重
   →联系玩家的其他敌人→多面夹击
   →如果实在守不住→谈判时设下陷阱条款"
  表现：全方位策略对抗，像真实人类高手一样
  结果：每一步都可能遭遇意外，需要玩家极谨慎
```

### 3.4 动态难度调节

```
动态难度系统（可选开启）：

目的：保持游戏始终"有挑战但不劝退"

调节机制：
  系统持续跟踪玩家的"成功指数"：
  
  成功指数 = (战胜率 + 扩张速度 + 经济增长率 + 外交成功率)/4
  
  成功指数持续过高(>80)→AI世界做出反应：
    ├── 其他国家对你的警惕提高→外交难度+15%
    ├── 邻国自动增加边防投入→更难入侵
    ├── 其他国家之间更容易结盟(针对你的合纵)
    └── 流寇/盗贼更频繁骚扰你的领土(趁乱打劫)
  
  成功指数持续过低(<30)→世界对你更宽容：
    ├── AI国家不太防备你→可能轻敌
    ├── 你更容易获得外交让步
    ├── 你的将领不太容易叛变(觉得你还有希望)
    └── 流寇不太敢惹你(你太穷没什么可抢的)
  
  但这种"宽容"有上限→不会变成"AI故意让你赢"

动态难度范围：
  动态调节在选定难度的±30%范围内
  极难难度不会因为玩家弱就降到简单水平
  简单难度不会因为玩家强就变成极难
```

### 3.5 反玩家策略（高难度）

```
高难度下AI的针对性策略：

1. 反单一兵种
   如果玩家大量使用骑兵→
   AI开始招募更多枪兵/在关键位置设拒马/避开平原交战

2. 反伏击
   如果玩家频繁使用伏击战术→
   AI增加侦察/斥候/不敢贸然进入森林

3. 反经济战
   如果玩家靠贸易积累大量财富→
   AI可能联合对玩家实施贸易禁运/提高关税

4. 反外交
   如果玩家外交太成功(盟友太多)→
   AI可能联合孤立你/离间你的盟友

5. 反斩首
   如果玩家习惯亲自冲锋斩将→
   AI会重点保护将领/用精锐部队围猎玩家

6. 学习型AI(仅极难)
   AI记录玩家的常用战术和习惯
   在后续战斗中做出针对性部署
   例：发现玩家总是从右翼进攻→下次AI加强右翼防御
   发现玩家从不使用水军→AI在沿海区域放松警惕
```

### 3.6 难度对游戏体验的影响

| 体验维度 | 简单 | 普通 | 困难 | 极难 |
|---------|------|------|------|------|
| 统一天下所需时间 | 10-15小时 | 20-30小时 | 40-60小时 | 80-120小时 |
| 可犯错误次数 | 很多 | 正常 | 少 | 2-3次大错就GG |
| 外交必要性 | 可选 | 建议 | 必须 | 生存必需 |
| 经济管理重要性 | 低 | 中 | 高 | 极高 |
| 间谍/情报必要性 | 不必 | 有用 | 重要 | 关键 |
| 战斗操作要求 | 休闲 | 中等 | 较高 | 很高 |
| 失败惩罚 | 轻微 | 正常 | 较重 | 严重 |
| AI计谋频率 | 从不 | 偶尔 | 频繁 | 持续 |
| 被多国夹击概率 | 0% | 10% | 30% | 60% |

### 3.7 玩家自定义AI难度

```
高级选项（可单独调节）：

  ├── AI经济效率：[低/中/高/极高]
  ├── AI军事侵略性：[和平/正常/好战/疯狂]
  ├── AI外交活跃度：[沉默/正常/活跃/纵横家级别]
  ├── AI计谋使用频率：[从不/偶尔/频繁/无孔不入]
  ├── AI作弊资源加成：[无/轻微/中等/显著]
  ├── 玩家资源惩罚：[无/-10%/-20%/-30%]
  ├── 动态难度：[关闭/开启/仅惩罚/仅帮助]
  └── AI战场战术水平：[新兵/老兵/精锐/天才]

预设难度 = 以上各项的组合预设
自定义 = 玩家逐项调节（适合有特定需求的玩家）
```

## AI难度UI

```
难度选择界面（开始新游戏时）：
  ├── 四个难度卡片：简单/普通/困难/极难
  ├── 每个卡片显示：
  │   ├── 难度描述
  │   ├── AI行为概要
  │   ├── 推荐玩家类型
  │   └── 预计统一时间
  ├── 高级选项折叠面板（可展开逐项调节）
  └── 难度无法在游戏中更改（防止成就滥用）

游戏中无法切换难度，但动态难度会自动调节。
```
