# AI系统概述

## 设计目标

> "世界不围着玩家转"——这是游戏五大设计支柱之一。AI系统是所有NPC、势力、城市生态和动态世界的智能引擎。七国会在玩家不作为时自主发展/交战/灭亡；平民有独立的作息和反应；战场上的每个士兵都会评估局势并做出战术决策。AI不是"让游戏变难"的工具，而是"让世界变活"的灵魂。

## AI设计原则

| 原则 | 含义 | 反例 |
|------|------|------|
| **目标驱动** | AI行为由目标驱动，而非随机 | 不是"有时征兵有时不征"，而是"因为要打仗所以征兵" |
| **信息不完全** | AI基于已知信息决策，有战争迷雾 | AI不会"知道"玩家的全部兵力部署 |
| **个性差异** | 不同AI有不同性格和决策倾向 | 秦王和韩王面对同样情况做出不同决策 |
| **可预测性** | AI行为有规律可循，高手可以预判 | 不是作弊，而是聪明 |
| **容错设计** | AI可以犯错(低级难度)，也可以很聪明(高级) | 不是所有AI都完美无缺 |
| **性能优先** | AI决策不能拖慢游戏性能 | 大量NPC的AI要分级计算 |

## AI分层架构

```
层级          决策频率        负责内容              技术方案
──────────────────────────────────────────────────────────
战略层(国家)   每月/每季      国家发展/外交/战争      Utility AI + GOAP
战役层(军团)   每天/每场战斗  军团移动/目标选择       Behavior Tree + EQS
战术层(战场)   每秒~每5秒    士兵个体行为/编队协作   State Tree + Mass AI
反应层(个体)   实时(每帧)    平民反应/对话/情绪      Behavior Tree + 脚本
```

## AI各子系统

### 一、势力AI [→161]
- 七国君主AI性格模型
- 国家自主运营（经济/军事/外交/人事/科技）
- 战争决策（何时宣战/何时求和/打谁）
- 外交决策（与谁结盟/何时背盟）
- 资源分配优先级
- 动态势力消长

### 二、平民与城市生态AI [→162]
- NPC昼夜作息系统（GTA式）
- 职业AI（农民/商人/工匠/士兵/学生/乞丐/艺人）
- NPC反应系统（对玩家行为/声望/犯罪）
- 城市活动与节庆
- 人群模拟（市场/庆典/逃难）

### 三、战场AI [→163]
- 士兵个体AI（攻击/防御/撤退/恐惧）
- 编队AI（保持阵型/响应指令/协同作战）
- 副将AI（接管指挥时的决策）
- 敌军AI（评估局势/选择战术/利用地形）
- 骑兵/弓兵/器械等兵种特化AI

### 四、动态世界系统 [→164]
- 自主世界事件（灾荒/瘟疫/政变/叛乱）
- 商队AI（自主贸易路线/规避风险）
- 流寇/山贼AI（刷新/劫掠/被剿灭）
- 难民潮（战争引发的连锁反应）
- 历史趋势模拟（秦国崛起的结构性因素）

### 五、AI难度与自适应 [→165]
- 四级难度设定（简单/普通/困难/极难）
- AI作弊程度控制（资源加成/信息优势/决策优化）
- 动态难度调节（玩家太强→AI更团结对抗）
- 学习型AI（记录玩家策略→针对性应对，仅高难度）

### 六、79位历史将领AI [→角色系统126]
- 每位将领的独立AI性格
- 名将专属战术偏好（白起偏好包围歼灭/孙膑偏好伏击）
- 将领之间的相性互动（朋友/敌人影响AI决策）
- 叛变/忠诚AI判定

## AI与各系统交互

| 关联系统 | AI角色 | 影响 |
|---------|--------|------|
| 内政系统 | 国家AI运营城池/分配资源 | AI驱动国家发展 |
| 外交系统 | 国家AI进行外交决策 | AI驱动国际格局 |
| 经济系统 | 商人AI/商队AI/市场AI | AI驱动经济循环 |
| 战斗系统 | 战场AI/士兵AI/副将AI | AI驱动战斗体验 |
| 角色系统 | NPC关系AI/忠诚AI | AI驱动人际关系 |
| 事件系统 | 动态世界事件AI | AI驱动沙盒叙事 |

## AI技术架构（UE5）

```
UE5 AI技术栈：

战略层：
  Utility AI System (效用评估) + GOAP (目标导向行动规划)
  每月评估所有可能行动的成本/收益→选择最优行动组合

战役层：
  Behavior Tree (行为树) + EQS (环境查询系统)
  军团移动、目标选择、战斗进入决策

战术层：
  State Tree (状态树) + Mass Entity (大规模实体)
  战场上千人士兵的个体行为

反应层：
  Behavior Tree + 脚本事件
  NPC对玩家行为的即时反应

感知系统：
  AI Perception System (视觉/听觉/伤害感知)
  战争迷雾 + 信息共享范围
```
