# 战场AI系统

## 设计目标

> 对标《骑马与砍杀》的战场AI。战场上的每个士兵不是无脑冲向敌人——他们会保持阵型、评估威胁、利用地形、在士气崩溃时逃跑。副将AI能独立指挥编队做出合理的战术判断。高难度下，敌军AI会像真实玩家一样思考。

## 系统架构

```
                    战场AI
                       │
    ┌──────┬──────┬────┼────┬──────┬──────┐
    ▼      ▼      ▼    ▼    ▼      ▼      ▼
  士兵AI  编队AI  副将  敌军  骑兵  远程   器械
  (个体)  (协同)  AI    统帅  AI    AI     AI
```

## 核心机制

### 3.1 士兵个体AI

#### 状态机

```
士兵状态机（State Tree）：

[待命] ──收到指令──→ [移动]
  │                    │
  └──敌军接近──→ [战斗]  │
                    │    │
                    ▼    ▼
                  [到达目的地] → [待命]

[战斗] 细分状态：
  ├── [接近敌人]：向最近敌人移动(保持队形约束)
  ├── [攻击]：进入武器范围→执行攻击动画
  ├── [格挡]：检测到来袭攻击→尝试格挡
  ├── [回避]：体力低/被包围→尝试后撤到友军后方
  ├── [士气动摇]：友军大量死亡/被包围→战斗效率下降
  ├── [溃逃]：士气归零→转身逃跑
  └── [重新集结]：逃跑后→在远处恢复→可能重新加入战斗

士兵感知（AI Perception）：
  ├── 视觉：前方120°锥形，距离30-80米(天气影响)
  ├── 听觉：360°，距离15-30米(战斗噪音)
  ├── 伤害感知：被攻击→立即知道攻击者位置
  └── 友军感知：始终知道编队内友军位置
```

#### 士兵个体决策

```
每1-3秒进行一次决策评估：

行动选择(基于效用分数)：

1. 攻击最近敌人：效用 = 基础60 + 距离×(1-距离/武器范围)×20
2. 转向攻击侧翼敌人：效用 = 40 + 侧面威胁×30
3. 保护自己(格挡/后撤)：效用 = 生命<50%→50 + (50-生命)/2
4. 协助被围攻的友军：效用 = 30 + 友军数量差×20
5. 保持阵型位置：效用 = 编队保持度×25
6. 溃逃：效用 = (士气=0时)→强制120(覆盖其他选择)

选最高效用行动执行。

个体能力差异：
  精锐士兵(T4/T5)：反应更快(每1.5秒评估)，命中率更高，士气更坚韧
  新兵(T0/T1)：反应慢(每3秒)，命中率低，士气脆弱
  名将指挥下：所有士兵反应加快(每2秒)，额外获得将领属性加成
```

### 3.2 编队AI

```
编队级AI(协调编队内几十到几百名士兵)：

编队保持：
  ├── 方阵：士兵约束在矩形区域内，前排交战/后排待命轮换
  ├── 圆阵：士兵约束在圆形区域，外圈举盾防御/全方向应敌
  ├── 锥阵：前方少量精锐集中突破/后方跟随扩大缺口
  └── 长蛇：单列行军，快速移动模式(不保持战斗阵型)

编队自动行为(无玩家指令时)：
  ├── 侦察：编队自动向前方派出散兵(轻步兵)探查
  ├── 接敌：敌人进入50米→自动转为战斗阵型
  ├── 轮换：前排战斗超过2分钟→自动与后排轮换(维持战斗力)
  ├── 追击：敌人溃逃→自动追击(如果玩家没有禁止)
  └── 重整：战斗间隙→自动整理阵型(恢复整齐)

编队协同：
  不同编队间有基本的协同意识：
  步兵在前→弓兵在后射击→骑兵在两翼→器械在后方
  一个编队被侧击→邻近编队自动转向支援(如果未被敌人牵制)
```

### 3.3 副将AI（接管指挥时）

```
当玩家将某编队委任给副将时，副将AI接管：

副将战术风格（由副将性格+属性决定）：

激进型(如白起类型)：
  偏好：主动进攻/锥阵突破/骑兵突击/追击溃敌
  战术：集中兵力击溃一点→扩大突破口→包围
  弱点：可能冒进被伏击/侧翼暴露

稳健型(如廉颇类型)：
  偏好：固守/方阵/圆阵/逐步推进
  战术：等待敌军先露出破绽→稳扎稳打
  弱点：可能错失战机/过于被动

灵活型(如孙膑类型)：
  偏好：多变性/伏击/诈败/侧翼迂回
  战术：用计谋扰乱敌军→在意想不到的方向攻击
  弱点：如果计谋失败→陷入被动

骑兵特化型(如赵奢类型)：
  偏好：骑兵冲锋/骚扰/打了就跑
  战术：利用速度优势反复冲击敌军薄弱点

副将AI决策质量：
  = 统率属性×0.5 + 智略属性×0.3 + 战斗经验×0.2
  
  好副将(统率80+)：AI决策接近人类高手
  中副将(统率50-70)：AI决策中规中矩
  差副将(统率<40)：AI可能做出错误判断
```

### 3.4 敌军统帅AI

```
敌方军队的AI统帅(对标玩家的指挥能力)：

战前评估：
  敌军AI在进入战场前评估：
  ├── 己方兵力 vs 敌方兵力
  ├── 地形分析(高地/河流/森林位置)
  ├── 选择初始阵型和部署位置
  └── 制定初步战术(防御/进攻/诱敌/撤退)

战中决策：
  每30-90秒进行一次战术评估：

  评估维度：
  ├── 兵力对比变化(已损失% vs 敌方损失%)
  ├── 关键位置控制(高地/渡口/桥梁)
  ├── 士气对比(各编队士气状态)
  ├── 将领存活情况
  └── 时间消耗(僵持太久→可能改变战术)

  战术调整：
  兵力优势>1.5→更频繁进攻
  兵力劣势<0.7→转为防御/寻求撤退
  侧翼被突破→调预备队填补
  主将阵亡→全军动摇→可能全线撤退
  有援军即将到达→拖延战术

敌军AI难度差异：
  简单：反应慢(90秒评估)，战术单一(只会正面进攻)，不使用计谋
  普通：正常反应(60秒)，使用基础战术，偶尔用计谋
  困难：快速反应(45秒)，积极利用地形和阵型相克，频繁使用计谋
  极难：极快反应(30秒)，接近人类高手，预判玩家战术
```

### 3.5 兵种特化AI

#### 骑兵AI

```
骑兵AI行为：

移动模式(无战斗)：
  ├── 利用速度优势：绕到敌军侧翼/后方
  ├── 保持距离：不在敌军步兵正面停留
  └── 等待冲锋时机：不急于投入战斗

战斗模式：
  ├── 冲锋判定：
  │   条件：距离合适(50-100米)+有冲锋空间+目标非架枪状态
  │   冲锋路径：直线加速→命中→穿过敌军→远处掉头→再次冲锋
  │   不冲锋时机：敌军在森林/沼泽/城内
  │
  ├── 如果冲锋后陷入混战→尝试脱离→重新冲锋
  └── 如果敌军架枪→绝不在正面冲锋→绕行

轻骑兵特殊：
  ├── 骚扰模式：接近→远程射击(弓骑)→拉开距离→再接近
  └── 追击模式：敌军溃逃→自动追击(速度优势)
```

#### 弓兵/弩兵AI

```
远程兵AI行为：

站位选择：
  ├── 寻找高地(+射程+视野)
  ├── 保持前方有友军遮挡(安全距离)
  ├── 弩兵偏好更近距离开火(穿透+精准)
  └── 弓兵偏好最大射程(高抛射)

目标选择：
  ├── 优先打击：密集编队/无盾单位/骑兵(大目标)
  ├── 其次打击：敌远程单位(反制火力)
  └── 避免打击：高举盾的单位/散开队形

弹药管理：
  弹药充足→自由射击
  弹药<30%→只射击高价值目标(敌军将领/密集队形)
  弹药耗尽→自动切换近战武器(短剑/匕首)

被近身反应：
  敌军接近<20米→自动后撤到友军后方
  如果无法后撤→切换近战→边打边撤
  弓兵近战能力极弱(伤害-40%)
```

#### 器械AI

```
攻城器械AI(投石机/弩炮等)：

目标优先级：
  1. 敌军器械(反制火力)
  2. 敌军密集编队(最大杀伤)
  3. 城门/城墙(攻城时)
  4. 箭塔/防御设施(攻城时)

射击时机：
  敌军进入射程→自动射击
  不射击混战中的敌我(避免误伤友军)
  敌军正在变阵(散乱)→优先射击

保护行为：
  器械前方布置步兵保护
  敌军接近器械→自动后撤(器械缓慢)
  器械被摧毁→操作士兵转为步兵
```

### 3.6 大规模士兵性能优化

```
千人战场性能策略（M&B式LOD）：

近景(0-50米，约20-50个士兵)：
  完整AI：State Tree全部状态+完整动画+碰撞检测

中景(50-150米，约50-200个士兵)：
  简化AI：减少决策频率(每5秒)/简化动画(关键帧)/简化碰撞

远景(150-500米，约200-800个士兵)：
  极简AI：仅团级决策(编队整体移动)/使用VAT(顶点动画纹理)/无个体碰撞

极远(500米+)：
  渲染为群体(用一个"编队"模型代表大量士兵)

Mass Entity系统(UE5)：
  远景士兵使用Mass Entity处理→CPU开销极小
  仅当士兵进入中景范围→切换为独立Actor+AI
  
目标：千人战场稳定60FPS(推荐配置)
```
