# 势力AI系统

## 设计目标

> 对标《三国志》的AI国家运营。七国不是等待玩家触发的静态布景——它们是自主的经济体、军事体和外交体。秦国会真正执行"远交近攻"，齐国会偏安通商，赵国会因多疑反复横跳。不依赖玩家，世界也能运转30年。

## 系统架构

```
                    势力AI (以秦国为例)
                           │
        ┌──────┬──────┬───┴───┬──────┬──────┐
        ▼      ▼      ▼       ▼      ▼      ▼
      经济AI  军事AI  外交AI  人事AI  科技AI  战争AI
      (发展   (征兵/  (结盟/  (任免/  (研发/  (开战/
       /税收) 部署)   谈判)   赏罚)   选择)   战术)
```

### 3.2 AI决策框架：效用评估系统

```
AI每月进行一次"战略评估"，选择最优行动组合：

战略评估流程：
  1. 评估当前国家状态（经济/军事/外交/科技/民忠）
  2. 识别威胁和机遇
  3. 生成可能的行动列表
  4. 对每个行动计算"效用分数"
  5. 选择效用最高的行动组合
  6. 执行→下月重新评估

效用分数 = Σ(目标权重 × 行动匹配度 × (1 + 性格修正) × 紧迫度)

示例：秦国评估"是否攻打韩国"
  统一天下权重：0.9
  攻打韩国匹配度：0.8（韩国弱+接壤）
  性格修正(高风险偏好)：+0.2
  紧迫度：0.7（韩国有铁矿石资源+扼守东出通道）
  
  效用分 = 0.9 × 0.8 × 1.2 × 0.7 = 0.605 (高分→决定打)

同时，秦国还评估"是否与齐国保持友好"
  统一天下权重：0.9
  保持友好匹配度：0.6（远交近攻策略）
  性格修正(低信任)：-0.1
  紧迫度：0.3（齐国暂时不构成威胁）
  
  效用分 = 0.9 × 0.6 × 0.9 × 0.3 = 0.146 (低分→不优先)
```

### 3.3 经济AI

```
AI国家每月的经济决策：

收入分配优先级（各国权重不同）：
  秦国：军费50% > 科技20% > 基建15% > 储备10% > 民生5%
  齐国：基建30% > 储备25% > 民生20% > 科技15% > 军费10%
  楚国：基建25% > 军费25% > 文化20% > 科技15% > 储备15%

内政指令选择：
  每月从可用指令中选择收益最高的3-5条执行
  选择依据：当前数值缺口 + 长期发展战略 + 当前危机

AI城池发展方向：
  秦国：边境=军事城 / 腹地=农业城 / 咸阳=综合
  齐国：沿海=商业城 / 临淄=科技城
  韩国：边境=军事城(防御) / 宜阳=科技城(冶铁)

灾荒应对：
  AI国库有储备粮→自动赈灾
  储备不足→可能向邻国买粮/向宗主国求援/放任(暴政型君主)
```

### 3.4 军事AI

```
AI军事决策循环：

1. 兵力评估
   当前总兵力 vs 目标兵力(按国家规模)
   兵力不足→征兵（征兵量=缺口/2，避免一次性征太多）
   
2. 部署策略
   边境城池：驻军权重40%
   首都：驻军权重30%
   内部城池：驻军权重20%
   机动兵力：10%（应对突发事件）

3. 战争决策
   │
   ├── 进攻性战争（主动开战）
   │   评估：目标国力/盟友数量/地形/时机
   │   条件：己方兵力>目标1.5倍 且 目标未与其他强国结盟
   │   秦国：频繁评估（每月）→扩张驱动
   │   齐国：极少评估（每年）→和平驱动
   │
   ├── 防御性战争（被动应战）
   │   被攻击→自动应战
   │   评估是否求援（向盟友/向宗主国）
   │
   └── 战争终止条件
       己方兵力<开战时的40% → 求和
       敌方兵力<开战时的30% → 可以提出对方投降条件
       占领敌方首都 → 敌方可选择臣服
       战争持续>24个月 → 双方厌战→可能自动停战

4. 战术AI（大地图军团层面）
   ├── 集中兵力 vs 分散进攻
   ├── 避开敌军主力 vs 正面决战
   ├── 利用地形（关隘防守/平原进攻）
   └── 名将效应（白起→偏好包围战术/廉颇→偏好固守）
```

### 3.5 外交AI

```
AI外交决策（详见140_外交系统）：

每月外交评估：
  1. 审视当前国际格局（谁强谁弱/谁跟谁结盟）
  2. 识别威胁（最强国/直接邻国/有仇的国家）
  3. 制定外交策略

各国策略倾向：
  秦国：远交近攻（与远方齐国结好+攻打邻国韩魏）
  齐国：左右逢源（与各国通商+尽量不卷入战争）
  楚国：坐山观虎斗（观望中原战争+趁机扩张南方）
  燕国：联赵抗齐或联齐抗赵（取决于当前最强邻国）
  韩国：骑墙（谁强依附谁/常换宗主）
  魏国：反复横跳（今天联秦明天联赵/根据利益切换）
  赵国：独狼（不太信任任何大国/靠骑兵自保）

AI背盟判定：
  背盟收益 > 背盟代价×性格修正 → 可能背盟
  秦国性格修正(实用主义)：背盟代价×0.6 → 更容易背盟
  齐国性格修正(商业信用)：背盟代价×1.5 → 几乎不背盟
  
AI接受同盟概率：
  见141_同盟与条约系统 3.3节公式
```

### 3.6 人事AI

```
AI官员任免：

选拔逻辑：
  空缺席位→从在野或现有官员中选拔
  选拔标准 = 属性匹配度×0.6 + 忠诚度×0.2 + 派系平衡×0.2

赏赐逻辑：
  武将忠诚<60 → 赏赐金/宝物
  文官忠诚<50 → 赏赐/升迁
  赏赐频率受国库充裕度影响

处罚逻辑：
  官员考核不合格→贬官/罢免
  发现贪污→按程度处分(秦国重刑/齐国从宽)
  发现谋反→立即抓捕/处决

AI君主也会犯错：
  可能任用庸才（识人不明）
  可能被谗言蒙蔽（处决忠诚将领）
  低级难度→AI更容易犯人事错误
```

### 3.7 科技AI

```
AI科技研发策略：

研发方向（各国侧重）：
  秦国：军事科技优先(铁兵器→弩机→攻城器械→精铁甲)
  齐国：经济科技优先(市集→商路→盐铁→货币统一)
  楚国：工艺科技优先(冶铁→造船→织造)
  韩国：军事+工艺(弩机→冶铁→炼钢→连弩)
  赵国：军事(骑兵革新→铁兵器→精铁甲)
  魏国：政治+军事(郡县→集权→铁兵器→精铁甲)
  燕国：农业+防御(铁犁→水利→筑城→...)

研发资源投入：
  国库收入×科技投入比例（各国不同）
  齐国15%/秦国8%/韩国12%

科技窃取（间谍）：
  AI也会派遣间谍窃取邻国科技
  见143_间谍与情报系统
```

### 3.8 动态势力消长

```
AI国家的兴衰循环：

崛起阶段：
  连续战胜+经济良好+外交稳定→国力上升
  国力上升→更加自信→更频繁扩张→可能成为"boss"

衰落阶段：
  连续战败/重大灾荒/内乱→国力下降
  国力下降→更加保守/更依赖盟友/更容易被攻击
  衰落持续→可能被灭国（被其他AI国家吞并）

国家灭亡：
  AI国家首都沦陷+主力被歼→可能灭亡
  灭亡后：领土被战胜国吞并
  该国将领：部分被杀/部分投降/部分逃亡
  影响：其他国家重新评估格局（"秦国灭了韩国，下一个可能是我们"）

玩家介入：
  玩家可以改变任何AI国家的命运轨迹
  玩家扶持弱国→可能逆转衰落
  玩家加速强国崩溃→改变历史大势
```

### 3.9 AI国家间的合纵连横（无需玩家触发）

```
自主合纵：
  条件：某国国力>其他任两国之和 + 该国霸业>60
  触发：纵横家苏秦/公孙衍在附近国家→自动发起合纵
  效果：3-6国自动结盟限制最强国
  
  示例：秦国国力超过赵+魏之和→
        苏秦游说六国→合纵形成→六国同时与秦关系-30→秦扩张受阻

自主连横：
  条件：合纵已形成 + 秦国有张仪
  触发：张仪出使合纵成员→逐个拉拢
  效果：某国退出合纵→合纵开始瓦解→秦国获得战略窗口

不需要玩家干预：
  这些事件在AI之间自主发生
  玩家可以参与（加入/破坏/利用）
  但即使玩家完全不管，战国格局也会动态演变
```

## AI势力表现（30年模拟预期）

```
如果玩家完全不干预（仅观察模式），30年后的战国格局预期：

高概率事件：
  ├── 秦国扩张：吞并韩国(15年内) + 魏国部分领土
  ├── 楚国南扩：吞并南方小国(越国残余)
  ├── 赵国北拓：吞并中山残部+与匈奴交战
  ├── 合纵形成→瓦解→再形成(2-3轮循环)
  ├── 齐国保持繁荣但军事不扩张
  └── 周天子最终被废(25-30年内，秦国或玩家)

低概率事件：
  ├── 韩国逆袭（几乎不可能，国力太弱）
  ├── 燕国崛起（除非玩家扶持）
  └── 六国真正团结灭秦（合纵历史上从未真正成功过）
```
