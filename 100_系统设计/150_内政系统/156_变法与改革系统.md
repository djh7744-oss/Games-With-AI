# 变法与改革系统

## 设计目标

> 战国时代最激动人心的主题之一就是变法——商鞅变法让秦国从西陲弱国变成虎狼之国，吴起变法让楚国一度强盛，胡服骑射让赵国成为军事强国。变法不是点一个按钮——它伴随着巨大的政治风险、利益集团的激烈反抗，以及可能的身死名裂。但成功的变法将彻底改变国家命运。

## 系统概述

变法是一种"结构性改革"，区别于日常的政策切换。变法的效果是永久性的——改变游戏的基本规则。但变法过程中需要面对来自既得利益集团的反对、旧制度的惯性、以及对变法者个人的反噬。历史上大多数变法者没有好下场——玩家需要比他们更聪明。

## 核心机制

### 3.1 变法类型

#### 历史变法（可重现）

| 变法 | 发起国 | 发起者 | 核心内容 | 受益群体 | 受损群体 | 历史结果 |
|------|--------|--------|---------|---------|---------|---------|
| **商鞅变法** | 秦 | 商鞅 | 废井田开阡陌/军功爵/连坐法/重农抑商/郡县制 | 农民/士兵/国君 | 旧贵族/世卿/商人 | 成功→秦强→商鞅被车裂 |
| **吴起变法** | 楚 | 吴起 | 削贵族/强军备/明法审令 | 国君/士兵 | 楚国大贵族 | 半成功→楚强→吴起被射杀 |
| **胡服骑射** | 赵 | 赵武灵王 | 改穿胡服/学骑射/建骑兵 | 骑兵/边境民众 | 传统贵族/中原文化派 | 成功→赵强→武灵王被困死沙丘 |
| **申不害变法** | 韩 | 申不害 | 术治/吏治改革/考核制度 | 国君/中层官僚 | 贵族/大官僚 | 半成功→韩中兴一段→申不害善终(极少) |
| **李悝变法** | 魏 | 李悝 | 尽地力/平籴法/法经 | 农民/小地主 | 大贵族/奸商 | 成功→魏战国初霸→李悝善终 |
| **邹忌改革** | 齐 | 邹忌 | 广开言路/赏罚分明 | 士人/中层官员 | 腐败官僚 | 成功→齐威王中兴 |

#### 自定义变法（玩家设计）

```
玩家可自定义变法方案：

1. 选择变法方向：
   ├── 经济改革（税制/土地/货币）
   ├── 军事改革（兵制/装备/编制）
   ├── 政治改革（官僚制/律法/集权）
   ├── 社会改革（奴隶制/等级制/习俗）
   └── 综合改革（多项并举，风险最大）

2. 设定改革力度：
   温和：效果50%但反对力度30%
   标准：效果100%反对力度100%
   激进：效果200%但反对力度300%（高风险高回报）

3. 准备变法条件：
   ├── 己方权力基础（自己是君主/权臣/改革大臣）
   ├── 支持者联盟（哪些派系支持你）
   ├── 资源储备（变法需要大量金/人才）
   └── 舆论准备（学派支持/民间宣传）
```

### 3.2 变法流程

```mermaid
flowchart TD
    A[提出变法方案] --> B[筹备阶段]
    B --> C[争取支持者]
    C --> D{支持力量足够?}
    D -->|是| E[颁布变法令]
    D -->|否| F{选择}
    F -->|妥协→缩小改革范围| G[修改方案→回到B]
    F -->|强行推进| E
    F -->|放弃变法| H[变法失败→政治信誉受损]
    E --> I[实施阶段 3-12月]
    I --> J{遭遇阻力?}
    J -->|旧贵族反对| K{选择应对}
    J -->|顺利推进| L[变法初步成功]
    K -->|镇压(武勇+法家)| M[旧贵族叛乱→需要平叛]
    K -->|妥协(魅力+儒家)| N[减缓改革速度→效果打折]
    K -->|收买(经营)| O[花费巨额金→平息反对]
    M --> P{平叛成功?}
    P -->|是| L
    P -->|否| Q[变法失败→变法者被清算]
    N --> L
    O --> L
    L --> R[变法效果生效]
    R --> S[变法者个人命运判定]
    S -->|成功善终| T[变法者声望+，安全]
    S -->|成功但被反噬| U[变法者被清算/处决]
    S -->|变法成果被废除| V[继任者废除变法→回到原点]
```

### 3.3 反对力量

```
变法中遇到的反对力量：

1. 旧贵族（最主要的反对者）
   反对原因：变法损害其世袭特权
   反抗方式：朝廷阻挠→地方叛乱→刺杀变法者
   应对方式：镇压/收买/妥协/架空
   风险：旧贵族往往有私兵和地方势力

2. 保守派官员
   反对原因：习惯旧制度，害怕变化
   反抗方式：消极怠工→集体辞职→暗中破坏
   应对方式：替换官员(用支持变法的新人)

3. 民众（短期可能反对）
   反对原因：不理解变法、害怕改变习惯、短期利益受损
   反抗方式：民忠下降→抗法→暴动
   应对方式：宣传变法好处/给短期利益/强力镇压

4. 外部势力
   反对原因：邻国不希望看到你强大
   反抗方式：支持反对派、趁乱进攻
   应对方式：变法期间保持强大军力威慑
```

### 3.4 变法风险评估

```
变法风险指数 = 改革力度 × 受损群体强度 × (1 - 支持力量/100) 
              × (1 - 变法者权力基础/100)

改革力度：温和30/标准60/激进90
受损群体强度：旧贵族势力(秦国20/楚国80/齐国50)
支持力量：支持变法的派系占比
变法者权力基础：自己的爵位+声望

风险指数评估：
  <20：低风险（几乎无阻力）
  20-40：中等风险（有反对但可控）
  40-70：高风险（激烈反对+可能平叛）
  70-90：极高风险（几乎必定叛乱）
  >90：自杀式改革（历史上变法者都死了）

变法者个人命运：
  即使变法成功→变法者100%被旧势力仇视
  君主在世→可保护变法者
  君主去世→变法者失去靠山→被清算概率60-90%
  
  变法者需要准备"后路"：
    急流勇退（变法成功后主动隐退）→安全但成果可能被废
    培养继承人→继续维持变法
    彻底清洗反对派→最安全但仁义-极多
```

### 3.5 变法效果

```
成功变法后的永久效果（例：商鞅变法的效果）：

经济层面：
  ├── 废除井田→农业产出永久+30%
  ├── 重农抑商→农业税收+40%
  ├── 开阡陌→土地利用率+25%
  └── 统一度量衡→交易成本-25%

军事层面：
  ├── 军功爵制→士兵士气永久+25%
  ├── 二十等爵→招募优质士兵+40%
  ├── 连坐法→逃兵率-60%
  └── 奖励军功→全民好战文化(武名获取+30%)

政治层面：
  ├── 郡县制→中央集权+50%(地方独立概率-80%)
  ├── 法令统一→行政效率+35%
  ├── 废除世卿世禄→人才流动性+100%
  └── 告奸法→犯罪率-50%

文化层面：
  └── 焚诗书而明法令→法家影响+100%，其他学派-30%

变法效果是永久的（不会被新政策覆盖）
但变法可被后续君主废除（回到变法前状态）
```

### 3.6 变法与国家发展阶段

```
不同国家阶段适合不同变法：

初期（国力弱/领土小）：
  适合：李悝式温和变法(发展经济+法制)
  不适合：商鞅式激进变法(没有足够的镇压能力)

上升期（国力增长/积极扩张）：
  适合：商鞅变法/吴起变法/胡服骑射(先军政策)
  风险：中等(有扩张红利可对冲改革阵痛)

鼎盛期（国力巅峰/领土广大）：
  适合：邹忌式微调改革(精耕细作)
  不适合：大动干戈的变法(系统太复杂→改不动)

衰落期（国力下降/危机显现）：
  适合：申不害式救命改革(术治/吏治/效率)
  风险：高(衰落期资源不多→改错一步可能加速灭亡)
```

### 3.7 变法失败处理

```
变法失败的可能结局：

1. 变法方案被否决（最温和的失败）
   结果：浪费准备时间+资源+政治信用
   下次变法更难(反对派更有组织)
   变法者本人安全

2. 变法中途夭折
   结果：已实施的部分可能回流
   社会混乱（新旧制度并存）→民忠-20
   变法者可能被问责

3. 变法引发叛乱→镇压失败
   结果：变法者被处决/流放
   变法完全被废
   国家陷入内战→国力大减

4. 变法成功但变法者被杀（商鞅/吴起的命运）
   结果：变法成果保留
   变法者本人牺牲
   继任者摘取果实
   玩家长远来看→变法成功但角色死亡→继承者受益

玩家选择：
  安全第一→温和变法(效果-50%但本人安全)
  理想主义→标准变法(正常风险)
  崇高牺牲→激进变法→成功则国家富强→自己被清算→值得吗？
```

## 历史变法触发条件

```
商鞅变法（秦国专属）：
  触发条件：秦国+法家学派加入+商鞅加入+君主支持
  前置任务：与旧贵族辩论(商鞅vs甘龙/杜挚)→君主决定支持变法
  关键节点：徙木立信(树立法令威信)

吴起变法（楚国专属）：
  触发条件：楚国+吴起加入+楚悼王在位
  前置任务：说服楚王+争取部分贵族中立

胡服骑射（赵国专属）：
  触发条件：赵国+赵武灵王在位+与游牧民族交战经验
  前置任务：说服王叔公子成+在军中试行胡服
```
