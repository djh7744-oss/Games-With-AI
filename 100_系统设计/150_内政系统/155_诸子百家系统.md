# 诸子百家系统

## 设计目标

> 诸子百家不只是文化标签。加入学派影响你可用的人物、政策、技能和外交选项。儒家偏仁政、法家偏集权、墨家偏兼爱非攻、道家偏无为而治……你的学派选择就是你的治国哲学。学派之间有争鸣、有融合、有打压。

## 系统概述

战国时代有十家主要学派：儒/法/墨/道/兵/纵横/名/阴阳/农/杂。玩家可加入一家学派（或多学兼修），获得该学派的专属被动效果、可选政策、代表人物任务和技能。学派影响力随玩家推广而变化，高影响力学派可主导国策方向。

### 3.3 加入学派

```
加入条件：
  1. 找到该学派的代表人物/学堂
  2. 满足属性要求（各学派不同）
     儒家：仁义≥30，智略≥40
     法家：智略≥45
     墨家：仁义≥40，工艺≥30
     道家：智略≥50
     兵家：统率≥30或武勇≥30
     纵横家：魅力≥50 (详见142)
     名家：智略≥55
     阴阳家：智略≥50
     农家：经营≥30或工艺≥30
     杂家：至少已加入2个其他学派
  3. 完成入门任务（各学派不同）
     儒家：完成一篇"仁政论"
     法家：参与一次变法/执法
     墨家：参与一次防御战/赈灾
     道家：隐居修行7天
     兵家：指挥一场胜利的战斗
     纵横家：成功游说一次
     名家：赢得一场辩论
     阴阳家：成功预测一次天气变化
     农家：亲自耕种/指导农业生产7天
     杂家：阅读4种不同学派的著作

学派等级：
  Lv1 弟子：获得学派被动BUFF(50%效果)
  Lv2 门徒：获得学派被动BUFF(100%)，解锁1项学派政策
  Lv3 学者：获得专属技能，解锁第2项学派政策
  Lv4 宗师：被动BUFF效果翻倍，可收徒
  Lv5 巨子/圣人：该学派所有成员与你关系+50，可主导学派方向

学派内部地位提升：
  著书立说：每部著作+1等级经验
  宣扬学说：每说服一人加入+经验
  实践学说：按学派核心理念行事+经验
```

### 3.4 学派冲突与融合

```
学派冲突矩阵：
  法家 vs 儒家：治国理念根本对立
    冲突表现：法家政策的推广自动降低儒家满意度
    可能事件：儒生上书反对变法→选择镇压(法家+)/妥协(儒家+)
  
  名家 vs 墨家：逻辑 vs 实用
    冲突表现：名家辩论攻击墨家"兼爱"理论的逻辑漏洞
    可能事件：稷下学宫大辩论→墨家暂时退出稷下
  
  激烈的学派之争 → 朝堂分裂 → 需要君主裁决
  君主偏袒一方 → 另一方人物出走/忠诚下降
  君主调和 → 杂家学派影响上升

学派融合（后期可能）：
  法家+儒家 → "外儒内法"（汉朝模式）
  道家+法家 → "黄老之治"（汉初模式）
  儒+墨+法 → 综合治国体系
  杂家 → 本身就是融合体

融合条件：
  玩家同时是两派宗师(Lv4+)
  两派代表人物关系≥友好
  完成"学派融合"特殊任务
```

### 3.5 学派与国策

```
学派影响力：

全国各学派影响力的总和 → 决定国家"思想倾向"

影响力计算：
  君主学派：40%权重
  官员中学派分布：35%权重
  民间学派流传：25%权重

思想倾向影响：
  法家主导→可选重刑/苛法/连坐等政策，对外扩张倾向
  儒家主导→可选宽政/礼乐/兴学等政策，外交和平倾向
  墨家主导→可选节用/非攻/尚贤，防御性国策
  道家主导→可选无为/减税/小国寡民，减少政府干预

  多学派均衡→国家灵活但单项不极致
  独尊一派→该派效果最大化，但压制其他学派
```

### 3.6 学派代表人物任务

```
每个学派有独特的任务线：

儒家：
  孔子周游列国→玩家拜访各国推广仁政→获得"圣人之徒"称号
  孟子见梁惠王→辩论"义利之辨"→推广王道思想

法家：
  商鞅变法→在秦国推行彻底变法→成功奖励巨大但阻力也巨大
  韩非著书→完成《韩非子》→成为法家集大成者

墨家：
  墨子救宋→阻止楚国攻打宋国(大规模守城战)→"非攻"实践

兵家：
  孙子著兵法→完成《孙子兵法》→全军获得"知己知彼"BUFF

道家：
  老子出关→在函谷关写下《道德经》→哲学成就
  庄子梦蝶→完成逍遥游→获得超然物外的特殊能力

```
