# 人事与官僚系统

## 设计目标

> 对标《三国志》的人事系统：得人者昌，失人者亡。有优秀的官员，城池才能繁荣；用错了人，再好的政策也会执行变味。人才选拔、考核、升迁、惩处——每一个环节都影响国家命运。

## 系统概述

玩家（作为君主或高层官员）需要任免官员管理国家。官员分为中央官（管全国事务）、地方官（管具体城池）、军事官（带兵打仗）。每位官员有自己的属性、忠诚、野心、性格，这些因素影响其治理效果和忠诚度。

### 3.2 官员属性

```
官员在治理中的属性影响：

任用官员时，其属性会产生以下效果：

城池治理（地方官）：
  统率：治安管理效率(治安下降速度-统率/10)
  智略：处理突发事件(灾害应对/叛乱预防)
  魅力：民忠自然增长(+魅力/20/月)
  经营：税收效率(+经营/3%) + 商业发展速度
  工艺：建设速度(+工艺/2%) + 工程质量

中央治理（中央官）：
  效果按该官员属性的50-80%影响全国
  如：治粟内史(经营)85 → 全国税收效率+85/3%×0.7=+19.8%

军事指挥（军事官）：
  统率：带兵上限+指挥能力
  武勇：该将军亲自参战时部队士气
  智略：避免伏击/计谋成功率
```

### 3.3 官员选拔

```
选拔方式：

1. 察举（推荐制）
  现有官员推荐人才→推荐质量=推荐者的智略/魅力
  可能推荐出优秀人才→也可能任人唯亲
  
2. 征辟（直接征召）
  君主主动征召在野名士
  需要：君主声望足够+对方愿意出仕
  如在野的诸葛亮→需要三顾茅庐

3. 军功晋升
  士兵/低级军官通过战功晋升
  最公平的晋升渠道（秦国军功爵制）

4. 科举萌芽（时代超前）
  战国晚期/统一后可试行
  通过考试选拔（智略判定）
  公平但传统贵族反对

5. 自荐
  在野人士主动求官
  需要评估真实能力vs自我吹嘘
  战国"毛遂自荐"模式
```

### 3.4 官员考核

```
考核周期：每年一次（年终考核）

考核指标（按官职类型）：

地方官：
  ├── 人口增长率（目标>2%/年）→ 不达标-评价
  ├── 税收增长率
  ├── 民忠维持（不能低于50）
  ├── 治安状况
  └── 重大事件处理（灾害/叛乱/战争）

中央官：
  按职责范围考核
  如：治粟内史→国库增长
      御史大夫→贪腐案件数量（越少越好）

考核结果：
  优秀(绩>85)：升迁推荐+赏赐
  合格(60-85)：留任
  不合格(<60)：降职/贬官/罢免
  极差(<30)：流放/处决（极端情况）

考核连续3年优秀→官员获得"能臣"称号→治理效果额外+10%
考核连续2年不合格→自动贬官
```

### 3.5 官员俸禄与贪污

```
俸禄制度：
  每位官员按品级领取俸禄（月俸）
  全国年俸禄支出 = Σ所有官员月俸×12

俸禄不足的后果：
  俸禄拖欠→官员忠诚集体下降
  低俸禄→贪污概率增加
  不发俸禄→几乎必定叛变/贪污失控

贪污系统：
  贪污概率 = 基础概率(10%) + (100-官员忠诚)/5% + (100-治安)/10%
            - 御史大夫监察效果
  
  贪污后果：
  被发现→降职/罢免/处决（取决于严重程度）
  未被发现→该城池税收暗减10-30%（流入官员私囊）
  
  反贪措施：
  任命御史大夫（降低全国贪污概率）
  定期考核（提高发现率）
  高薪养廉（俸禄高→贪污概率降低）
  严刑峻法（一旦发现→严惩，威慑效应）
```

### 3.6 官员派系

```
官员间有隐藏的"派系倾向"：

主要派系类型：
  ├── 宗室派：王族成员→忠诚度通常高，但能力参差不齐
  ├── 军功派：靠战功升迁→统率高，轻视文官
  ├── 士人派：靠学问入仕→智略高，可能"清谈误国"
  ├── 外戚派：通过联姻上位→能力一般，但关系硬
  ├── 客卿派：外国人入仕→能力可能很强，但忠诚存疑
  └── 地方派：各地豪强→在本地势力大，但可能尾大不掉

派系影响：
  同一派系的官员互相推荐(+好感)
  不同派系可能互相攻击(朝堂争斗)
  一派独大→君主被架空的风险
  派系平衡→君主有更多掌控权

玩家可操作：
  提拔被压制的派系→恢复平衡
  借一派之手打击另一派
  但任何操作都有代价
```

### 3.7 任命与罢免的声望影响

```
任命：
  任命贤能(高属性+高忠诚)→仁义+2,信义+2
  任命庸才(低属性)→信义-3(识人不明)
  任命亲信(高忠诚低能力)→属下议论(其他官员忠诚-3)
  任命外国人→该国关系+5

罢免：
  罢免无能者→仁义+3(为民除害)
  罢免能臣(无过而罢)→信义-10(兔死狗烹)
  流放官员→其派系成员忠诚-10
  处决官员→其家族永久仇恨(-100)
```

## 人事管理UI

```
官员总览面板：
  ├── 按官职类型筛选（中央/地方/军事）
  ├── 每位官员显示：头像/姓名/官职/主要属性/忠诚度/俸禄/考核评价
  ├── 排序功能：按属性/按忠诚/按考核
  └── 操作按钮：任命/升迁/贬官/赏赐/训诫/罢免/流放/处决

任命界面：
  选择空缺官职→从可任命的候选人中选择
  候选人显示：属性/忠诚/当前职位/与空缺席位的匹配度
  任命确认→官员就职

考核界面：
  年终弹出考核报告
  每城/每部门考核结果
  优秀者高亮(绿色)/不合格者标红
  批量处理：全合格→赏赐/全不合格→大清洗
```
