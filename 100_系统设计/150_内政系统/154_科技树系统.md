# 科技树系统

## 设计目标

> 对标《文明》系列的科技树：六大分支，48项科技，每项科技解锁新能力/新兵种/新建筑/新政策。科技不是线性的——玩家可以选择国家的发展方向（先军事征服还是先经济繁荣？）。科技传播机制让落后国家有机会追赶。科技之间存在协同效应，完成特定组合解锁强大的"科技协同"加成。

## 系统概述

科技树分为六大分支，每分支8项科技，共48项。科技分为三个时代（古典→争鸣→帝国），时代进阶需要完成该时代足够的科技。研发速度取决于资源投入、人才储备、已有科技基础。科技可通过三种途径获得：自主研究、邻国传播、间谍窃取。

## 核心机制

### 3.0 科技时代系统

```
时代进阶：

第一时代：古典时代（起始）
  可用科技：各分支T1-T3（共18项基础科技）
  特征：战国初期的技术水平
  进阶条件：完成≥12项古典科技

第二时代：争鸣时代
  可用科技：各分支T4-T6（共18项中级科技）
  特征：百家争鸣驱动的技术爆发
  进阶条件：完成≥12项争鸣科技

第三时代：帝国时代
  可用科技：各分支T7-T8（共12项终极科技）
  特征：大一统帝国的超级工程和终极发明
  时代效果：所有已完成科技效果+25%（集大成）

时代影响：
  时代越靠后，科技研发基础花费越高(+50%/时代)
  但国家基础设施也越强→总体研发速度在帝国时代最快
  跨时代研发(跳过时代)→花费×3（不推荐）
```

### 3.1 六大科技分支

```
                    科技树总览
                        │
    ┌───────┬───────┬───┴───┬───────┬───────┐
    ▼       ▼       ▼       ▼       ▼       ▼
  农业    军事    经济    政治    文化    工艺
 (8项)   (8项)   (8项)   (8项)   (8项)   (8项)

每分支科技链结构：
  T1 ──→ T2 ──→ T3 ──→ T4 ──→ T5
   │       │       │       │
   └──→ T6 ──→ T7 ──→ T8 (终极)

分支间交叉前置：
  经济T7(海贸) 需要 工艺T5(造船术)
  政治T8(车同轨) 需要 经济T5(货币统一)
  经济T3(盐铁专营) 需要 农业T3(水利) 或 军事T1(铁兵器)
  工艺T5(造船术) 需要 经济T2(商路开拓)
```

### 3.8 科技研发机制

```
研发速度 = 基础研发力 × 科技修正 × 人才修正 × 投入修正 × 时代修正

基础研发力：1.0/天
科技修正：已完成的该分支科技数×5%（科技树越深越慢）
人才修正：国家技术值/50 + 是否有相关科学家(有则+0.3)
投入修正：投入金/2000/天 + 投入竹简/书籍(每卷+0.05/天，限10卷)
时代修正：古典×1.0 / 争鸣×0.7 / 帝国×0.5（越高级越难）

研发模式：

1. 自主研究
   选择科技 → 投入资源 → 研发进度条逐渐填充
   可排队研发(最多3项) → 完成一项自动开始下一项
   研发中可暂停/切换(已投入资源不退还，保留进度50%)

2. 突破(Inspiration)
   某些游戏行为可触发"科技灵感" → 特定科技研发速度+50%持续30天
   例：
     连续3次成功使用火攻 → 火器初探灵感触发
     完成一次跨国贸易利润>200% → 商路开拓灵感触发
     建造10座水利设施 → 都江堰模式灵感触发
     一次招降3名敌将 → 察举制灵感触发
     瘟疫中成功救治>1000人 → 医学灵感触发
   
   灵感是一次性的(每项科技最多触发一次灵感)

3. 被动传播
   邻国已完成科技 → 每年有概率自动获得进度
   传播概率 = 邻国技术值/500 + 文化科技加成(百家争鸣+30%)
   传播速度 = 邻国研发该科技时速度的20-50%
   
   两国关系越好 → 传播越快
   两国敌对 → 传播可能中断

4. 间谍窃取
   派遣间谍潜入科技先进国 → 窃取特定科技(见143_间谍与情报系统)
   风险：间谍被抓 → 两国关系恶化 → 可能引发外交危机
   成功后：获得目标科技50-80%的研发进度

5. 科技交易
   在外交中可要求对方转让科技(作为条约条件)
   对方接受率取决于：关系+该科技的战略价值+你提供的条件
   转让后：立即获得该科技(无需研发)
   
   代价：通常非常昂贵(对方不会轻易转让核心技术)
   例：要求秦国转让冶铁术 → 可能需要割让2座城池或支付50000金
```
