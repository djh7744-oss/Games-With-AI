# 军事科技详解

## 设计目标

> 军事科技直接影响战场胜负。不只是"伤害+15%"这样的数值变化——每项军事科技都改变兵种体系、解锁新战术选项、重塑战争方式。从铁兵器取代铜兵器，到火药武器改变战争形态，军事科技是军事实力的核心驱动力。

---

## T1 铁兵器（古典时代）

```
研发需求：60天 / 3000金
前置：无
解锁：T2铁制兵种 / 铁兵器(装备)

效果详解：
  ├── 全军武器伤害+15%（铜→铁的材质跃升）
  ├── 解锁T2兵种招募(秦锐士/楚申息之师/齐技击之士等)
  ├── 铜兵器退役→旧铜兵器可熔铸为铜钱(回收50%价值)
  ├── 解锁冶铁坊建筑前置需求
  └── 武器耐久+30%(铁比铜耐用)

历史背景：
  战国中期，铁兵器逐渐取代青铜兵器。
  韩国宜阳以冶铁闻名，天下强弓劲弩皆出于韩。
  秦国在商鞅变法后大规模推广铁兵器标准化生产。

战略意义：
  铁兵器是军事科技的入门，几乎所有后续军事科技依赖它。
  不研究铁兵器→长期使用T1铜兵器→军事严重落后。
  通常是最优先研发的军事科技。

各国特色：
  韩国：研发时间-30%(60→42天)，宜阳铁资源优势
  秦国：研发花费-25%(3000→2250金)，标准化生产
  楚国：铁兵器伤害额外+5%(宛城铁矿)
```

## T2 弩机（古典时代）

```
研发需求：90天 / 5000金
前置：铁兵器
解锁：弩兵兵种 / 弩装备

效果详解：
  ├── 解锁弩兵(独立兵种线)
  ├── 弩破甲+20%（穿透盔甲的能力）
  ├── 弩射程+15%（比弓更远更精准）
  ├── 守城时弩兵伤害额外+20%(城墙高度优势)
  └── 解锁制弩坊(建筑)

弩 vs 弓的选择：
  弓：射速快(1.2秒/箭)，精度一般，适合压制射击
  弩：射速慢(2.5秒/发)，精度高，穿甲强，适合精准打击
  战术选择：开阔战场→弓；攻城/守城→弩

兵种解锁：
  秦国：秦强弩手(T2弩兵，破甲30%)
  韩国：韩劲弩手(T2弩兵，精度+15%，韩国弩天下第一)
  魏国：魏强弩手(T2弩兵，连弩前置)

战略意义：
  弩机是"弱者对抗强者"的关键科技——弩兵可以在安全距离上
  击穿重甲，让精锐重步兵不再无敌。
  韩国凭此科技在强邻环伺中生存了200年。
```

## T3 骑兵革新（古典时代）

```
研发需求：90天 / 6000金
前置：无(独立起点)
解锁：重骑兵 / 马上战斗技巧

效果详解：
  ├── 骑兵冲锋伤害+25%
  ├── 解锁重骑兵兵种(T2-T3)
  ├── 解锁马场改良(战马品质+1级)
  ├── 马上弓箭精度惩罚-15%（骑射改善）
  └── 骑兵移动速度+10%

不依赖铁兵器：骑兵革新可独立研发
  如果先研发骑兵革新→有强力骑兵但步兵装备差
  如果先研发铁兵器→步兵强但骑兵弱
  均衡发展→两翼齐飞

兵种解锁：
  赵国：赵铁骑(T2重骑兵，冲锋+30%，赵国骑兵立国)
  秦国：秦战车兵(T3战车，需骑兵革新+铁兵器)
  燕国：燕骑射手(T2弓骑兵)

战略意义：
  骑兵革新是赵国和秦国扩张的关键。
  赵国胡服骑射本质上就是骑兵革新的先行者。
  没有骑兵革新的国家→只能被动挨打→无法追击溃敌。
```

## T4 连弩（争鸣时代）

```
研发需求：150天 / 12000金
前置：弩机
解锁：连弩兵 / 速射战术

效果详解：
  ├── 解锁连弩兵(3连发，射速+60%)
  ├── 连弩弹药消耗×2（射得快但也费弹药）
  ├── 守城时连弩效果翻倍(一人操作，城墙狭小空间优势)
  ├── 解锁"箭雨"指令：连弩编队齐射→覆盖区域持续伤害
  └── 心理威慑：连弩声音让敌军士气-5(持续的"嗖嗖"声)

战术维度：
  3连发后的装填间隙(4秒)→敌军可能趁机冲锋
  配合步兵保护装填间隙→形成"射击-保护-装填-射击"循环
  连弩+火攻=终极守城组合

兵种解锁：
  韩国：韩强弩锐士(T3连弩兵，暴击率+20%，穿甲50%)
  魏国：魏连弩手(T3连弩兵，3连发)
  秦国：秦国后期可研发(但比韩国晚一个时代)

历史超越：
  连弩在战国时代实际上尚未普及(诸葛亮改良是后来的事)
  游戏中作为"架空科技"→如果韩国存活足够久→可提前发明连弩
  这是"玩家改变历史"的体现
```

## T5 攻城器械（争鸣时代）

```
研发需求：180天 / 18000金
前置：铁兵器 + 骑兵革新
解锁：投石机/冲车/井阑/云梯(标准化)

效果详解：
  ├── 解锁投石机：远距离破坏城墙/箭塔
  ├── 解锁冲车：近距离撞击城门/城墙
  ├── 解锁井阑：移动箭塔，弓弩手高台射击
  ├── 云梯升级：从简易梯子→标准化攻城云梯
  ├── 攻城效率+40%(所有攻城方式)
  └── 解锁"器械制造所"建筑

器械战术选择：
  投石机：射程200米→先轰击城墙/箭塔→破坏防御后再步兵进攻
  冲车：需推到城门下→需要步兵保护→高风险高回报
  井阑：推到城墙附近→己方弓弩手登上→与守军对射→掩护攻城
  组合使用最佳：投石机轰→井阑压制→冲车破门→步兵冲锋

资源消耗：
  制造一台投石机：50木材+15精铁+8人操作+12小时
  制造一台冲车：30木材+10精铁+6人操作+6小时
  制造一座井阑：40木材+5精铁+10人操作+8小时

战略意义：
  无攻城器械→攻城只能靠围困(耗时数月)
  有攻城器械→攻城缩短到数天→战略机动性大增
  攻城器械+数量优势=几乎必胜(除非守方有同等级城防)
```

## T6 精铁甲（争鸣时代）

```
研发需求：120天 / 10000金
前置：铁兵器
解锁：T3兵种 / 重甲装备

效果详解：
  ├── 士兵防御+25%（铁甲取代铜甲/皮甲）
  ├── 重甲单位移速惩罚-15%(精铁更轻更强)
  ├── 解锁T3兵种(秦铁鹰锐士/楚若敖军/魏武卒精锐等)
  ├── 重步兵对弓箭的防御额外+15%
  └── 解锁制甲坊(高级)

防御体系升级：
  T0布衣(5%) → T1皮甲(15%) → T2铜甲(30%) → T3铁甲(45%) → 精铁甲(60%)
  每升一级→战场上存活率大幅提升

兵种解锁：
  秦国：秦铁鹰锐士(T3重步，防御+15%，秦独有精锐)
  魏国：魏武卒精锐(T3重步，士气<20时攻击+25%)
  楚国：楚若敖军(T3重步，王室禁军)

战略意义：
  精铁甲让重步兵成为战场核心。
  高防御+高士气→正面几乎无敌。
  但克制方式：弩(穿甲)+骑兵冲锋(速度克制重甲慢速)+火攻
```

## T7 筑城术（帝国时代）

```
研发需求：150天 / 10000金
前置：攻城器械
解锁：高级城防 / 砖石城墙

效果详解：
  ├── 解锁砖石城墙(Lv3城防→Lv4)
  ├── 城防上限+50%
  ├── 守城伤亡-20%(城墙提供更好掩体)
  ├── 解锁瓮城设计(城门内第二道防线)
  ├── 解锁马面(凸出墙台，侧面射杀攀城者)
  └── 围城时间延长+50%(守军更耐久)

城防升级路径：
  木栅土墙(Lv1) → 夯土城墙(Lv2) → 砖石城墙(Lv3) → 
  重砖瓮城(Lv4) → 三重城墙(Lv5，需超级工程)

守城科技 vs 攻城科技：
  筑城术(T7) 对抗 攻城器械(T5)：守方优势
  但：如果攻方有火器(T8)+攻城器械→城墙的时代终结
  筑城术是守势国家的终极保障

战略意义：
  韩国和燕国最需要此科技(防守型国家)
  有了筑城术→即使兵力劣势3:1也能守住
  秦国也需要(守住已占领土不被反攻)
```

## T8 火器初探（帝国时代）

```
研发需求：300天 / 50000金
前置：连弩 + 攻城器械
解锁：火药武器(时代超前)

效果详解：
  ├── 解锁火药配方(硫磺+硝石+木炭)
  ├── 火攻伤害+100%（火药助燃）
  ├── 解锁"火箭"升级配件(弓箭+火药→射程+30%，命中后燃烧)
  ├── 解锁"火铳兵"(T5架空兵种，远程范围爆炸伤害)
  ├── 投石机可投掷火药罐(范围爆炸+燃烧)
  └── 对城墙伤害+50%(火药炸墙)

历史架空性：
  火药在战国尚未发明(唐末才出现)
  游戏允许玩家通过科技树"提前"发明火药→改变历史走向
  这是"如果战国时代发明了火药会怎样"的架空玩法

火器时代的战争变革：
  城墙→火药炸墙→城防优势大减→围城战缩短
  重甲→火药爆炸→重甲不再是绝对防御→军事革命
  密集阵型→火药范围伤害→阵型战术重新思考

伦理代价：
  研发和使用火器→仁义-20(火药杀人不仁)
  某些学派强烈反对(墨家/儒家)
  但军事效果无可置疑→玩家需要权衡

火器初探 + 终极工艺 = 可制造"铁火炮"(巨型火药武器)
火器初探 + 攻城器械 = 攻城效率+100%(碾压级)
火器初探 + 连弩 = 解锁"火箭连弩"(架空终极武器)
```

## 军事科技研发策略

```
速攻流（先军策略）：
  铁兵器(T1) → 骑兵革新(T3) → 精铁甲(T6) → 攻城器械(T5)
  优势：快速形成军事优势，适合早期扩张
  风险：科技树偏科，经济/文化落后

防御流（守势策略）：
  铁兵器(T1) → 弩机(T2) → 精铁甲(T6) → 筑城术(T7)
  优势：守城能力极强，以少胜多
  风险：缺乏进攻能力，被动挨打

技术流（架空路线）：
  铁兵器(T1) → 弩机(T2) → 连弩(T4) → 攻城器械(T5) → 火器(T8)
  优势：终极军事科技，碾压一切传统军队
  风险：研发时间长(总计930天)，期间兵力空虚

均衡流（推荐）：
  铁兵器(T1) + 骑兵革新(T3) 同时研发
  → 精铁甲(T6) → 攻城器械(T5)
  既能进攻也能防守，最稳妥
```
