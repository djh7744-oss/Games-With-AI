# 内政系统概述

## 设计目标

> 对标《三国志》系列的内政深度 + 《文明》系列的科技树与政策卡系统。内政不只是"点按钮涨数值"——每座城池有独立的人口、经济、民忠循环；每位官员都有自己的野心和忠诚；每项政策都有受益者和受害者。治国如烹小鲜，火候过了会糊，火候不够不熟。

## 系统架构

```
                        内政系统
                            │
    ┌───────┬───────┬───────┼───────┬───────┬───────┐
    ▼       ▼       ▼       ▼       ▼       ▼       ▼
  城池管理  人事与   律法与   科技树   诸子百家  变法与   国家
  与运营   官僚系统  政策系统  (48项)  (10学派)  改革    财政
```

## 核心循环

```
征收资源(税/粮/劳力) → 投入建设(农业/商业/军事/科技) → 人口增长/满意度变化 
    → 制定政策(调节平衡) → 应对外部挑战(战争/外交/灾荒) → 再次征收
```

## 设计支柱

| 支柱 | 含义 | 参考对标 |
|------|------|---------|
| **城池自主化** | 每城有独立的人口/经济/民忠循环，不是统一数值 | 三国志系列 |
| **人才是根本** | 官员质量决定治理效率，人才选拔与考核制度 | 三国志人事系统 |
| **政策有代价** | 每项政策有获益方和受损方，没有免费的改革 | 文明系列政策卡 |
| **科技驱动发展** | 48项科技分六大分支，国家科技水平决定国力上限 | 文明系列 |
| **学派影响国策** | 诸子百家不只是文化标签，加入学派影响可选政策 | 历史考据 |
| **变法改变规则** | 重大改革可重塑游戏规则，但伴随巨大风险 | 历史教训 |

## 内政各子系统

### 一、城池管理与运营 [→151]
- 每城七大数值（人口/农业/商业/技术/民忠/城防/治安）
- 内政指令（开发/征召/建设/安抚/搜索）
- 人口增长与迁移
- 城市专业化（农业城/商业城/军事城/科技城）
- 灾荒与瘟疫应对

### 二、人事与官僚系统 [→152]
- 官员任免与考核
- 官僚体制（中央官/地方官/军事官）
- 俸禄与贪污
- 举荐与科举萌芽
- 官员派系与内斗

### 三、律法与政策系统 [→153]
- 法律体系（轻刑/中刑/重刑/苛法）
- 政策卡系统（经济/军事/文化/社会 四大类）
- 政策冲突与兼容
- 执法力度调节

### 四、科技树系统 [→154]
- 六大分支：农业/军事/经济/政治/文化/工艺
- 48项科技，各8项
- 科技研发速度与条件
- 科技传播与窃取

### 五、诸子百家系统 [→155]
- 十家学派：儒/法/墨/道/兵/纵横/名/阴阳/农/杂
- 加入学派的条件与收益
- 学派代表人物与任务线
- 学派之争与融合

### 六、变法与改革系统 [→156]
- 商鞅变法/吴起变法/胡服骑射等历史改革
- 改革的支持者与反对者
- 改革实施的风险与回报
- 改革失败的处理

## 与国家策略各系统的交互

| 关联系统 | 交互方式 | 影响 |
|---------|---------|------|
| 经济系统 | 税收/封地/贸易受内政政策影响 | 内政决定经济效率 |
| 外交系统 | 国策影响外交选项和信誉 | 内政→外交姿态 |
| 战斗系统 | 军事政策/科技影响军队质量 | 内政→军事实力 |
| 角色系统 | 官员任免与角色管理 | 内政=角色在国家层面的体现 |
| 声望系统 | 治国效果影响仁义/霸业/信义 | 内政成果→声望变化 |
