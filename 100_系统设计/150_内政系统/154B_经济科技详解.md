# 经济科技详解

## 设计目标

> 经济科技决定国家富裕程度。军事科技让你能打赢战争，经济科技让你能支撑战争。齐国凭商业立国，不靠军事扩张也能成为战国最富庶的国家——这就是经济科技的力量。

---

## T1 市集（古典时代）

```
研发需求：30天 / 1000金（最便宜最快的科技）
前置：无
解锁：市集建筑 / 商业活动基础

效果详解：
  ├── 全国商业收入+10%
  ├── 解锁市集建筑(每城可建1个，商业+5/月)
  ├── 商路可见(地图上自动标注已知商路)
  ├── 物价信息可获取(看到邻城物价)
  └── 商铺投资解锁(Lv1小摊→Lv5垄断)

历史背景：
  战国时代商业迅速发展，临淄(齐都)有7万户，市集繁荣。
  "天下熙熙，皆为利来；天下攘攘，皆为利往。"

战略意义：
  市集是所有经济科技的起点。
  不研发市集→无法发展商业→经济永远停留在农业阶段。
  研发仅需30天→几乎总是第一或第二个研发的科技。
```

## T2 商路开拓（古典时代）

```
研发需求：60天 / 3000金
前置：市集
解锁：新商路 / 商队优化

效果详解：
  ├── 可发现3条隐藏商路(高利润路线)
  ├── 商队容量+30%(每队多运30%货物)
  ├── 商路地图显示危险区域(流寇/战区标注)
  ├── 解锁"派遣商队"指令自动化
  └── 商队护卫上限+50%(更多士兵保护)

隐藏商路：
  研发后自动发现3条未标注在普通地图上的商路
  隐藏商路通常利润更高但风险也更大
  例如：蜀道(巴蜀→关中)→运药材，利润200%但栈道危险
        楚越山路(楚→百越)→运珍珠/香料，利润250%但有瘴气

自动化商队：
  研发前：只能手动派遣商队(每次都要选择路线/货物/护卫)
  研发后：可设定"自动商队"(选择商品类型→商队自动找最佳路线)
  节省大量微操时间
```

## T3 盐铁专营（古典时代）

```
研发需求：90天 / 5000金
前置：商路开拓（需要水利或铁兵器作为隐性前置→确保有盐或铁）
解锁：盐铁专营政策

效果详解：
  ├── 解锁"盐铁专营"政策卡
  ├── 国家垄断盐铁收入：+2000~5000金/月(根据国家盐铁产量)
  ├── 盐铁价格由国家统一制定(不再随市场波动)
  ├── 私盐/私铁→违法→治安-5(但可派兵打击→治安恢复)
  └── 敌国无法从你这里购买盐铁(战略物资禁运)

政策效果分析：
  盐铁专营初期：国库暴增→可投入军事/建设
  盐铁专营中期：民间盐铁价格上涨→民忠-5(民众负担加重)
  盐铁专营后期：走私泛滥→需要投入资源打击走私

历史原型：
  齐国管仲首创"盐铁专营"→齐国由此富强。
  汉武帝时桑弘羊将其推向极致→"盐铁之利，二十倍于古"。
  现代经济学认为这是"国有企业"的雏形。

战略意义：
  盐铁专营是国家财政收入的最大单次提升。
  掌控盐铁资源→他国军事受限(没有铁造兵器)→战略优势。
  但过度依赖→经济结构单一→一旦失去盐铁产地→国库崩溃。
```

## T4 均输平准（争鸣时代）

```
研发需求：120天 / 8000金
前置：盐铁专营
解锁：物价调控能力

效果详解：
  ├── 物价波动幅度-50%(政府可干预市场)
  ├── 灾荒时粮价不暴涨(政府开仓平抑)
  ├── 丰收时粮价不暴跌(政府收购维持价格)
  ├── 解锁"常平仓"建筑(每城可建，粮食储备+500)
  └── 商人对你国家的好感+10(稳定的市场环境)

运作机制：
  粮价过低→损害农民→政府按最低价收购→存入常平仓
  粮价过高→损害市民→政府开仓按最高价卖出→平抑物价
  政府从中可赚取差价(低买高卖)→额外财政收入

常平仓：
  每城可建1座常平仓
  容量500石粮食
  在灾荒/战争时可动用储备
  相当于"国家粮食保险"

历史原型：
  战国李悝在魏国推行"平籴法"——最早的均输平准。
  汉代桑弘羊完善为"均输平准"制度。
  
战略意义：
  均输平准让国家经济从"靠天吃饭"变成"可控系统"。
  灾荒不再是毁灭性打击→赈灾有储备→民忠稳定。
  商人更愿来你国家做生意→商业持续增长。
```

## T5 货币统一（争鸣时代）

```
研发需求：180天 / 15000金
前置：均输平准
解锁：统一货币 / 交易效率提升

效果详解：
  ├── 废除国内多种货币→统一为一种(如秦半两)
  ├── 交易成本-20%(不需要频繁兑换)
  ├── 跨城贸易效率+30%(同一货币结算)
  ├── 税收效率+15%(统一货币便于统计)
  ├── 伪造货币难度+50%(标准化铸币工艺)
  └── 他国商人使用你的货币→你的货币影响力+

货币统一前后对比：
  统一前：秦国用半两/楚国用蚁鼻钱/齐国用刀币→跨城贸易需兑换
          兑换手续费3-8%→吃掉贸易利润
          不同货币含铜量不同→信任问题
  
  统一后：全国一种货币→交易直接结算
          货币=国家信用→你的货币越稳定→越多国家愿意使用
          铸币权在手→可调节货币供应量→控制通胀

历史原型：
  秦始皇统一六国后"车同轨、书同文、统一度量衡、统一货币"。
  半两钱成为全国标准→中国经济一体化迈出第一步。

战略意义：
  货币统一是帝国形成的标志之一。
  统一前你是"联盟"→统一后你是"国家"。
  他国商人使用你的货币→你的经济影响力→软实力。
```

## T6 钱庄体系（争鸣时代）

```
研发需求：150天 / 10000金
前置：货币统一
解锁：金融系统 / 资本运作

效果详解：
  ├── 解锁钱庄建筑(每城可建，提供借贷+存储)
  ├── 闲置资金>10000金→自动生息(月息0.3%)
  ├── 可向钱庄贷款(额度=资产×50%，年利率8-15%)
  ├── 解锁"汇票"系统(跨城转账不需要运现金)
  └── 商业投资回报率+20%(更多资本工具)

钱庄运作：
  存款：玩家将金存入钱庄→获得月息
        存款越多→利息收入越高→鼓励长期储蓄
  贷款：玩家可从钱庄借款→用于投资/战争
        还款压力→促使玩家更高效使用资金
  汇票：在A城存金→凭汇票在B城取金→安全跨城转移资金
        商队不需要携带大量现金→降低被劫损失

金融风险：
  钱庄挤兑：如果大量储户同时提款→钱庄可能破产
  贷款坏账：借款人无法偿还→钱庄损失→可能影响存款安全
  通货膨胀：政府滥发货币→货币贬值→存款购买力下降

历史背景：
  战国时代已有"子钱家"(放贷人)→齐国孟尝君曾在薛地放贷。
  但真正的钱庄体系是在唐代才形成(柜坊/飞钱)。
  游戏中作为"经济科技终极形态"的铺垫。
```

## T7 海贸（帝国时代）

```
研发需求：200天 / 20000金
前置：商路开拓 + 造船术(工艺T5)
解锁：海上贸易网络

效果详解：
  ├── 解锁3条海上商路(齐楚海路/渤海商路/吴越海路→南海商路)
  ├── 海上商路收入+50%(海运量大+成本低)
  ├── 新奢侈品流入：南海珍珠/香料/象牙/珊瑚
  ├── 沿海城池商业上限+20(海上贸易带来的繁荣)
  └── 可建造"港口"建筑(沿海城池专属)

海贸 vs 陆贸：
  海贸优势：运量大(一艘船=10辆车)/成本低/不受流寇骚扰
  海贸劣势：受海盗威胁/风暴风险/需要港口设施/造船成本高
  海贸适合：齐国/楚国/燕国(沿海国家)
  海贸不适合：韩国/魏国(内陆国家，只能通过河道间接参与)

海上商路详情：
  齐楚海路：临淄→会稽，运盐/丝绸南下，珍珠/漆器北上
  渤海商路：蓟城→临淄，皮毛/人参南下，盐北上
  南海商路：会稽→南海，漆器南下，珍珠/香料/象牙北上(最远+最危险)

历史原型：
  齐国和吴越地区在战国已有活跃的海上贸易。
  齐国海盐通过海运输送到楚国和燕国。
  "海上丝绸之路"的萌芽。
```

## T8 天下市集（帝国时代）

```
研发需求：300天 / 50000金（最贵科技之一）
前置：全部经济科技(8项)
解锁：终极经济体系

效果详解：
  ├── 所有商路收入+30%（叠加海贸=总共+80%）
  ├── 跨国贸易无额外关税(你的商队在他国享受"最惠国待遇")
  ├── 自动寻找最佳贸易路线(AI自动调整商队路线追求利润最大化)
  ├── 商路可视范围+50%(商队走过的路线自动探索周边)
  ├── 解锁"经济霸权"外交选项(用经济手段迫使小国屈服)
  └── 其他国家使用你的货币的概率+50%

经济霸权：
  如果你控制>50%的国际贸易→解锁"经济霸权"能力
  对其他国家的经济制裁：禁止你控制的所有商路→目标国经济崩溃
  商业吸引力：小国自动向你靠拢(经济依附→政治臣服)

天下市集的效果：
  你的都城成为"天下商业中心"
  所有商路自动汇聚到你的国家
  商人云集→商业税收暴增→月收入可能突破10万金
  其他国家在经济上对你依赖→军事上不敢轻易开战
  (开战=贸易中断=他们也损失巨大)

历史超越：
  这是"全球化"在战国时代的架空体现。
  如果战国有国家完成了"天下市集"→统一可能不需要战争
  经济统一→货币统一→文化统一→政治统一(和平路径)
```

## 经济科技研发策略

```
商人流（经济优先）：
  市集(T1) → 商路(T2) → 盐铁(T3) → 均输(T4) → 钱庄(T6)
  优势：经济起飞快，资金充裕
  适用：商人出身/齐国

帝国流（统一准备）：
  市集(T1) → 商路(T2) → 货币统一(T5) → 天下市集(T8)
  优势：统一后经济一体化快速
  适用：秦国统一后

海权流（沿海国家）：
  市集(T1) → 商路(T2) → 造船术(工艺T5) → 海贸(T7)
  优势：海上贸易垄断
  适用：齐国/楚国
```
