# 政治科技详解

## 设计目标

> 政治科技决定国家治理能力。优秀的制度让庸才也能有效运转国家，落后的制度让天才也寸步难行。从郡县制取代分封制，到察举制打破世袭垄断——政治科技每一步都在重塑权力的分配方式。

---

## T1 郡县制（古典时代）

```
研发需求：60天 / 3000金
前置：无
解锁：中央集权体制基础

效果详解：
  ├── 中央集权+，地方叛乱概率-20%
  ├── 可直接任命/罢免县令(不用经过贵族同意)
  ├── 地方税收更高效上缴中央(减少中间截留)
  ├── 每新增一县→自动纳入行政管理体系
  └── 解锁"县令"官职(地方官体系)

郡县 vs 分封：
  分封制：把土地封给功臣/宗室→他们有独立治权→长期尾大不掉
  郡县制：中央直接派人管理→地方官可随时调换→中央掌控力强
  过渡期：很多国家两种制度并存(秦国就是郡县与封地并行)

历史原型：
  商鞅变法在秦国推行郡县制→秦国集权程度远超六国。
  这也是秦国能"集中力量办大事"(修长城/修驰道/打统一战争)的制度基础。

战略意义：
  郡县制是国家从"部落联盟"进化到"中央集权国家"的关键科技。
  不研发郡县制→永远被贵族掣肘→无法有效动员全国资源。
  研发后→行政效率质变→但旧贵族强烈反对(需要应对)。
```

## T2 集权（古典时代）

```
研发需求：90天 / 5000金
前置：郡县制
解锁：政策槽位+1 / 行政效率

效果详解：
  ├── 政策卡槽位+1(从3→4)
  ├── 行政效率+15%(政令从中央到地方的时间缩短)
  ├── 政令传达延迟-50%(驿传系统改善)
  ├── 君主对地方的控制力+→地方官不敢擅权
  └── 解锁"丞相"可代理部分政务(减轻微操负担)

行政效率的含义：
  效率100%→政令在1天内传达到所有城池
  效率50%→政令在2-3天内到达→边远城池延迟更长
  集权→效率从50%→65%→政令快2天

驿传系统：
  研发后自动解锁"驿站"网络
  从首都到边境的驿道→每隔30里一个驿站
  换马不换人→政令快速传达
  秦国"驰道"的前身

战略意义：
  集权让你能"同时做更多事"(政策槽位+1)
  也让你的命令能更快被执行→在战争中尤其重要
  集权+郡县→中央对地方的控制达到巅峰→但也要小心"暴政"
```

## T3 察举制（古典时代）

```
研发需求：120天 / 6000金
前置：集权
解锁：人才选拔制度

效果详解：
  ├── 人才选拔效率+30%(候选人更多更快)
  ├── 寒门入仕渠道(不再只有贵族才能当官)
  ├── 人才池+40%(在野人才更愿意出仕)
  ├── 人才品质分布改善(更多中高品质人才出现)
  └── 解锁"举荐"指令(现任官员可举荐人才)

察举 vs 世袭：
  世袭：贵族(世卿世禄)→人才来源限贵族阶层→人才质量取决于运气
  察举：地方推荐有才德者→朝廷考核→择优录取→人才来源扩大
  但察举也可能演变为"推荐自己人"(任人唯亲)

历史原型：
  汉代察举制(举孝廉)→取代秦代的军功爵制(仅战时有效)
  察举制让平民中的优秀者有机会入仕→社会流动性增加

游戏中的察举：
  每年一次"察举"事件→各地推荐人才→玩家面试/考核
  选择：择优录取(可能漏掉大器晚成者)/全部录用(良莠不齐)
  察举质量=地方官魅力+智略→官员越好推荐的越好
```

## T4 俸禄制（争鸣时代）

```
研发需求：90天 / 4000金
前置：察举制
解锁：官员薪酬制度

效果详解：
  ├── 官员贪污概率-30%(有稳定收入→不需要冒险贪)
  ├── 俸禄支出透明化(清楚知道每个官员花了多少钱)
  ├── 官员满意度+10%(按时发薪→工作积极性提升)
  ├── 解锁"俸禄调节"功能(可按品级调整俸禄标准)
  └── 考核结合俸禄→优秀者加薪→不合格者降薪→自动激励

俸禄 vs 无俸禄：
  无俸禄：官员靠"灰色收入"(贪污/受贿/榨取民脂)→腐败温床
  有俸禄：官员有稳定合法收入→但俸禄太低→还是会贪
  关键在于：俸禄水平能否覆盖官员体面生活的开销

历史原型：
  商鞅变法后秦国推行"以粟出禄"→按粮食计算俸禄。
  之后演变为按品级固定俸禄→中国官僚俸禄制度的雏形。

游戏平衡：
  俸禄水平需要与其他支出平衡→太高→国库压力→太低→贪腐
  理想水平：中等品级官员的俸禄=其辖区内中等家庭的年收入的2-3倍
```

## T5 考课制（争鸣时代）

```
研发需求：150天 / 8000金
前置：俸禄制
解锁：官员绩效考核

效果详解：
  ├── 官员考核系统强化(从年度评估→季度评估)
  ├── 绩效差者自动触发"降职"建议(AI提醒)
  ├── 官僚效率+20%(人人知道被考核→更努力工作)
  ├── 考核数据可视化(每个官员的KPI雷达图)
  └── 解锁"功绩簿"(记录官员历年考核结果→辅助升迁决策)

KPI指标（地方官）：
  人口增长率 / 税收增长率 / 民忠变化 / 治安维持 / 灾害应对 / 建设完成度
  每个指标权重可自定义（如：战争时期治安权重增大）

KPI指标（中央官）：
  按职责范围（如治粟内史→国库增长/财政健康度）
  
考核结果应用：
  连续3次优秀→自动提示升迁(俸禄翻倍+满意度提升)
  连续2次不合格→自动提示降职(降薪+满意度下降)
  极差(连续3次不合格)→自动触发"免职"事件

历史原型：
  秦代已有"上计"制度(地方官每年向中央汇报政绩)
  考核是法家"术治"的核心→申不害变法的重要内容

战略意义：
  考课制让官僚体系从"靠关系"变为"靠绩效"。
  好人被提拔→坏人有压力→整体效率提升。
  但如果考核指标设计不当→可能助长"面子工程"。
```

## T6 密折制度（争鸣时代）

```
研发需求：180天 / 10000金
前置：考课制
解锁：秘密监察系统

效果详解：
  ├── 官员可匿名举报(鼓励揭发腐败/阴谋)
  ├── 反腐效率+50%(更多线索→更快发现)
  ├── 密折直达君主(不经中间官僚层级→无法截留)
  ├── 解锁"情报简报"(每月一次，汇总所有密折和反间谍情报)
  └── 副作用：猜忌氛围+5%(人人自危→官员忠诚-5)

正面效果：
  御史大夫+密折制度=几乎零容忍的反腐体系
  官员知道有人可能在暗中盯着→不敢肆意妄为
  历史上雍正皇帝通过密折制度维持了极高效率的统治

负面效果：
  人人自危→官员不敢做决策(怕被举报)
  可能被利用→诬告忠良→君主需要甄别
  猜忌氛围→创新和冒险精神下降

使用建议：
  密折制度+高薪俸禄=最佳组合(高薪养廉+密折监察)
  密折制度+低薪俸禄=恐怖统治(官员因穷而贪→被举报→被处决→循环)
  暴君模式：密折+重刑+苛法→史书上最黑暗的统治
```

## T7 文字统一（帝国时代）

```
研发需求：200天 / 15000金
前置：集权
解锁：全国标准化文字

效果详解：
  ├── 全国行政效率+20%（用同一文字书写政令）
  ├── 外交沟通改善(对方能看懂你发的外交文书)
  ├── 文书标准化(政令格式统一→减少误解)
  ├── 文化影响力+15%(你的文字成为区域标准)
  └── 解锁"文字推广"政策(加速其他国家的文字同化)

文字统一的意义：
  统一前：七国各有文字变体(大家都从甲骨文演化但分支不同)
          秦国文书→楚国官员看不懂→需要翻译→效率低
  统一后：一种标准文字(小篆/秦篆)→所有人都能看懂
          文化交流加速→国家认同感提升→"我们是一国人"

历史原型：
  秦始皇"书同文"→李斯创小篆→程邈创隶书→统一文字标准。
  这是"中国"这个概念从地理名词变成文化共同体的关键一步。

战略意义：
  文字统一是帝国长期稳定的基础。
  有了统一文字→政令无阻→文化传播→身份认同→反抗意识减弱。
  没有文字统一→统一也只是军事征服→迟早分裂。
```

## T8 车同轨书同文（帝国时代）

```
研发需求：360天 / 50000金（最昂贵科技）
前置：文字统一 + 货币统一(经济T5)
解锁：帝国终极一体化

效果详解：
  ├── 全国所有行政效率+50%(帝国运转如精密机器)
  ├── 驰道网络解锁(全国主干道标准化→移速+30%)
  ├── 统一度量衡(交易再无欺诈→国民信任+)
  ├── 解锁"帝国政令"(超级政策卡，每项效果+50%但切换冷却1年)
  ├── 解锁"巡狩"功能(君主巡视全国→每停留一城→该城BUFF持续1年)
  └── 文化影响力+50%，国家认同度+80%

车同轨：
  所有车辆的轮距标准化(6尺=约1.38米)
  驰道宽度标准化(可容两车并行)
  运输效率质的飞跃→物流成本-40%

书同文：
  在全国范围内推广标准文字
  所有官方文书必须使用标准文字
  设立"学室"→培训公文写作→培养标准化官僚

统一度量衡：
  长度：寸/尺/丈 → 全国统一
  容量：升/斗/石 → 全国统一
  重量：铢/两/斤/钧/石 → 全国统一
  市场交易从此有一个标准→商业信任度飙升

帝国政令：
  车同轨书同文完成后解锁的终极政策卡组：
  ├── 驰道令：全国行军速度+30%(军事)
  ├── 书同文令：科技传播速度+100%(文化)
  ├── 度量衡令：全国商业收入+30%(经济)
  └── 郡县直辖令：中央可直接管理任意城池(政治)

历史意义：
  秦始皇的这一系列统一措施→奠定了此后2000年
  中国"大一统"的制度基础。这是人类历史上最大规模
  的标准化运动之一。
  
  在游戏中→这是通往"完美胜利"的最后一块拼图。
```

## 政治科技研发策略

```
集权流（中央控制优先）：
  郡县制(T1) → 集权(T2) → 文字统一(T7) → 车同轨书同文(T8)
  优势：中央控制力极强→帝国难以分裂
  适用：统一后的帝国维持

吏治流（官僚效率优先）：
  郡县制(T1) → 集权(T2) → 察举(T3) → 俸禄(T4) → 考课(T5)
  优势：官员体系高效→国家自动运转良好
  适用：领土较大需要完善官僚系统时

法家流（严刑峻法）：
  郡县(T1) → 集权(T2) → 考课(T5) → 密折(T6)
  配合：重刑法律+法家学派→恐怖但高效的统治
  适用：法家路线/秦国

全面流（统一必备）：
  全部8项科技都要完成→这是统一后维持帝国的必要条件
  顺序：郡县→集权→察举→俸禄→考课→密折→文字统一→车同轨
  总计研发时间：1220天(约3.3年)→实际需要5-8年(考虑资源分配)
```
