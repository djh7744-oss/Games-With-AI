# 朝贡与臣服系统

## 设计目标

> 战国时代的"国际秩序"核心是朝贡与臣服——小国向大国称臣纳贡以换取保护，大国接受朝贡以确立霸主地位。这不是简单的"吞并"，而是一种等级化的依附关系。对玩家而言，臣服可以是权宜之计（卧薪尝胆），也可以是战略选择（大树底下好乘凉）。

## 系统概述

朝贡与臣服体系定义了强国与弱国之间的附庸关系。附庸国向宗主国定期朝贡，宗主国提供军事保护和不吞并的承诺（当然，可以不遵守）。附庸关系有不同等级，从名义臣服到完全附庸。附庸国可在条件成熟时寻求独立，宗主国也可逐步吞并附庸国。

## 核心机制

### 3.1 臣服等级

| 等级 | 名称 | 朝贡频率 | 朝贡内容 | 宗主义务 | 附庸自主权 | 外交独立 | 军事自主 |
|------|------|---------|---------|---------|-----------|---------|---------|
| Lv1 | **名义臣服** | 每年一次 | 象征性贡品(500-2000金) | 不攻击即可 | 高 | 有(有限制) | 有 |
| Lv2 | **朝贡国** | 每半年 | 金+特产+可能质子 | 军事保护(对外) | 中 | 需宗主批准 | 有(防御) |
| Lv3 | **附庸国** | 每季 | 年收入15%+军事支援 | 全面军事保护 | 低 | 无(宗主代理) | 需宗主批准 |
| Lv4 | **傀儡国** | 每月 | 年收入30%+军事指挥权 | — | 极低 | 无 | 无(宗主控制) |
| Lv5 | **行省/郡县** | 无(直接管辖) | 全部税收归宗主 | — | 无(已被吞并) | 无 | 无 |

### 3.2 臣服建立方式

```
方式一：武力胁迫
  1. 军事击败对方（占领首都或消灭主力）
  2. 提出臣服条件
  3. 对方判定：
     剩余军力<10%→几乎必接受
     剩余军力10-30%→概率50-80%
     剩余军力>30%→可能拒绝（再打一仗）
  4. 对方接受→臣服关系建立
  5. 对方拒绝→继续战争或灭亡对方

方式二：外交劝服
  1. 派遣使者劝服小国臣服
  2. 成功率 = 霸业声望/3 + 魅力/4 + 两国国力差距
  3. 劝服优势：
     不战而屈人之兵（仁义声望+）
     保留对方国家架构（后续管理更方便）
  4. 劝服劣势：
     对方可能只是名义臣服(Lv1)
     需要长期维护关系

方式三：危机依附
  1. 小国面临灭国危机（被他国攻击）
  2. 向你求援→你以"臣服"为条件
  3. 小国在"被灭"与"臣服于你"之间选择
  4. 通常选择臣服（好过被灭）

方式四：继承统一
  1. 通过联姻获得继承权
  2. 对方君主死后无嗣→你的配偶/子女继承
  3. 两国和平合并（最优方式，但极罕见）

方式五：长期渗透
  1. 对目标国长期施加影响
  2. 策反其大臣→扶持亲己方派系
  3. 逐步从Lv1名义臣服→Lv2→Lv3→...→最终吞并
```

### 3.3 宗主国与附庸国的权利与义务

#### 宗主国

```
宗主国权利：
  ├── 收取朝贡（金/物资/特产）
  ├── 附庸国军事通行权（自动）
  ├── 附庸国战争时→宗主可决定是否参战
  ├── 附庸国外交限制（Lv2+需宗主批准）
  ├── 附庸国君主继位需宗主认可（Lv3+）
  ├── 可征用附庸国军队（Lv3+）
  ├── 可在附庸国境内驻军（Lv3+）
  └── 逐步吞并附庸国（Lv3→Lv4→Lv5）

宗主国义务：
  ├── 附庸国被攻击→宗主有义务保护
  │   （不履行→信义-20，附庸可能独立/投靠其他强国）
  ├── 附庸国灾荒→宗主应援助（不援助→仁义-10）
  ├── 附庸国内乱→宗主可派兵平叛（维持附庸稳定）
  └── 不主动攻击附庸国（自毁体系）
```

#### 附庸国

```
附庸国权利：
  ├── 获得宗主军事保护
  ├── 其他国家不敢轻易攻击（忌惮宗主）
  ├── 内部自治（Lv1-2有较大自主权）
  ├── 可向宗主请求援助（金/粮/军事）
  └── 可在条件成熟时寻求独立

附庸国义务：
  ├── 按时朝贡（拖欠→关系恶化→宗主可能惩罚）
  ├── 外交需宗主批准（Lv2+）
  ├── 宗主战争时→可能需要出兵支援
  ├── 不得与宗主的敌国交往
  └── 接受宗主驻军（Lv3+）

附庸国不满积累：
  宗主征用过多→不满+
  宗主不履行保护义务→不满++
  宗主干涉内政→不满+
  宗主频繁要求额外贡品→不满+
  宗主与其他国家联姻(感觉被冷落)→不满+（小国心态）

  不满达到阈值→附庸寻求独立
```

### 3.4 朝贡系统

#### 朝贡内容

```
基础贡品（每年/半年/季度/月，按臣服等级）：
  ├── 金：按年收入百分比
  ├── 粮食：500-5000石
  ├── 特产：该国特产（如韩国铁器/楚国漆器）
  ├── 美女/宫女：1-5名（可选）
  └── 质子：送王子为质（Lv2+）

特殊贡品（重大事件/宗主庆典时额外朝贡）：
  ├── 名马/名剑/珍宝
  ├── 对方急需的战略物资（战马/铁/盐）
  ├── 技术/图纸（罕见）
  └── 割让部分领土（实质性的臣服升级）

贡品价值计算：
  贡品总值 = 金 + 物资估值 + 特产估值 + 美女/质子估值
  贡品总值不足→宗主不满→关系下降
  贡品总值超额→宗主满意→可能给予赏赐/减贡
```

#### 朝贡流程

```
朝贡日流程：

1. 朝贡日期到达（自动提醒）
2. 附庸国准备贡品（系统自动计算）
3. 朝贡使团出发→抵达宗主国都城
4. 朝贡仪式（可选参加/跳过）：
   ├── 附庸使者跪拜→献上贡品清单
   ├── 宗主验收→表达满意/不满
   ├── 宗主赏赐回礼（可选，仁义+）
   └── 宗主提出新要求（可选）
5. 朝贡完成→更新关系

朝贡仪式（玩家作为宗主时可参与）：
  接受朝贡→霸业声望+1-5（根据附庸数量）
  拒绝朝贡→附庸国恐慌（以为你要灭他）
  赏赐回礼→仁义+2-5，附庸满意度+
  提出额外要求→附庸不满+，但短期利益+
```

### 3.5 附庸国独立

```
附庸寻求独立的触发条件：
  ├── 不满积累到阈值（>70/100）
  ├── 宗主国陷入大战（无力镇压）
  ├── 宗主国连续战败（威慑力下降）
  ├── 有其他强国支持（提供独立保证）
  └── 宗主国长期不履行保护义务

独立方式：
  1. 和平独立（请求宗主批准）
     条件：关系尚可+宗主正忙于其他事务
     宗主同意→和平分手，关系回到中立
     宗主拒绝→可能引发其他附庸的不满

  2. 武力独立（宣布独立+武力对抗宗主）
     宗主派兵镇压→独立战争
     附庸胜利→完全独立+获得国际认可
     附庸失败→臣服等级+1级（惩罚性升级）

  3. 转投他国（换个宗主）
     新宗主提供保护承诺
     旧宗主震怒→可能与新宗主开战
     "二虎竞食"策略的典型应用

附庸独立后：
  前宗主获得对该国的"再征服"战争借口（12个月内有效）
  其他附庸国看在眼里→可能效仿（连锁独立）
  该国对其他国家的臣服邀请更敏感（刚获得自由）
```

### 3.6 吞并附庸

```
宗主逐步吞并附庸的路径：

Lv1→Lv2(朝贡国)：要求增加朝贡频率
  条件：附庸满意/恐惧
  对方拒绝→关系-10，不满+15

Lv2→Lv3(附庸国)：要求驻军+外交控制权
  条件：附庸恐惧/依赖
  对方拒绝→可能引发军事镇压

Lv3→Lv4(傀儡国)：要求军事指挥权+更高朝贡
  条件：附庸极度恐惧/已被完全控制
  对方拒绝→大概率武力镇压→灭亡

Lv4→Lv5(吞并)：废除附庸君主→设为郡县
  条件：霸业≥70/附庸已被傀儡化>2年
  后果：
    直接吞并→其他附庸国恐慌→可能集体寻求独立
    仁义-20~40（灭人宗庙，不仁不义）
    但领土+税收永久归于你

和平吞并策略（减少负面效果）：
  联姻+继承（最佳）
  附庸主动请求并入（极罕见，需宗主极其仁义+附庸面临巨大威胁）
  分阶段吞并（每次吞一小块，减少震动）
```

## 朝贡与臣服网络UI

```
臣服体系总览（文明式）：
  ├── 势力层级图：

  ├── 臣服等级+持续时间

  ├── 当前满意度/不满来源
  ├── 独立倾向（低/中/高/极高）
  └── 操作按钮：要求额外贡品/提升臣服等级/赏赐/威胁/释放独立
```

## 与其他系统的交互

| 关联系统 | 交互方式 | 影响 |
|---------|---------|------|
| 外交系统 | 附庸国外交由宗主代理 | 臣服=外交绑定 |
| 战斗系统 | 宗主有义务保护附庸 | 附庸被攻→宗主参战 |
| 经济系统 | 朝贡=稳定收入来源 | 附庸越多收入越多 |
| 声望系统 | 吞并/善待影响仁义/霸业 | 方式决定声望变化 |
| 会盟系统 | 附庸国可参与会盟 | 宗主带附庸一起 |
