# 间谍与情报系统

## 设计目标

> 对标《孙子兵法》的"五间俱起"理论。《孙子兵法·用间篇》将间谍分为五类：因间、内间、反间、死间、生间。每种间谍有不同的用途、风险和回报。情报是战争的第一要素——知己知彼，百战不殆。

## 系统概述

玩家可派遣间谍潜入敌国执行情报收集和破坏任务。间谍需要时间建立潜伏网络，潜伏越久能获取的情报越机密。敌方也有反间谍系统，可发现并清除己方间谍。信息不对称是核心设计原则——你永远不知道对方知道多少。

## 核心机制

### 3.1 五间体系

#### 因间（利用当地人）

```
定义：利用敌国普通民众/商人/旅人作为情报来源

建立方式：
  收买当地商人：500-2000金/人
  在酒馆招募线人：100-500金/次情报
  利用已有的商业网络（商人出身优势）

情报质量：★★☆☆☆（低精度，可能有误报）
情报类型：市场物价/民忠状况/驻军大致数量
被反间谍发现概率：5%（不好排查平民）

优势：成本低、铺开快、被发现概率低
劣势：情报精度差、无法获取军事机密
```

#### 内间（收买敌方官员）

```
定义：收买/策反敌方官员提供情报

建立方式：
  策反敌国官员（低忠诚/功利型性格优先）
  步骤：
    1. 间谍接触目标官员（需要先有基础间谍网络）
    2. 评估：忠诚度<50 / 有不满(长期不升迁/不赏赐) / 野心型性格
    3. 提出条件：金/官职/封地/安全保证
    4. 对方判定：基于忠诚/贪欲/风险评估
    5. 成功→该官员成为内间

维护：
  每月支付"信息费"：100-2000金
  内间被发现→被处决→线索可能追溯到玩家
  官员升迁→能获取更高级情报
  
情报质量：★★★★☆（高精度，直达核心）
情报类型：军事部署/科技进度/外交秘密/君主决策
被反间谍发现概率：15-25%（官员行动受限）
```

#### 反间（双重间谍）

```
定义：利用敌方的间谍为己方服务

方式一：捕获后策反
  1. 反间谍系统捕获敌方间谍
  2. 选择：处决/关押/策反
  3. 策反成功→敌方间谍成为双面间谍
  4. 他继续向原主人发送假情报（你控制情报内容）
  5. 同时向你提供真情报

方式二：主动渗透
  派遣己方间谍假装被对方策反→打入敌对方

反间的价值：
  可向敌方输送假情报（如：虚假的兵力部署/虚假的进攻方向）
  假情报被采信→敌方做出错误决策→战略优势

风险：
  反间被发现（敌方也在反间谍）→情报战升级
  双面间谍可能再次反转→三重间谍
```

#### 死间（牺牲型间谍）

```
定义：执行必定暴露的任务，以牺牲为代价达成目标

死间任务类型：
  ├── 刺杀：刺杀敌国君主/名将
  │   成功率：10-30%（极高风险）
  │   后果：成功→敌国陷入继承危机；失败→间谍必死+两国关系-80
  │
  ├── 重大破坏：炸毁敌国核心武库/粮仓/决堤
  │   成功率：20-40%
  │   后果：成功→敌军战斗力大降；失败→间谍死亡
  │
  ├── 传播假情报至死：故意被俘→携带假情报→敌方信以为真
  │   成功率：情报被采信30-50%
  │   后果：间谍必死（敌方发现被骗后处决）
  │
  └── 离间到死：伪造"谋反证据"→让敌国君主处决自己的名将
      成功率：15-25%
      后果：成功→敌方自毁长城；失败→间谍被处决

死间要求：
  忠诚度必须≥90（百分百忠诚才愿意赴死）
  死间执行后→仁义声望-5~15（使用死间手段不光彩）
```

#### 生间（活着回来的间谍）

```
定义：执行任务后能够安全返回的间谍

生间任务类型：
  ├── 常规情报收集（军事/经济/外交）
  ├── 建立间谍网络（发展下线）
  ├── 策反敌方官员
  ├── 侦查地形/城防
  └── 接触特定NPC传递信息

生间特点：
  可重复使用（经验积累→每次任务成功率提升）
  长期潜伏→获取情报质量和数量随潜伏时间增长
  需要伪装身份（商人/学者/艺人/难民）

生间风险：
  潜伏时间越长→被发现概率累积
  敌方反间谍越强→越难潜伏
  每次情报传回→可能被截获→追踪到间谍
```

### 3.2 间谍派遣与管理

#### 派遣流程

```
1. 招募/训练间谍
   ├── 在酒馆招募（随机品质，花费低）
   ├── 从军中选拔（智略高的士兵→训练成间谍）
   ├── 收买已有间谍网络的"中间人"
   └── 纵横家/门客兼职间谍（张仪曾为秦间谍）

2. 选择目标城池/势力
   ├── 军事目标：军营/武库/边境关隘
   ├── 经济目标：都城市场/钱庄/仓库
   ├── 政治目标：王宫/官衙/贵族府邸
   └── 外交目标：对方外交使团

3. 选择掩护身份
   ├── 商人：适合经济情报（商人出身有加成）
   ├── 学者/士人：适合政治情报
   ├── 艺人/游侠：适合流动侦查
   ├── 流民/难民：适合潜入城中
   └── 敌方士兵(假冒)：适合军事情报（高难度）

4. 派遣+潜伏期
   潜伏期：15-60天（建立情报网络）
   潜伏成功后→开始产出情报

5. 情报传回
   ├── 定期传回（如：每10天一次情报）
   ├── 紧急传回（重大发现立即回报，但暴露风险+）
   └── 亲自回报（间谍返回→当面汇报最安全）
```

#### 间谍属性

```
间谍关键属性：
  智略：影响任务成功率、情报精度、被发现的抵抗能力
  魅力：影响策反/发展下线的能力
  忠诚：决定是否会被策反/叛变
  经验：影响所有行动成功率（每完成一次任务+经验）

间谍品质：
  新手：智略<50 → 成功率低，容易被发现
  合格：智略50-70 → 可执行常规任务
  精锐：智略70-85 → 可执行高风险任务
  大师：智略85-95 → 可执行刺杀/策反
  谍王：智略95+ → 几乎不可能被发现

间谍暴露概率 = 基础暴露率 × (100 - 间谍智略)/100 
               × 敌方反间谍能力 × 行动风险系数
```

### 3.3 情报类型与精度

#### 情报类型

| 情报类型 | 获取难度 | 潜伏时间要求 | 信息精度范围 | 时效性 | 示例 |
|---------|---------|------------|-------------|--------|------|
| 市场物价 | 极低 | 即时 | 95-100% | 7天 | "邯郸粮价暴涨50%" |
| 民忠/民心 | 低 | 5天 | 85-95% | 15天 | "大梁民忠约65，有不满" |
| 驻军大致数量 | 低 | 10天 | 70-90% | 10天 | "咸阳约驻军5000-8000" |
| 科技进度 | 中 | 20天 | 60-85% | 30天 | "楚国正在研发炼钢术" |
| 军事调动 | 中 | 15天 | 60-80% | 5天(紧急) | "秦军主力向函谷关集结" |
| 外交秘密条约 | 高 | 30天 | 50-75% | 20天 | "齐楚可能已签署秘密同盟" |
| 君主决策 | 极高 | 60天 | 40-70% | 10天 | "秦王计划明年春攻赵" |
| 敌军完整部署 | 极高 | 45天 | 55-80% | 7天 | "长平秦军部署图" |

#### 情报精度

```
情报精度说明：
  90-100%：确认情报（可据此做出决策）
  70-89%：高可信度（基本可靠，但可能有偏差）
  50-69%：中度可信（需要交叉验证）
  30-49%：低可信度（仅作参考）
  <30%：不可靠（可能是假情报/误报）

情报交叉验证：
  两个独立来源同时报告→精度+15%
  三个独立来源+→精度+25%
  间谍智略越高→初始精度越高
```

### 3.4 破坏行动

```
间谍可执行的破坏行动（除情报收集外）：

军事破坏：
  ├── 焚烧粮仓 → 敌军粮草减少30-60%
  ├── 破坏武库 → 敌军武器/装备损失20-40%
  ├── 毒杀战马 → 敌军骑兵战斗力-30%
  ├── 破坏攻城器械 → 延缓敌方攻城7-14天
  └── 在敌军水源投毒 → 敌军生命持续-5%/天

城防破坏：
  ├── 破坏城门 → 攻城时城门耐久-30%
  ├── 收买守城将领 → 战时打开城门
  ├── 破坏城墙(凿洞) → 城墙耐久-15%
  └── 散布恐慌 → 守军士气-20

经济破坏：
  ├── 囤积居奇 → 目标城物价异常波动
  ├── 散布谣言 → 引发挤兑/抢购
  └── 伪造货币 → 扰乱当地经济

政治破坏：
  ├── 离间君臣 → 降低将领忠诚
  ├── 伪造谋反证据 → 让君主处决自己的将领
  └── 散布对君主不利的谣言 → 民忠-10~30

破坏行动风险：
  每次破坏行动→暴露概率+15-40%
  破坏失败→间谍可能被当场抓获
  大规模破坏→敌方反间谍力度加倍
```

### 3.5 反间谍系统

```
防御性反间谍：

反间谍强度 = 君主智略/3 + 间谍总管智略/2 + 反间谍投入金/1000

反间谍措施：
  ├── 任命间谍总管（智略高的部下）→ 全局反间谍+20-50%
  ├── 边境盘查 → 敌方间谍潜入概率-30%
  ├── 定期排查 → 每30天自动扫描一次境内间谍
  │   发现概率 = 反间谍强度 - 敌方间谍智略
  ├── 悬赏举报 → 民众举报间谍概率+20%
  ├── 加密通信 → 己方情报被截获概率-40%
  └── 陷阱情报 → 故意泄露假情报→追踪泄密源→揪出内鬼

捕获间谍后的处理：
  ├── 审问(智略判定) → 可能供出同伙/上线
  ├── 处决 → 威慑其他间谍(敌方间谍潜伏成功率-10%持续3个月)
  ├── 关押 → 可用来交换己方被俘间谍
  ├── 策反(变成双面间谍) → 反间作战
  └── 释放(故意放走→跟踪→发现整个间谍网)

己方间谍保护：
  定期更换暗号 → 降低被破译概率
  单线联系 → 一人被捕不会牵连整个网络
  紧急撤离方案 → 暴露后可安全撤回
```

## 情报UI

```
间谍网络总览：
  ├── 已派遣间谍列表（位置/状态/潜伏时间/最近情报）
  ├── 地图标注间谍位置（不同颜色表示不同类型）
  ├── 情报汇总面板（按国家/类型分类）
  ├── 敌情预警（紧急情报高亮闪烁）
  └── 风险评估（每个间谍的暴露概率）

情报详情面板：
  情报标题+来源(哪个间谍)+精度+时效性+交叉验证状态
  "根据3个独立来源确认：秦军主力约5万人正向长平集结"
```

## 与其他系统的交互

| 关联系统 | 交互方式 | 影响 |
|---------|---------|------|
| 战斗系统 | 情报影响战前部署 | 知己知彼→战术优势 |
| 外交系统 | 间谍可破坏他国同盟/获取秘密条约 | 情报→外交决策 |
| 经济系统 | 间谍可获取物价情报/破坏经济 | 情报→贸易优势 |
| NPC系统 | 策反/离间影响NPC忠诚和行为 | 间谍→人事影响 |
| 科技系统 | 间谍可窃取科技进度 | 科技情报→追赶策略 |
