# 纵横家与游说系统

## 设计目标

> 纵横家是战国时代最特殊的群体——凭三寸不烂之舌改变天下格局。苏秦佩六国相印主导合纵，张仪为秦连横瓦解联盟。本系统让玩家能招募、培养、派遣纵横家，体验凭外交手腕改变历史的快感。

## 系统概述

纵横家（或称说客/辩士）是执行外交任务的专业人才。玩家可招募纵横家、派遣他们出使他国进行游说。游说结果取决于纵横家的个人能力（魅力/智略）、玩家声望、两国关系、提出的条件等因素。历史上著名的纵横家NPC（苏秦/张仪/公孙衍/苏代等）有专属技能和任务链。

## 核心机制

### 3.1 纵横家角色

#### 纵横家属性要求

```
纵横家的核心属性：
  魅力（权重60%）：说服力、建立信任
  智略（权重40%）：分析利害、设计说辞

优秀纵横家的标准：
  魅力≥70 + 智略≥60 → 合格使者
  魅力≥85 + 智略≥75 → 一流纵横家
  魅力≥95 + 智略≥90 → 国之重器（苏秦/张仪级别）
```

### 3.2 游说任务类型

| 游说任务 | 目标 | 难度 | 耗时 | 风险 | 成功奖励 |
|---------|------|------|------|------|---------|
| 结盟说 | 说服他国同意结盟 | 中-极高 | 10-30天 | 被扣留 | 达成同盟 |
| 解盟说 | 说服他国与第三方断交 | 高 | 15-25天 | 激怒对方 | 瓦解敌方同盟 |
| 中立说 | 说服他国在战争中保持中立 | 中 | 10-20天 | — | 减少敌方潜在盟友 |
| 参战说 | 说服他国加入己方战争 | 高-极高 | 15-30天 | 被扣留 | 获得战争盟友 |
| 割地说 | 说服他国割让领土 | 极高 | 20-40天 | 被驱逐 | 不战而获领土 |
| 臣服说 | 说服小国主动臣服 | 高 | 15-30天 | — | 获得附庸国 |
| 纳贡说 | 说服他国同意进贡 | 中 | 10-20天 | — | 获得持续收入 |
| 质子说 | 说服他国送质子 | 中-高 | 15-25天 | — | 获得人质保障 |
| 破坏说 | 破坏他国之间的同盟 | 高-极高 | 15-30天 | 被驱逐/扣留 | 瓦解敌方联盟 |
| 劝降说 | 说服敌国君主投降 | 极高 | 20-60天 | 被杀 | 和平吞并敌国 |

### 3.3 游说成功模型

```
游说成功率 = (使者魅力 × 0.6 + 使者智略 × 0.4) / 2 
           × 任务基础难度修正 
           × 双方关系修正
           × 利益一致性修正
           × 条件吸引力修正
           × 技能修正
           × 随机波动(±15%)

详细分解：

1. 任务基础难度修正：
   结盟说：×1.2 / 解盟说：×0.8 / 中立说：×1.0
   参战说：×0.7 / 割地说：×0.4 / 臣服说：×0.6
   纳贡说：×1.0 / 质子说：×0.9 / 破坏说：×0.6
   劝降说：×0.2（几乎不可能，除非对方已穷途末路）

2. 双方关系修正：
   血盟友好×1.5 / 友好×1.2 / 中立×1.0 / 紧张×0.5 / 敌对×0.2

3. 利益一致性修正：
   提议对对方非常有利×1.8 / 有利×1.3 / 中立×1.0
   不利×0.5 / 严重不利×0.2

4. 条件吸引力修正：
   附带金/地/联姻/质子等→每项+0.1~0.25

5. 技能修正：
   纵横家专属技能：+0.15~0.40
   玩家魅力系技能节点(纵横捭阖)：+0.15
   使者与对方君主有私人好感：+好感/500

6. 最终成功率上限：95%，下限：1%
```

### 3.4 游说流程与风险

```
游说流程：

Step 1：选择使者
  ├── 必须是有纵横能力的角色（魅力+智略综合≥100）
  ├── 可选玩家亲自出使（消耗时间+有被扣留风险）
  └── 可选纵横家NPC（苏秦/张仪等）

Step 2：准备说辞
  ├── 选择游说策略：
  │   ├── 晓之以利（强调利益）：成功率+10%，对方要求更多
  │   ├── 动之以情（强调关系）：成功率+5%，长期关系改善
  │   ├── 胁之以威（强调武力）：成功率+15%，但关系-20（恐惧≠友谊）
  │   ├── 晓之以理（强调道理）：成功率+0%，但仁义声望+5
  │   └── 诱之以利（行贿对方官员）：成功率+20%，花费金，可能暴露
  ├── 准备礼物（金/宝物/美女）→成功率+
  └── 准备条件（割地/联姻/质子）→成功率+

Step 3：派遣出使
  ├── 使者出发→路上时间（距离/200天）
  ├── 到达目标国都城→开始游说
  ├── 游说时间：基础10-40天（取决于任务类型）
  └── 期间玩家可收到使者回报（中途调整条件）

Step 4：游说结果
  ├── 大成功(>90%)：对方不仅同意，还额外让步
  ├── 成功：对方同意提案
  ├── 部分成功：对方同意但有额外要求（讨价还价）
  ├── 失败：对方拒绝
  └── 大失败(<5%)：对方震怒→使者被扣留/驱逐

使者风险：
  扣留：使者被扣押（需要赎金赎回/外交施压/武力营救）
  驱逐：使者被驱逐出境（1年内不得再出使该国）
  被杀：极低概率（死间/对世仇国家出使时），导致两国关系-60
  被收买：使者被对方策反（极低概率，取决于使者忠诚）
```

### 3.5 合纵连横模拟

#### 合纵机制

```
合纵定义：
  3个以上国家联合起来遏制最强国（通常针对秦国）

合纵触发条件：
  ├── 某国国力超过其他任两国之和
  ├── 某国连续吞并>2国
  ├── 某国霸业声望>70
  └── 纵横家主动发起（苏秦专属能力）

合纵发起流程：
  1. 发起国（或纵横家）评估"最大威胁"
  2. 向可能的参与国发出合纵邀请
  3. 各国分别判定是否加入
     ├── 与威胁国接壤→加入概率+30%
     ├── 与威胁国有仇→加入概率+25%
     ├── 与威胁国有同盟→加入概率-60%
     └── 国力太小→加入概率+20%（依附集体安全）
  4. 参与国>=3→合纵成立
  5. 合纵同盟生效→对威胁国的统一阵线

合纵效果：
  成员间互不侵犯+贸易优惠
  对威胁国：外交封锁+共同防御条约
  威胁国对合纵任一成员开战→所有成员自动参战

合纵瓦解条件：
  ├── 威胁国被削弱（国力降到第一的80%以下）
  ├── 纵横家张仪使用连横策略分化
  ├── 成员间发生战争
  ├── 威胁国拉拢某个成员（贿赂/割地/联姻）
  └── 自带领袖苏秦死亡/被策反
```

#### 连横机制

```
连横定义：
  强国（秦）通过分化拉拢，瓦解合纵联盟

连横策略（张仪专属）：
  策略一：远交近攻
    与远处国家结盟→攻击近处国家→逐个击破
  
  策略二：以利诱之
    向合纵中的某国提出极优厚条件→该国退出合纵
    例：秦承诺归还某地→某国退出→合纵瓦解→秦食言（张仪欺楚）
  
  策略三：以威逼之
    向合纵中的弱国施压→不退出就灭你
    成功率基于：己方军力/对方军力比

  策略四：制造矛盾
    在合纵成员间挑拨→让他们互相猜忌→合纵自然瓦解

连横成功率：
  目标与威胁国有仇恨：+30%
  目标国力弱小：+25%
  提供的条件很优厚：+20~50%
  张仪亲自出马：+40%
  合纵刚成立(<6个月)：-25%（热情未退）
  合纵已经很久(>24个月)：+25%（合纵疲劳）
```

### 3.6 纵横学派

```
纵横学派是诸子百家之一：

加入条件：
  ├── 魅力≥60 + 智略≥50
  ├── 找到纵横家师父（鬼谷子/苏秦/张仪等）
  └── 完成入门任务（成功游说一次）

学派等级：
  Lv1 弟子：游说成功率+5%
  Lv2 门客：可同时派遣2名使者
  Lv3 辩士：解锁"威胁"策略（武力威胁成功率+20%）
  Lv4 策士：解锁"连横/合纵"主导能力
  Lv5 纵横家：游说冷却时间-50%，可远程游说（不亲自到场）

鬼谷子传说：
  鬼谷子是纵横家的祖师（苏秦+张仪的师父）
  隐居在鬼谷（地图隐藏地点）
  找到鬼谷子→获得"鬼谷子兵法"→游说能力大幅提升
  鬼谷子出现条件：纵横学派Lv4+智略≥90
```
