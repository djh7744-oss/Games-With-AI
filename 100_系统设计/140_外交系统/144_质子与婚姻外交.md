# 质子与婚姻外交

## 设计目标

> 质子与联姻是战国时代最核心的外交保障手段。送儿子去敌国做人质是一种"以血为誓"的最高级别保证；联姻则将两个家族/国家绑定在一起。这些手段不仅是外交工具，更是充满戏剧性的叙事元素。

## 第一部分：质子系统

### 3.1 质子制度概述

> "质子"即人质——将自己家族成员（通常是儿子/弟弟）送到对方国家作为履约保证。战国时代质子制度极为普遍：秦始皇的父亲异人（子楚）就曾在赵国当质子。

```
质子作用：
  ├── 作为条约/同盟的终极保证
  ├── 表示臣服（小国向大国送质子）
  ├── 互换质子（两国互送→最高信任）
  └── 质子=外交杠杆（对方不敢轻易违约）

质子类型：
  ├── 君主的儿子（最有价值）
  ├── 君主的弟弟/侄子（次等价值）
  ├── 重臣之子（低等价值）
  └── 君主本人（极少见，如越王勾践入吴为质）
```

### 3.2 送质子规则

```
送质子条件：
  ├── 己方必须有合适的质子人选(儿子/弟弟/侄子>10岁)
  ├── 质子人选年龄一般为10-40岁
  └── 送质子可大幅提升条约谈判成功率(+20~35%)

质子人选价值评估：
  嫡长子（继承人）→ 价值：100分
  嫡子 → 80分
  庶子 → 60分
  弟弟 → 70分
  侄子 → 45分
  重臣之子 → 30分

质子在外的生活：
  ├── 被安置在对方都城（通常是王宫或贵族府邸）
  ├── 待遇有：
  │   ├── 上宾待遇（关系好时）
  │   ├── 普通待遇（正常）
  │   └── 囚禁待遇（关系恶化时）
  ├── 质子可学习对方文化/技能（侧面收益）
  └── 质子可建立对方国内的人脉（长期收益）
```

### 3.3 质子相关事件

```
质子事件链：

1. 质子出逃
   触发条件：两国关系恶化至敌对+质子意识到危险
   逃回母国→条约立刻破裂→两国关系-50→可能直接开战
   逃跑失败→质子被处决/更严密囚禁

2. 质子死亡
   自然死亡：对方需通报+送回遗体
   意外死亡：母国可能怀疑是谋杀→关系恶化
   被处决(关系极度恶化时)：→母国获得"复仇"战争借口(开战时己方士气+30)

3. 质子成为对方国君
   异人(子楚)模式：质子→被吕不韦投资→成为秦王
   质子与对方贵族联姻→在对方国内获得势力
   质子继位母国→拥有双重身份→两国自动成为血盟

4. 质子赎回
   花费巨额金赎回质子
   用战俘交换
   对方同意释放（关系改善）

5. 质子价值变化
   新儿子出生→原本的庶子变成"可送的质子"
   嫡长子死亡→次子晋升→质子人选价值变化
```

## 第二部分：婚姻外交

### 3.4 联姻类型

| 联姻类型 | 发起方 | 效果 | 持续时间 | 破裂代价 |
|---------|--------|------|---------|---------|
| **嫁女** | 任意 | 两国关系+20，同盟接受率+25% | 配偶在世时 | 配偶死亡→关系-10 |
| **娶妻** | 任意 | 关系+15，获得配偶家族支持 | 配偶在世时 | 离婚→关系-25 |
| **互换联姻** | 双方 | 你嫁女给我+我嫁女给你→关系+35 | 双配偶在世 | 一方死亡→关系-5 |
| **宗室联姻** | 君主间 | 两国君主家族通婚→血盟关系 | 多代持续 | 破坏→关系-40 |
| **政治纳妾** | 强方 | 弱方送女儿为妾→臣服关系 | 妾在世时 | 妾被休→关系-30 |

### 3.5 联姻的外交效果

```
联姻带来的外交加成：
  联姻国被攻击→你有"参战理由"（仁义不损）
  联姻国请求援助→拒绝时信义-15（比普通拒绝多罚5）
  联姻国之间贸易→关税额外-20%
  联姻国之间军事通行→自动授予
  联姻国→你的盟友→联姻国更容易接受你的同盟邀请

联姻网络策略：
  同时与多国联姻→创建"联姻网络"
  你的子女→嫁/娶多个国家→你是"亲家公/岳父"
  联姻网络=天然同盟体系

负面效果：
  两个联姻国之间开战→你夹在中间（两边都是亲家）
  选择一方→另一方关系-40
  保持中立→两边都不满意（关系各-15）
```

### 3.6 联姻外交事件

```
特殊联姻事件：

1. 吕不韦模式
   投资在外国做质子的王子→帮他回国继位→获得巨大回报
   玩家需要：识人之明+大量资金+耐心等待

2. 和亲失败
   联姻后对方虐待你的女儿/姐妹→你有理由开战
   获得"复仇"战争借口→开战时己方士气+20
   仁义声望不降（大家理解你是为了家人）

3. 继承权联姻
   娶对方君主独女→君主的唯一继承人是你妻子
   君主死后→你的妻子继位→你成为君主配偶
   你的子女→两国王位的继承人→可能统一两国

4. 联姻破裂
   联姻配偶死亡→关系自然下降
   离婚/废妃→严重外交后果
   配偶出轨→你的反应决定一切（开战/原谅/政治利用）
```

## 质子与联姻策略

```
外交杠杆使用：

短期目标（获得条约）：
  质子 > 联姻 > 赔款 > 割地 > 进贡
  质子是最快让对方信任的方式

长期目标（维持和平）：
  联姻 > 质子 > 进贡 > 赔款 > 割地
  联姻维持时间长（配偶在世=数十年）

最高级保证（互换质子+联姻）：
  你送儿子去他家 + 你娶他女儿 = 双重绑定
  这种关系下违约→失去儿子+失去妻子+彻底失信
  几乎不可能被背叛

风险对冲：
  质子年龄15-20→对方会好好养着他10-30年
  质子年龄>40→没几年价值了，对方可能放弃
  联姻娶强势公主→对方不敢轻易翻脸
```

## 与角色继承系统的关联

```
质子/联姻与玩家角色死亡的交互：

玩家角色死亡→子嗣继承后：
  如果新角色曾被送去当质子→保留在该国的人脉关系
  如果新角色的母亲是联姻对象→该国是新角色的舅舅家
  这些关系在继承后继续存在（甚至可能强化）

质子成为继承人：
  玩家长子在外国当质子→玩家死后→质子回国继承
  回国路上的风险→可能被拦截/刺杀
  质子继位后→与质押国的关系特殊（曾是质子vs曾是质押国）
```

## 与其他系统的交互

| 关联系统 | 交互方式 | 影响 |
|---------|---------|------|
| 婚姻系统 | 联姻与婚姻系统共享对象 | 政治婚姻 |
| 外交系统 | 质子/联姻作为条约保障 | 外交杠杆 |
| 继承系统 | 质子可能成为继承人 | 角色→国家 |
| 声望系统 | 质子待遇影响仁义声望 | 善待质子=仁义 |
