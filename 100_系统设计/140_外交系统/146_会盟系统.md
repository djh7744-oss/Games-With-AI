# 会盟系统

## 设计目标

> 会盟（或称盟会）是战国时代最具戏剧性的外交场景——多国君主齐聚一堂，唇枪舌剑、歃血为盟。苏秦主导的"六国合纵"是历史上最著名的会盟。本系统让玩家能发起、参加、主导会盟，在会议上通过外交手腕实现战略目标。

## 系统概述

会盟是3个以上国家参与的多边外交会议。会盟有明确的议题（合纵抗秦/贸易协定/瓜分领土/互不侵犯等），由盟主主持，各国表态。会盟结果可能是成功达成协议、部分达成、或不欢而散。会盟的维持需要后续外交努力，历史上大多数会盟以瓦解告终。

## 核心机制

### 3.1 会盟类型

| 会盟类型 | 发起条件 | 最少参与国 | 核心议题 | 持续时间 | 历史原型 |
|---------|---------|-----------|---------|---------|---------|
| **合纵会盟** | 存在一个超级强国威胁 | 3 | 联合遏制最强国 | 直到威胁解除或瓦解 | 苏秦六国合纵 |
| **称霸会盟** | 霸业声望>60 | 3 | 承认盟主为霸主 | 12-36月 | 齐桓公葵丘会盟 |
| **互不侵犯会盟** | 多国边境紧张 | 3 | 签订多边互不侵犯 | 24-48月 | 弭兵之会 |
| **贸易会盟** | 经济需求 | 2+ | 降低关税/统一货币/开放商路 | 12-36月 | — |
| **瓜分会盟** | 有共同的扩张目标 | 2-3 | 约定瓜分某国领土 | 直到目标达成 | 三家分晋(非会盟但类似) |
| **拥立会盟** | 周天子更替/虚弱 | 3+ | 拥立新天子/确立新秩序 | 长期 | 诸侯拥立周平王 |
| **讨伐会盟** | 某国严重违约/暴行 | 3+ | 联合讨伐共同敌人 | 3-6月(战争期) | 五国伐秦 |

### 3.2 会盟发起与筹备

```
会盟发起流程：

Step 1：发起动议
  条件：
  ├── 发起国霸业声望≥40 或 信义声望≥60
  ├── 发起国至少与2个其他国家关系≥善意
  └── 有明确的会盟议题（不能"无题而盟"）

Step 2：选择会盟地点
  选项：
  ├── 发起国都城（东道主，会盟费用自担，但掌控议程）
  ├── 中立地点（洛阳/周室领地，大家都放心）
  ├── 轮流主办（每次会盟国家轮流，但效率低）
  └── 影响最大的城池（如合纵抗秦通常选在函谷关以东某地）

Step 3：发出邀请
  向目标国家发出会盟邀请
  接受概率 = 关系/2 + 共同利益 + 对议题的兴趣 - 距离惩罚
  距离惩罚 = 路程/100（太远的国家不愿意来）

Step 4：各国确认参加
  至少3国同意→会盟可成立
  少于3国→会盟流产（信义-5，发起国丢面子）

Step 5：会盟日期确定
  给各国留出准备时间：15-30天
  准备期间可进行会前外交（私下沟通/拉票/收买）

Step 6：会盟召开
  所有参与国使团/君主到达会盟地点
  进入会盟模式（特殊UI界面）
```

### 3.3 会盟会议流程

```
会盟日流程：

第一阶段：入盟仪式
  ├── 各国君主/使者入席
  ├── 盟主（发起国）致开幕辞
  ├── 歃血仪式（象征性）：
  │   杀牲口取血→涂在嘴边→宣读盟誓→对天发誓
  │   （此仪式后毁约=对天不敬→信义惩罚×2）
  └── 确定会盟议程

第二阶段：议题讨论
  盟主提出议题→各国依次表态

  表态类型：
  ├── 赞成：完全同意
  ├── 有条件赞成：同意但有附加要求
  ├── 弃权/观望：暂时不表态
  ├── 反对：明确不同意
  └── 退席抗议：直接离开会盟（严重羞辱盟主）

  表态受以下影响：
  ├── 与该国关系的当前状态
  ├── 议题对他是利是弊
  ├── 其他国家的态度（随大流效应）
  ├── 盟主的威逼利诱（会前外交的结果）
  └── 该国当前面临的其他问题

第三阶段：讨价还价
  盟主根据各国反馈修改提案
  私下与个别国家沟通
  让步+施压+拉拢结合

第四阶段：决议表决
  投票规则：
  ├── 简单多数（>50%）：常规决议
  ├── 绝对多数（>66%）：重要决议
  └── 全员一致（100%）：歃血盟约（最高级别）

第五阶段：盟约签署
  通过决议→各国签署盟约
  歃血盟约→惩罚最严厉
  普通决议→惩罚较轻

第六阶段：会盟结束
  成功→所有参与国关系改善
  部分成功→参与国关系微调
  失败→发起国威望损失
```

### 3.4 会盟中的外交博弈

```
会盟中的策略手段：

1. 会前外交（会盟前15-30天）
   ├── 私下拜访各国君主→提前获取支持
   ├── 利益交换（你支持我的提案，我支持你的提案）
   ├── 秘密许诺（会后给你好处）
   ├── 威胁（不支持我→后果自负）
   └── 收买对方使臣（在会中替你说话）

2. 会中策略
   ├── 主导议程（盟主特权→决定讨论顺序）
   ├── 拉拢中间派（对犹豫不决的国家额外说服）
   ├── 孤立反对派（让其他国家都表态支持→反对派感到压力）
   ├── 临时修改提案（根据反馈灵活调整）
   ├── 以退为进（假装放弃→让对方感觉"不通过损失大"）
   └── 突然袭击（提出对方没准备的新议题→在来不及思考时要求表决）

3. 会后跟进
   ├── 会盟决议需要后续落实
   ├── 督促各国履行盟约
   └── 下次会盟（定期会盟机制）
```

### 3.5 盟主选举与争夺

```
盟主产生方式：

方式一：发起国自动成为盟主
  你发起的会盟→你是盟主
  但其他大国可能不服→影响会盟效果

方式二：选举盟主
  各国投票选举会盟盟主
  投票权重 = 霸业声望/2 + 信义声望/3 + 国力排名分
  盟主不是发起国→发起国可能觉得被"篡权"

方式三：轮流盟主
  定期会盟→每次轮值
  避免一家独大

盟主争夺（会盟中最激烈的斗争）：
  两个大国都想当盟主→会盟可能分裂
  一方拉拢一半国家→另一方拉拢另一半
  调解失败→可能变成两个对立的会盟
  这正是"合纵"和"连横"在会盟层面的体现
```

### 3.6 会盟的维持与瓦解

```
会盟维持：

会盟生效期间：
  成员国受盟约约束
  违反盟约→信义惩罚（歃血盟约×2）
  盟主有责任监督执行

定期会盟（如每年一次）：
  固定时间+轮流地点
  讨论新议题+评估旧议题执行情况
  类似"年度峰会"

会盟瓦解：

瓦解原因（按常见度排序）：
  1. 强国的拉拢分化（某个成员被收买→退出会盟）
     → 张仪连横策略的典型运用
  2. 成员国之间利益冲突（会盟内部出现矛盾）
  3. 威胁消失（会盟目标达成或威胁不再）
  4. 盟主衰落（盟主不再有能力领导）
  5. 大规模违约（多个国家同时不履约→会盟名存实亡）
  6. 成员国被灭（他国吞并会盟成员→会盟减员）

瓦解后果：
  部分瓦解：某国退出→其他国家关系-10
  全面瓦解：会盟解散→"会盟不可靠"的印象→下次组织更难
  会盟失败后→30天内无法再发起同类会盟
```

### 3.7 历史上的著名会盟（游戏中的事件）

```
游戏中的会盟事件（可触发或自然发生）：

1. 苏秦合纵（公元前333年）
   苏秦游说六国→六国合纵抗秦
   苏秦佩六国相印→会盟达成
   玩家触发条件：苏秦加入+六国关系≥善意+秦霸业≥50
   效果：六国对秦国形成包围→秦国15年内不敢东出

2. 张仪连横破合纵
   张仪入秦→提出连横策略
   张仪出使各国→逐个拉拢→合纵瓦解
   玩家触发条件：张仪加入+秦国+对有合纵目标
   效果：合纵瓦解→秦国可逐个击破

3. 五国伐秦（公元前318年）
   公孙衍发起→魏/赵/韩/燕/楚五国攻秦
   玩家触发条件：秦国过于强大+5国联合
   效果：对秦国发动联合战争

4. 弭兵之会（公元前546年）
   （春秋末期，但机制可复用）
   晋楚两大国+多个小国→约定停止争霸战争
   玩家触发条件：两个大国都厌倦长期战争
   效果：长期和平协议
```

## 会盟UI

```
会盟大厅界面：
  ├── 中央圆桌（显示参会国家席位）
  ├── 盟主席位突出（居中/上位）
  ├── 各国君主/使者的3D模型或立绘
  ├── 当前议题+提案内容（中央展示）
  ├── 各国态度预览（赞成/反对/弃权的颜色标记）
  ├── 投票倒计时
  └── 操作按钮：提出新议题/私下沟通/修改提案/发起表决/退会

会盟历史记录：
  过去的会盟列表+决议+是否还在生效
  各国出席记录
  会盟中的著名事件（如某国退席抗议）
```
