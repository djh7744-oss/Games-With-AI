# 人生路线任务

## 设计目标

> 对标《太阁立志传5》的多路线主线任务。六条人生路线各有一系列里程碑任务，每条路线8个关键里程碑覆盖从新手到巅峰的完整旅程。路线间可自由切换——你在霸道路线上可以同时推进商道路线。任务不仅引导玩法，更是角色在战国乱世中"活出自己故事"的叙事引擎。

---

## 路线一：霸道 —— 从士兵到帝王

```
适合出身: 贵族/游侠/农民
核心属性: 统率+武勇

阶段1: 初入军营 [统率≥10]
  任务1.1「投军」: 找到征兵官→完成基础训练→正式入伍
       奖励: 什长身份+铜剑+100金
  任务1.2「初阵」: 随军参加第一场战斗→击杀3名敌人
       奖励: 统率+2, 武勇经验+5, 军中声望+10
  任务1.3「新兵试炼」: 完成3次巡逻/剿匪任务
       奖励: 百夫长身份, 统率+3

阶段2: 崭露头角 [统率≥30, 武勇≥25]
  任务2.1「独立指挥」: 获得第一支独立部队(50人)→完成一次小规模战斗
       奖励: 校尉军衔, 统率+5, 武名+10
  任务2.2「军功积累」: 累计击杀500→积累军功
       奖励: 将军衔, 爵位晋升至大夫
  任务2.3「大战考验」: 参与一场大规模战役→表现优异
       奖励: 上将军头衔, 封地×1(村庄), 统率+8
  任务2.4「名将之路」: 指挥一场关键战役并获胜
       奖励: 封侯提名, 武名+30, 专属称号"百战"

阶段3: 封疆大吏 [统率≥55, 爵位≥中卿]
  任务3.1「受封」: 获得封地→建设封地(完成3项建设)
       奖励: 封地正式归属, 经营+5
  任务3.2「灭国之战」: 作为主要指挥官参与灭国战役
       奖励: 关内侯, 霸业+40, 武名+50
  任务3.3「功高震主」: 声望>君主声望时触发→选择: 继续效忠/自立
       效忠: 君主赏赐+信义+20
       自立: 建立自己的势力(需要足够封地和军队)
  任务3.4「称帝」: 统一天下→称王→称帝
       奖励: 帝号, 游戏通关(霸道路线完成)
       
关键分支:
  「功高震主」阶段是霸道路线的核心道德选择
  效忠→以臣子身份辅佐君主统一→"忠臣结局"
  自立→推翻旧主→"霸王结局"→仁义-40但霸业+80
```

## 路线二：王道 —— 不战而屈人之兵

```
适合出身: 士人/贵族/商人
核心属性: 魅力+智略

阶段1: 入仕 [魅力≥25, 智略≥30]
  任务1.1「拜师」: 加入某学派→完成学业→获得毕业认可
       奖励: 学派等级提升, 智略+3
  任务1.2「献策」: 向君主提交一份内政/外交方略→获得采纳
       奖励: 大夫衔, 魅力+3, 信义+5
  任务1.3「初试锋芒」: 完成一次外交斡旋(调解两国争端)
       奖励: 外交官身份, 信义+10, 仁义+5

阶段2: 纵横捭阖 [魅力≥45, 智略≥50]
  任务2.1「合纵」: 联合3个以上国家对抗强国→成功缔结同盟
       奖励: 上卿, 信义+20, 魅力+5
  任务2.2「和平吞并」: 通过外交使某小国主动归附
       奖励: 霸业+20(不战而扩土), 仁义+15
  任务2.3「化解危机」: 在战争一触即发时通过外交化解
       奖励: 信义+25, 国际声望+30
  任务2.4「万国来朝」: 5个以上国家主动与你建交
       奖励: 列侯, 称号"天下信之"

阶段3: 不战而王 [魅力≥70, 信义≥70]
  任务3.1「天下归心」: 各国人民主动拥护你
       奖励: 仁义+30, 所有在野NPC好感+20
  任务3.2「和平统一」: 不发动战争实现统一(通过外交/臣服/禅让)
       奖励: 帝号, "圣王结局"→仁义永久锁定100
  任务3.3「天子」: 受周天子禅让→成为天下共主
       奖励: 王道通关, "三代之治"成就

关键分支:
  王道vs霸道: 使用武力威胁+20%效率但仁义-
  纯粹的和平统一极其困难(但也极其荣耀)
```

## 路线三：侠道 —— 十步杀一人

```
适合出身: 游侠/农民
核心属性: 武勇

阶段1: 江湖初入 [武勇≥30]
  任务1.1「酒馆初遇」: 接到第一个赏金任务→完成
       奖励: 武勇+3, 赏金100金
  任务1.2「名声渐起」: 完成10个赏金任务→解锁高级任务
       奖励: 称号"侠客", 武名+15
  任务1.3「名剑传说」: 听到关于某把名剑的传说→开始追寻
       奖励: 名剑线索, 解锁隐藏任务链

阶段2: 侠客行 [武勇≥50, 武名≥30]
  任务2.1「行侠仗义」: 完成一个保护弱者的重大任务
       奖励: 仁义+15, 武名+20, 民众好感+30
  任务2.2「结识豪杰」: 招募3名游侠同伴→组成侠客团
       奖励: 解锁"侠客团"机制(可组队战斗)
  任务2.3「豪门恩怨」: 被某贵族赏识→成为门客→接触政治
       奖励: 政治入口, 魅力+5
  任务2.4「荆轲路线」: 接受刺杀某国君主的任务(机会仅一次)
       接受: 进行高风险刺杀→无论成败千古留名→游戏进入特殊结局
       拒绝: 继续侠道→走向"侠之大者"

阶段3: 侠之大者 [武勇≥75, 仁义≥50]
  任务3.1「民间领袖」: 收服各路豪杰→建立民间武装
       奖励: 独立武装力量, 统率+8
  任务3.2「起义/招安」: 选择建势力或接受招安→成为将军/封君
       奖励: 爵位/封地/正规军
  任务3.3「侠道霸业」: 以侠道为基础建立政权
       奖励: 侠道通关, "侠王结局"

关键分支:
  荆轲路线=接受任务后无法回头
  成功: 武名+100, 但仁义-80(弑君), 后续进入逃亡/隐居线
  失败: 角色被处决→触发继承(如果无继承人→游戏结束)
```

## 路线四：商道 —— 吕不韦之路

```
适合出身: 商人/士人
核心属性: 经营

阶段1: 白手起家 [经营≥30]
  任务1.1「第一桶金」: 赚取第一个1000金利润
       奖励: 经营+3, 称号"行商"
  任务1.2「商路建立」: 建立第一条稳定商路(同一条路线走3趟)
       奖励: 经营+5, 解锁商路管理
  任务1.3「商铺开业」: 在任一城池购买/租赁第一间商铺
       奖励: 月被动收入开始

阶段2: 富甲一方 [经营≥55, 资产≥20000]
  任务2.1「商会加入」: 加入或创建商会→成为商会首领
       奖励: 商会网络, 商业情报共享
  任务2.2「垄断第一桶」: 成功垄断一种商品(控制50%+供应)
       奖励: 经营+8, 商业影响力+30
  任务2.3「跨国商网」: 在≥3个国家拥有商铺
       奖励: 称号"巨贾", 解锁跨国贸易特权
  任务2.4「投资政治」: 赞助某国王子/大臣→获得政治影响力
       奖励: 政治入口, 信义-10(利益交换)

阶段3: 富可敌国 [经营≥80, 资产≥100000]
  任务3.1「幕后之手」: 你的财富足以影响一国政策
       奖励: 解锁"经济附庸"外交选项
  任务3.2「吕不韦选择」: 投资一位流落在外的王子→帮他回国继位
       成功: 该国成为你的附庸→"现代吕不韦"结局
  任务3.3「商道天下」: 用经济手段统一战国(经济霸主)
       奖励: 商道通关, "陶朱公"成就
```

## 路线五：匠道 —— 鬼斧神工

```
适合出身: 工匠/农民
核心属性: 工艺

阶段1: 学徒 [工艺≥25]
  任务1.1「拜师学艺」: 找到名师→完成入门考核→成为正式学徒
       奖励: 工艺+3, 解锁基础锻造
  任务1.2「第一件作品」: 锻造出第一件绿色品质装备
       奖励: 工艺+5, 称号"匠人"
  任务1.3「修复古物」: 修复一件破损的古代器物(蓝色+)
       奖励: 工艺+5, 发现隐藏技术线索

阶段2: 名匠 [工艺≥50]
  任务2.1「名匠之作」: 锻造出蓝色品质装备并卖出高价
       奖励: 工艺+8, 称号"名匠", 订单增加
  任务2.2「宫廷征召」: 被君主征召为宫廷工匠→打造御用器物
       奖励: 爵位(士), 稳定月俸
  任务2.3「技术革新」: 发明一项新技术(改进锻造/铸造/纺织)
       奖励: 工艺+10, 该技术以你命名, 科技传播
  任务2.4「工坊帝国」: 在≥3城开设工坊, 雇佣≥50工匠
       奖励: 大规模生产能力

阶段3: 宗师 [工艺≥80]
  任务3.1「神兵出世」: 锻造出金色品质装备(非名剑)
       奖励: 称号"宗师", 武名+20(铸剑也是武)
  任务3.2「十大名剑重铸」: 收集碎片→重铸历史上的名剑
       奖励: 传说武器+历史成就
  任务3.3「改变战争格局」: 发明颠覆性军事技术(连弩量产/火药/铁甲舰)
       奖励: 匠道通关, "改变世界的人"成就
```

## 路线六：隐道 —— 著书立说

```
适合出身: 士人/贵族
核心属性: 智略+魅力

阶段1: 求学 [智略≥30]
  任务1.1「拜入师门」: 加入某学派→完成基础学业
       奖励: 智略+5, 学派影响力+
  任务1.2「游学四方」: 拜访≥3国学宫→与不同学派学者交流
       奖励: 智略+5, 视野开阔(解锁跨学派学习)
  任务1.3「初试小论」: 完成一篇小型论文/辩论文
       奖励: 学派内声望+10

阶段2: 名士 [智略≥55, 学派等级≥3]
  任务2.1「著书立说」: 撰写第一部完整著作(需6-12月)
       奖励: 智略+10, 文化影响力+30
  任务2.2「学宫辩论」: 在大型辩论中击败不同学派学者
       奖励: 称号"辩才", 学派影响力+50%
  任务2.3「收徒传道」: 招收≥5名学生→培养成才
       奖励: 魅力+8(名师效应), 学派传承
  任务2.4「经世致用」: 你的学说被某国君主采纳为国策
       奖励: 历史级成就感

阶段3: 宗师/圣贤 [智略≥85]
  任务3.1「百家之宗」: 你的学说影响力超过其他学派
       奖励: 学派领袖地位, 称号"子"(如"X子")
  任务3.2「万世师表」: 你的著作被后世奉为经典
       奖励: 隐道通关, 文化胜利条件之一
  任务3.3「思想统一」: 不用武力, 用思想统一天下
       奖励: "百世之师"成就, 终极隐道结局
```

## 交叉路线设计

```
六条路线不互斥。玩家可以同时推进多条路线。

路线间的交叉里程碑:

🇨🇳 霸道 × 商道: 将军+商人 = 以战养商 → 解锁"军商合一"特殊路线
   专属任务: "战争投机"——在战争中为双方供货
   
🇨🇳 霸道 × 匠道: 将军+工匠 = 兵器革新者 → 装备品质+20%
   专属任务: "兵器革新"——为军队打造专属装备线

🇨🇳 王道 × 隐道: 外交官+学者 = 圣贤 → 文化影响力×2
   专属任务: "经世致用"——学说被多国采纳

🇨🇳 侠道 × 霸道: 游侠+将军 = 马上封侯 → 招募成本-30%
   专属任务: "招安"——从江湖到庙堂

🇨🇳 商道 × 王道: 商人+政治家 = 吕不韦 → 经济附庸
   专属任务: "政治投资"——资助王子回国继位

🇨🇳 匠道 × 隐道: 工匠+学者 = 自然哲学家 → 科技突破
   专属任务: "格物致知"——把工艺经验总结为理论
```
