# 任务生成系统

## 设计目标

> 对标《太阁立志传5》的程序化任务生成。通过模板+参数填充，系统能产生无限种不重复的任务，每个任务都基于当前世界状态和玩家情况生成——护送的不是抽象"商队"而是具体NPC，讨伐的不是抽象"山贼"而是当前真实存在的流寇团伙。每个任务都有理由、有背景、有后果。

---

## 第一部分：任务模板系统

### 1.1 模板结构

```
任务模板 = 任务类型框架 + 参数槽 + 生成条件 + 难度公式

模板定义了：
  ├── 任务名称格式: "[动词][目标]在[地点]"
  ├── 任务描述: 由模板句拼接AI填充
  ├── 目标参数: 需要哪些具体实体(NPC/地点/物品)
  ├── 生成条件: 什么情况下可以生成此任务
  ├── 难度公式: 如何计算任务难度(1-5星)
  ├── 基础奖励公式: 金/声望/物品的基准值
  └── 可能的分支: 执行过程中的意外事件概率
```

## 第二部分：参数填充系统

### 2.1 实体填充规则

```
参数填充流程：

1. 确定任务地点
   基于当前玩家位置/国家/周边区域 → 选择合适的城池/地点

2. 选择关联NPC
   从世界NPC池中根据条件筛选:
   ├── NPC必须在任务相关区域
   ├── NPC状态必须允许参与此任务(非死亡/非囚禁/非战斗中)
   ├── 任务类型匹配NPC职业(商人→送货/讨债，学者→辩论/著书)
   └── 如果无匹配NPC → 自动生成临时NPC(Local NPC池)

3. 选择关联物品/商品
   基于经济系统的物价表 → 选择当前有差价/紧缺的商品
   基于装备系统 → 选择难度匹配的武器/宝物作为奖励

4. 确定威胁/敌人
   基于流寇系统 → 选择当前真实存在的流寇团伙
   基于AI系统 → 选择当前活跃的敌对势力/NPC
   如果无真实威胁 → 生成临时威胁(Random Encounter)

5. 填充叙事细节
   根据以上参数 → AI拼接叙事描述
   确保每个任务读起来像"真实发生的事件"而非"系统生成的模板"
```

### 2.2 任务难度计算公式

```
任务难度(★) = RoundUp( (战斗强度 + 技能要求 + 时间压力 + 风险评估) / 4 )

战斗强度:
  无战斗: 0
  遭遇少量弱敌(流寇小股): 1-2
  遭遇中等敌军(正规军小队): 3-4
  遭遇强敌(名将/大军): 5

技能要求:
  仅需基础属性(走路/对话): 1
  需要特定属性≥30: 2
  需要特定属性≥60+特定技能: 3-4
  需要宗师级属性≥85+多项技能: 5

时间压力:
  无时间限制: 0
  宽松(>15天): 1
  正常(7-14天): 2
  紧迫(3-6天): 3
  紧急(1-2天): 4
  刻不容缓(<1天): 5

风险评估:
  基本无风险: 0
  低风险(可能损失金钱): 1-2
  中风险(可能损失兵力/受伤): 3
  高风险(可能死亡/被俘/声望严重受损): 4
  致命风险(几乎必定有人死亡): 5
```

### 2.3 奖励计算公式

```
基础奖励金 = 难度^2 × 100 × (1 + 玩家等级/50) × 区域经济系数

基础声望 = 难度 × 5 × (1 + 声望修正)

基础经验 = 难度 × 10 × (1 + 对应属性/100)

区域经济系数:
  都城/大商业城: ×1.5
  郡城: ×1.2
  县城: ×1.0
  边境/小城: ×0.7

额外奖励概率:
  稀有物品(蓝色): 10% + 难度×3%
  史诗物品(紫色): 2% + 难度×2%
  传说物品(金色): 概率极低(仅特定隐藏任务)
  NPC好感提升: 完成任务时"表现超出预期"→触发
  额外赏金: 任务发布者特别满意→+20-50%基础金
```

## 第三部分：任务分支与连锁

### 3.1 任务中意外事件

```
任何任务在执行过程中都有概率触发意外：

护送任务中:
  30%概率: 遭遇流寇(标准→触发战斗)
  15%概率: 商队首领请求绕路(去另一个城市→报酬变化)
  10%概率: 商队中有隐藏身份的NPC(揭露→触发支线)
  5%概率: 天气骤变→行程延长→可能超时

讨伐任务中:
  20%概率: 山贼实际人数比情报多(难度升级)
  10%概率: 山贼首领提出谈判(交钱买命→仁义选择)
  5%概率: 发现山贼背后有地方官员包庇(→触发调查任务)
  2%概率: 山贼其实是反抗暴政的义士(→道德困境)

寻人任务中:
  15%概率: 找到人但他不愿意回去(需要说服→魅力判定)
  10%概率: 失踪者被发现已死亡(→变为调查死亡原因)
  5%概率: 失踪者发现了宝藏/秘境(→触发探索任务)
```

### 3.2 任务连锁(Quest Chain)

```
完成任务后→基于结果触发后续任务：

护送商队成功→
  "商人很满意→推荐更多商人找你→解锁高级商队护送(更高报酬)"
  或 "你在目的地发现商机→触发商会投资任务"

讨伐山贼成功→
  "山贼供出幕后黑手→触发调查官员贪腐任务(→可能引发政治地震)"
  或 "山贼投降请求收编→道德选择：收编为私兵/送官府/放了"

完成国家任务→
  "君主赏识→更多国家任务→更高的官职和信任"
  或 "任务中发现君主秘密→道德选择：揭发/保密/利用"

完成学派任务→
  "学派内声望提升→解锁更高难度学术任务→最终拜入宗师门下"

任务链设计原则：
  第一环总是简单的(引导玩家了解任务类型)
  后续环逐渐升级(难度+叙事深度+道德复杂性)
  最终环有重大影响(改变NPC命运/影响国家格局/解锁独特奖励)
```
