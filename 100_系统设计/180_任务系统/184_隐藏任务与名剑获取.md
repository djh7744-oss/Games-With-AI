# 隐藏任务与名剑获取

## 设计目标

> 对标《上古卷轴》的探索式隐藏任务。名剑、宝物、秘境不是任务日志里的待办事项——而是需要玩家通过线索碎片、NPC对话、古书记载拼凑出来的"发现"。找到它们的过程本身就是奖励。

---

## 第一部分：十大名剑获取任务

### 1.1 湛卢 —— 仁道之剑

```
名剑效果: 自动完美格挡(无需方向匹配)——全游戏最顶级的防御效果

获取线索链(三段):
  线索1(酒馆): "听说江南有位隐士，手持一把通体漆黑的长剑，格挡之时如臂使指"
  线索2(学宫): 古籍记载"湛卢，仁道之剑也。剑成而精光贯天，日月争耀"
  线索3(墨家): 墨家巨子提及"湛卢择主，非仁者不可得"

获取任务「仁道试炼」:
  地点: 楚国会稽山深处 → 找到隐士居所
  隐士考验(三个选择):
    考验1: 隐士派山贼袭击附近村庄(假扮)→你是帮隐士还是救村民?
    考验2: 隐士请你评价历任剑主(秦皇/汉武/唐宗)→需展示历史知识
    考验3: 一个自称湛卢持有者的人向你挑战→剑其实是假的→真剑在隐士手中
  
  通过条件: 仁义≥80 + 三个考验中至少两个选择"仁"的选项
  失败: 隐士婉拒→"湛卢尚未等到它的主人"→仁义+10后可以再试
  
  获得: 湛卢剑 + 隐士传授"完美格挡"被动技能(即使不用湛卢也有概率触发)
```

### 1.2 鱼肠 —— 刺客之剑

```
名剑效果: 刺击伤害+50%，可藏于袖中(隐蔽携带)

获取线索链:
  线索1(酒馆): "专诸刺王僚时用的那把短剑，据说还在吴国故地"
  线索2(侠客): 老刺客提及"鱼肠剑藏于鱼腹之中，专诸以此刺杀王僚"
  线索3(古墓): 吴国古墓中发现"鱼肠剑碎片"→其实是仿品→真剑线索指向

获取任务「刺客传承」:
  地点: 吴国故地(会稽/姑苏一带)
  条件: 加入刺客组织(需要侠道路线或与刺客组织建立关系)
  
  任务链:
    1. 证明你的刺客资质(完成3次高级暗杀→不被发现)
    2. 找到专诸后裔→他隐藏了鱼肠剑
    3. 专诸后裔的考验: 刺杀一位鱼肉百姓的暴虐官员
       不杀平民→不惊动守卫→成功击杀后消失于黑夜
    4. 完成考验→获得鱼肠剑+专诸刺杀术(刺击暴击率+15%)

  注意: 如果玩家仁义≥60→专诸后裔拒绝→"你不配持有这把剑"
        鱼肠剑是"以暴制暴"的象征→仁义太高反而不适合
```

### 1.3 巨阙 —— 力士之剑

```
名剑效果: 重击可破防+击退+小范围溅射

获取线索链:
  线索1(铁匠铺): "宜阳有个传说，有把剑重一百二十斤，非力士不能举"
  线索2(军旅): 秦国老兵说"见过一把巨剑，一剑下去砸碎了三面盾牌"
  线索3(古战场): 在古战场发现巨剑使用过的痕迹(石头上巨大的剑痕)

获取任务「力士试炼」:
  地点: 韩国宜阳铁矿山
  条件: 武勇≥90 (力量不够根本拿不动)
  
  试炼:
    1. 在宜阳矿山找到被巨石压住的巨阙剑(地震导致山洞塌方)
    2. 需要推开巨石(武勇判定)或使用杠杆(智略判定)
    3. 举起巨阙剑(武勇判定→武勇90+才能挥动)
    4. 试剑: 用巨阙劈开试剑石→成功3次重击破石
  
  失败: 如果武勇不够→巨阙纹丝不动→围观者嘲笑→武名-5
        但可以以后武勇够了再来试(剑一直在那)
  
  获得: 巨阙剑 + 称号"力能扛鼎"
```

### 1.4 太阿 —— 威道之剑

```
名剑效果: 每3次命中→下次攻击必暴击

获取线索链:
  线索1(学宫): 古书"太阿，欧冶子与干将为楚王所铸，威道之剑"
  线索2(楚国): 楚国贵族提到"太阿剑随楚王下葬了"→盗墓者线索
  线索3(盗墓者): 找到盗墓者→他说"太阿的墓有机关，我进不去"

获取任务「王者之墓」:
  地点: 楚国某王陵(地图隐藏→需要盗墓者带路)
  
  任务链:
    1. 找到盗墓者→说服他带路(金/威胁/交易)
    2. 进入王陵→机关迷宫(智略判定→走错一步触发陷阱)
    3. 墓室守护者: 古代机关人→战斗
    4. 最终墓室: 太阿剑在楚王石棺上→选择:
       A. 直接取剑(触发诅咒→永久debuff"王者之怒":士气-10)
       B. 先向楚王致敬/祭祀(礼毕后诅咒解除→正常取剑)
    5. 获得太阿剑+楚王铠甲(金色品质)
  
  风险: 盗墓→仁义-20(如果被发现)
        但太阿剑+楚王甲的价值远超仁义损失
```

### 1.5-1.10 其余六剑概要

```
干将莫邪(双剑):
  获取: 楚地铁匠世家任务线→寻找失踪的铸剑师→发现他被囚禁铸剑
  选择: 救出他(得到干将)或救出他妻子(得到莫邪)或两人都救(双剑)
  效果: 双持时干将+攻速20%/莫邪+伤害20%

龙渊:
  获取: 探索龙渊剑池→击败守护龙兽(巨型鳄鱼)→淬龙血→剑成
  效果: 连招每段伤害递增10%(最高+50%)

工布:
  获取: 工匠路线专属→天下百兵试炼→削断100把武器→工布现世
  效果: 对方武器耐久消耗×3(15击必断)

胜邪:
  获取: 战场1000人斩→剑灵唤醒→降服邪灵→胜邪认主
  效果: 每击杀+2攻击/场(上限+20); 但使用过久→剑灵反噬

纯钧:
  获取: 王道路线→诸侯会盟→受赠或夺取
  效果: 对方攻击-15%(心理压制); 贵族社交+30; 附带外交BUFF
```

## 第二部分：隐藏宝物任务

### 2.1 孙子兵法(完整版)

```
宝物效果: 统率+10(永久), 全军获得"知己知彼"BUFF(士气+5持续)

获取线索:
  线索1(齐国学宫): 据说孙武后人还保留着完整的兵法手稿
  线索2(吴国故地): 孙武曾为吴国练兵→吴国旧臣后裔可能知道下落
  线索3(兵家NPC): 孙膑提及祖传兵法"有十三篇，现存十一篇，两篇失传"

获取任务「寻找失传的兵法」:
  1. 找到孙武后人(隐居在齐国乡间)
  2. 后人说有两篇在他被秦军掳走的祖父身上→被带到了咸阳
  3. 前往咸阳→在秦国档案库(或黑市)寻找手稿
  4. 找到手稿→拼合完整13篇
  5. 获得: 「孙子兵法(完整版)」+ 统率永久+10
```

### 2.2 和氏璧

```
宝物效果: 持有者霸业声望获取+50%, 魅力+8

获取线索:
  线索1(楚国): 传说"和氏璧在楚国王宫中"
  线索2(赵国): 后来"和氏璧辗转到了赵国"
  线索3(秦国): 秦国提出"以十五城换和氏璧"(蔺相如完璧归赵的由来)

获取任务「天下至宝」:
  1. 和氏璧在赵国王宫中→需要极高外交关系或盗取
  2. 秦国也会出价→如果你代表秦国→需要智略判定
  3. 多种获取方式:
     外交购买(极高金+高关系)
     盗取(间谍任务→高风险)
     战争夺取(灭赵→和氏璧作为战利品)
     政治交易(用城池/联姻/质子交换)
  4. 获得后→世界事件"和氏璧易主"→影响国际格局
```

### 2.3 鬼谷子隐居地

```
发现效果: 找到鬼谷子→学习纵横术(魅力+智略+10, 外交成功率+20%)

触发条件: 非常隐蔽——没有任何任务提示

发现方式:
  线索碎片分散在:
    纵横家NPC对话(苏秦/张仪偶尔提到"师父")
    古书中提及"鬼谷"地名(模糊)
    一位老隐士提到"云梦山中有高人"
  
  拼凑线索→前往云梦山→在山中寻找(需要智略+运气)
  找到鬼谷→鬼谷子出现→他说"你找到我,便是缘分"
  
  鬼谷子教学(7天):
    每天学习一项纵横术
    第7天→鬼谷子赠「鬼谷子」竹简→然后飘然而去(下次再也找不到)
  
  这是全游戏最隐蔽的隐藏内容之一
  设计意图: 让极少数真正深入探索的玩家获得惊喜
```

## 隐藏任务的设计哲学

```
1. 渐进式发现
   不是"找到一个卷轴→任务弹出"
   而是"拼凑N个线索碎片→玩家自己意识到'这里有东西'"

2. 社区共享
   隐藏内容极其隐蔽(单个玩家几乎不可能找到全部)
   鼓励玩家社区分享发现(像《艾尔登法环》的社区解谜)

3. 有意义的奖励
   不是"找到隐藏+5%属性"
   而是"找到鬼谷子→完全改变你的外交玩法"
   隐藏奖励改变游戏体验,不只是数值

4. 尊重玩家智慧
   没有任何"提示弹窗"
   没有任何"附近有秘密"的UI标记
   全靠观察力+推理+运气+社区协作
```
