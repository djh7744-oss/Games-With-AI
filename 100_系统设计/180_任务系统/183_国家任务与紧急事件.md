# 国家任务与紧急事件

## 设计目标

> 对标《三国志》系列的国家任务系统。当你身居官职，国家不再只是背景——君主会委派你执行各种任务，从平凡的征税到决定国运的决战。紧急事件不会等待你有空——敌军入侵时，你在千里之外经商也必须做出反应。国家需要你。

---

## 第一部分：国家任务

### 1.1 君主委派任务

```
任务由AI君主基于国家需求自动生成：

军务类:
  征集新兵: 前往[X城]征集[数量]新兵→训练→编入军队
  平定叛乱: [X城]爆发叛乱→率军平叛
  防御边境: 前往[X关隘]驻防[天数]→防范[某国]入侵
  突袭敌军: 趁[某国]不备→突袭[目标]→速战速决
  护送军粮: 护送军粮从[A]到前线[B]→确保不被劫

政务类:
  安抚灾民: [地区]遭遇灾荒→开仓赈济/组织恢复
  审理案件: 调查[X事件]→找到真凶→公正裁决
  推行新政: 在[X城]推行[某政策]→克服地方阻力
  考核官员: 前往[X城]考核县令政绩→如实汇报
  征收赋税: 前往[区域]征收已拖欠的税款

情报类:
  出使他国: 携带国书出使[某国]→交涉[议题]→回国复命
  侦查敌情: 潜入[某国]→收集[军事情报]→安全返回
  搜捕奸细: [某国]间谍潜入→排查搜捕→维护国安
  传递密信: 携带绝密文书→送到[指定NPC]手中→不能泄密
```

### 1.2 国家任务等级与权限

```
任务等级匹配官职:

士/大夫: 基础任务(征税/护送/新兵训练)
下卿/中卿: 中级任务(平叛/外交/侦查)
上卿: 高级任务(主持新政/灭国战役)
关内侯+: 国家级任务(主持变法/统帅三军/与国君共商国是)

拒绝国家任务:
  可拒绝但后果严重:
    拒绝1次: 君主好感-15
    拒绝3次: 爵位降级
    拒绝5次+: 以"抗命"论处→可能被免职或处决
  (紧急任务不可拒绝→拒绝=叛国)
```

### 1.3 国家长期任务

```
长期任务(没有具体时限，但影响国家发展):

富国强兵: 在[X]年内将国家军力提升至[X]
  进度追踪: 兵力/装备/训练度三个指标

备战备荒: 在[X]年内囤积[X]粮草
  进度追踪: 粮仓储备量

变法维新: 协助君主完成[某变法]的全国推行
  进度追踪: 变法在各城的实施进度

伐交伐谋: 在[X]年内孤立[某国](使其盟友≤1)
  进度追踪: 目标国的外交关系数量

人才强国: 在[X]年内为国家招募[N]名良才
  进度追踪: 新招募人才数量+质量

奖励: 完成长期任务→君主大量赏赐+爵位晋升
      失败→可能降职(但不会严厉惩罚，因为是"长期"任务)
```

## 第二部分：紧急事件

### 2.1 国家紧急事件

```
事件触发基于世界状态→立即通知玩家→强制响应:

🔴 敌国入侵 (最高紧急)
  触发: 敌军越过边境→进入我国领土
  通知: 信使飞马传书→"秦军已过函谷关!"
  选项:
    A. 立即率军回防 (推荐，需在[X]天内到达)
    B. 派副将率军 (如果玩家太远赶不回来)
    C. 无视 (等同于叛国)
  
  后果:
    成功防御→爵位晋升+君主赏赐
    防御失败→失地+君主迁怒→降爵
    无视→视为叛国→全国通缉

🔴 君主遇刺/驾崩 (最高紧急)
  触发: 本国君主被刺杀或自然死亡
  通知: 丧钟响起→"王薨!"
  影响:
    继承危机→各派系争夺王位→需要选择支持一方
    外部威胁→邻国趁虚而入→边境告急
  
  你的选择:
    支持合法继承者(嫡长子)→信义+但可能站错队
    支持强者→现实主义但信义-
    自立为王(需要足够实力)→最高风险最高回报

🟠 大规模叛乱
  触发: 某区域民忠<20持续3月+有组织者
  通知: 地方官员急报→"[X地]反了!"
  
  你的任务:
    率军平叛→选择镇压/谈判/招安
    (见164_动态世界系统-叛乱系统)

🟠 邻国突然宣战
  触发: AI国家发动突然袭击
  通知: 边境烽火点燃→"狼烟起!"
  
  额外风险: 如果玩家恰好在该国境内→可能被扣留/驱逐

🟡 重大灾荒
  触发: 旱灾/洪水/蝗灾→粮食减产>50%
  通知: 地方官员上报→"大饥!"
  
  你的任务:
    组织赈灾→调配粮食→防止饥民暴动
    也可: 忽视(仁义-50，大量人口死亡)
```

### 2.2 个人紧急事件

```
个人层面的突发事件(不影响国家但影响玩家):

🔴 封地遭攻击
  触发: 敌国/流寇攻入你的私人封地
  通知: 封地管家急报→"封地告急!"
  选项:
    A. 立即返回保卫(放弃当前任务)
    B. 派军队防守(如果有驻军)
    C. 放弃封地(损失惨重→封地可能有去无回)

🟠 家人/配偶遇险
  触发: 家人被绑架/暗杀/病危
  通知: 家书→"速归!"
  
  这是情感驱动的事件，不是利益计算
  回去→仁义+但可能耽误国家大事
  不回去→家庭关系破裂+仁义-+可能遗憾终生

🟡 被通缉/追杀
  触发: 声望过低/犯罪/被政敌陷害
  通知: 卫兵贴出告示→"悬赏[玩家名]!"
  
  紧急对策:
    A. 自首→接受审判(可能是死刑)
    B. 逃亡→离开该国(流亡者身份)
    C. 反击→找出幕后黑手→洗清罪名

🟡 部将叛变
  触发: 忠诚<30的关键部下突然叛变
  通知: 亲信来报→"[将领名]叛投[某国]!"
  
  后果:
    叛变将领带走部分军队/封地
    该国获得你的情报
    需要评估: 追击惩罚叛徒/放任不管/策反回来
```

### 2.3 紧急事件优先级系统

```
紧急度分级(UI用颜色和图标):

🔴 P0 - 存亡攸关 (必须立即处理)
  敌军入侵/君主驾崩/你被刺杀
  触发后→强制暂停其他任务→弹出全屏通知
  如果不在[X]天内响应→自动按默认选项处理(通常是最坏的)

🟠 P1 - 重大危机 (强烈建议处理)
  叛乱/宣战/封地被灭
  通知→HUD顶部闪烁警告
  如果不在[X]天内响应→负面后果自动生效

🟡 P2 - 需要关注 (建议处理但非强制)
  灾荒/家人遇险
  通知→任务日志高亮
  响应期较长(15-30天)

⚪ P3 - 信息通报 (知悉即可)
  商人提供情报/邻国内斗/他国结盟
  通知→消息列表记录
  不需要响应(但可以主动利用)
```

## 第三部分：世界事件任务

### 3.1 全球性事件

```
定期发生的世界级事件(每5-10年可能在游戏中触发):

⚔️ 霸主之战: 两个最强国爆发全面战争
  所有国家被迫选边站→中立也需要表态
  玩家任务: 为本国争取最佳战略位置

📜 天下之会: 周天子召集天下会盟
  各国君主或使者齐聚洛阳→讨论重大议题
  玩家任务: 在会盟中为本国争取利益(见146_会盟系统)

🌊 大灾之年: 全球范围的自然灾害集中爆发
  多个国家同时受灾→国际赈灾合作/趁火打劫
  玩家任务: 选择合作(仁义)或利用(霸业)

💀 名将凋零: 一代名将(白起/廉颇/孙膑等)自然老死
  该国军事实力大降→邻国虎视眈眈
  玩家任务: 抓住机会/援助友邦/招募其门徒
```

### 3.2 可选参与的世界事件

```
这些事件触发后→玩家可选择是否参与:

"黄金台之约": 燕昭王筑黄金台求贤→天下人才涌向燕国
  玩家选择: 去燕国求职/留在本国/趁机挖燕国墙角

"稷下论道": 齐国举办大型学术辩论→诸子百家云集
  玩家选择: 参加辩论(智略)/观战学习/经商牟利

"宜阳兵器展": 韩国举办武器贸易展→天下买家齐聚
  玩家选择: 采购最新装备/展示自己的武器/刺探技术

"函谷关商路": 秦国开放函谷关商路→东西贸易大通道
  玩家选择: 组织商队通过/担忧秦国军事威胁
```
