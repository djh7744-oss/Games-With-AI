# 声望与爵位系统

## 设计目标

> 声望是你在世人眼中的形象——仁义之君、信义之人、百战名将、霸道征服者。爵位是官方对你的认可——从一介布衣到裂土封王。两者互相影响但不绑定：你可以有声望无爵位（名士隐士），也可以有爵位无声望（暴虐之将）。

## 第一部分：四维声望

### 1.1 声望概述

```
四维声望独立计算，范围 -100 到 100：

仁义：他人对你的道德评价
信义：他人对你的信用评价
武名：他人对你的武力评价
霸业：他人对你的征服力评价

声望不是"越高越好"——高声望有好处也有代价。
不同声望组合塑造不同的人设。
```

### 1.2 仁义声望

```
仁义代表：善待他人、仁政爱民、慈悲为怀。

获得方式：
  释放俘虏（不杀/不卖）：+2~10/次
  赈灾（花费金+粮食）：+5~20/次
  减免赋税：+1/月（持续时）
  保护平民不受兵灾：+5~15/次
  拒绝屠城：+15
  收养孤儿/资助学堂：+3~8/次
  医疗救助：+2~5/次

失去方式：
  屠城：-50~-100
  处决俘虏：-5~-15/次
  强征粮食(从平民)：-3~-10/次
  放任士兵劫掠：-15~-30
  加税：-1/月（持续时）
  见死不救：-2~-10/次

仁义效果：
  高声望(≥60)：
    平民好感+30，招募在野武将成功率+25%
    民忠+15，人口迁入+20%
    敌对势力百姓对你的军队抵抗意愿-40%
    仁君之名 → 周边小国主动归附概率+20%
  
  低声望(≤-30)：
    平民仇视(-30好感)，招募困难(-25%)
    占领城市后叛乱概率+40%
    治下民忠持续下降
    被邻国宣传"暴君" → 外交困难

仁义特殊事件：
  仁≥80：触发"万民请愿"——平民自发为你提供情报/隐藏敌人
  仁≥95：触发"圣人之名"——诸子百家中儒家/墨家主动与你合作
  仁≤-80：触发"天下共讨"——其他国家可能组成"讨伐暴君"联盟
```

### 1.3 信义声望

```
信义代表：言出必行、守信重诺、交易公平。

获得方式：
  履行外交承诺：+10~20/次
  按时还贷：+3~8/次
  遵守停战协议：+5/月（持续时）
  兑现对NPC的承诺：+5~15/次
  公平交易（不欺诈）：+1~3/次
  守城不退（答应守城就死战不退）：+15

失去方式：
  撕毁盟约：-30~-60
  背刺盟友：-50
  违约不还贷：-15~-30
  许下承诺不兑现：-10~-25/次
  使用欺诈手段交易：-5~-15/次
  向敌军投降后反叛：-40

信义效果：
  高声望(≥60)：
    外交承诺被对方接受概率+30%
    贷款利率-40%（银行信任你）
    商业伙伴愿意赊账额度×3
    属下忠诚保持+20%
    条约谈判时对方让步更多
  
  低声望(≤-30)：
    无人愿与你结盟（外交成功率-50%）
    贷款利率+100%（或根本借不到）
    属下更容易被策反（忠诚判定-30%）
    商人拒绝与你大额交易

信义特殊事件：
  信≥80：触发"一诺千金"——可以口头承诺替代书面盟约（但违约损失×3）
  信≥95：触发"天下信之"——你的承诺比书面条约更有分量
  信≤-80：触发"失信之人"——即使说的是真话NPC也半信半疑
```

### 1.4 武名声望

```
武名代表：战场上的威名、个人的武力、军事成就。

获得方式：
  单挑胜利：+3~10/次（对手越强越多）
  斩杀敌将：+5~15/次
  指挥战役胜利：+5~20/场（规模越大越多）
  以少胜多：+10~30/场
  灭国：+50
  连续胜利（不败纪录）：每10连胜+10

失去方式：
  单挑失败：-5~15/次
  战役失败：-5~20/场
  被俘：-15~-30
  逃跑（从战场）：-20~-40
  全军覆没：-30
  被以少胜多：-25

武名效果：
  高声望(≥60)：
    敌军面对你时士气-15（闻风丧胆）
    己方士兵士气+10（追随战神）
    招募勇将成功率+35%
    单挑时对手更容易接受（"能与你一战是我的荣幸"）
  
  低声望(≤-30)：
    敌军轻视你（但也没什么特殊加成）
    己方士兵士气-10（对统帅没信心）
    单挑时对手可能拒绝（"你不配"）

武名特殊事件：
  武≥80：触发"威震天下"——其他国家不敢轻易与你开战
  武≥95：触发"战神"——你的存在本身就是战略威慑
  武≤-80：触发"笑柄"——市井小民都在嘲笑你的军事才能
```

### 1.5 霸业声望

```
霸业代表：征服的野心、领土的扩张、统一天下之势。

获得方式：
  占领城池：+5~15/城
  灭国：+40~80
  扩大领土：按面积计算
  称王/称帝：+30/50
  迫使小国臣服：+10~20/国
  修筑长城/大型军事工程：+10~20

失去方式：
  失去领土：-10~30/城
  被夺回封地：-15
  属国背叛独立：-20/国
  割地求和：-30
  被贬爵：-20~-50

霸业效果：
  高声望(≥60)：
    小国臣服概率+40%
    敌国谈和时愿意割地赔款
    称王/称帝条件满足更快
    你的领土内叛乱概率-30%
  
  低声望不影响（霸业低只是说明你没有扩张野心，不影响已有权力）

霸业特殊事件：
  霸≥80：触发"问鼎中原"——周天子可能主动禅让
  霸≥100：触发"天命所归"——天下大势朝你倾斜
```

### 1.6 声望组合人设

| 声望组合 | 人设 | NPC反应 |
|---------|------|--------|
| 仁义高+武名高 | 儒将（如吴起、乐毅） | 全民爱戴 |
| 仁义高+霸业高 | 圣王（如周文王） | 天下归心 |
| 武名高+霸业高 | 霸王（如项羽、白起） | 畏惧+尊敬 |
| 信义高+仁义高 | 君子（如信陵君、平原君） | 各方信任 |
| 武名高+信义低 | 枭雄（如曹操型） | 又怕又防 |
| 霸业高+仁义低 | 暴君（如秦始皇刻板印象） | 恐惧统治 |
| 全高 | 天命之子 | 几乎所有人尊敬 |
| 全低 | 无名之辈 | 无人关心 |

## 第二部分：爵位系统

### 2.1 九级爵位

| 等级 | 爵位 | 封地规模 | 可驻军 | 年俸(金) | 晋升条件 |
|------|------|---------|--------|---------|---------|
| Lv1 | 士 | 无 | 0 | 0 | 入仕/从军起 |
| Lv2 | 大夫 | 村庄×1 | 50人 | 500 | 累积击杀100 + 属国认可 |
| Lv3 | 下卿 | 村庄×3 | 200人 | 1500 | 累积击杀500 + 统率≥30 |
| Lv4 | 中卿 | 县城×1 | 500人 | 4000 | 指挥5场胜利 + 统率≥45 |
| Lv5 | 上卿 | 县城×2 | 1500人 | 10000 | 指挥15场胜利 + 统率≥60 |
| Lv6 | 关内侯 | 郡城×1 | 5000人 | 30000 | 灭国/史诗成就 + 霸业≥30 |
| Lv7 | 列侯 | 郡城×3 | 15000人 | 80000 | 统一地区 + 霸业≥60 |
| Lv8 | 王 | 全国 | 全国兵力 | — | 称王(需推翻周天子) + 霸业≥80 |
| Lv9 | 帝 | 天下 | 天下兵马 | — | 统一天下 + 称帝 + 霸业≥95 |

### 2.2 爵位获取途径

```
途径一：军功爵（秦国特色，其他国部分采用）
  斩杀敌人→累积军功→自动晋升
  秦国的"二十等爵"简化版
  每级有明确军功要求
  最公平但也最血腥的途径

途径二：封赏（各国通用）
  君主直接封赏（需要君主赏识+推荐人）
  联姻获得（娶公主→驸马都尉→进一步封赏）
  献地/献金（向君主贡献大量资源）

途径三：自立称王（最高风险）
  拥有足够领土+兵力+霸业声望
  推翻周天子/不被任何国家制约
  称王后其他国家态度：
    盟友→可能翻脸（王不见王）
    敌国→立即开战（"乱臣贼子天下共讨之"）
    中立→观望

途径四：和平禅让
  周天子主动禅让（霸业+仁义达标）
  前朝君主禅让（极罕见，需要特殊条件触发）
```

### 2.3 爵位特权

| 爵位 | 特权 |
|------|------|
| 士 | 可进入官府，可向君主上书 |
| 大夫 | 拥有第一块私人封地，可招募少量门客 |
| 下卿 | 封地扩大，可任免封地内低级官员 |
| 中卿 | 可参与朝政(有投票权)，可统领千人军团 |
| 上卿 | 可担任国相/上将军，有独立外交权（与外国谈判） |
| 关内侯 | 食邑(税收归己)，可铸造私人兵器 |
| 列侯 | 世袭罔替(子孙继承)，独立铸币权 |
| 王 | 独立国家(国际法主体)，开战/结盟/立法 |
| 帝 | 天下共主——全图统一 |

### 2.4 爵位与声望的互动

```
爵位晋升受声望影响：

军功爵（依赖武名）：
  武名越高→军功爵晋升越快→君主更愿封赏

封赏爵（依赖信义+魅力）：
  信义高→君主信任你→封赏大方
  信义低→君主防范你→封赏吝啬

自立称王（依赖霸业+武名）：
  霸业高→称王后其他国家不敢轻举妄动
  仁义低→称王后被邻国联合讨伐

爵位与声望冲突：
  高等爵位+低声望 → 被嘲讽为"暴发户"
  低等爵位+高声望 → 民间自发尊重，"无冕之王"
```

## 与其他系统的交互

| 关联系统 | 交互方式 | 影响 |
|---------|---------|------|
| 封地经济 | 爵位决定封地规模 | 爵位→封地→经济 |
| 军团指挥 | 爵位决定最大带兵数 | 爵位→军力 |
| 外交系统 | 爵位和声望影响外交选项 | 声望+爵位=外交能力 |
| 婚姻系统 | 高声望+高爵位=更多联姻选项 | 声望→联姻质量 |
| NPC关系 | 声望直接影响NPC态度 | 声望=社会评价 |
