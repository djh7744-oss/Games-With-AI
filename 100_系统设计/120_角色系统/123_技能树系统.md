# 技能树系统

## 设计目标

> 对标《太阁立志传5》的技能修行：六条技能树，每条5层，共30个可选节点。技能点有限（60点满级=60点），玩家必须在深度（单线升满）和广度（多线兼顾）之间做选择。

## 系统概述

角色每升1级获得1技能点（等级上限60=60技能点）。六条技能树（战斗/统帅/智谋/魅力/经营/工艺）各5层，每层有多个节点可选。高等级节点需要前置节点+属性门槛。玩家可自由分配技能点，形成独特的角色Build。

## 技能树一：战斗系（侧重武勇）

> 提升个人战斗力。适合亲自冲锋陷阵的玩家。

```
Lv1（入门 — 无门槛）
├── 近战精通：所有近战武器伤害+8%
├── 远程精通：弓弩射速+15%，精度+10%
└── 生存专家：生命上限+15%，包扎恢复+30%

Lv2（进阶 — 需要武勇≥25）
├── 骑术专精：马上攻击伤害+15%，马匹操控+20%
├── 双持战斗：可双持单手武器，攻速+20%但防御-10%
└── 重甲适应：穿重甲时移速惩罚-50%

Lv3（专家 — 需要武勇≥40）
├── 致命一击：暴击率+10%，暴击伤害+25%（从150%→175%）
├── 闪避大师：闪避无敌帧+0.05秒，闪避体力消耗-30%
├── 破甲专精：破甲率+15%（可与武器破甲叠加）
└── 坚韧体质：体力恢复速度+50%，体力<30时攻击不减

Lv4（大师 — 需要武勇≥60）
├── 武器大师：可在战斗中切换武器无冷却，每种武器首次攻击+20%伤害
├── 不屈意志：生命<20%时→防御+50%+攻击+25%（绝地反击）
└── 旋风连斩：击杀敌人后下一次攻击伤害+30%（可连续触发）

Lv5（宗师 — 需要武勇≥75，消耗2技能点）
└── 万人之敌：单挑时伤害+50%，斩杀敌将→敌全军士气-30%，队友士气+15%
```

## 技能树二：统帅系（侧重统率）

> 提升带兵指挥能力。适合走霸道路线的玩家。

```
Lv1（入门 — 无门槛）
├── 练兵：部队训练速度+30%，新兵初始士气+10
├── 阵型基础：变阵时间-15%
└── 后勤管理：军粮消耗-15%，行军速度+10%

Lv2（进阶 — 需要统率≥25）
├── 骑兵指挥：骑兵编队攻击+12%，冲锋伤害+20%
├── 弓兵指挥：弓兵编队射程+10%，齐射伤害+15%
└── 步兵指挥：步兵编队防御+12%，方阵效果+20%

Lv3（专家 — 需要统率≥45）
├── 伏击战术：伏击成功率+25%，伏击伤害+30%
├── 攻城指挥：围城时敌军士气下降加速+50%，攻城器械伤害+20%
├── 急行军：全军移速+20%持续12小时，冷却24小时
└── 地形利用：全军获得当前地形优势×1.5倍

Lv4（大师 — 需要统率≥65）
├── 多线指挥：可同时给3个不同编队下达不同指令（无此技能只能逐个指挥）
├── 精英训练：T3+部队全属性+10%
└── 战场应变：变阵可在被攻击时完成（正常会被打断），变阵时间-30%

Lv5（宗师 — 需要统率≥80，消耗2技能点）
└── 千军万马：带兵上限+50%，可同时指挥3支独立编队独立行动
```

## 技能树三：智谋系（侧重智略）

> 提升计谋/辩论/科技。适合走王道或隐道路线的玩家。

```
Lv1（入门 — 无门槛）
├── 辩才：辩论成功率+15%，献策被采纳率+20%
├── 洞察：识别敌方计谋概率+25%，发现伏击概率+20%
└── 博闻强记：阅读竹简速度×2，学习效率+20%

Lv2（进阶 — 需要智略≥25）
├── 火计精通：火计成功率+25%，火攻伤害+30%
├── 诱敌深入：诈败成功率+30%，诱敌至伏击圈后敌军混乱时间×2
└── 谣言散布：流言效果+30%，敌城民忠下降加速+50%

Lv3（专家 — 需要智略≥45）
├── 反间计：策反敌将成功率+25%，被反间概率-50%
├── 空城计：兵力劣势>3:1时可使用，30%概率吓退敌军
├── 驱虎吞狼：挑拨两国开战成功率+20%
└── 天时预测：可提前1天预知天气变化，据此调整战术

Lv4（大师 — 需要智略≥65）
├── 连环计：计谋可连环使用，第二个计谋成功率+20%
├── 运筹帷幄：可在战略地图上远程指挥间谍行动（正常需到场）
└── 奇门遁甲：获得特殊计谋"落石""迷雾""地陷"

Lv5（宗师 — 需要智略≥80，消耗2技能点）
└── 神机妙算：所有计谋冷却时间-50%，计谋成功率+15%，每场战斗可使用计谋数+2
```

## 技能树四：魅力系（侧重魅力）

> 提升社交/外交/招募/领导力。适合走王道或商道路线的玩家。

```
Lv1（入门 — 无门槛）
├── 社交手腕：NPC初始好感+15，对话选项增多
├── 识人之明：可查看NPC隐藏属性（忠诚倾向/野心/性格弱点）
└── 风度翩翩：参加社交活动消耗-30%，获得情报+50%

Lv2（进阶 — 需要魅力≥25）
├── 外交辞令：外交成功率+20%，条约条件优化(对方让步+10%)
├── 三寸不烂之舌：劝降成功率+25%，招降俘虏概率+20%
└── 领袖魅力：属下忠诚度衰减速度-30%，属下叛变概率-40%

Lv3（专家 — 需要魅力≥45）
├── 纵横捭阖：可同时与5国保持同盟（正常上限3国）
├── 一呼百应：招募在野武将时→多人响应概率+30%
├── 化敌为友：战后招降敌将成功率+25%（包括名将）
└── 民心所向：封地民忠+10，人口迁入率+20%

Lv4（大师 — 需要魅力≥65）
├── 一言兴邦：每成功外交一次→本国声望+3（可累积）
├── 群臣归心：直属属下全属性+5%，永远不会背叛
└── 倾国倾城：异性NPC相关判定+30%，特殊剧情触发

Lv5（宗师 — 需要魅力≥80，消耗2技能点）
└── 天下归心：被动光环——30里内所有友方NPC士气+10，敌方非将领单位10%概率不战而降
```

## 技能树五：经营系（侧重经营）

> 提升经商/封地管理/投资。适合走商道路线的玩家。

```
Lv1（入门 — 无门槛）
├── 商业嗅觉：可看到商品在当前城的"公道价"（不怕被宰）
├── 精打细算：所有交易手续费-20%，讨价还价下限+5%
└── 货比三家：城内物价信息自动获取（不需要亲自跑市场）

Lv2（进阶 — 需要经营≥25）
├── 商路开拓：可发现隐藏商路，商队效率+20%
├── 投资达人：商铺投资回报率+25%，投资回本周期-30%
└── 库存管理：商队携带货物上限+50%，货物损耗-30%

Lv3（专家 — 需要经营≥45）
├── 垄断大师：同一商品买入>100件时，该城该商品售价+15%/每100件
├── 钱生钱：闲置资金(>10000金)在钱庄自动生息(月息0.5%)
├── 砍价大师：还价上限+15%（总额可达55%）
└── 实业家：封地内所有生产建筑产出+20%

Lv4（大师 — 需要经营≥65）
├── 富可敌国：个人资产>10万金时→在任意城池获得"上宾"待遇
├── 商业帝国：名下商铺≥10间时→所有商铺收入+30%，连锁经营效果
└── 危机商机：战争/灾荒时自动获得"紧急物资供应"任务（超高利润）

Lv5（宗师 — 需要经营≥80，消耗2技能点）
└── 陶朱之富：可向国家提供贷款(利率自定)，贸易路线收益+100%，经济独立于国家
```

## 技能树六：工艺系（侧重工艺）

> 提升锻造/建造/发明。适合走匠道路线的玩家。

```
Lv1（入门 — 无门槛）
├── 基础锻造：可锻造绿色品质装备，材料消耗-10%
├── 巧手：建造速度+20%，修理效率+30%
└── 慧眼识材：采集资源时有15%概率获得额外材料

Lv2（进阶 — 需要工艺≥25）
├── 精锻技术：可锻造蓝色品质装备，绿色品质暴击率+15%
├── 攻城器专家：建造攻城器械速度+30%，器械耐久+20%
└── 机关术：解锁基础机关陷阱（捕兽夹/绊索/落坑）

Lv3（专家 — 需要工艺≥45）
├── 百炼成钢：可锻造紫色品质装备，蓝色品质暴击率+20%
├── 名剑仿制：可仿造十大名剑的"仿品"（紫品质，威力80%）
├── 建筑大师：封地建造时间-30%，建筑维护费-25%
└── 火器初探：解锁火药制作（硫磺+硝石+木炭）

Lv4（大师 — 需要工艺≥65）
├── 神兵锻造：可锻造金色品质装备（非名剑），紫色暴击率+25%
├── 移动工坊：可在野外搭建临时工坊（修理/制造基础装备）
└── 攻城革命：解锁配重投石机（射程+40%）和铁火炮（范围爆炸）

Lv5（宗师 — 需要工艺≥80，消耗2技能点）
└── 鬼斧神工：可重铸损坏的名剑，制造专属神兵（命名权），所有锻造品质+1级
```

## 技能树总览

```
等级上限：60级 = 60技能点

全修方案（均衡型）：
  每条树点到Lv3 = 3×6 = 18点（剩42点）
  均衡发展，没有短板但也没有极致优势

深度专精方案（极致型）：
  一条树点到Lv5 = 8点（Lv5消耗2点）
  剩余52点分配到其他5条树
  在某一领域达到巅峰

双修通才方案（推荐）：
  两条树点到Lv5 = 16点
  两条树点到Lv3 = 12点
  剩余32点自由分配
  在两个领域达宗师，两个领域达专家

技能重置：
  可在学宫/宗师处重置技能点
  重置费用 = 已使用点数 × 500金
  冷却时间：30天（不能频繁洗点）
```

## 特殊：隐藏技能节点

```
满足特殊条件后解锁的隐藏节点（消耗2技能点）：

1. 万人敌(战斗)  + 千军万马(统帅) = 解锁隐藏"霸王"
   效果：可同时亲自冲锋+全军指挥，无切换惩罚

2. 神机妙算(智谋) + 天下归心(魅力) = 解锁隐藏"圣贤"
   效果：计谋和外交可结合使用（一边谈判一边使用离间计）

3. 陶朱之富(经营) + 鬼斧神工(工艺) = 解锁隐藏"国师"
   效果：经济产出和工艺品质互相加成（+20%）

4. 万人敌(战斗) + 鬼斧神工(工艺) = 解锁隐藏"剑圣"
   效果：个人战斗力受工艺属性加成（工艺/2=额外武勇伤害修正）
```

## 与其他系统的交互

| 关联系统 | 交互方式 | 影响 |
|---------|---------|------|
| 属性系统 | 技能树有属性门槛 | 属性不达标无法点高级技能 |
| 战斗系统 | 战斗系/统帅系直接增强战力 | 技能=战斗力倍率 |
| 经济系统 | 经营系增强贸易/投资 | 技能=经济效率 |
| 声望系统 | 魅力系增强外交/招募 | 技能=社交能力 |
| 工艺锻造 | 工艺系解锁高级锻造 | 技能=制造能力 |
