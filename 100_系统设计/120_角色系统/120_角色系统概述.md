# 角色系统概述

## 设计目标

玩家可以从6种出身中选择，在战国乱世中自由发展。角色系统需要支持：
- 开局差异感（贵族开局 ≠ 农民开局，不是换皮）
- 长期成长感（从底层到巅峰的跨越需要真实的时间和努力）
- 多路线兼容（一个存档可体验多条人生路线）

## 六维属性

```
上限：100（基础）+ 20（物品/事件加成）= 120

成长方式：
  统率：每指挥5场战斗→概率+1 (概率=100-当前值)
  武勇：每单挑胜利10次/亲手杀敌200→概率+1
  智略：每成功使用计谋5次/完成1部著作→概率+1
  魅力：每成功外交5次/民心高→概率+1
  经营：每赚取10000金净利润→概率+1
  工艺：每锻造10件精良装备/建造5座大型工程→概率+1

年龄修正：
  15-30岁：全属性正常成长
  30-50岁：武勇成长-50%，智略+经营+工艺成长+50%
  50-60岁：武勇开始下降(-1/5年)，统率+智略加速
  60岁以上：武勇持续下降，经验→智略
```

## 六条技能树

每条技能线5层，共30个可选节点。每升一级获得1技能点（等级上限60=60技能点）。

### 战斗系（侧重武勇）
```
Lv1: 近战精通 | 远程精通 | 生存专家
Lv2: 骑术专精 | 双持战斗 | 重甲适应
Lv3: 致命一击 | 闪避大师 | 破甲专精
Lv4: 武器大师 | 不屈意志 | 旋风连斩
Lv5: 万人之敌 —— 单挑时伤害+50%，斩杀敌将后敌全军士气-30%
```

### 统帅系（侧重统率）
```
Lv1: 练兵 | 阵型基础 | 后勤管理
Lv2: 骑兵指挥 | 弓兵指挥 | 步兵指挥
```

## 年龄与继承

- 角色从15-25岁开始（根据出身不同）
- 基础寿命60-75岁（随机）
- 武勇≥80→寿命+5，多次重伤→寿命-5~15
- 角色死亡→由子嗣继承（需有子嗣），否则游戏结束
- 婚姻系统：可联姻各国宗室/贵族 → 政治影响 + 子嗣
