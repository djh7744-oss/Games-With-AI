# NPC关系系统

## 设计目标

> 79位历史将领，数百个NPC，每个人都有对你的看法。关系不是简单的"好感度条"——相性、利益、恐惧、尊敬，多种因素交织形成复杂的人际网络。一场背叛可能源于三年前的一次失信。

## 系统概述

每个NPC与玩家有独立的关系值（-100到100），受好感度、忠诚度、相性匹配度、利益一致性等多因素影响。玩家可通过赏赐/交流/共同作战/联姻等方式培养关系，也可能因忽略/失信/利益冲突导致关系破裂。关系影响招募、叛变、外交、交易等几乎一切NPC交互。

## 核心机制

### 3.1 NPC关系维度

#### 好感度（Affection）

```
范围：-100（极度厌恶）到 100（生死之交）

基础好感 = 魅力/5 + 声望修正 + 出身修正

声望修正：
  仁义高→正派NPC+好感，反派NPC-好感
  武名高→武将+好感，文人-好感
  信义高→几乎所有NPC+好感

好感变化：
  + 赏赐(金/物品)：+1~10/次（递减）
  + 共同战斗胜利：+3~8/次
  + 帮助NPC完成任务：+5~15
  + 善待其亲友：+3~10
  + 称赞/认同其观点：+1~3
  - 忽略/冷落：-1~3/月（关系需要维护）
  - 背叛/欺骗：-20~-50
  - 杀其亲友：-60~-100
  - 频繁拒绝请求：-5~10/次
```

#### 忠诚度（Loyalty）

```
范围：0-100（仅对已招募/已归顺的NPC）

基础忠诚 = 50 + 魅力/5 + 信义/5 + 对待部下的方式

忠诚影响：
  忠诚≥90：永不叛变，被俘也不投降
  忠诚70-89：稳定，但在极端条件下可能动摇
  忠诚50-69：可被策反（需要敌方高魅力/高条件）
  忠诚30-49：主动叛变概率5%/月
  忠诚<30：几乎必定叛变

忠诚变化：
  + 赏赐(每500金+1，递减) 
  + 封官许愿(+5~15)
  + 满足其野心(+10~20)
  + 长期共同作战(+1~5/年)
  + 救命之恩(+30一次性)
  - 长期不赏赐/不升迁：-1~3/月
  - 被其他势力策反：-10~30
  - 其亲友被处决/流放：-15~40
  - 封地被收回：-20~40
  - 君主失信(承诺未兑现)：-10~25
```

#### 相性（Compatibility）

```
每个NPC有隐藏的"相性值"（1-100），类似三国志的相性系统。

相性差 = |玩家相性 - NPC相性|

相性差影响：
  0-15：相性极合 → 好感获取×1.5，忠诚稳定
  16-30：相性较合 → 好感获取×1.2
  31-50：一般 → 正常
  51-70：相性不合 → 好感获取×0.7，容易产生矛盾
  71-100：水火不容 → 好感获取×0.4，即使高忠诚也可能突然背叛

相性不可改变（代表价值观/性格的根本差异）
```

### 3.2 NPC性格类型

| 性格 | 相性倾向 | 关注点 | 容易提升关系的方式 | 容易降关系的行为 |
|------|---------|--------|-------------------|----------------|
| **忠义型** | 信义高的君主 | 信义+仁义 | 守信/善待部下 | 失信/残暴 |
| **功利型** | 能给他利益的人 | 金钱+爵位 | 赏赐/升迁 | 忽略/不赏 |
| **野心型** | 不限制他发展的人 | 权力+独立性 | 封地/自主权 | 压制/架空 |
| **名誉型** | 武名高的君主 | 武名+荣誉 | 共同作战/赞赏 | 打败仗/羞辱 |
| **享乐型** | 让他舒服的人 | 金钱+美女 | 赏赐/赐婚 | 艰苦环境 |
| **淡泊型** | 不打扰他的人 | 自由+安静 | 偶尔问候 | 频繁打扰/强求 |
| **复仇型** | 有共同敌人的人 | 敌人的敌人 | 帮他复仇 | 与仇人和解 |

### 3.3 关系网络

```
NPC之间也有独立的关系网络：

朋友：互相增好感，一方受辱另一方关系-，可互相推荐
敌人：互相减好感，招募一方→另一方忠诚-
师徒：传承关系，师父→徒弟属性成长加速
父子/兄弟：家族关系，最牢固的联盟
情敌/仇人：极难调和，必须二选一

玩家影响NPC间关系：
  同时招募互为朋友的NPC → 双方忠诚各+10
  同时招募互为敌人的NPC → 双方忠诚各-20，可能引发内斗
  在A面前称赞B → A对B好感变化
```

### 3.4 招募系统

#### 招募渠道

```
1. 在野招募（NPC当前无归属）
   条件：好感≥60 + 满足其加入条件
   方式：亲自拜访→展示诚意→邀请加入
   拒绝后冷却：30天

2. 登用（三国志式）
   派遣魅力高的部下去招募目标
   成功率 = 部下魅力/2 + 与目标的好感/5
   可同时派多人提高概率

3. 劝降（敌方将领）
   条件：敌军士气低/我方包围/目标忠诚<50
   方式：遣使劝降/亲自劝降
   成功率 = 魅力/3 + 智略/4 + 信义/5 + 各种条件加成

4. 俘虏招降
   战后俘虏→选择"招募"
   成功率 = 魅力/3 + 仁义/4 + 俘虏忠诚度-（越低越容易）
   仁义高→降将忠诚更高

5. 联姻招募
   娶武将之女→该武将忠诚+30，可招募其族人

6. 推荐招募
   NPC朋友→推荐其他在野NPC→跳过好感门槛

7. 慕名而来
   高声望→在野NPC主动投奔
   触发条件：武名>85或仁义>85或信义>85
```

#### 招募后管理

```
初次加入：忠诚度初始化（基于招募方式+相性+声望）

入职管理：
  ├── 安排职位（封官→忠诚+）
  ├── 安排封地（给封地→忠诚++，收回→忠诚---）
  ├── 赏赐（金/宝物/美女）
  ├── 联姻（最牢固的绑定）
  ├── 派系平衡（避免一派独大）
  └── 冷板凳（不给任务→忠诚每月-1~2）

辞退/解雇：
  主动辞退→该NPC好感-30，其朋友好感-15
  流放→处决般的恨意，好感-80
  处决→其亲友永久仇恨(-100)
```

### 3.5 叛变与策反

#### 叛变条件

```
NPC叛变判定（每月检测一次，忠诚<50时）：

叛变概率 = (50 - 忠诚度) × 0.5% + 敌方策反加成

叛变类型：
  ├── 投敌：带着部队/封地投靠敌方
  ├── 自立：宣布独立(需要自己有封地+部队)
  ├── 叛逃：抛弃一切自己跑了(最轻微)
  └── 刺杀：野心型极端→刺杀君主后夺权(极罕见)

叛变前兆（给玩家预警）：
  ├── 拒绝指令(开始不听指挥)
  ├── 私下接触敌方使者(间谍报告)
  ├── 忠诚度持续下降趋势(面板可见)
  └── 对君主表示不满(对话中)

阻止叛变：
  赏赐/升迁/满足要求 → 临时提高忠诚
  将家属移入都城(人质) → 叛变概率-50%
  先发制人 → 提前免职/流放/处决(但可能引起其他人不满)
```

#### 策反敌方将领

```
策反操作：
  派遣间谍接触目标（需要间谍网络）
  条件：目标忠诚<70 + 己方魅力>目标魅力
  策反成功率 = 智略/4 + 魅力/4 + 信义声望/5 - 目标忠诚/2

策反投入：
  基础资金：1000金
  承诺：封官许愿/联姻/封地（兑现与否→影响信义）

策反后果：
  成功→敌方将领投奔，敌国君主震怒→关系恶化
  失败→目标忠诚+5(更坚定了)，己方间谍暴露
```

### 3.6 师徒与传承

```
师徒系统：

拜师条件：
  ├── NPC愿意收徒（好感≥70，空闲时间）
  ├── 学生在对应属性上<师父
  └── 同领域（武勇→武勇/智略→智略）

师徒效果：
  学生：在师父所在领域的属性成长速度+50%
  师父：声望+，学生出师后有成就感
  师徒关系：永久

传承系统（角色死亡时）：
  ├── 技能传承：子嗣继承人物50%已解锁的技能节点
  ├── 属性传承：子嗣获得已故角色的属性加成10%
  ├── 装备传承：名剑/宝物传给继承人
  └── 故事传承：已故角色的声望50%保留为"家族声望"
```

## NPC交互界面

```
NPC详情面板：
  ├── NPC头像+名字+身份
  ├── 属性：五维属性（统/武/智/魅/经/工）
  ├── 关系值：
  │   ├── 好感度条(-100~100)
  │   ├── 忠诚度条(0-100)（仅部下）
  │   └── 相性显示(合/中/不合)
  ├── 性格标签
  ├── 当前状态(在城/在军/在封地/失踪/被俘)
  ├── 关系网络(朋友/敌人/家族)
  ├── 需求/诉求(当前最想要什么)
  └── 互动按钮：赏赐/交谈/赠礼/任务/联姻/辞退

指挥官关系总览：
  ├── 所有部下一览表(按忠诚排序)
  ├── 忠诚度变化趋势(上升/下降标记)
  ├── 派系图(谁跟谁好/谁跟谁不好)
  └── 叛变风险评估(红色标记高危人物)
```

## 与其他系统的交互

| 关联系统 | 交互方式 | 影响 |
|---------|---------|------|
| 声望系统 | 声望影响NPC初始好感 | 声望→第一印象 |
| 婚姻系统 | NPC可以结婚/联姻 | 婚姻→关系绑定 |
| 战斗系统 | 共同战斗增加好感/忠诚 | 战友情→关系 |
| 外交系统 | NPC关系影响外交选项 | 私人关系→国家关系 |
| 继承系统 | 继承人继承部分NPC关系 | 父辈关系→子女关系 |
