# 属性与成长系统

## 设计目标

> 属性不是静态的数字——每一次战斗、每一次外交、每一次交易都在塑造你的能力。成长是"用进废退"的，有年龄的印记，有天赋的上限，也有突破极限的可能。

## 系统概述

角色拥有六维属性：统率/武勇/智略/魅力/经营/工艺。属性值范围0-120（100基础+20物品/事件加成）。属性通过"使用即成长"的方式提升，成长速度随属性值升高而递减（越接近上限越难提升）。年龄影响属性的成长速率和上限。

## 核心机制

### 3.1 六维属性详解

#### 统率（Command / CMD）

```
统率决定：
  ├── 带兵上限：基础50人 + 统率×5人/每编队
  ├── 阵型指挥能力：统率越高，变阵越快（-0.5%/点）
  ├── 指令响应速度：部队响应延迟减少
  ├── 军团士气基础值：统率/5
  └── 委任副将的AI质量

统率成长方式：
  每指挥5场战斗(自己为主要指挥官) → 概率+1
   概率 = 100 - 当前统率值
   例：统率40时→60%概率+1；统率80时→20%概率+1
  
  每指挥一场大规模战役(>500人) → 概率+1（额外判定）
  阅读兵书(竹简) → 直接+经验（统率经验独立计算）
  观摩名将指挥 → 概率+1（白起/廉颇/孙膑等）
```

#### 武勇（Might / MGT）

```
武勇决定：
  ├── 个人战斗伤害：每点+1.5%
  ├── 体力上限：100 + 武勇/5
  ├── 暴击率：武勇/200
  ├── 单挑时各项判定优势
  └── 可使用重型武器的门槛

武勇成长方式：
  每单挑胜利10次 → 概率+1（概率=100-当前武勇）
  每亲手击杀200名敌人 → 概率+1
  每击败一名武勇高于自己的对手 → 高概率+1（概率+30%）
  每领悟一项新战斗技能 → +1（确定）
  长期不战斗 → 武勇缓慢下降（-1/年，武勇≥80时触发）
```

#### 智略（Intellect / INT）

```
智略决定：
  ├── 计谋成功率：基础+智略/3%
  ├── 科技研发速度：+智略/2%
  ├── 辩论/献策成功率
  ├── 陷阱/伏击发现率：智略/2%
  └── 读书学习效率

智略成长方式：
  每成功使用计谋5次 → 概率+1
  每完成一部著作（写书/论著） → 概率+1
  每与智略高于自己的NPC辩论 → 概率+1（无论输赢都涨）
  每阅读10卷竹简（不同内容） → 概率+1
  每参与一次成功的变法/改革 → +1（确定）
```

#### 魅力（Charisma / CHA）

```
魅力决定：
  ├── 外交成功率：+魅力/3%
  ├── NPC初始好感：+魅力/5
  ├── 招募成功率：+魅力/4%
  ├── 讨价还价中的友善加成
  ├── 部下忠诚保持：魅力/10（月衰减减少）
  └── 异性NPC好感加成

魅力成长方式：
  每成功外交5次 → 概率+1
  每招募一名在野武将 → 概率+1
  每成功调解一次冲突 → 概率+1
  每参与一次大型社交活动(盟会/庆典) → 概率+1
  每获得一次民众爱戴 → 概率+1
  衣着打扮（装备加成，非永久）→ 临时提升
```

#### 经营（Economy / ECON）

```
经营决定：
  ├── 贸易利润加成：+经营/2%
  ├── 讨价还价幅度：+经营/2%
  ├── 封地税收效率：+经营/3%
  ├── 投资回报率：+经营/4%
  ├── 商队效率：经营/5%（减少运输损耗）
  └── 可管理的商铺/工坊数量上限

经营成长方式：
  每赚取10000金净利润 → 概率+1
  每成功达成一笔超额利润交易(>100%利润) → 概率+1
  每开设一家新商铺/工坊 → 概率+1
  每垄断一种商品 → +1（确定）
  每经营封地一年(净收入为正) → 概率+1
  赔钱 → 也可能学到教训（净亏损5000金→概率+1）
```

#### 工艺（Craft / CRAFT）

```
工艺决定：
  ├── 锻造装备品质上限
  ├── 建造速度：+工艺/2%
  ├── 修理效率与消耗
  ├── 攻城器械建造速度
  ├── 发明新配方的概率
  └── 可使用的最高级工具/建筑

工艺成长方式：
  每锻造10件精良品质(绿+)装备 → 概率+1
  每建造5座大型工程(建筑/攻城器械) → 概率+1
  每发明一项新配方/技术 → +1（确定）
  每成功修理一件史诗(紫+)装备 → 概率+1
  每向名师学习一次 → 概率+1
```

### 3.2 成长公式

#### 核心成长概率

```
属性提升概率 P = (100 - 当前属性值) / 100

简化：
  属性0-20：成长概率 80-100% → 几乎是确定成长
  属性20-40：成长概率 60-80% → 快速成长
  属性40-60：成长概率 40-60% → 稳定成长
  属性60-80：成长概率 20-40% → 缓慢成长
  属性80-95：成长概率 5-20% → 极慢成长
  属性95-100：成长概率 0-5% → 需要特殊事件才能突破

成长触发间隔：
  同一属性成长后有冷却时间（防止连续爆发）
  冷却时间 = 触发该成长所需的行为次数 × 0.5
  例：战斗5次→统率+1→未来2-3场战斗不会触发统率成长
```

#### 出身加成对成长的影响

```
农民出身：所有属性成长概率 +20%（修正后）
  例：属性50时→基础概率50%→农民修正→60%

工匠出身：工艺成长概率额外 +30%
商人出身：经营成长概率额外 +30%
游侠出身：武勇成长概率额外 +30%
士人出身：智略成长概率额外 +30%
贵族出身：魅力成长概率额外 +30%
```

### 3.4 属性突破（超越100）

```
基础属性上限：100
突破方式：

物品加成（最多+15）：
  名剑/玉佩/称号/坐骑 → 临时加成
  例：湛卢剑 → 武勇+8（突破上限）
      孙子兵法 → 统率+5
      玉璧佩饰 → 魅力+3

事件加成（最多+5，永久）：
  完成史诗任务 → 某属性上限永久+2
  名将传授 → 统率/武勇上限+2
  学派宗师 → 智略上限+2
  发明革命性技术 → 工艺上限+2

最终上限：100 + 20 = 120
```

### 3.5 属性训练

#### 主动训练（花费时间+金钱）

```
训练方式（在安全区主动修炼）：

演武场训练（提升武勇）：
  花费：100金/次
  时间：1天/次
  效果：武勇经验+1-3（积累10经验→武勇+1）
  NPC教官可加速训练

学宫读书（提升智略）：
  花费：竹简购买费+学费
  时间：3-7天/卷
  效果：智略经验+2-5/卷

社交练习（提升魅力）：
  参加宴会/社交活动
  花费：200金/次
  效果：魅力经验+1-2

经商模拟/实战（提升经营）：
  小额贸易实践
  效果：经营经验+1

工坊实践（提升工艺）：
  制作装备/建筑模型
  效果：工艺经验+1-3
```

### 3.6 属性与技能树关联

```
每条技能树有属性门槛：

战斗系技能Lv3 → 需要武勇≥40
战斗系技能Lv5 → 需要武勇≥70

统帅系技能Lv3 → 需要统率≥50
统帅系技能Lv5 → 需要统率≥80

智谋系技能Lv3 → 需要智略≥45
魅力系技能Lv3 → 需要魅力≥45
经营系技能Lv3 → 需要经营≥45
工艺系技能Lv3 → 需要工艺≥45

各系Lv5 → 需要对应属性≥75
```

## 与其他系统的交互

| 关联系统 | 交互方式 | 影响 |
|---------|---------|------|
| 出身系统 | 出身决定初始属性 | 起点不同 |
| 技能树 | 属性达到门槛解锁技能 | 属性→技能 |
| 战斗系统 | 武勇影响伤害，统率影响带兵 | 属性→战力 |
| 经济系统 | 经营影响贸易利润 | 属性→收益 |
| 年龄系统 | 年龄修正属性成长 | 时间→属性变化 |
