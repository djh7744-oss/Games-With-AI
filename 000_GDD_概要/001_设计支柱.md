# 设计支柱 —— 所有设计决策的北极星

## 五大支柱

```
        自由人生
          ▲
         / \
    历史感 ← → 策略深度
         \ /
          ▼
       战斗手感
          |
       沙盒世界 (基石)
```

## 支柱详解

### 支柱一：自由人生

> "王侯将相，宁有种乎？"

**要求**：
- 6种出身对应6种不同的开局体验，不是换皮
- 每条人生路线都有专属任务链、专属NPC、专属结局
- 路线之间可交叉（农民→士兵→将军→封君→经营商业）
- 不存在"最优路线"，每条路都有独特优势

**红线**：
- 不能让游侠出身的人必须先去从军才能推进主线
- 不能让商人出身的人无法获得社会尊重
- 不能有"堵死"的情况——任何时候都有至少2条可选路径

### 支柱二：沙盒世界

> "世界不围着玩家转。"

**要求**：
- 7国AI自主：征兵/交战/结盟/灭亡，不依赖玩家触发
- 经济自主：物价根据供需自然波动
- NPC自主：将领可叛变、平民可迁移、商队可被劫
- 玩家不作为时，世界依然有趣——甚至更乱

**红线**：
- 不能有"等待玩家触发"的世界事件
- 不能让AI国家永远保持初始边界
- 玩家不能同时是所有国家的盟友

### 支柱三：历史感

> "玩家认得这是战国，学者也认得。"

**要求**：
- 地名、人名、事件基于史实
- 武器/盔甲/建筑参考考古发现
- 社会制度（军功爵制/郡县制/井田制等）真实还原
- 但允许玩家改变历史走向（架空路线）

**红线**：
- 不能出现三国时期或明清时期的元素
- 不能为了"好看"而完全脱离史实（如魔幻铠甲）
- 架空路线需要有逻辑推导（"如果赵括赢得了长平之战会怎样"）

### 支柱四：策略深度

> "高手和新手的决策结果截然不同。"

**要求**：
- 内政/外交/经济/军事 四个维度不是各自独立而是互相影响
- 每个决策都有 trade-off（短期利益 vs 长期代价）
- AI 对手会根据难度等级展示不同层次的策略
- 信息不完全对称（有战争迷雾、有间谍、有欺骗）

**红线**：
- 不能有"永远正确的选择"
- 不能让战斗能力完全碾压策略选择

### 支柱五：战斗手感

> "砍到肉和砍到空气，必须有质的区别。"

**要求**：
- 对标骑马与砍杀2的打击感
- 四方向攻击方向由鼠标控制，不是按键
- 骑兵冲锋有速度加成，重甲有厚重感
- 千人战场不掉帧（性能硬指标）

**红线**：
- 不能让战斗变成"数值对撞"
- 玩家个人操作水平必须显著影响战斗结果

---

## 设计决策优先级

当设计产生冲突时，按以下优先级裁决：

```
1. 自由人生 > 历史感
   → 如果历史事实限制了玩家选择的自由度，优先自由度
   → 例：历史上女性不能从军，游戏里可以（用难度和挑战平衡）

2. 战斗手感 > 策略深度
   → 如果复杂的策略系统牺牲了战斗的即时反馈，简化策略
   → 例：阵型系统10种阵型，但战斗中玩家只需选择3种常用

3. 沙盒世界 > 所有
   → 如果某个功能需要"世界为玩家暂停"，重新设计
   → 例：玩家在战役中时，外部世界时间仍然流逝
```
