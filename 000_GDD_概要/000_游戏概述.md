# 战国 · 乱世风云 — 游戏概述

## 一句话描述

> 在战国乱世中，以任意出身开启人生，用你的方式改变历史。

## 游戏类型

- **主类型**：开放世界沙盒 RPG
- **副类型**：历史策略 + 军团战斗 + 角色养成
- **平台**：PC (Steam/Epic/WeGame)，后续扩展 PS5/Xbox
- **引擎**：Unreal Engine 5.5+
- **视角**：第三人称（战斗）+ 鸟瞰（战略大地图）
- **人数**：单人

## 核心体验

```
玩家在战国七雄并立的开放世界中：

1. 选择出身（贵族/士人/商人/游侠/工匠/农民）
2. 加入任意国家或自立门户
3. 从底层成长为将军、巨富、刺客宗师、学派领袖……
4. 自由探索中国核心区域（关中→中原→江南→巴蜀→燕北）
5. 指挥千人军团（骑马与砍杀式战斗）
6. 参与合纵连横的外交博弈（三国志式策略）
7. 没有固定结局 —— 你的人生由你定义
```

## 设计支柱

| 支柱 | 含义 | 检验标准 |
|------|------|---------|
| **自由人生** | 6种出身 × 6条路线，每条路都被尊重 | 每种出身是否有独特的专属事件链？ |
| **沙盒世界** | 世界不围着玩家转，国家自主发展/交战/灭亡 | 玩家不作为时，世界是否依然有趣？ |
| **历史感** | 真实地名、人物、事件，但允许架空 | 历史学者能否认出这个时代的灵魂？ |
| **策略深度** | 内政/外交/经济/军事多层次决策 | 高手和普通玩家的选择是否有显著差异？ |
| **战斗手感** | 对标骑马与砍杀的打击感 | 砍到人和砍空气的区别是否足够清晰？ |

## 游戏世界

### 时间跨度
战国时代（公元前475年-前221年），254年历史。玩家角色自然寿命60-80年，通过子嗣继承系统可跨越整个战国时代。

### 地理范围
九大区域：关中(秦核心)/中原(魏韩周)/河北(赵)/山东(齐)/荆楚(楚核心)/巴蜀/山西/燕北/吴越。16km×14km无缝战略大地图，40+主城，80+县/关隘。

### 七国势力
| 国 | 都城 | 军事评级 | 经济评级 | 特色 |
|----|------|---------|---------|------|
| 秦 | 咸阳 | S | B | 军功爵制/标准化/远交近攻 |
| 楚 | 郢都 | A | B | 水军最强/地广人稀/漆器丝绸 |
| 齐 | 临淄 | C | S | 稷下学宫/海盐垄断/商业立国 |
| 赵 | 邯郸 | A | B | 胡服骑射/骑兵最强/悲壮坚韧 |

## 玩家旅程

### 六种出身 × 六条路线 = 无限人生

**六种出身** (详见 [121_出身系统](../100_系统设计/120_角色系统/121_出身系统.md)):
贵族/士人/商人/游侠/工匠/农民。每种出身有独特的初始属性、起始位置、初始人脉、社会待遇和专属任务链。出身不锁定路线，但影响起点难度。

**六条人生路线** (详见 [310_主线剧情](../300_叙事设计/310_主线剧情/310_六条人生路线主线任务.md)):
| 路线 | 核心属性 | 终极目标 | 对标历史 |
|------|---------|---------|---------|
| 霸道 | 统率+武勇 | 从士兵到帝王 | 白起/王翦/秦始皇 |
| 王道 | 魅力+智略 | 不战而屈人之兵 | 苏秦/蔺相如/信陵君 |
| 侠道 | 武勇 | 十步杀一人→侠之大者 | 荆轲/专诸/朱亥 |
| 商道 | 经营 | 吕不韦之路 | 吕不韦/白圭/陶朱公 |
| 匠道 | 工艺 | 鬼斧神工→改变战争 | 欧冶子/干将/郑国 |
| 隐道 | 智略 | 著书立说→百世之师 | 孔子/韩非/鬼谷子 |

**经典交叉路径**:
- 农民→士兵→将军→获封→经营封地→富国强兵
- 游侠→被赏识→门客→入仕→将军→封君
- 商人→捐官→入仕→朝政→幕后操控(吕不韦路线)

### 游戏中的一天

```
黎明: 在邯郸的酒馆醒来。楼下市集已经热闹起来。
      收到一封家书——母亲病了，需要药材。

上午: 接了一个护送商队的活儿(去临淄，顺路找药)。
      出城时看到秦军的征兵告示。

中午: 官道上遇到小股山贼。战斗。山贼头目是个逃兵。
      他说家里老母病了才落草。你给了他十两银子。

下午: 到达临淄。卖掉货物，赚了800金。
      在药铺买到母亲需要的药材。

傍晚: 在临淄酒馆听说——秦军攻韩了。宜阳被围。
      如果宜阳失守...算了，明天再说。

深夜: 给母亲写了回信。寄出。
      看着窗外的月亮。明天——又会发生什么?
```

## 设计哲学

### 玩家叙事主权
设计师搭建舞台(世界背景/NPC/历史事件)和道具(武器/金钱/领土)，然后退后一步——让玩家自己去"演出"。没有"官方结局"，你的结局就是你的结局。详见 [360_叙事系统设计](../300_叙事设计/360_叙事系统设计/360_叙事系统概述.md)

### 道德灰度
没有"好/坏"选择，只有"不同的价值观"。每个重大选择都有合理的理由和真实的代价。玩家应该纠结——这正是好叙事的标志。

### 历史与架空
基于真实历史，但允许玩家改写。架空分支需要逻辑推导——不是"魔法改变历史"，而是"在那个时代，如果做了不同选择"。
