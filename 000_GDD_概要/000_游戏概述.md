# 战国 · 乱世风云 — 游戏概述

## 一句话描述

> 在战国乱世中，以任意出身开启人生，用你的方式改变历史。

## 游戏类型

- **主类型**：开放世界沙盒 RPG
- **副类型**：历史策略 + 军团战斗 + 角色养成
- **平台**：PC (Steam/Epic/WeGame)，后续扩展 PS5/Xbox
- **引擎**：Unreal Engine 5.5+
- **视角**：第三人称（战斗）+ 鸟瞰（战略大地图）
- **人数**：单人

## 核心体验

```
玩家在战国七雄并立的开放世界中：

1. 选择出身（贵族/士人/商人/游侠/工匠/农民）
2. 加入任意国家或自立门户
3. 从底层成长为将军、巨富、刺客宗师、学派领袖……
4. 自由探索中国核心区域（关中→中原→江南→巴蜀→燕北）
5. 指挥千人军团（骑马与砍杀式战斗）
6. 参与合纵连横的外交博弈（三国志式策略）
7. 没有固定结局 —— 你的人生由你定义
```

## 设计支柱

| 支柱 | 含义 | 检验标准 |
|------|------|---------|
| **自由人生** | 6种出身 × 6条路线，每条路都被尊重 | 每种出身是否有独特的专属事件链？ |
| **沙盒世界** | 世界不围着玩家转，国家自主发展/交战/灭亡 | 玩家不作为时，世界是否依然有趣？ |
| **历史感** | 真实地名、人物、事件，但允许架空 | 历史学者能否认出这个时代的灵魂？ |
| **策略深度** | 内政/外交/经济/军事多层次决策 | 高手和普通玩家的选择是否有显著差异？ |
| **战斗手感** | 对标骑马与砍杀的打击感 | 砍到人和砍空气的区别是否足够清晰？ |

| 子系统 | 参考游戏 | 借鉴核心 |
|--------|---------|---------|
| 军团战斗 | 骑马与砍杀2 | 四方向砍杀、骑兵冲锋、士气系统 |
| 战略地图 | 三国群英传系列 | 鸟瞰大地图、军团移动、遭遇入战场 |
| 国家策略 | 三国志系列 | 内政/外交/人事/计谋 |
| 地图探索 | 英雄无敌系列 | 战争迷雾、资源点、野怪遭遇 |
| 角色成长 | 太阁立志传5 | 多职业路线、身份跃迁、技能修行 |
| 开放世界 | GTA V / 大镖客2 | 无缝世界、NPC生态、动态事件 |
| 经商贸易 | 大航海时代4 | 跨城物价差、商路安全 |
| 科技树 | 文明系列 | 多分支科技、政策卡系统 |

## 内容规模

| 类别 | 数量 |
|------|------|
| 可玩势力 | 7大主要势力 + 周 + 少数民族 |
| 历史将领 | 79位（含专属技能） |
| 城池 | 40+主城 + 80+县/关隘 |
| 兵种 | 7国各T0-T5兵种线 + 特殊兵种 |
| 诸子学派 | 10家（儒/法/墨/道/兵/纵横/名/阴阳/农/杂） |
| 科技树 | 48项（农业/军事/经济/政治/文化/工艺各8项） |
| 武器类型 | 剑/刀/戟/戈/矛/弓/弩/盾 + 十大名剑 |
| 人生路线 | 6条（霸道/王道/侠道/商道/匠道/隐道） |
| 历史战役(DLC) | 8个（长平/邯郸/伊阙/鄢郢/即墨/马陵/乐毅伐齐/秦灭六国） |
