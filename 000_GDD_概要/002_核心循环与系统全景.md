# 核心循环与系统全景

## 一、三层核心循环

### 微观循环(秒-分钟): 个人战斗

```
装备检查 ──→ 遭遇敌人 ──→ 四方向攻击/格挡 ──→ 击杀/被击退
    ↑                                              │
    └──────────── 获得经验/装备/金钱 ←──────────────┘

玩家体验: 每一次挥剑都有"重量感"
对标: 骑马与砍杀2
```

### 中观循环(小时-天): 军团与城池

```
招募/训练军队 ──→ 战略移动 ──→ 遭遇战/攻城 ──→ 占领/掠夺
      ↑                                            │
      └────── 获得领土/资源/声望 ──────────────────┘
                    │
                    └──→ 发展城池经济 ──→ 更多税收 ──→ 更强军队

玩家体验: 从一个小军官成长为一方霸主
对标: 三国群英传 + 三国志 + 骑马与砍杀
```

### 宏观循环(月-年): 国家与天下

```
制定国策 ──→ 发展科技/经济 ──→ 外交合纵连横 ──→ 发动战争/和平吞并
    │                                                    │
    └──────── 国力消长/格局改变 ─────────────────────────┘
                            │
                            └──→ 角色老去 → 子嗣继承 → 循环延续

玩家体验: 一生的征战与经营，影响整个战国的命运
对标: 三国志 + 文明 + 十字军之王
```

## 三、系统依赖关系

```
玩家的一个"回合"(游戏日)可能涉及的系统:

上午: 
  1. [110战斗] 在邯郸城外遭遇山贼 → 战斗
  2. [180任务] 完成后获得赏金+发现一封密信 → 触发调查任务

下午:
  3. [420城池] 回邯郸市集卖出战利品 → [130经济] 物价波动
  4. [126 NPC关系] 在酒馆遇到廉颇 → 对话 → 好感变化

傍晚:
  5. [151内政] 邯郸太守请求你协助赈灾 → [153政策] 选择赈灾方式
  6. [164动态世界] 收到消息: 秦军攻打韩国宜阳 → 国际格局变化

深夜:
  7. [180任务] 决定是否前往韩国参战 → [140外交] 影响各国关系
  8. [存档] 一天结束 → 世界时间推进
```

## 四、八年合计

| 维度 | 内容 | 时间跨度 |
|------|------|---------|
| 战斗 | 数百场大小战斗，从山贼到灭国之战 | 即时 |
| 角色 | 从无名之辈到帝王/圣贤/巨贾/宗师 | 30-60年(游戏内) |
| 策略 | 七国博弈，合纵连横，变法图强 | 数十年 |
| 经营 | 从一间商铺到跨国商网 | 10-20年 |
| 感情 | 结识NPC，建立友谊/爱情/仇恨 | 一生 |
| 传承 | 子嗣继承，家族延续 | 跨越世代 |
