# 战国时代开放世界游戏 · Unreal Engine 5 实现路线图

> 基于《战国游戏完整设计文档》(17篇/142章节)
>
> **游戏本体**：战国沙盒 —— 扮演任意身份在战国乱世中自由生存、发展、崛起
> **DLC扩展**：历史战役剧本 —— 长平之战、邯郸保卫战、伊阙之战等
>
> 技术栈：**Unreal Engine 5.5+** (引擎) + **Blender 4.x** (建模/贴图) + **C++/Blueprint** (编程)

# Phase 0：环境搭建与工具链（2-4周）

## 0.1 软件安装与配置

### 引擎
- **Unreal Engine 5.5+** — 从 Epic Games Launcher 安装
  - 勾选：C++ 开发工具、Windows调试工具
  - 推荐配置：64GB RAM + RTX 4070+ + SSD 1TB+

### 3D建模
- **Blender 4.x LTS** — blender.org 下载
  - 必要插件：TexTools、Node Wrangler、Rigify、Auto-Rig Pro

### 版本控制
- **Git + Git LFS**（大文件管理）
- **Perforce**（团队协作推荐，UE5原生支持更好）

### 其他工具
- **Substance Painter** — PBR贴图绘制
- **Quixel Megascans** — 免费照片级扫描资产（UE5内置）
- **World Machine / Gaea** — 程序化地形生成
- **Wwise / FMOD** — 音频中间件

> 核心参考：**三国群英传** 的大地图 + **英雄无敌** 的探索  
> 对应设计文档 第二篇：地图与地形

## 3.1 地图设计理念 —— 双层级地图

```
┌──────────────────────────────────────────────────────────────────┐
│                                                                  │
│  【战略地图层】(三国群英传风格)                                   │
│  ├── 鸟瞰视角（可旋转/缩放）                                     │
│  ├── 展现整个战国中国版图                                        │
│  ├── 城池/关隘/渡口/资源点清晰可见                                │
│  ├── 玩家军团以棋子/模型在地图上移动                              │
│  ├── 敌方军团/商队/流寇也在地图上活动                            │
│  ├── 时间流逝：昼夜/季节影响可见                                  │
│  └── 遭遇敌军→进入战场（实例关卡）                                │
│                                                                  │
│  【探索层】(英雄无敌风格)                                         │
│  ├── 战争迷雾：未探索区域被雾覆盖                                 │
│  ├── 接触式探索：军团移动到新区域→迷雾揭开                         │
│  ├── 地图互动点：                                                 │
│  │   ├── 资源堆：木材/石材/金矿(占领后定期产出)                    │
│  │   ├── 野怪营地：流寇/山贼/戎狄(击败获得战利品)                  │
│  │   ├── 遗迹/古墓：探索可能获得宝物/名剑/竹简                     │
│  │   ├── 隐士居所：拜访诸子/隐士→学习技能/获得任务                │
│  │   ├── 驿站/客栈：恢复体力/招募同伴/打听情报                    │
│  │   └── 马场/铁匠铺/药铺：特殊功能建筑                           │
│  └── 英雄无敌式宝屋：山寨/地宫/古战场(独立探索关卡)                │
│                                                                  │
│  【城市层】(GTA V式)                                              │
│  ├── 进入城池→切换到城内自由探索                                  │
│  ├── 可在街道行走，与NPC对话                                      │
│  ├── 访问：酒馆/商铺/官府/兵营/学宫/市集/铁匠铺                   │
│  └── 城市生态：昼夜循环 + NPC作息                                 │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
```

## 3.2 地图规模

| 方案 | 尺寸 | 城池密度 | 适合 |
|------|------|---------|------|
| **A: 完整版** | 64km×64km | 120+城池 | AAA大团队 |
| **B: 精简版** (推荐) | 16km×14km | 40+主城+60+据点 | 独立团队 |
| **C: 最小版** (MVP) | 8km×8km | 15主城+20据点 | 原型验证 |

> 建议：先用方案C做Phase 1-3原型，方案B做本体正式版，后续扩展至方案A


## 3.4 群英传式战略地图交互

```
【大地图上的可见要素】
├── 城池图标（3D模型缩略 + 名字 + 所属国旗）
├── 关隘/渡口（特殊通行节点）
├── 军队模型（按兵种混合显示，数量感）
├── 商队/运输队（NPC经济单位）
├── 流寇/野怪（随机刷新，英雄无敌式）
├── 季节视觉效果（雪/雨/落叶/花）
└── 战争迷雾（未探索区灰色覆盖）

【大地图操作】
├── 点击空地→移动到该处（群英传式点击移动）
├── 点击城池→进入城池（或围攻）
├── 点击敌军→追击/战斗/回避
├── 点击资源点→占领/拾取
├── 右键拖拽→旋转视角
├── 滚轮→缩放（从近景~全国视野）
└── 快捷键切换：战略地图 / 当前任务追踪 / 国家关系图
```

## 3.5 英雄无敌式探索机制

```
├── 战争迷雾系统：
│   ├── 初始：仅本国产出发地周围可见
│   ├── 扩展方式：军团移动/派遣斥候/间谍/盟友共享
│   └── 已探索区域：地形可见，但单位需重新"侦察"才更新
│
├── 英雄无敌式地图物件（DataTable定义）：
│   ├── 资源矿场：石矿/铁矿/金矿/伐木场/马场/盐田——占领后每回合产出
│   ├── 野怪营地：难度1-5星，击败获得经验+战利品+俘虏
│   ├── 宝箱/遗物：随机宝物/金币/名剑碎片
│   ├── 智慧石/古碑：永久属性+1/解锁科技/获得技能
│   ├── 传送门(驿站)：快速移动到远方已发现驿站
│   ├── 隐士小屋：学习技能/接受特殊任务
│   └── 龙潭虎穴(山寨/古墓)：独立战斗副本，击败Boss获得神器
│
└── 探索UI：小地图雷达 + 发现物提示(英雄无敌式弹窗)
```

# Phase 4：军团战斗系统（3-6个月）

> 核心参考：**骑马与砍杀** 的战场指挥 + 个人战斗  
> 对应设计文档 第六篇(兵种)/第十一篇(阵法)

## 4.1 M&B式个人战斗深化

```
在Phase 1基础上扩展：
├── 完整武器类型：剑/刀/戟/戈/矛/枪/弓/弩/盾
├── 武器切换/投掷/捡取/耐久
├── 单挑系统（M&B式阵前叫阵→双方将领1v1→士兵围观）
│   ├── 胜利：敌全体士气-30%
│   └── 失败：己方士气-30%，可能被俘
└── 十大名剑专属效果（每把独特获取任务+特效）
```

## 4.2 M&B式兵团指挥

```
【指挥系统】—— 参考M&B2的指挥UI
├── F1：移动/驻守/跟随
├── F2：阵型（方阵/圆阵/锥阵/鹤翼/鱼鳞/长蛇/雁行/偃月）
├── F3：射击模式（自由射击/齐射/停射）
├── F4：武器切换（近战/投掷/盾墙）
├── F5：冲锋/稳步前进/后撤
├── F6：委任副将（AI接管指定部队）
│
├── 部队编组：步兵/弓兵/骑兵/器械 分别编队
├── 战场小地图：实时显示各编队位置
├── 士气系统：每队独立士气值 + 全局士气
└── 溃逃：士气归零 → 部队溃散 → 可被追杀/俘虏

【战斗进入方式】—— 群英传式
├── 战略地图上双方军团相遇
├── 短暂加载进入战场实例关卡（根据地形生成战场）
├── 地形取自战略地图的当前位置（平原/山地/森林/河岸）
└── 战斗结束→返回战略地图（时间推进）
```

## 4.3 战场规模与LOD

```
├── 战斗规模设定（玩家可选）：
│   ├── 小型：每方50-100人
│   ├── 中型：每方200-500人（推荐）
│   ├── 大型：每方500-1000人
│   └── 史诗：每方1000-3000人（高端PC）
│
├── 士兵LOD策略（M&B式）：
│   ├── 近景(0-50m)：完整骨骼动画
│   ├── 中景(50-150m)：简化动画
│   ├── 远景(150-500m)：静态Mesh + VAT
│   └── 极远(500m+)：Imposter
```

## 4.4 城邑战/攻城战

```
├── 攻城器械：云梯/冲车/投石机/井阑/弩炮/攻城塔
├── 城防体系：木栅→土城→砖城→石城→都城
```

# Phase 6：国家策略系统（3-5个月）

> 核心参考：**三国志系列** (9/11/14) 的内政/外交/人事/计谋  
> 这是游戏"深度"的核心所在

## 6.1 三国志式内政系统

```
【城池运营】—— 参考三国志的城池数值管理
├── 每座城池有独立数值：
│   ├── 人口：影响税收/征兵上限/劳力
│   ├── 农业：粮食产出
│   ├── 商业：金钱收入
│   ├── 技术：科技研发速度
│   ├── 民忠：影响叛乱概率
│   ├── 城防：城墙耐久
│   └── 驻军：防守兵力
│
├── 内政指令（每回合可执行）：
│   ├── 开发农业：消耗金+劳力→农业值+
│   ├── 发展商业：消耗金→商业值+
│   ├── 征召士兵：消耗人口+金→增加驻军
│   ├── 修筑城墙：消耗金+石材→城防+
│   ├── 安抚民心：消耗金→民忠+
│   └── 搜索人才：概率发现隐藏武将/在野人士
│
└── 封地管理（玩家个人领地独立于国家）
```

## 6.2 三国志式外交系统

```
参考三国志的外交界面：

├── 外交行动：
```


## 8.2 太阁式技能修行


# 附录A：经典游戏参考深度解析

## A.1 骑马与砍杀 —— 战斗手感的黄金标准

```
M&B对本作的核心启发：
├── 四方向战斗：鼠标移动决定攻击方向（不是按键）
├── 骑砍速度加成：马速×武器伤害的倍率曲线
├── 战场指挥：F键指令系统，简单但深度极高
├── 士气连锁溃逃：一队溃→邻队动摇→全军崩
├── 俘虏/赎金/战后处置：让每场战斗都有经济意义
└── 沙盒叙事：没有强制剧情，但"玩"出了故事

需要改进的地方（M&B的弱点）：
├── M&B缺乏战略层→我们用三国群英传+三国志补充
├── M&B经济太简单→我们加入大航海式贸易
└── M&B角色成长单调→我们加入太阁式多路线
```

## A.2 三国群英传 —— 战略地图的最佳参考

```
群英传对本作的核心启发：
├── 大地图：鸟瞰视角+自由点击移动+军团碰撞即战斗
├── 城池可见模型+旗帜/兵力数
├── 搜索/内政/外交的统一界面
├── 千人战画面（虽然是简化版，但感觉宏大）
└── 武将技+兵种相克（我们的版本：阵法+地形+士气）
```

## A.3 三国志系列 —— 策略深度的标杆

```
三国志对本作的核心启发：
├── 回合制策略+即时战斗的混合
├── 人事系统：忠诚度/相性/登用/流放
├── 外交系统：同盟/婚姻/劝降/二虎竞食
├── 内政系统：城池数值管理+开发指令
├── 计谋系统：离间/流言/驱虎吞狼
└── 势力消长：AI国家会自主发展/扩张/灭亡
```

## A.4 英雄无敌系列 —— 地图探索的典范

```
英雄无敌对本作的核心启发：
├── 战争迷雾+拾取资源堆
├── 地图上的功能建筑（矿场/宝库/智慧石/隐士）
├── 野怪守卫宝藏（难度星级标注）
├── 每周资源产出（我们的版本：每月/每季）
└── 探索的快感——翻过山丘看到新区域的期待感
```

## A.5 太阁立志传5 —— 角色扮演的珠峰

```
太阁对本作的核心启发：
├── 多职业自由切换（武士/商人/忍者/海贼/剑豪/茶人...）
├── 身份跃迁的成就感（足轻→家老→城主→大名）
├── 技能修行系统（拜访师傅+实战+读书）
├── 卡片收集（称号/技能/人物/名胜 全都有收集价值）
├── 个人故事与国家历史交织
└── 极简画面+极深系统（证明：系统深度 >> 画面精度）
```
