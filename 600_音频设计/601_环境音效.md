# 环境音效设计

## 一、九大Biome环境音

```
关中(黄土高原+渭河平原):
  基调: 干燥的风声 + 远处麦浪
  细节: 窑洞口的回响/渭河水声(靠近河流)
  动物: 麻雀/喜鹊
  天气: 风干燥(中高频突出)
  昼夜: 白天鸟鸣活跃/夜晚蟋蟀

中原(黄河冲积平原):
  基调: 广阔平原的风 + 远处农耕声
  细节: 水渠流水/风过麦田的沙沙声
  动物: 燕子/布谷鸟
  人为: 远处牛铃/村庄犬吠
  天气: 风温和(中频平衡)

河北(华北平原北端):
  基调: 草原风声 + 白桦林沙沙
  细节: 草浪声/马群远处嘶鸣
  动物: 鹰啸/野兔/狼嚎(夜)
  天气: 北风凛冽(低频多)
  特色: 冬季极静(雪地吸音)

山东(丘陵+沿海):
  基调: 海浪+海鸥+渔船
  细节: 盐田风车/沙滩浪声
  人为: 渔市嘈杂(远处)/船帆鼓风声
  动物: 海鸥/燕鸥
  天气: 海风湿润

荆楚(长江流域):
  基调: 江水声+竹林风+雾气
  细节: 水滴竹叶/蛙鸣/水车
  动物: 猿啼(远处)/白鹭/蝉鸣(夏)
  天气: 雨声频繁/雾中声音柔和
  特色: 神秘幽深感(低频混响多)

巴蜀(四川盆地):
  基调: 山谷回响+溪流+鸟鸣
  细节: 梯田流水/栈道风声
  动物: 金丝猴/熊猫(极罕见触发)/杜鹃
  天气: 雾中声音闷/雨后清晰

山西(黄土高原):
  基调: 沟壑风声(呼啸) + 矿场锤击(远处)
  细节: 窑洞回音/黄土崩塌(偶尔)
  动物: 鹰/野兔/蝎子
  人为: 矿场/冶铁坊(宜阳)
  特色: 风过沟壑的呜咽声

燕北(草原+山地):
  基调: 极寒风声+草原+松林
  细节: 雪落/冰裂/狼嚎
  动物: 狼/鹿/雪狐
  特色: 冬季几乎无声(积雪吸音)/风声极强

吴越(江南水乡):
  基调: 水声+船桨+桑叶风
  细节: 小桥流水/渔歌/纺织机
  动物: 水鸟/青蛙/鱼跃
  人为: 船工号子(远处)/市集
  特色: 湿润/宁静/生活气息浓厚
```

## 二、天气音效

```
晴天: 标准环境音(各Biome默认)
  风: 微风(2-3级)

多云: 环境音略微衰减(高频-2dB)
  风: 中风(4-5级)

雨天:
  雨滴: 屋顶/地面/水面三种溅射声(RTC)
  雷: 远处闷雷(3-10秒间隔)+近处炸雷(偶尔)
  环境: 动物声消失/人声减少/雨声主导
  风: 伴随阵风

雪天:
  环境: 极静(背景噪声-10dB)
  细节: 雪花落地微声/脚下踩雪/呼出白气的呼吸声
  风: 寒风(高频切削声)

雾天:
  环境: 声音变"闷"(低通滤波, cutoff 2000Hz)
  细节: 远处声音更模糊/近处水滴声

暴风雨:
  环境: 压倒性的风雨声
  细节: 雷声密集/树木折断/水浪拍岸
```

## 三、城池环境音

```
村庄(Lv1):
  基调: 鸡鸣犬吠/远处农耕
  细节: 水井辘轳/打谷场/孩童嬉闹
  NPC: 3-5人对话(模糊)

乡镇(Lv2):
  基调: 小市集嘈杂+铁匠铺
  细节: 叫卖声/织机/牲畜
  NPC: 8-15人模糊对话

县城(Lv3):
  基调: 市集热闹+官衙肃静
  细节: 马车/士兵操练(兵营)/学堂读书声
  NPC: 20-30人模糊对话+3-5清晰NPC

郡城(Lv4):
  基调: 繁华商业+文化气息
  细节: 钱庄算盘/书肆翻书/酒馆喧闹
  NPC: 40-60人模糊对话+5-10清晰NPC

都城(Lv5):
  基调: 都城特有的宏大感+权力感
  细节: 朝钟/王宫礼乐(远处)/大军操练
       各国都城独特音:
         咸阳: 兵器工坊+四方馆各国口音
         临淄: 稷下辩论声+海盐市
         郢都: 楚乐+漆器工坊
         邯郸: 马市+丛台操演
         大梁: 运河码头+书肆
         蓟城: 寒风+马场
         新郑: 冶铁+劲弩试射
  NPC: 80-150人模糊对话+10-20清晰NPC
```

## 四、战略地图音效

```
地图移动:
  军团移动: 脚步声/马蹄声(随地形变化)
  经过城池: 城池环境音(随距离衰减)
  经过森林: 鸟鸣+树叶
  经过河流: 水声

地图交互:
  发现新地点: 短促的"发现"提示音(编钟一声)
  遭遇敌军: 战鼓渐强
  资源点占领: 确认音+资源采集声
  战争迷雾揭开: 微弱的"纸张揭开"声
```
