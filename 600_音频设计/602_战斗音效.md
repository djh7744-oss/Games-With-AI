# 战斗音效设计

## 一、兵器音效

### 武器 Swing (挥舞破风声)

```
剑: 尖锐的"咻——"(高频，细)
刀: 厚重的"呜——"(中低频，粗)
戟: 沉重的"呼——"(低频，长)
矛: 集中的"嗖——"(中频，直)
弓: 弓弦"嘣——"(短促，弹性)
弩: 弩机"咔嗒——嗖!"(机械+飞行)
```

### 兵器命中

```
砍击命中肉体:
  层1(低频): 沉闷的"噗"(100-300Hz)
  层2(中频): 撕裂声(500-1500Hz)
  层3(高频): 血溅(2000-5000Hz)
  随机: 骨骼碎裂(概率触发，低沉"咔嚓")

砍击命中盔甲:
  金属碰撞"铛——"(800-3000Hz, 有长尾音)
  随盔甲等级: 皮甲闷/铜甲中/铁甲高/精铁甲极高

刺击命中:
  层1: 穿透声"嗤——"(比砍击更集中)
  层2: 拔出时"噗嗤"(可选)

格挡/武器碰撞:
  剑vs剑: 清脆"叮——"
  刀vs刀: 厚重"铛——"
  戟vs盾: 沉重"咚——"
  完美格挡: 额外高频泛音(更亮的"叮!")
```

### 兵器特殊音效

```
暴击: 命中音叠加+音调升高+短暂混响增强
处决: 慢动作期间的"心跳声"→处决瞬间全部音效短暂静音(0.2s)→爆发
名剑: 每把名剑有独特的命中音色
  湛卢: 命中时伴随微弱钟磬余音
  鱼肠: 命中时伴随液体渗透声
  巨阙: 命中时伴随低频冲击波
```

## 二、战场氛围

```
开战前:
  战鼓(低频，有节奏，渐强)
  号角(长音，宣告开战)
  士兵列阵脚步声
  旗帜在风中猎猎作响

战斗中:
  千人混战: 大量兵器碰撞+喊杀声(融合成"战场噪层")
  骑兵冲锋: 马蹄震地(低频)+马嘶+骑兵喊杀
  弓弩齐射: 弓弦齐响"嘣——"+箭矢飞行"咻——"+命中反馈
  攻城: 投石机的"嘎吱——轰!"+城墙被砸的"轰隆——"
  士气崩溃: 溃逃的混乱脚步声+恐惧的喊叫

近距离战斗(玩家周围30m):
  清晰层次: 可分辨个体兵器声+喊杀声
  空间化: 3D定位(听声辨位)

远距离战场(30m+):
  混合成"战场噪层"(Battle Ambience)
  随距离衰减+低通滤波
```

## 三、技能与特殊音效

```
火攻:
  点火: 火把/火油罐破裂(玻璃/陶器碎裂+"轰——")
  燃烧: 火焰持续燃烧声(低频持续)
  蔓延: 火焰蔓延时木材爆裂"噼啪"

毒:
  施放: 毒液喷射"嘶——"
  持续: 腐蚀声(高频"滋滋")
  效果: 中毒者痛苦声

士气技能:
  激励: 战鼓节奏加快+号角激昂
  恐惧: 低沉不和谐音+"心跳声"加速
```

## 四、单挑音效

```
单挑专属:
  阵前叫阵: 号角宣告+马蹄踏地
  进入决斗圈: 观众喧哗渐弱→决斗圈内"安静区"(仅两人呼吸+脚步声)
  兵器交锋: 每击都清晰(无战场噪层干扰)
  名剑特效: 名剑音效在单挑中更为显著
  胜负瞬间: 慢镜+心跳+静音→爆发
```
