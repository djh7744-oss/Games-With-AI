# 语音与对话设计

## 一、语音类型

```
战斗语音(Combat VO):
  攻击呐喊(短): "哈!" / "杀!"
  受击: 闷哼/痛呼(分轻/重/致命)
  技能施放: 技能名/战吼
  击杀: "下一个!" / "弱!"
  死亡: 临终惨叫/遗言

指令语音(Command VO):
  军团指令: 编队指令/阵型切换/冲锋/撤退
  回应: "遵命!" "是!"

剧情语音(Cinematic VO):
  Lv4/Lv5叙事场景: 完整配音(逐句)
  关键NPC台词: 全部配音
  普通NPC: 仅首句配音+文字

环境语音(Ambient VO):
  市集叫卖: "新鲜的鱼!", "上好的丝绸!"
  酒馆闲聊: 模糊多人对话(不可分辨具体内容)
  士兵巡逻: "站住! 什么人!"
```

## 二、配音标准

```
配音语言: 中文(普通话)为主
方言运用:
  秦: 略带陕西口音(古朴感)
  楚: 略带湖北/湖南口音
  齐: 标准普通话(文化中心)
  赵: 略带河北口音
  其他: 标准普通话(避免过度方言化)

录音标准:
  录音室: 专业录音棚(隔音+吸音)
  麦克风: 大振膜电容麦(Neumann U87或同级)
  距离: 15-30cm(带防喷罩)
  电平: -12dBFS(平均), 峰值<-3dBFS
  格式: 48kHz/24bit Mono

后期:
  降噪: 轻量(iZotope RX)
  压缩: 轻量(2:1~3:1)
  EQ: 低切80Hz/高频适当提升(清晰度)
  混响: 场景匹配(室内/室外/战场)
```

## 三、关键角色配音方向

```
白起:
  声线: 中年男声，低沉，缓慢，几乎不带感情
  关键词: 冷酷、精确、少言
  示例: "传令。全军——杀。"
        (每一个字之间停顿，最后"杀"字很轻)

荆轲:
  声线: 青年男声，沙哑，有酒意
  关键词: 不羁、深情、决绝
  示例: "风萧萧兮易水寒——(停顿)壮士一去兮——不复还。"

孙膑:
  声线: 中年男声，沉稳，偶尔露出痛苦
  关键词: 隐忍、智慧、复仇的冷静
  示例: "师兄。马陵道——就是你葬身之地。"

商鞅:
  声线: 中年男声，坚定、清晰、不带商量余地
  关键词: 理性、执着、孤独
  示例: "法之不行，自上犯之。太子犯法——师傅代受刑。"

田单:
  声线: 中年男声，普通人的声音→逐渐变得坚定
  关键词: 平凡、坚韧、创造奇迹
  示例: "我不是将军。我只是一个管市场的小吏。
        但即墨需要一个人站出来。"

嬴政:
  声线: 青年→中年男声，威严、多疑、偶尔孤独
  关键词: 权力、孤高、不容置疑
  示例: "朕为始皇帝。后世以计数，二世三世至于万世——传之无穷。"
```

## 四、语音本地化

```
支持语言:
  简体中文(默认)
  繁体中文
  English
  日本語
   한국어

本地化策略:
  剧情语音: 全部重新录制(每种语言独立配音)
  战斗语音: 保留中文"杀!"(文化特色)+翻译关键指令
  环境语音: 本地化市集叫卖/酒馆闲聊

字幕:
  始终显示(可关闭)
  语音语言与字幕语言可独立选择
  例: 中文语音+英文字幕
```
