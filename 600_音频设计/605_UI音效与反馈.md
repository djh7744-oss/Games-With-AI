# UI音效与反馈设计

## 一、UI音效风格

```
战国UI音效关键词:
  - 木质共鸣(木牍/竹简感)
  - 金石之声(编钟/玉磬短音)
  - 墨迹/书写(书简翻阅感)
  - 铜钱/刀币(金属清脆)

设计原则:
  1. 每个UI操作有明确声音反馈
  2. 正向操作(确认/升级) → 上升音调
  3. 负向操作(取消/错误) → 下降音调
  4. 不超过0.5秒(UI音效应短促)
```

## 二、UI音效清单

```
按钮:
  通用点击: 木牍轻叩"笃"(50ms)
  确认: 玉磬短音"叮—"(200ms,上升音)
  取消/关闭: 木牍合上"嗒"(80ms)
  悬停: 极轻微"嗡"(可选,30ms)

面板:
  打开: 竹简展开"哗啦"(300ms)
  关闭: 竹简卷起"唰"(200ms)
  切换Tab: 书页翻动(100ms)

通知:
  普通消息: 编钟轻敲"叮"(200ms)
  重要消息: 编钟重击"铛——"(500ms,有余音)
  警告: 铜钲急敲"铛铛!"(短促重复)
  错误: 低沉"嗡——"(下降音)

数值变化:
  金币增加: 刀币碰撞"叮铃"(短)
  经验增加: 上升的音阶(3个音快速连奏)
  升级: 辉煌的编钟和弦(1s)
  受伤: 低沉鼓点"咚"(100ms)
  死亡/重伤: 沉重战鼓一击+余音衰减(2s)

战斗UI:
  命中确认: 短促"唰"(配合伤害数字飘出)
  暴击: 额外高频"叮!"+短暂闪亮音
  技能冷却完毕: 清脆"叮—"
  士气变化: 上升音(增益)/下降音(减益)

战略地图UI:
  军团选择: 旗帜猎猎声(短)
  移动指令: 马蹄声(短)
  资源采集: 对应资源声(矿=镐击/木=斧砍/粮=收割)
  发现新地点: 编钟+简短的"发现"旋律(1s)
```

## 三、触觉反馈(手柄)

```
配合UI音效的振动:

按钮点击: 轻微脉冲(10ms)
确认: 双脉冲(短-短)
取消: 单脉冲稍长(20ms)
警告: 三脉冲(短-短-短)
伤害受击: 按伤害量振动(轻/中/重)
暴击: 强烈脉冲+持续振动(50ms)
```
