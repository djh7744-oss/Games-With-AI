# 音乐系统设计

## 一、七国主题音乐

```
秦国《铁血秦风》:
  调式: 羽调(La调式，深沉)
  核心乐器: 编钟(低音)/战鼓/铜钲/号角
  速度: 慢板→进行曲(渐变)
  情感: 从西陲贫弱到虎狼之国的史诗感
  结构: 低沉压抑(建国)→节奏渐强(变法)→战鼓激昂(崛起)→编钟辉煌(统一)
  时长: 4:30

楚国《云梦楚歌》:
  调式: 楚调(七声，有变徵/变宫，神秘感)
  核心乐器: 瑟/箫/编钟/竹笛
  速度: 中板，有流动感
  情感: 南方大国的神秘与哀愁
  结构: 云梦雾气(引子)→楚宫宴乐→水军战歌→屈原投江后的哀歌→"楚虽三户"的倔强尾声
  时长: 4:00

齐国《稷下风华》:
  调式: 宫调(Do调式，庄严)
  核心乐器: 古琴/笙/筝/箫
  速度: 中板，优雅从容
  情感: 繁华富庶的文化自信
  结构: 临淄早市→稷下论道→海盐贸易→渔歌唱晚
  时长: 3:30

赵国《胡服骑射》:
  调式: 商调(Re调式，激昂)
  核心乐器: 号角/战鼓/胡笳/琵琶
  速度: 快板，马蹄奔腾的节奏
  情感: 热血、悲壮、永不言败
  结构: 草原日出→骑兵操演→长平悲歌→邯郸坚守→最后冲锋
  时长: 4:00

燕国《北疆孤鸿》:
  调式: 羽调变体(La调，苍凉)
  核心乐器: 胡笳/马头琴式长鸣/骨笛
  速度: 慢板，辽阔悠远
  情感: 北方边疆的孤寂与坚韧
  结构: 燕山风雪→黄金台→乐毅伐齐→荆轲刺秦→易水余音
  时长: 3:30

魏国《中原之声》:
  调式: 徵调(Sol调式，庄重)
  核心乐器: 筑/磬/篪/古琴
  速度: 中板，稳重大气
  情感: 文化底蕴与四面楚歌的焦虑
  结构: 大梁繁华→西河学派→人才流失的忧虑→魏武卒最后的荣耀
  时长: 3:00

韩国《劲弩悲歌》:
  调式: 角调(Mi调式，激越中带悲)
  核心乐器: 竹笛/钟/缶/奚琴
  速度: 中快，有精密机械感
  情感: 精巧与悲剧并存
  结构: 宜阳冶铁→劲弩试射→申不害变法→被秦攻破的终章
  时长: 2:30

周室《天子九鼎》:
  调式: 黄钟正声(上古音阶)
  核心乐器: 编钟/玉磬/埙/柷
  速度: 极慢，庄严肃穆
  情感: 八百年周朝的落日余晖
  结构: 九鼎辉煌→诸侯不朝→洛阳残阳→周室终结
  时长: 2:00
```

## 二、自适应音乐系统

### 分层音乐

```
每首主题曲分为4层(Layers):

Layer 1 环境层(最底层): 
  持续的低音(Drone)/氛围和弦
  和平区域/探索时 → 仅Layer 1播放

Layer 2 律动层:
  节奏乐器(鼓/打击乐)加入
  进入城池/遭遇NPC → Layer 1+2

Layer 3 旋律层:
  主旋律乐器加入
  战斗/重要事件 → Layer 1+2+3

Layer 4 高潮层:
  全部乐器+合唱(如果适用)
  史诗时刻/Boss战/结局 → 全部Layer

过渡:
  层级切换时使用 Wwise Transition(淡入淡出)
  过渡时间: 2-4秒(平滑切换)
```

### 动态音乐参数

```
音乐受以下游戏参数实时影响:

威胁度(Danger Level):
  0(安全): Layer 1
  1-3(警戒): Layer 1+2
  4-6(战斗): Layer 1+2+3
  7-10(致命): 全部Layer

情绪强度(Emotion Intensity):
  剧情场景中控制音乐情绪
  RTCP(Real-Time Parameter Control)驱动音量/混响/滤波器

地域(Biome):
  切换国家主题曲
  切换时使用 Wwise State(平滑过渡，8-16小节)

时间(昼夜):
  白天: 明亮调(高频略多)
  夜晚: 安静调(低频略多，音量-3dB)
```

## 三、特殊场景音乐

```
主菜单: 《乱世风云》(编钟+管弦乐，史诗感，4:00)

荆轲刺秦: 《易水寒》(独奏箫+古琴，极简，2:30)
  → 不使用Layer系统，固定演奏
  → 全游戏唯一"不循环"的场景音乐(播完即静)

长平坑杀: 《四十万》(无音乐，仅风声，1:00静默)
  → 全游戏唯一完全静默的场景

统一结局: 《天下》(七国主题各一句→融合→辉煌终章，5:00)

角色死亡: 《逝水》(单乐器缓慢演奏——角色所属国的乐器)
```
