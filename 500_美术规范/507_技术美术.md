# 技术美术规范

## 一、PBR材质标准

### 材质模板

```
主材质(Master Material):
  功能: 所有材质实例的基础
  参数:
    BaseColor: Texture2D + TintColor
    Normal: Texture2D (DirectX格式)
    ORM: Texture2D (R=AO, G=Roughness, B=Metallic)
    DetailNormal: 可选细节法线(微表面)
    Dirt/Wetness: 污渍/湿润叠加层

材质实例(Material Instance):
  每个资产一个MI → 继承主材质 → 仅调整参数
  禁止为每个资产创建独立Materia(浪费编译时间)

材质函数(Material Function):
  复用模块: MF_DirtOverlay / MF_Wetness / MF_DamageWear
```

### 材质校验

```
PBR值校验规则:
  BaseColor暗色: <50 sRGB (木炭/黑铁)
  BaseColor亮色: <240 sRGB (雪/白墙)
  Roughness: 0.15(光滑金属) ~ 1.0(粗糙木头)
  Metallic: 0(非金属)或1(金属)，无中间值(除过渡区)

常见错误:
  ✗ 木材Metallic≠0 → 会产生错误反射
  ✗ 金属BaseColor太亮(>200 sRGB) → 违反物理
  ✗ 粗糙度贴图未使用 → 所有表面一样光滑
```

## 二、LOD策略

```
Static Mesh LOD:
  LOD0: 100% △ (近景0-20m)
  LOD1: 50% △ (中景20-80m)
  LOD2: 20% △ (远景80-200m)
  LOD3: 5% △ (极远200m+)
  Nanite(UE5): 自动LOD → 仅对大型建筑/地形使用

Skeletal Mesh LOD:
  LOD0: 100% △ + 完整骨骼
  LOD1: 60% △ + 简化骨骼
  LOD2: 30% △ + 移除面部/布料
  LOD3: 10% △ + VAT/Imposter

材质LOD:
  与几何LOD联动
  LOD2+: 使用更简单的材质(减少贴图采样)
  LOD3: 使用烘焙的简化材质
```

## 三、性能标准

### 帧预算

```
目标: 60FPS = 16.6ms/帧

分配:
  CPU(Game+AI): 5ms
  CPU(Render Thread): 4ms
  GPU: 6ms
  Other: 1.6ms

GPU预算细项(6ms):
  几何(Geometry): 1.5ms
  光照(Lighting): 1.5ms (Lumen)
  后处理(Post Process): 1ms
  特效(VFX): 1ms
  阴影(Shadows): 0.5ms (Virtual Shadow Maps)
  UI: 0.5ms
```

### Draw Call预算

```
战略地图: <2000 Draw Calls
城池内部: <3000 Draw Calls
战场: <2500 Draw Calls
探索副本: <1000 Draw Calls

使用Nanite → Draw Call大幅减少
使用ISM(Instanced Static Mesh) → 大量重复物件(如树/石头)
```

### 内存预算

```
VRAM (8GB卡):
  Textures: 3GB
  Geometry: 2GB (含Nanite数据)
  Render Targets: 1.5GB
  Other: 1.5GB

贴图池(Texture Pool):
  使用UE5 Texture Streaming
  根据距离动态调整贴图分辨率
  Pool Size: 3000MB(8GB卡)
```

## 四、碰撞规范

```
复杂碰撞(Complex Collision):
  仅用于需要精确碰撞的物件(如: 室内可拾取的小物件)
  
简单碰撞(Simple Collision):
  大多数物件 → Box/Sphere/Capsule碰撞体
  建筑: Box碰撞
  柱子: Cylinder碰撞
  树木: Capsule(树干)+Sphere(树冠)

角色碰撞:
  Capsule: 半径35cm, 高度180cm (玩家)
  ＜骑马＞: 半径50cm, 高度250cm

武器碰撞:
  使用Physics Asset的碰撞体
  仅在攻击帧开启碰撞检测(性能优化)
```

## 五、导出规范

### Blender → UE5

```
单位: 1 Blender Unit = 1 UE Unit = 1cm
朝向: -Y Forward, +Z Up
缩放: Apply Scale (1,1,1)
三角面: 导出时自动三角化

FBX导出设置:
  √ Selected Objects
  √ Apply Modifiers
  √ Triangulate
  Scale: 1.00
  Forward: -Y
  Up: Z

贴图导出(Substance Painter → UE5):
  BaseColor: sRGB, 8bit PNG
  Normal: Linear, 8bit PNG (DirectX格式)
  ORM: Linear, 8bit PNG (R=AO,G=Roughness,B=Metallic)
  
  (UE5 ORM: OcclusionRoughnessMetallic 压缩格式)
```

### 命名验证

```
资产导入前自动检查:
  □ 命名符合规范 (SK_/SM_/T_/M_/MI_/AN_)
  □ 无中文字符
  □ 无特殊符号(除下划线)
  □ 面数在预算内
  □ 贴图分辨率符合规范
  □ 材质使用Material Instance
```

