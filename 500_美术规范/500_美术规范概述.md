# 美术规范概述

## 视觉方向

> "历史感，而非魔幻感。" 战国的美术应该让玩家感受到真实的历史厚重——青铜的铁锈、丝绸的质感、黄土的粗粝。不做"金光闪闪"的魔幻战国，而做"可以闻到尘土和血腥味"的真实战国。

## 三大视觉支柱

| 支柱 | 含义 | 参考来源 |
|------|------|---------|
| **材质真实** | 铜是铜，铁是铁，丝绸是丝绸。每种材质有独特的PBR属性 | 考古文物/博物馆参考 |
| **七国差异** | 每国有独特的色调、纹样、建筑风格 | 地域文化考据 |
| **时代感** | 战国末期的技术进步（铁器取代铜器）体现在视觉上 | 历史时间线 |

## 全局技术标准

| 项目 | 标准 | 备注 |
|------|------|------|
| 引擎 | UE 5.5+ | Lumen + Nanite |
| 建模 | Blender 4.x LTS | 统一单位: 1BU=1cm |
| 贴图 | Substance Painter | PBR金属/粗糙度工作流 |
| 雕刻 | ZBrush (可选) | 高模烘焙到低模 |
| 场景 | Gaea/World Machine | 程序化地形 |
| 贴图分辨率 | 角色2K/环境Tileable 2K/地面4K VT | 见各子系统规范 |
| 纹理格式 | BaseColor(Normal)/Normal(DirectX)/ORM(R=AO,G=Roughness,B=Metallic) | ORM三合一贴图 |
| 命名规范 | 见各子系统 |
| 版本控制 | Git LFS (大文件) / Perforce (团队) | 美术源文件>500MB用Perforce |

## 美术资产命名总规范

```
格式: [类型]_[国家]_[名称]_[变体]_[LOD]

类型码:
  SK = Skeletal Mesh (骨骼网格体)
  SM = Static Mesh (静态网格体)
  T = Texture (贴图)
  M = Material (材质)
  MI = Material Instance (材质实例)
  AN = Animation (动画)
  BP = Blueprint (蓝图)

示例:
  SK_Qin_Soldier_Heavy_LOD0       (秦国重步兵LOD0)
  SM_Chu_Building_Palace_01       (楚国宫殿建筑01)
  T_Zhao_Leather_Armor_BC         (赵国皮甲BaseColor贴图)
  M_Weapon_Iron_Sword             (铁剑材质)
  AN_Horse_Gallop                 (马匹奔跑动画)
```
