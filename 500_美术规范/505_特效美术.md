# 特效美术规范

## 一、战斗特效

### 兵器命中特效

```
砍击命中(肉体):
  粒子: 血雾(方向性喷射，沿砍击方向)
  颜色: 深红 #8B0000 → 鲜红 #DC143C
  大小: 10-30个粒子/次，0.3-0.8秒消散
  伴随: 微弱的武器轨迹残影

砍击命中(盔甲):
  粒子: 火星(金属碰撞)
  颜色: 橙黄 #FFA500 → 白 #FFFFFF
  声音: 金属碰撞声(优先级高于肉体声)
  伴随: 镜头微震(手柄震动)

刺击命中:
  粒子: 集中型血束(比砍击更集中)
  伴随: 命中停顿(Hit Stop 0.05s)

暴击:
  粒子: 更大范围+更亮+慢动作
  伴随: 屏幕边缘微红(受重伤感觉)
  声音: 特殊打击音效

处决:
  特效: 专属处决动画+镜头慢镜(0.3s)
  粒子: 大量血液+武器轨迹
  声音: 特殊处决音效
```

### 武器轨迹

```
普通攻击: 半透明武器拖尾(与武器颜色一致)
重攻击: 更明显的拖尾+轻微冲击波
名剑: 每把名剑有独特的轨迹颜色/形状
  湛卢: 蓝色流光拖尾
  鱼肠: 绿色毒液拖尾
  巨阙: 金色重击冲击波
```

### 技能特效

```
火攻: 
  粒子: 火焰(UE5 Niagara + 流体模拟)
  颜色: 橙→红→黑烟
  范围: 基于技能等级缩放
  持续: 5-30秒(可被扑灭)

毒:
  粒子: 绿色毒雾+腐蚀液滴
  颜色: 绿 #3a8a2a → 暗绿 #1a4a1a
  效果: 目标皮肤变绿(材质参数变化)

士气效果:
  增益: 金色光点上升(友方)
  减益: 暗紫色光点下降(敌方)
  图标: 角色头顶士气图标(颜色标识)
```

## 二、环境特效

### 天气粒子

```
雨:
  粒子: 雨滴(细长条，方向一致)
  密度: 小雨500粒子/大雨2000粒子
  伴随: 地面溅射/湿润反射/雾

雪:
  粒子: 雪花(六角形或圆形软粒子)
  密度: 小雪300/大雪1500
  伴随: 地面白色材质覆盖/呼出白气(角色)

雾:
  Volumetric Fog (UE5体积雾)
  浓度: 轻雾(可视500m)/浓雾(可视100m)

风:
  植被摆动: 树木/草丛的顶点动画
  沙尘: 沙漠地区的沙粒粒子
```

### 城池氛围

```
市集: 飘浮的微小尘埃(阳光透过窗户)/炊烟
酒馆: 暖色灯光+壁炉火焰+吊灯摇晃
工坊: 锻造火花+烟囱白烟+热浪扭曲(折射)
战场: 硝烟/燃烧残骸/飘扬的旗帜(布料物理)
```

## 三、UI特效

```
界面过渡:
  面板打开: 竹简展开动画(从右向左)
  面板关闭: 竹简卷起动画
  弹窗: 从中心缩放弹出

数值飘字:
  伤害: 红色大字，向上飘+淡出
  治疗: 绿色大字，向上飘+淡出
  暴击: 更大+更亮+震屏

奖励获得:
  金币: 刀币图标+数字飘起
  装备: 装备图标从中心放大→飞入背包位置
  声望: 对应声望颜色光效(仁义蓝/武名红等)
```

## 四、特效性能预算

```
同屏粒子上限:
  PC: 5000粒子
  主机: 3000粒子
  
单特效预算:
  普通攻击: <50粒子, <0.5ms GPU
  技能: <200粒子, <2ms GPU
  环境(持续): <1000粒子, <3ms GPU
  
  使用GPU粒子(Niagara) → 高效
  CPU粒子仅用于需要碰撞检测的特效(如溅血)
```
