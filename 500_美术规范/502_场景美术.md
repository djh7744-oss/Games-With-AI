# 场景美术规范

## 一、地形与Biome

## 二、建筑风格

### 七国建筑特征

```
秦国建筑:
  风格: 厚重实用，直线条，少装饰
  材质: 夯土墙+木结构+黑瓦
  色调: 土黄+暗褐+黑色
  代表: 咸阳宫(高台建筑，体量巨大)
  特色: 标准化(秦砖规格统一)/功能性优先

楚国建筑:
  风格: 华丽繁复，曲线柔美，高台+干栏式
  材质: 木材+红漆+金饰+青铜饰件
  色调: 深红+金色+木原色
  代表: 章华台(楚国著名高台建筑)
  特色: 漆画装饰/凤鸟纹样/干栏式(南方防潮)

齐国建筑:
  风格: 雅致富丽，比例优雅
  材质: 精加工木材+白墙+青瓦
  色调: 白色+蓝色+木色
  代表: 稷下学宫(学术殿堂)
  特色: 学宫/庭院深深/藏书楼

赵国建筑:
  风格: 坚固实用，军事化风格
  材质: 夯土+石材+粗木
  色调: 土黄+灰褐+皮革色
  代表: 丛台(军事操演高台)
  特色: 马厩/训练场/城墙厚重

燕国建筑:
  风格: 低调内敛，适应寒冷
  材质: 石材+厚土墙+茅草/瓦顶
  色调: 灰色+白色+暗褐
  代表: 黄金台(求贤台)
  特色: 厚墙小窗(保暖)/皮毛装饰

魏国建筑:
  风格: 庄重典雅，文化气息浓厚
  材质: 精石材+木材+青铜
  色调: 紫色+金色+灰色
  代表: 大梁城(繁华中原都市)
  特色: 书肆/文雅装饰/运河码头

韩国建筑:
  风格: 精巧细致，小而精
  材质: 精细加工木石+铁饰件
  色调: 橙色(宜阳铁锈色)+黑色+木色
  代表: 宜阳冶铁坊
  特色: 工坊密集/精密铁艺装饰

周室建筑:
  风格: 礼制严格，古朴庄重
  材质: 青铜+粗石+夯土
  色调: 铜绿+黄色+赭石
  代表: 洛阳王城/明堂
  特色: 青铜礼器/宗庙/九鼎
```

### 城池建筑分级LOD

```
Lv1村庄: 5-8栋茅屋+水井+打谷场 (总面数<5万△)
Lv2乡镇: 15-20栋建筑(木/土混合)+小市集 (总面数<15万△)
Lv3县城: 城墙+30-50栋+官衙+兵营+市集 (总面数<40万△)
Lv4郡城: 砖石城墙+100+栋+分区明确 (总面数<100万△)
Lv5都城: 王宫+城墙瓮城+200+栋+地标建筑 (总面数<200万△)

城内用Nanite处理大件静态物(建筑/城墙)
室内独立关卡(非流式)→精细资产
```

```
  LOD0(近景): 完整3D树(2000-5000△)+风动顶点动画

  LOD2(远景): Billboard(2面片)+法线贴图

```

## 四、光照与环境氛围

```
时间光照:
  黎明(4-6): 冷蓝→暖橙渐变/雾浓度高/体积光明显
  白昼(6-18): 标准日光/色温5500K/阴影清晰
  黄昏(18-20): 暖橙→紫蓝/长阴影/金色边缘光
  夜晚(20-4): 月光(色温4100K)/火把光源/灯笼点光源

天气氛围:
  晴天: 标准Lumen全局光照
  多云: 软阴影/Directional Light强度×0.7
  雨天: 反射增强(湿地面)/雾气/天空光偏冷
  雾天: Volumetric Fog浓度高/可视距离压缩
  雪天: 地面白色材质覆盖/天空光偏蓝

城池氛围:
  咸阳: 严肃/冷峻/暗色调
  临淄: 温暖/繁华/金蓝色调
  郢都: 神秘/湿润/雾气弥漫
  邯郸: 厚重/军事感/凛冽北风
```
