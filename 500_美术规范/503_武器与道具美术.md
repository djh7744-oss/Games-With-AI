# 武器与道具美术规范

## 一、武器建模标准

### 面数预算

| 武器类型 | 近景△ | 中景△ | 远景△ | 备注 |
|---------|--------|--------|--------|------|
| 剑/刀(单手) | 1500-2500 | 800 | 300 | 含剑鞘 |
| 戟/戈/矛(长杆) | 2500-4000 | 1200 | 400 | 长杆+头部 |
| 弓 | 2000-3000 | 1000 | 300 | 含弦(骨骼) |
| 弩 | 3000-5000 | 1500 | 500 | 精密机关 |
| 盾 | 1500-3000 | 800 | 300 | — |
| 名剑(传说) | 4000-8000 | 2000 | 800 | 高精度展示用 |
| 匕首 | 800-1500 | 400 | 150 | 小型 |

### 七国武器视觉差异

```
秦国武器: 厚重实用
  剑: 宽刃，简约无纹饰，铁灰色/黑色
  戟: 粗壮，标准化(各戟几乎一模一样)
  弩: 大尺寸，结构简单但坚固
  色调: #3a3a3a(铁) + #1a1a1a(黑) + #8a7a5a(木柄)

楚国武器: 装饰华丽
  剑: 剑身略长，有纹饰/镶嵌/错金工艺
  戟: 戟头有兽纹或凤鸟纹
  弓: 长弓，弓身缠绕丝线(红/金色)
  色调: #8B0000(红) + #DAA520(金) + #4a3a2a(深色木)

齐国武器: 工艺精巧
  剑: 剑身轻盈，护手精美
  刀: 刃口锋利(高光明显)
  色调: #1a3a5c(蓝) + #C0C0C0(银) + #5a4a3a(木)

赵国武器: 骑兵特化
  刀: 弯刀设计(马上砍击)+皮革缠柄
  矛: 骑枪，超长杆(3m+)+枪缨
  色调: #2d5a27(绿) + #8B4513(皮革) + #5a4a3a(木)

燕国武器: 耐寒设计
  剑/刀: 柄部裹皮毛(防冻手)
  弓: 复合弓(适应温差)
  色调: #6b6b6b(灰) + #F5F5F5(白皮毛) + #4a3a2a(木)

魏国武器: 精细锻造
  剑: 百炼钢纹路可见(法线贴图)
  戟: 精密编缀连接
  色调: #4a1a6b(紫) + #B8860B(黄铜) + #5a4a3a(木)

韩国武器: 精密技术
  剑: 极薄刃口(高光锋利感)
  弩: 精密机关(齿轮/弹簧视觉)
  色调: #c75b1a(橙/铁锈) + #2a2a2a(黑) + #3a3a3a(精密铁)
```

## 二、武器材质球

```
铁器(通用):
  BaseColor: 暗灰 #3a3a3a
  Roughness: 0.4-0.6 (锻造面较粗)
  Metallic: 1.0
  细节: 锤锻痕迹(法线贴图)/局部锈迹

精铁:
  BaseColor: 深灰 #2a2a2a
  Roughness: 0.3-0.4 (更光滑)
  Metallic: 1.0
  细节: 细密锻纹(法线)/刃口高亮

百炼钢:
  BaseColor: 暗灰带纹理 #2a2a2a (带折叠纹)
  Roughness: 0.2-0.35
  Metallic: 1.0
  细节: 层叠纹路(特殊法线贴图)/刃口极高光

青铜:
  BaseColor: 铜绿/金铜 #b87333 → 氧化绿 #5a8a4a
  Roughness: 0.5-0.7
  Metallic: 1.0
  细节: 锈斑/铜绿

黄金(镶嵌/装饰):
  BaseColor: 金色 #DAA520
  Roughness: 0.15-0.25
  Metallic: 1.0
  细节: 抛光面/磨损边角

木材(柄/鞘):
  BaseColor: 木色 #5a4a3a
  Roughness: 0.7-0.9
  Metallic: 0.0
  细节: 木纹(法线)/使用磨损

皮革(缠柄):
  BaseColor: 皮革色 #6a4a3a
  Roughness: 0.8-1.0
  Metallic: 0.0
  细节: 皮纹/缝合线
```

## 三、道具与物件

### 城内物件

```
商铺道具:
  算盘/钱币(铜币/刀币/布币不同造型)
  竹简/毛笔/墨/砚台
  秤/斗/升(度量衡器具)
  布料卷/丝绸匹
  陶罐/漆器盒

酒馆道具:
  酒坛/酒碗/酒壶(陶器)
  长桌/长凳(粗木)
  灯笼/烛台
  菜刀/砧板/炉灶

官衙道具:
  官印/令箭/令牌
  文案/笔架/竹简堆
  惊堂木/刑具(夹棍/枷锁)
  旗帜/兵器架

民居道具:
  灶台/水缸/米缸
  床榻/被褥/枕头(布制)
  纺车/织机
  农具(锄/镰/犁)
```

### 物件LOD

```
城内物件(Static Mesh):
  LOD0: 完整模型(100-500△/件)
  LOD1: 简化(50-200△)
  LOD2: 极简(20-50△)
  大量物件用Nanite自动LOD
  NPC手持物件: 仅LOD0(近景始终可见)

物件命名:
  SM_[城池等级]_[区域]_[物件名]_[序号]
  例: SM_Lv4_Market_Stool_01 (郡城市集凳子01)
```

## 四、武器特效视觉

```
品质光效:
  白色: 无光效
  绿色: 微弱绿色边缘光(仅在高光角度可见)
  蓝色: 蓝色微光+轻微粒子(小光点)
  紫色: 紫色光晕+粒子拖尾+轻微辉光
  金色: 金色光晕+粒子旋转+Bloom效果
  暗金: 暗金色波动光+专属粒子特效

名剑特效(每把独特):
  湛卢: 剑身半透明+蓝色流光
  鱼肠: 毒液绿色滴落粒子
  巨阙: 重击时地面裂纹+冲击波
  太阿: 剑身电光缠绕(架空)
```
