# UI与界面美术规范

## 一、视觉风格

```
战国UI风格关键词:
  - 竹简/木牍质感 (背景纹理)
  - 青铜纹样 (边框/装饰)
  - 篆书字体 (标题/大字)
  - 水墨/丹青色调 (黑/白/朱砂红)
  - 刀币/布币/圜钱 (图标元素)
  - 战旗/战鼓 (战斗UI元素)
```

## 二、色彩体系

```
主色调:
  背景: 深棕 #2a1a0a (竹简色)
  面板: 米黄 #f5e6c8 (绢帛色)
  边框: 暗铜 #8B7500
  文字主: 墨黑 #1a1a1a
  文字辅: 朱砂红 #c43a2a (重点/警告)

功能色:
  可交互: 金色 #DAA520
  不可交互: 灰色 #8a8a8a
  成功/正面: 翠绿 #3a8a3a
  警告: 朱砂 #c43a2a
  危险: 深红 #8B0000
  友方: 蓝色 #3a6aB4
  敌方: 红色 #B43a3a
  己方: 金色 #DAA520
```

## 三、字体规范

```
中文字体:
  标题: 篆书风格(自研或授权) → 用于: 主菜单/国家名/爵位
  正文: 仿宋/楷体 → 用于: 对话/任务描述/日志
  数字: 清晰的等宽字体 → 用于: 数值/伤害/金钱

英文字体(本地化):
  标题: Trajan Pro (罗马体，匹配篆书气质)
  正文: serif字体 (匹配仿宋气质)

字号规范(参考,基于1080p):
  主标题: 48px
  副标题: 32px
  面板标题: 24px
  正文: 18px
  小字/备注: 14px
```

## 四、UI组件库

```
按钮:
  普通: 木牍纹理底+篆书文字+悬停高亮(金色描边)
  重要: 红色底+金色文字
  禁用: 灰色底+灰色文字

面板:
  背景: 竹简/绢帛纹理(半透明90%)
  边框: 青铜纹样边框(四角有青铜饰件)
  标题栏: 深色木纹理+篆书标题
  滚动条: 竹简卷轴样式

图标:
  风格: 简洁线描(白描)+单色填充
  尺寸: 24×24(小)/48×48(中)/96×96(大)
  示例: 剑(战斗)/鼎(内政)/令箭(任务)/刀币(经济)

进度条:
  样式: 竹简展开/卷轴展开
  填充: 朱砂红色从左到右
  背景: 暗色木纹理

弹窗:
  背景: 深色半透明遮罩(70%黑)
  面板: 居中竹简/绢帛面板
  按钮: 底部居中排列
```

## 五、关键界面设计

```
主菜单:
  背景: 动态战国地图(水墨风格)+飘落竹简粒子
  标题: 篆书大字"战国·乱世风云"
  按钮: 竖排竹简式按钮(新游戏/继续/设置/退出)
  氛围: 古琴背景音乐+编钟偶尔响起

HUD(战斗):
  左上: 血条(红色竹简展开)+体力条(绿色)
  右上: 小地图(水墨地形风格)
  底部中: 武器栏(刀币形状图标)+道具(布袋图标)
  底部左: 当前指令提示
  中央: 命中反馈(飘字伤害/暴击特效)

HUD(战略):
  顶部: 资源栏(金/粮/兵 刀币/米袋/令箭图标)
  右上: 时间/日期(篆书+天干地支)
  右下: 军团状态(编队图标+士气条)
  左上: 任务追踪(竹简展开)

角色面板:
  中央: 角色立绘或3D模型
  左侧: 六维雷达图(八卦阵样式)
  右侧: 装备槽(身体部位图示)
  下方: 属性详情(竹简滚动列表)
```

## 六、UI适配

```
分辨率适配:
  最低: 1280×720 (16:9)
  推荐: 1920×1080 (16:9)
  理想: 2560×1440 (16:9)
  超宽: 3440×1440 (21:9, HUD拉伸/居中)

UI缩放: 支持0.8x-1.5x用户自调
色盲模式: 红/绿弱 → 增加图标+纹理区分(非纯依赖颜色)

本地化UI调整:
  英文: 文字比中文长30-50% → 按钮/面板需预留空间
  日文: 可使用部分汉字 → 字体需包含日文汉字
```
