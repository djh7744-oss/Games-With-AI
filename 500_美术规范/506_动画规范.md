# 动画规范

## 一、角色动画

### 骨架规范

```
必须兼容 UE5 Mannequin 骨架
额外骨骼(非Mannequin标准):
  武器挂点: weapon_r (右手)/weapon_l (左手)/weapon_back (背)
  盾牌挂点: shield_l (左臂)/shield_back (背)
  披风骨骼: cloak_01~04 (4根骨骼驱动布料)
  头发骨骼: hair_01~06 (长发角色)
  面部: 标准FACS BlendShape (52个基础表情)

骨架层级:
  Root → Pelvis → Spine_01~03 → Neck → Head
       → Clavicle_L/R → UpperArm_L/R → LowerArm_L/R → Hand_L/R
       → Thigh_L/R → Calf_L/R → Foot_L/R
```

### 动画清单

```
玩家角色动画(约200+):
  移动:
    待机(idle)×3(放松/警戒/疲惫)
    慢走/快走/跑步/冲刺(各×8方向: 前/后/左/右/左前/右前/左后/右后)
    蹲下移动×4方向
    跳跃/落地/翻越

  战斗(近战):
    轻击×4(左砍/右砍/上劈/下刺)
    中击×4
    重击×4(蓄力)
    连招过度动画×6(轻→轻/轻→重/重→重等)
    格挡×4(对应四个方向)
    完美格挡×1(特殊timing)
    闪避×4(前/后/左/右)
    受击×6(头部/躯干/四肢×前后)
    处决×4(每武器类型一个)

  战斗(远程):
    拉弓/放箭/装填(弩)/瞄准idle

  马上:
    上马/下马
    马上idle/慢走/快走/疾驰
    马上攻击×2(左砍/右砍)
    骑枪冲刺(架枪→命中)
    落马/受惊

  社交:
    行礼/跪拜/交谈/大笑/哭泣/愤怒
    饮酒/进食/读书/写字

NPC动画:
  复用玩家动画(士兵/将领)
  平民特有: 耕作/纺线/叫卖/乞讨/读书/扫地
  动物: 马(见下文)/狗/鸡/牛(基础)
```

### 马匹动画

```
马匹动画(约50+):
  移动: 站立/慢走/快走/慢跑/疾驰/跳跃
  战斗: 受惊/前蹄抬起/倒地/死亡
  交互: 被抚摸/吃草/甩头/嘶鸣

马上骑手动画:
  与马匹动画同步(IK控制骑手身体)
  骑手动画+马匹动画=骑马完整动画
```

### 动画融合

```
 locomotion: Blend Space 2D (速度+方向)
 战斗移动: Blend Space (strafe+forward)
 攻击连招: Animation Montage + Slot节点
 受击: 叠加在 locomotion上的部分骨骼动画(上半身受击+下半身保持移动)
 马上: 骑手+马匹 两个AnimBP同步(通过SyncGroup)
```

## 二、面部动画

```
FACS BlendShape:
  基础52个 → 可组合出>100个表情
  
关键表情:
  中性/微笑/大笑/悲伤/愤怒/恐惧/惊讶/轻蔑/痛苦/决绝

对话面部动画:
  口型同步: 基于音频自动生成(UE5 MetaHuman或第三方)
  眼球: 视线追踪(看对话对象)
  眨眼: 随机间隔2-5秒

过场动画:
  Lv4/Lv5叙事场景 → 手工制作面部动画(关键帧)
  使用UE5 Sequencer录制
```

## 三、布料与物理

```
布料:
  披风/长袍/旗帜: Chaos Cloth
  性能预算: 每角色<2000布料顶点

头发:
  短发: 蒙皮到头部骨骼(无物理)
  长发: 部分骨骼+Chaos Cloth(马尾/辫子)

武器挂件:
  鞘/箭囊: 骨骼挂载(蒙皮)，少量物理震动

链甲/鳞甲:
  下半部分: Chaos Cloth模拟
  上半部分: 蒙皮(性能优化)
```

## 四、性能标准

```
LOD动画:
  LOD0(近景): 完整骨骼+完整BlendShape
  LOD1(中景): 简化骨骼(移除额外骨骼)+简化BlendShape(仅主要表情)
  LOD2(远景): 移除面部+布料 → 仅身体移动
  LOD3(极远): VAT(顶点动画纹理)/Imposter → 无骨骼

动画更新频率:
  LOD0: 每帧
  LOD1: 每2帧
  LOD2: 每5帧
  LOD3: 静态/贴图

骨骼数量上限:
  角色: <150 bones
  马匹: <80 bones
  士兵(近景): <100 bones
```
