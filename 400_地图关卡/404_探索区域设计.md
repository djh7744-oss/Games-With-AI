# 探索区域设计

## 设计目标

> 对标《英雄无敌》的地图探索乐趣。在战略大地图的战争迷雾下，隐藏着各种可探索的地点——山寨、古墓、秘境、隐士居所。这些探索区域不是"任务目的地"——而是玩家自己"发现"的惊喜。

## 一、探索点类型

### 山寨/流寇营地

```
规模: 小型关卡(0.5km×0.5km)
外观: 木栅围墙+瞭望塔+营帐
敌人: 流寇/山贼/逃兵(10-50人，1-3星难度)
探索方式: 进入山寨关卡 → 击败敌人 → 搜刮战利品
奖励: 金+物资+可能俘虏+可能获得情报(山寨头目知道附近有什么)
刷新: 被清剿后30-60天可能被新流寇占据(如果该区域治安<50)
```

### 古墓/王陵

```
规模: 室内关卡(线性迷宫)
外观: 封土堆+石碑+可能有的盗洞
敌人: 机关陷阱(非战斗)/守墓机关人(罕见)/盗墓者(竞争)
探索方式: 进入古墓 → 破解机关(智略判定) → 到达墓室 → 获取宝物
奖励: 古代文物(可卖出高价)/名剑碎片(集齐可重铸)/竹简(科技)
风险: 机关伤害/墓室坍塌/盗墓被发现→仁义声望-10
重要古墓:
  - 某楚王陵(太阿剑线索)
  - 古蜀王墓(三星堆式青铜文明)
  - 始皇陵雏形(架空，可能触发隐藏剧情)
```

### 秘境

```
类型: 
  - 鬼谷(云梦山): 找到鬼谷子 → 学习纵横术
  - 桃花源: 与世隔绝的和平村落 → 仁义+20
  - 龙脉: 风水宝地 → 在此建都可获国运Buff
  - 蓬莱(东海): 海上仙山(架空) → 长生术线索
  - 昆仑虚(西域): 古代遗迹 → 失传科技

发现方式: 极隐蔽。无地图标记。靠NPC对话碎片+古书线索+运气
设计哲学: 只有1%的玩家会发现，但这就是价值所在
```

### 隐士居所

```
类型:
  - 隐士(学者): 可学习智略/辩论技巧
  - 隐士(武将): 可学习武艺/单挑技巧
  - 隐士(工匠): 可学习高级锻造配方
  - 隐士(纵横家): 可学习游说技巧

交互: 与隐士对话 → 可能需要完成一个小考验 →
      隐士传授技能/知识 → 隐士可能离开(去下一个地方云游)
```

### 自然奇观

```
地点(散布在地图上，发现后永久标记):
  - 泰山: 封禅之地(君主可在此祭祀→霸业+20)
  - 黄河壶口瀑布: 壮观景观(首次到访→智略+2)
  - 长江三峡: 险峻水道(首次通过→勇气事件)
  - 华山: 险峻山峰(攀登→体力+5永久)
  - 洞庭湖: 中国最大淡水湖(首次到访→魅力+2)
  - 蓬莱海市: 海上蜃楼(极罕见→运气事件)

设计意图:
  不是"有用"的。没有任何战斗/资源/任务。
  纯粹是为了让玩家感受"这个世界真大"。
  某种意义上是"设计师给玩家的礼物"。
```

## 二、探索奖励设计

```
奖励分级:
  Lv1(常见): 金/物资/普通装备(山寨/流寇营地)
  Lv2(少见): 蓝色品质装备/特殊材料(古墓/山寨头目)
  Lv3(稀有): 紫色品质装备/名剑碎片/隐藏技能(特定古墓/秘境)
  Lv4(传说): 金色品质/永久属性+/改变游戏体验(鬼谷/蓬莱/龙脉)

探索引导:
  绝不直接告诉玩家"这里有什么"
  通过酒馆闲聊/古书片段/NPC对话中的模糊线索
  "听说云梦山有高人..."
  "我在宛城收了一卷古简，上面画了一个奇怪的墓..."
  "爷爷说，他年轻时在东海见过一座山，但第二天就消失了..."

探索日志:
  玩家发现探索点 → 自动记录在"探索日志"中
  包括: 发现日期/地点描述/获得的奖励/你的感想
  这是玩家个人的"探险记录"
```

## 三、探索区域的程序化生成

```
并非所有探索点都是手工设计的:

手工设计: 约20个(古墓/秘境/重要探索点)
  每个都是关卡设计师+叙事设计师合作的成果

程序化生成: 山寨/流寇营地(约50-100个点位)
  模板: 山寨布局×敌人数×奖励等级
  参数: 位置(基于世界状态: 治安低的区域更多流寇)
       敌人强度(基于玩家等级)
       奖励(基于难度)

混合: 约30个(隐士居所/自然奇观)
  位置手工指定(合理的世界位置)
  内容程序化(隐士的类型/教授的技能随机)
```
