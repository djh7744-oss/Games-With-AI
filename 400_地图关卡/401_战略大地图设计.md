# 战略大地图设计

## 设计目标

> 对标《三国群英传》的鸟瞰大地图 +《英雄无敌》的探索乐趣。玩家在战略地图上移动军团、发现资源、遭遇敌军。战争迷雾覆盖未探索区域，城池/关隘/资源点清晰可辨。这是玩家"看天下"的视角。

## 一、视角与操作

```
默认视角: 鸟瞰(俯视45°~90°可调)
可旋转: 右键拖拽旋转
可缩放: 滚轮从近景(500m)到全国视野(50km)
移动: 点击目标地点→军团自动寻路

快捷键:
  Space: 暂停/恢复时间流逝
  M: 切换战略地图/当前任务追踪
  Tab: 切换战争迷雾/势力范围/地形视图
  Ctrl+数字: 保存当前视角(1-4)，按数字键快速切换
```

## 二、地形系统

### 地形类型

```
8种基础地形(影响移速/战斗/伏击):

平原(移速100%，骑兵最佳):
  视觉: 黄绿色开阔地带
  分布: 中原/河北
  战略: 无伏击/骑兵冲锋无惩罚/适合大规模会战

丘陵(移速85%，伏击+15%):
  视觉: 起伏的矮丘
  分布: 过渡地带
  战略: 弓兵射程+10%/轻微伏击可能

山地(移速55%，伏击+40%):
  视觉: 灰色岩体+稀疏植被
  分布: 太行山/秦岭/南方山区
  战略: 弓兵射程+20%/骑兵几乎不可用/强伏击

森林(移速60%，伏击+50%):
  视觉: 深绿色密林
  分布: 楚地/巴蜀
  战略: 火攻伤害+50%/弓兵射程-50%

沼泽(移速35%，重甲不可通过):
  视觉: 暗绿色水洼地
  分布: 云梦泽/沿海滩涂
  战略: 限制重甲/移速极低

河流/渡口(移速30%，攻击方-30%):
  视觉: 蓝色水道+浅滩
  分布: 黄河/长江/淮河/汉水沿岸
  战略: 关键战略节点/必须抢占渡口

沙漠/戈壁(移速70%，体力×2夏):
  视觉: 黄褐色沙地
  分布: 西北边境(少量)
  战略: 限制大规模行军

雪地(移速50%，非耐寒部队士气-2%/日):
  视觉: 白色覆雪
  分布: 燕北/冬季北方
  战略: 燕国优势地形/冬季作战限制
```

### 地形与生物群系(Biome)

```
9大Biome(决定整体视觉+植被+天气):

关中: 黄土高原+渭河平原 → 暖色调/阔叶林/麦田
中原: 黄河冲积平原 → 金黄色/农田/城镇密集
河北: 华北平原北端 → 草原过渡带
山东: 丘陵+沿海 → 绿色+蓝色/鱼盐/城镇富裕
荆楚: 长江流域 → 深绿/水网/雾气/竹林
巴蜀: 四川盆地 → 青绿/梯田/湿润
山西: 黄土高原 → 灰黄/矿产/沟壑
燕北: 草原+山地 → 枯黄/寒冷/人烟稀少
吴越: 江南水乡 → 碧绿/水网/稻田/丝绸
```

## 三、战争迷雾系统

```
迷雾层级:

第一层: 完全未知(黑色)
  从未探索过的区域
  玩家不知道那里有什么——城池? 山脉? 敌军?
  
第二层: 曾经探索(灰色)
  曾经有己方单位经过，但现在没有视野
  地形可见(知道那里有座山)，但单位不可见(不知道现在有没有敌军)
  
第三层: 当前可见(明亮)
  己方单位/盟友/间谍正在提供实时视野
  所有单位/资源点可见

视野来源:
  己方军团: 周围20-50里(取决于军团规模/天气/地形)
  己方城池: 周围15-30里(取决于城池等级)
  斥候/侦察兵: 周围30-80里(速度快但脆弱)
  盟友: 共享盟友视野(需要外交协议)
  间谍: 提供目标城池附近的视野(需要提前派遣)
```

## 四、军团移动

```
移动速度计算:
  基础速度 = 军团中最慢兵种的速度 × 地形修正 × 天气修正
  
行军模式:
  正常行军: 移速100%，体力消耗正常，士气无变化
  急行军: 移速140%，体力消耗×2，到达时疲劳(士气-5)
  谨慎行军: 移速60%，体力消耗×0.5，伏击发现率+30%

军团在地图上的表现:
  单军团: 一个"棋子"(带兵种图标+数量标签)
           秦国锐士图标上写"5000"
  多军团: 多个棋子在附近 → 上级军团长的旗帜更高
  未交战: 棋子按编队排列
  交战中: 棋子停止移动，周围有战斗动画(烟尘/火光微缩)
```

## 五、资源点与地图物件

```
英雄无敌式地图物件(散布在地图上):

资源矿场(占领后每回合产出):
  ⊙ 石矿: 石材50石/月
  ⊙ 铁矿: 铁矿石20石/月
  ⊙ 金矿: 金矿5石/月(高价值)
  ⊙ 伐木场: 木材80石/月
  ⊙ 马场: 战马3匹/月
  ⊙ 盐田: 海盐30石/月

功能建筑(进入后触发):
  🏠 驿站: 快速旅行(传送到另一个已发现的驿站)
  🏠 酒馆: 招募同伴/接任务/打听情报
  🏠 铁匠铺: 修理装备/购买武器
  🏠 药铺: 购买药材/治疗伤病
  🏠 隐士居所: 学习技能/获得任务

遭遇点(接触触发):
  ⚔️ 流寇营地: 1-3星，击败获得战利品
  ⚔️ 山贼山寨: 3-5星，击败获得大量战利品+俘虏
  🏴 古墓: 探索获得宝物/名剑碎片
  🏴 遗迹: 探索获得科技/竹简
  📜 智慧石: 触碰后某属性+1(一次性)
  🗿 古碑: 阅读后获得历史知识(文化+)
  ⛲ 温泉: 军团体力恢复+50%(一次性)

刷新规则:
  资源点: 固定位置，占领后持续产出
  遭遇点: 每30天刷新一次(不同位置)
  一次性物件(智慧石/古碑): 使用后消失，不再刷新
```

## 六、地图信息层级

```
玩家可切换的图层:

【地形层】默认: 显示地形+城池+己方军团
【势力层】: 各国领土用不同颜色标记
          秦=黑/楚=深红/齐=蓝/赵=绿/燕=灰/魏=紫/韩=橙/周=黄
【外交层】: 显示各国关系
          绿色连线=盟友/红色连线=战争/黄色虚线=条约/无色=中立
【商路层】: 显示可用的商路
          粗线=安全/虚线=危险/红线=中断
【科技层】: 显示科技传播方向
          箭头从高科技国指向低科技国
【迷雾层】: 仅显示战争迷雾(不显示其他信息)
```
