# 城池与据点设计

## 设计目标

> 对标《GTA V》的城市内部自由探索。每一座城不只是一个"图标"——走进城门后是另一个世界。市集喧嚣、酒馆热闹、官衙肃穆、城墙巍峨。40+主城各有特色: 咸阳的威严、临淄的繁华、郢都的神秘、邯郸的坚韧。

## 一、城池分级与布局

### 五级城池

```
Lv1 村庄(×20+): 人口<500
  建筑: 民居×5-8/水井/打谷场
  功能: 征兵基础/粮食收购/基础情报
  特色: 田园风光，村民日常劳动
  加载: 极简(总资产<50个物件)

Lv2 乡镇(×15+): 人口500-2000
  建筑: 民居×15-20/市集雏形/铁匠铺/酒馆/乡亭
  功能: 基础贸易/基础锻造/基础任务
  特色: 小市集的热闹

Lv3 县城(×20+): 人口2000-5000
  建筑: 城墙(夯土)/官衙/兵营/市集/酒馆×2/学堂/药铺
  功能: 完整商业/锻造/任务/驻军
  特色: 麻雀虽小五脏俱全

Lv4 郡城(×15+): 人口5000-15000
  建筑: 砖石城墙/瓮城/官衙/大兵营/大市集/商会/酒馆×3/
        学宫/钱庄/驿站/仓库/铁匠铺×2/
  功能: 区域中心/高级商业/高级锻造/高级任务/学派活动
  特色: 繁华的战国城市

Lv5 都城(×7): 人口15000-35000
  建筑: 砖石城墙+瓮城+护城河/王宫/朝堂/大兵营/大市集/
        商会总部/酒馆×4/稷下学宫(齐)/黄金台(燕)等独特建筑/
        钱庄/驿站/仓库/铁匠铺×3/药铺×2/奢侈品店
  功能: 全部功能最顶级
  特色: 每座都城有独特的"地标建筑"和视觉风格
```

### 七国都城独特地标

```
咸阳(秦): 咸阳宫(黑色+金色，严肃规整)
          四方馆(接待六国使臣和客卿的地方)
          兵器工坊(秦国标准化兵器生产中心)

郢都(楚): 楚王宫(红色+金色，繁复华丽)
          云梦台(楚王宴饮和祭祀的高台)
          漆器工坊(楚国特产)

临淄(齐): 齐王宫(蓝白色，雅致富丽)
          稷下学宫(战国最大的学术中心)
          海盐市(齐国海盐交易中心)

邯郸(赵): 赵王宫(绿色，质朴坚固)
          丛台(赵武灵王观看骑兵操演的高台)
          马市(战马交易中心)

大梁(魏): 魏王宫(紫色，庄重文雅)
          鸿沟码头(连接黄河与淮河的运河码头)
          书肆街(魏国文化出版中心)

蓟城(燕): 燕王宫(灰色，低调内敛)
          黄金台(燕昭王筑台招贤——即使燕昭王已不在，台还在)
          辽东马场(燕国战马驯养地)

新郑(韩): 韩王宫(橙色，精巧细致)
          宜阳冶铁坊(韩国冶铁中心——游戏中最先进的铁匠铺)
          劲弩工坊(韩国弩兵装备的生产中心)
```

## 二、城池内部设计

### 功能区域划分

```
每座Lv3+城池包含以下区域:

【官衙区】政治中心
  建筑: 官衙/朝堂(都城)/监狱/驿传
  NPC: 官员/卫兵/求见的百姓
  活动: 接官府任务/举报犯罪/交税/申请觐见君主
  氛围: 严肃/安静

【市集区】商业中心
  建筑: 市集广场/商铺×N(不同商品)/钱庄/仓库
  NPC: 商人/顾客/搬运工/乞丐
  活动: 买卖/讨价还价/打听物价情报/接商会任务
  氛围: 嘈杂热闹
  时间: 早市(8-12点最热闹)/晚市(16-19点)

【工匠区】手工业中心
  建筑: 铁匠铺/制甲坊/纺织坊/陶瓷坊
  NPC: 工匠/学徒
  活动: 锻造/修理/定制装备/学习工艺
  氛围: 叮当锤声

【学宫区】文化中心(Lv4+)
  建筑: 学宫/书肆/辩论场
  NPC: 学者/学生/学派人士
  活动: 学习/辩论/著书/接学宫任务
  氛围: 安静/书香
  仅在部分大城有

【酒馆/客栈区】社交中心
  建筑: 酒馆×N/客栈
  NPC: 酒馆老板/旅人/游侠/情报贩子
  活动: 喝酒/住宿/打听情报/接酒馆任务/招募同伴
  氛围: 温暖/热闹(夜间最活跃)

【军营区】军事中心
  建筑: 兵营/训练场/武库/马厩
  NPC: 士兵/军官/新兵
  活动: 征兵/训练/购买军备
  氛围: 纪律严明

【城墙上】防御设施
  建筑: 城墙/箭塔/城门楼/瓮城
  NPC: 守城士兵/巡逻队
  活动: 巡逻/防御战
  视野: 可观城外远景
```

### NPC分布密度

```
不同区域的NPC密度(以Lv4郡城为例):

市集区(早市): 80-150人 (最密集)
市集区(傍晚): 40-60人
官衙区: 10-20人
工匠区: 15-30人
学宫区: 10-25人
酒馆区: 20-40人(夜晚更多)
军营区: 20-50人
城墙: 每100米1-2人

性能优化:
  远景NPC(>50m): 使用LOD模型+简化AI(每3秒一次决策)
  中景NPC(20-50m): 标准AI
  近景NPC(<20m): 完整AI+可互动
```

## 三、关隘与渡口

### 关隘设计

```
十大名关(战略要地，独立的Lv3-Lv4级要塞):

函谷关(秦): 战国最重要的关隘
  地形: 两侧绝壁，中间狭窄通道
  防御: 城墙横亘山谷，箭塔密布
  战役: 从未被攻破的传奇
  
武关(秦楚): 秦岭中的险关
  地形: 山地+森林
  特色: 秦楚战争的关键通道

井陉(赵): 太行八陉之一
  地形: 山中峡谷
  特色: 赵国通往山西的关键通道

居庸塞(燕): 燕山中的关隘
  地形: 山地
  特色: 燕国北疆屏障

虎牢关(韩/周): 洛阳东面屏障
  地形: 黄河渡口+关隘
  特色: 《三国演义》中吕布战三英的地方(但战国时已存在)

萧关(秦): 秦国北方屏障
大散关(秦): 秦蜀通道(通往巴蜀)
方城(楚): 楚国北方防线
平阴(齐): 齐国西面屏障
雁门(赵): 赵国北疆关隘(李牧在此防御匈奴)
```

### 渡口设计

```
主要渡口(控制河流交通):

孟津(黄河): 黄河最重要的渡口
  位置: 洛阳北
  战略: 控制孟津=控制黄河中游的通行

蒲坂(黄河): 秦晋之间
  位置: 关中通往山西
  战略: 秦国东出需要控制此渡口

白马津(黄河): 中原北部
  位置: 魏国北部
  战略: 魏国防御北方的重要渡口

采石(长江): 楚国南方
  位置: 长江下游
  战略: 楚国南方防线

夏口(长江): 楚国中部
  位置: 汉水入长江处
  战略: 楚国水军基地
```

## 四、城池昼夜循环

```
时间影响城池氛围:

4:00-6:00 黎明: 城门开/商人开始摆摊/炊烟升起
6:00-8:00 清晨: 农民进城/市集开始/士兵换班
8:00-12:00 早市: 最热闹的时段/所有商铺营业
12:00-14:00 午时: 短暂的安静(午饭时间)
14:00-18:00 午后: 持续营业/工匠最忙的时段
18:00-20:00 黄昏: 市集收摊/晚市开始
20:00-24:00 夜晚: 酒馆最热闹/一些商铺关门/巡逻加强
0:00-4:00 深夜: 几乎无人/只有更夫和巡逻兵/盗贼活动

天气影响:
  晴天: 标准氛围
  雨天: NPC减少30%/街道泥泞/酒馆更拥挤
  雪天: NPC减少50%/火盆取暖/少数NPC在户外
```
