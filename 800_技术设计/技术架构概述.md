# 技术架构概述

## 引擎与语言

- **引擎**：Unreal Engine 5.5+
- **核心语言**：C++ (游戏逻辑) + Blueprint (脚本/UI/快速迭代)
- **数据驱动**：DataTable / DataAsset / CSV
- **版本控制**：Git + Git LFS (或 Perforce)

## 模块架构

```
WarringStates.uproject
├── Source/
│   ├── WarringStates/              # 核心游戏模块
│   │   ├── Core/                   # 基础框架(GameMode/GameState/GameInstance)
│   │   ├── Player/                 # 玩家角色(Character/Controller/Camera)
│   │   ├── Combat/                 # 战斗系统
│   │   ├── AI/                     # AI系统(BehaviorTree/EQS/StateTree)
│   │   ├── Economy/                # 经济系统
│   │   ├── Diplomacy/              # 外交系统
│   │   ├── Strategy/               # 战略地图
│   │   ├── Narrative/              # 任务/对话
│   │   ├── TechTree/               # 科技树
│   │   └── UI/                     # UI系统
│   └── WarringStatesEditor/        # 编辑器工具
├── Content/
│   ├── Characters/                 # 角色资产
│   ├── Maps/                       # 地图/关卡
│   ├── Weapons/                    # 武器
│   ├── UI/                         # UI资源
│   ├── VFX/                        # 特效
│   ├── Audio/                      # 音频
│   └── DataTables/                 # 数据表
└── Plugins/                        # 第三方插件
```

## 关键技术决策

| 决策点 | 选择 | 理由 |
|--------|------|------|
| 大世界方案 | World Partition | UE5原生支持，自动流式加载 |
| 士兵渲染 | Mass Entity (远景) + Skeletal (近景) | 千人战性能需求 |
| AI框架 | Behavior Tree + State Tree + EQS | 战场AI + 战略AI分层 |
| 存档方案 | SaveGame + SQLite | 支持复杂数据 + 快速查询 |
| 对话系统 | 自研 Dialogue Tree | 需要高度定制(分支+条件+滚动) |
| UI框架 | Common UI + UMG | UE5推荐方案 |
| 网络 | 单机（首发），预留多人接口 | 首发聚焦单机体验 |
| 音频 | Wwise (如预算允许) / MetaSounds | 动态音乐+环境音效 |

## 性能目标

| 配置 | 分辨率 | 帧率 | 战场规模 |
|------|--------|------|---------|
| 最低 | 1080p | 30 FPS | 100 vs 100 |
| 推荐 | 1440p | 60 FPS | 300 vs 300 |
| 理想 | 4K | 60+ FPS | 1000 vs 1000 |

## 数据流架构

```
策划 (Excel/CSV) → UE5 DataTable → C++/Blueprint → 运行时
                                       ↕
美术 (Blender/Substance) → FBX/PNG → UE5 Asset → 运行时
                                       ↕
音频 (DAW/Wwise) → WAV/BNK → UE5 Audio → 运行时
```
