# 架空历史分支

## 设计目标

> 游戏不是历史教材。"如果当时..."是架空历史的核心命题。玩家可以改写长平之战的结果、拯救被赐死的白起、让屈原不去投江、帮助韩国在夹缝中崛起。每个架空分支都需要有逻辑推导——不是"魔法改变历史"，而是"在那个时代，如果做出了不同的选择，事情确实可能朝另一个方向发展"。

---

## 架空分支设计原则

```
1. 基于历史的合理推导
   不是"如果赵括突然变聪明了"，而是"如果赵括在被围前听取了部下的建议"
   不是"如果韩国突然变强了"，而是"如果韩国的劲弩技术被提前量产"
   架空分支的起点必须是"历史上真实存在的可能性"

2. 蝴蝶效应
   一个选择的改变 → 引发连锁反应 → 整个历史走向不同
   例: 赵括没死在长平 → 赵国保留更多兵力 → 
       邯郸保卫战压力减小 → 信陵君不需要窃符救赵 →
       魏国政局不受影响 → ...最终可能改变统一进程

3. 有限度的架空
   不"为所欲为"。游戏的架空限于"战国格局的变化"——
   不出现"外星人降临帮助韩国"、"魔法改变战争"
   尊重历史的"结构性力量"——秦国确实有地理、制度、资源的结构性优势
   架空不是取消这些优势，而是给其他国家一个"可能对抗"的机会

4. 代价原则
   改变历史是有代价的——通常是巨大的代价。
   救白起 → 得罪秦国君主 → 你成为秦国公敌
   改变长平 → 改变历史进程 → 面对不可预知的连锁反应
   大团圆结局不存在。每个"好的改变"都有对应的代价。

5. 蝴蝶效应的可视化
   每次重大架空选择 → 触发"历史分叉"界面
   显示: "如果选择了A → 会发生..."
   但不显示B路线的内容(保持神秘感)
```

## 十大架空分支 (Key What-If Scenarios)

### 分支1: 长平之战——如果赵括赢了?

```
历史: 赵括全军覆没，赵军40万被坑杀，赵国元气大伤

架空触发: DLC长平之战中，赵括在关键节点做出不同选择:
  - 未全军渡河追击 (保持主力在安全区域)
  - 识破白起的佯败 (智略判定)
  - 在包围圈合龙前突围成功 (统率判定)

架空结果:
  短期(1-5年):
    赵国保留30万+主力 → 秦国长平之战未取得决定性胜利
    白起被秦国召回问责 → 秦王怀疑白起"故意不胜"
    赵国的国际地位上升 → "唯一正面挡住了秦国的国家"
    
  中期(5-15年):
    秦国转向先灭韩魏 → 赵国得到喘息
    赵国利用喘息期 → 胡服骑射深化 → 骑兵进一步强化
    但赵国朝堂斗争依旧 → 李牧 vs 郭开 → 内部矛盾爆发
    
  长期(15-30年):
    秦国统一进程被延迟10-15年
    但秦国的结构性优势(制度/地理/资源)依然存在
    最终秦国仍可能统一——但赵国将成为最后被灭的国家
    (历史上赵国是第二个被灭的)
    
  极端架空(概率极低):
    赵国联合楚魏 → 反攻秦国 → 攻入函谷关 →
    秦国退回关中 → 战国格局回到"均势"状态 →
    可能再延续100年的分裂时代

游戏内效果:
  → 赵国兵种获得永久BUFF"长平幸存者之魂"
  → 白起在本体中"可招募"条件改变 (被秦王冷落后可能出走)
  → 秦灭赵的触发时间延迟
```

### 分支2: 荆轲刺秦——如果他成功了?

```
历史: 荆轲刺秦失败，图穷匕见，被秦王斩杀

架空触发: 荆轲刺秦成功 (匕首刺中秦王 vs 刺杀失败)
  成功条件(极苛刻):
    荆轲: 武勇判定 > 秦王的警觉判定
    如果玩家陪同: 双人配合 → 成功率翻倍但不保证
  
刺杀成功的后果:
  短期:
    嬴政死亡 → 秦国陷入继承危机
    嬴政的儿子们争夺王位 → 可能内战
    秦国暂停对外扩张 → 六国获得喘息之机
    
  中期:
    李斯(秦国重臣)可能扶持傀儡继位 or 自立 → 秦国政治动荡
    如果秦国分裂内战 → 六国趁机反攻 → 可能夺回部分领土
    
  长期:
    即使嬴政死了 → "统一"的历史趋势仍可能由他人完成
    秦国积累的制度优势不会因为一个君主死亡而消失
    "始皇帝"这个名字不会出现 → 但"统一者"仍会出现
    
  最极端的架空:
    秦国陷入长期内战 → 分裂为多个小国 →
    赵国/楚国成为新的最强国 → "统一者"来自赵国或楚国 →
    中国历史完全改写 → 可能出现"楚始皇"或"赵始皇"

游戏内效果:
  → 秦国陷入"继承战争"事件 → 所有秦将忠诚-30%
  → 如果你参与刺杀→仁义-100，武名永久+100
  → 解锁隐藏结局: "弑君者的诅咒"
```

### 分支3: 屈原不死——如果他选择了活下去?

```
历史: 屈原投汨罗江自尽

架空触发: 在屈原投江前与他对话 → 说服他"活下去"
  说服条件: 魅力≥75 + 智略≥65 + 对楚文化有深刻理解
            (不是简单的"别死"——需要给他一个"活下去的理由")

活下去的屈原:
  他不再参与楚国政治 → 但他继续写作
  他的后期作品更加深沉 ——"活着的痛苦比死亡更需要勇气"
  他成为楚国文化的象征 → 激励了几代楚人
  当楚国被灭时 → 屈原写了一篇《哀郢》(历史上真有其作)
  他活到了楚国灭亡 → 见证了项燕的最后一战 →
  在楚国灭亡后，他默默地做了一个"楚文化的记录者"——
  把楚国的历史、文化、传说记录下来 → 留给了后世。

游戏内效果:
  → 屈原继续创作 → 楚国文化影响力持续上升 → 影响战后文化格局
  → 解锁屈原后期著作《楚辞续编》
  → 如果玩家是楚国人 → 楚国灭亡后 → 获得特殊传承
```

### 分支4: 韩信如果崛起——小国的逆袭?

```
历史: 韩国是第一个被秦灭掉的(前230年)

架空奇迹: 如果韩国在某个关键节点"翻盘"
  翻盘的可能路径:
    1. 宜阳冶铁技术被提前武器化 → 韩国劲弩大规模量产
       → 韩军防御能力质变 → 秦国攻韩代价过高
       
    2. 韩非的政治主张被韩王采纳 → 韩国变法成功
       → 韩国从"恐惧求生"变成"坚强自保"
       
    3. 韩国利用列强矛盾 → 在大国之间巧妙周旋
       → "以小事大"的极致智慧 → 成为谁也吞不掉的硬骨头

架空结果:
  韩国可能不会成为"统一者"——实力差距太大
  但韩国可能成为"最后一个被灭的国家"
  甚至在统一进程中"主动投降换取优待"
  → 韩国宗庙得以保存 (历史上被彻底消灭)

游戏内效果:
  → 韩国兵种获得永久BUFF"劲弩天下"(守城无敌)
  → 韩非不死 → 继续著书 → 法家思想提前成熟
  → 解锁隐藏政策"韩国模式"——以技术优势弥补国力劣势
```

## 架空分支的实现

```
技术实现:
  1. 条件检测: 系统持续监测"架空触发条件"是否满足
  2. 分支激活: 条件满足 → 历史分叉事件触发
  3. 蝴蝶效应链: 一系列后续事件自动引发
  4. 历史偏差值: 记录"偏离历史程度" → 影响结局评价

历史偏差值 (Divergence Score):
  0-20: 微调历史 (人物命运不同但大势不变)
  20-50: 显著偏离 (某个国家命运改变)
  50-80: 重大偏离 (整个国际格局不同)
  80-100: 完全架空 (战国历史彻底改写)
  
  偏差值越高 → 结局越独特 → 但逻辑一致性要求越高
  (保证高偏差值的结局仍是"合理的",而非"随机的")
```
