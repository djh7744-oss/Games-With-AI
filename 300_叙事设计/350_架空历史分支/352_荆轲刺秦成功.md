# 荆轲刺秦成功

## 历史分叉点

```
历史: 图穷匕见→荆轲抓住秦王袖子→秦王挣脱→绕柱而逃→荆轲被斩杀

分叉条件(概率极低，需要多重因素同时成立):
  1. 荆轲武勇判定 > 秦王警觉判定(基础概率约15%)
  2. 如果玩家陪同荆轲 → 成功率翻倍(约30%)
  3. 如果使用真·鱼肠剑(非仿品) → 成功率额外+15%
  4. 秦王的御医不在场(药囊投掷是历史上救了秦王的关键)
  
  最大成功率: 45%(双人+真鱼肠+御医不在)
  最小成功率: 8%(单人+仿品+正常情况)
```

## 刺杀成功的瞬间

```
咸阳宫。大殿。

荆轲左手抓住秦王的衣袖，右手从地图尽头抽出匕首。
匕首的尖端泛着幽蓝的光——淬了剧毒。

秦王嬴政想挣脱——但这一次，袖子没有撕裂。
匕首刺入了秦王的胸膛。

大殿死寂。

荆轲松开了秦王的袖子。秦王踉跄后退——手捂着胸口——看着荆轲。
嬴政: "...你是谁?"
荆轲: "燕人荆轲。"
嬴政: "为什么?"
荆轲: "为了燕国。为了六国。为了天下不再有战争。"
嬴政倒下了。

大殿里所有人都愣住了。然后——
一个太监尖叫:"刺客! 抓刺客!"
卫兵涌入。荆轲没有抵抗。他靠在铜柱上，和历史上一样靠着铜柱。
只是这一次——他成功了。
```

## 连锁反应

```
第一天: 咸阳宫封锁。嬴政死亡的消息暂时保密。
第二天: 消息泄露。咸阳震动。六国震动。
第三天: 秦国朝堂分裂——谁来继承王位?

继承危机:
  嬴政的长子扶苏(在北方监军)→ 被赵高/李斯秘不发丧
  幼子胡亥→ 赵高扶持
  嬴政的其他儿子→ 各有支持者
  
  (历史上这正是秦二世而亡的原因之一——继承制度不明确)

一个月后: 秦国陷入内战。
  扶苏在蒙恬的支持下 → 宣布自己是合法继承人
  胡亥在赵高/李斯的支持下 → 据守咸阳
  双方对峙。秦国的统一进程——暂停了。

六国的反应:
  赵国: "天赐良机! 收复失地!"
  楚国: "秦国乱了! 我们的机会来了!"
  燕国: 太子丹得知荆轲成功→痛哭→"壮士..."
       但燕国不敢居功——怕秦国报复
  齐国: 观望。"等他们打完再说。"
  韩国: (已经被灭了，无法反应)
  魏国: 趁机收复部分领土

一年后: 秦国仍然在内战。但谁也没有能力灭掉对方。
        扶苏控制北方(蒙恬军团)，胡亥控制关中(秦国的核心)。
        六国开始收复失地。秦国十五年的扩张——在一年之内丧失大半。

十年后: 战国格局回到了约前250年的状态。
        秦国不再是"不可战胜的虎狼之国"。
        六国各自恢复。
        但——他们又开始互相攻打了。
        老毛病又犯了。

三十年后: 历史的"惯性"开始显现。
          秦国的制度优势依然存在(商鞅变法没有因为嬴政死亡而消失)。
          某个新的霸主会崛起。
          可能是秦国——如果在内战中胜出。
          可能是赵国——如果趁机做大了。
          可能是楚国——如果终于解决了内部问题。
          
          统一仍然会发生。但不是"秦灭六国"。
          可能是"赵灭六国"。可能是"楚灭六国"。
          
          始皇帝不叫嬴政。叫别人。
```

## 荆轲的结局

```
荆轲被秦军抓住——但没有被立刻处死。

因为秦国在内战中，没人有心思处理他。
他被关在咸阳的大牢里。

几个月后。扶苏(在内战中占据优势后)提审了荆轲。
扶苏: "你杀了我父亲。"
荆轲: "是的。"
扶苏: "我该杀你。但我想知道——值得吗?"
荆轲: "我不知道。我只知道燕国还能活几年。"
        "至于天下——天下永远不会因为一个人的死而太平。"
扶苏沉默了很久。最后说: "放了他。"

荆轲被释放了。他走出了咸阳的大牢。
外面——咸阳的街道。人们还在生活。世界还在运转。
他杀了一个皇帝。世界没有变得更好。也没有变得更坏。
只是不同了。

荆轲回到了燕国。太子丹在易水边等他。
太子丹: "你回来了。"
荆轲: "回来了。"
太子丹: "...值得吗?"
荆轲没有回答。他看着易水——和出发那天一样的水。
高渐离击筑。不再是变徵之声——而是缓慢的、悠长的声音。
```

## 游戏内实现

```
触发条件:
  DLC长平之战后的"荆轲刺秦"特殊章节
  玩家可以选择:
    A. 扮演荆轲 → 执行刺杀
    B. 扮演陪同者 → 双人刺秦
    C. 扮演劝阻者 → 尝试改变历史

成功率计算:
  基础15% + 双人+15% + 真鱼肠+15% + 御医不在+5% = 最大50%

蝴蝶效应:
  成功 → 秦国继承战争事件链
        → 六国收复失地 → 国际格局重置
        → 统一延迟30-50年
        → 统一者可能不是秦

游戏结局分支:
  "荆轲的遗产"结局: 如果秦国最终被赵国或楚国统一
  "始皇帝不叫嬴政"结局: 如果新统一者来自其他国家
  "永远的战国"结局: 如果统一从未发生(概率极低的隐藏结局)
```
