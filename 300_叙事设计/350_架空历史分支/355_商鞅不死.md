# 商鞅不死

## 历史分叉点

```
历史: 秦孝公死后→商鞅被诬告谋反→逃亡→被抓→车裂

分叉条件:
  节点A: 秦孝公临终前——留下了保护商鞅的遗诏
         触发: 秦孝公与商鞅关系≥90 + 商鞅变法成功度>80%
  节点B: 商鞅逃亡时——成功逃出秦国
         触发: 商鞅智略判定 > 追兵的搜索
  节点C: 新君秦惠文王选择——不杀商鞅
         触发: 玩家(如果在秦国为官)成功说服秦王
```

## 分叉A: 遗诏保护

```
秦孝公临终。商鞅跪在榻前。
孝公: "商君...我死后，他们不会放过你的。"
商鞅: "臣知道。但变法已成。臣死而无憾。"
孝公: "不。你不能死。"
      孝公从枕下取出一卷诏书。
      "这是寡人的遗诏。新君继位后——商鞅归隐，任何人不得加害。"
      商鞅接过诏书。他的手在颤抖。

秦惠文王继位。旧贵族们欢呼——可以杀商鞅了。
然后遗诏被宣读。贵族们的笑容凝固了。
惠文王沉默了很久。最终说:"先王之命。寡人不敢违。"
商鞅被允许活着。但他必须离开秦国。

归隐后的商鞅:
  商鞅去了魏国(他年轻时待过的地方)。隐居在西河。
  他不再参与政治。但他继续写作——
  把变法的经验、教训、所有的细节写下来。
  
  这部著作比《商君书》更完整、更坦诚。
  里面不仅写了"如何使国家强大"——
  还写了"变法的代价是什么"。
  
  他开始反思: "我制定的法律太严苛了。"
  "人们遵守法律不是因为认同，是因为恐惧。"
  "恐惧维持的秩序——能持续多久?"
  
  这部反思著作被秘密保存。直到他死后才被发现。
  后世称其为《商君后书》——法家思想从"严刑峻法"到"宽严相济"的转型。

商鞅活到了七十多岁。他看到了秦国统一天下。
他看到的秦法——已经不是他当年制定的模样。
变了很多。松弛了很多。但也保留了核心。
商鞅说:"法必须变。不变的法，不是好法。"
这是他一生的总结。
```

## 分叉B: 逃亡成功

```
商鞅在被追捕时——找到了他的一个老部下。
老部下: "商君，往这边走。有一条小路出关。"
商鞅: "你为什么要帮我? 我是全国通缉犯。"
老部下: "因为您制定的法律——让我这个平民的儿子，有机会当上军官。"
        "我欠您的。"

商鞅逃出了秦国。逃到了楚国。
楚悼王已经死了。吴起的变法也失败了。但楚国的贵族还记得吴起。
他们看到一个和吴起同样命运的人——变法者，被旧势力反扑。
楚国贵族没有收留商鞅。但也没有出卖他。
商鞅在楚国的一个小村庄里隐居下来。

他每天都在反思: 为什么变法者都没有好下场?
吴起被射杀。自己被车裂(如果没逃出来的话)。
"变法的根本问题——不是法律本身。是既得利益者。"
"他们不会自愿放弃特权。法律再多，他们也能找到漏洞。"

商鞅在楚国写下了他的最后著作——不是关于法律，是关于"利益"。
"天下的根本问题不是'没有好法律'，
 而是'好法律被坏利益阻止'。"
 
这成了中国政治思想史上最早的利益集团分析。
```

## 分叉C: 秦王不杀

```
玩家在秦国为官。你站在年轻的秦惠文王面前。
旧贵族们已经递上了弹劾商鞅的奏章——堆得像小山一样。

秦王: "你也来劝寡人杀商鞅?"
玩家: "不。臣来劝大王不杀。"

秦王: "为什么?"
玩家(三个选项):
  A. "商鞅之法使秦国强大。杀商鞅——等于告诉天下，
      为秦国做出贡献的人不得好死。以后谁还敢为秦国效力?"
  B. "大王可以保留商鞅的法，杀掉商鞅的人。
      但大王想过没有——商鞅死了，他的学生还在。
      他们会记住——是谁杀了他们的老师。"
  C. "大王。商鞅已经老了。让他在软禁中度过余生。
      既不违背旧贵族的诉求，也不背负'杀功臣'的恶名。"

秦王沉默了很久。最终点头。

商鞅被软禁在咸阳的一处宅院中。他可以读书、写作、会客。
但不能离开咸阳。
旧贵族们不满意——他们想看到商鞅死。
但秦王不允许。商鞅活着，但"消失"了。

五年后。商鞅在软禁中自然死亡。
临死前，他把自己的变法笔记交给了秦王。
笔记的最后一页写着:
"臣一生所为，得罪了天下所有贵族。
 但臣所立之法，让秦国的每一个平民有了上升的通道。
 大王。不要让这条路断了。
 这条路断了——秦国就回到了商鞅变法之前。"
```

## 游戏内实现

```
触发条件:
  - 玩家在秦国为官 + 秦孝公在位期间与商鞅建立高关系
  - 或: 新君继位时玩家成功劝谏
  - 或: 玩家帮助商鞅逃亡

蝴蝶效应:
  商鞅不死 → 法家思想继续发展(从"严苛"到"宽严相济")
           → 秦国变法更加完善
           → 统一后的秦法可能不那么严苛
           → 秦朝的寿命可能延长

  商鞅反思著作 → 后世影响 → 法家与中国政治更深层的融合
  
  如果商鞅逃亡到其他国家 → 可能帮助其他国家变法
                        → 改变秦国的"制度独大"优势
```
