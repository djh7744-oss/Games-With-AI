# 纵横家与奇人传记

> 他们不属于任何一个国家——他们属于"天下"。他们的武器是三寸不烂之舌，他们的战场是君主的朝堂。一句话可以让两国结盟，一句话可以让百年和平化为灰烬。他们是纵横家。

---

## 一、鬼谷子 —— 隐于云雾中的百家之师

```
没有人知道他真正的名字。
他是苏秦的师父，张仪的师父，孙膑的师父，庞涓的师父。
他教出了四个改变战国格局的人——然后消失了。
                                   ——关于鬼谷子

基本信息:
  生卒: 不详 (神秘到连生卒都无考)
  国籍: 不详 (据说是周人或楚人)
  最高官职: 无(终身不仕)
  历史定位: 纵横家之祖/孙膑庞涓苏秦张仪的共同老师

出身与早年:
  关于鬼谷子，历史留下的信息极少——少到像一个传说。
  他的名字来自他隐居的地方: "鬼谷"(在今河南云梦山)。
  他可能姓王名诩，也可能不叫这个名字。没有人确定。
  
  他一生隐居，不入仕途，不当官，不参战。
  他做的一件事是: 收徒，教学。
  他教出来的四个学生:
  - 苏秦: 佩六国相印，合纵联盟的缔造者
  - 张仪: 秦国丞相，连横策略的执行者
  - 孙膑: 军事天才，《孙膑兵法》作者
  - 庞涓: 魏国名将(虽然结局不好)

  这四个人加起来，主导了战国中期几乎所有重大事件。
  而他们的老师，鬼谷子，却像一个影子——只存在于他们的回忆中。

性格画像:
  鬼谷子的真实性格已成谜。但从他的教学方法可以推断出一些东西。
  他同时教了苏秦(合纵)和张仪(连横)——两个完全相反的策略。
  他同时教了孙膑(善)和庞涓(恶)——两个最终反目的同门。
  
  他似乎不在乎学生们"学成之后站哪一边"。
  他只是教给他们"术"和"道"——至于怎么用，是他们自己的选择。
  这是一种极其超脱的教育理念——也是一种极其冷酷的放任。

在游戏中的呈现:
  出现场景: 鬼谷(地图隐藏地点，云梦山深处)
  不可招募(他不属于任何势力)
  可以: 找到他→学习→获得全部纵横术和兵法的"源泉"
  发现方式: 拼凑碎片线索→找到鬼谷→"你找到我，便是缘分"
  教学: 在鬼谷跟随他学习7天→每天获得一项技能
        第7天→他说"你该走了"→赠你《鬼谷子》→然后
        "下次来，你可能就找不到我了。"
  这是全游戏最隐蔽的隐藏内容——绝大多数玩家通关都不会发现
```

## 二、苏秦 —— 六国相印

```
"使我有洛阳二顷田，安能佩六国相印?"
  (如果我在洛阳有两顷田地，怎么会去佩六国的相印呢?)
                                   ——苏秦

基本信息:
  生卒: ? - 前284年(被刺杀)
  国籍: 周(洛阳人)→六国
  最高官职: 同时担任六国丞相(史无前例后无来者)
  历史定位: 合纵策略的缔造者/纵横家第一人

出身与早年:
  苏秦是洛阳人，出身贫寒。年轻时他在鬼谷子门下学习纵横术。
  学成后，他四处游说——但没人听他的。穷困潦倒地回到家乡，
  妻子不理他，嫂子不给他做饭，父母不跟他说话。
  
  苏秦受到了极大的刺激。他把自己关在屋子里，
  翻出师父给的《鬼谷子》。读了一遍又一遍。
  困了就用锥子刺自己的大腿——血流到脚跟，剧痛让他清醒。
  这就是"悬梁刺股"中"刺股"的由来。
  
  一年后，苏秦再次出发。这一次，他成功了。

性格画像:
  苏秦是一个被"穷"和"被看不起"驱动的人。
  他追求的不是权力本身——而是"证明自己"。
  这就是他说"使我有洛阳二顷田"的真正含义:
  如果我有退路，我就不会拼到这种程度。因为没有退路，所以必须成功。
  
  这种动机让他成为了最伟大的外交家，
  但也让他成为了一个无法停下来的人。

关键人生节点:

  游说六国: 苏秦首先去了燕国。他对燕文公说:
            "秦国之所以不打燕国，是因为中间隔着赵国。
             如果赵国灭亡，燕国就是下一个。"
            燕文公被说服→资助苏秦去游说其他国家。
            
            苏秦去了赵国、韩国、魏国、齐国、楚国——
            对每个国家他说了不同的"故事"，但核心逻辑都一样:
            "秦国太强了，你单独打不过。合纵——大家联合起来。"
            
            奇迹发生了。六国君主被他说服。
            苏秦成为了"纵约长"(合纵联盟的秘书长)，
            同时担任六国的丞相。
            
            他回洛阳时，各国都派使者护送。
            周天子听说他回来了，命人清扫街道，
            亲自到郊外迎接。他的家人跪在路边不敢看他。
            
            苏秦笑着问嫂子:"为什么以前不理我，现在跪我?"
            嫂子说:"因为你以前穷，现在有钱有地位..."
            苏秦笑了。这句话概括了他的一生。

  被刺杀: 合纵联盟维持了15年后崩溃(张仪的连横策略瓦解了它)。
          苏秦在齐国被刺客刺伤，临终前他对齐王说:
          "我死后，把我的尸体车裂，对外说'苏秦是燕国间谍'——
           刺客听到后一定会站出来领赏。"
           齐王照做了。刺客果然自投罗网。
           苏秦在临死前，用最后一计为自己报了仇。

在游戏中的呈现:
  招募条件: 找到鬼谷→获得苏秦线索→发起合纵(★★★★★)
           (极难: 苏秦不效忠任何一国，他只在合纵联盟中存在)
  可学技能: 六国相印(可同时任多国官职)/悬梁刺股(学习速度+50%)
  专属剧情: 发起合纵联盟→苏秦主导→你作为他的助手或副使
```

## 三、张仪 —— 欺楚的纵横家

```
"只要我舌头还在，就够了。"
             ——张仪(年轻时被打后对妻子说的话)

基本信息:
  生卒: ? - 约前310年
  国籍: 魏→秦
  最高官职: 秦国丞相
  历史定位: 连横策略的缔造者/苏秦的对手与朋友

出身与早年:
  张仪年轻时在楚国，有一次和楚相一起喝酒。
  楚相丢了一块玉璧，门客们怀疑是张仪偷的:"张仪穷，品行不好!"
  他们把张仪打了个半死。张仪回到家，妻子哭着说:
  "你要是不读书游说，怎么会受这种屈辱!"
  
  张仪张开嘴，问妻子:"你看我的舌头还在不在?"
  妻子笑了:"在。"
  张仪说:"舌头在，就够了。"
  
  后来张仪去了秦国。苏秦(他的老同学)正在推行合纵。
  但苏秦暗中资助了张仪——因为他需要一个"对手"来让合纵更有价值。
  张仪知道后，没有生气。他对苏秦说:
  "只要你活着，我绝不破坏你的合纵。"
  这是两个天才之间的默契——各为其主，各尽其才。

性格画像:
  张仪比苏秦更不择手段。
  苏秦至少还讲"信义"——张仪连信义都可以拿来当武器。
  
  他欺骗楚怀王: "秦国愿意割让六百里土地，换取楚国与齐国断交。"
  楚怀王信了→与齐国断交→然后张仪说:"我说的是六里，不是六百里。"
  楚怀王大怒→发兵攻秦→大败→从此楚国一蹶不振。
  
  这就是"张仪欺楚"。一个帝国被一个骗局摧毁了。
  张仪对此毫无愧疚——"外交就是欺骗的艺术。"
  
  但张仪也不是完全没有底线的人。
  苏秦死后，张仪才全力推动连横策略——他遵守了"苏秦在时不破坏合纵"的承诺。

在游戏中的呈现:
  招募条件: 秦国+有连横需求+魅力≥55(★★★★★)
           或在张仪落魄时(被打后在楚国)招募(★★)
  可学技能: 欺楚(假承诺对方采信率+50%)/连横之策
  专属剧情: 张仪欺楚→你可以参与这个骗局/揭发它/利用它
```

## 四、吕不韦 —— 历史上最成功的风险投资人

```
"投资一个国家，回报率是多少？"
                                   ——吕不韦

基本信息:
  生卒: ? - 前235年(自杀)
  国籍: 卫→赵→秦
  最高官职: 秦国相国，封文信侯
  历史定位: 中国历史上最著名的商人政治家

出身与早年:
  吕不韦是卫国人，大商人。他走南闯北，积累了巨大财富。
  但在那个时代，商人地位很低——有钱但没地位。
  
  他在邯郸做生意时，遇到了秦国派到赵国的人质:异人(子楚)。
  异人是秦太子的儿子，但不是嫡子，也不受宠——
  所以被送来当人质(战国的惯例)。他在赵国过得很惨。
  
  吕不韦回家后问父亲:
  "耕田之利几倍?" 父:"十倍。"
  "珠玉之赢几倍?" 父:"百倍。"
  "立国家之主赢几倍?" 父:"无数。"
  
  吕不韦说:"我现在去投资一个国家。(今力田疾作，不得暖衣余食；
           今建国立君，泽可以遗世。)"
  
  于是他把全副身家——所有的钱——押在了异人身上。

性格画像:
  吕不韦是史上最极端的"高风险投资者"。
  他为异人做了三件事:
  1. 给异人大笔钱改善生活(让他在赵国过得好一点)
  2. 去秦国运作，让异人认华阳夫人为母(华阳夫人是秦太子宠妃但无子)
  3. 把自己的妾室(赵姬)送给异人——赵姬后来生了嬴政
  
  异人后来真的回国继位了(秦庄襄王)。
  吕不韦被封为丞相、文信侯，食邑十万户。
  投资回报率? 无法计算。

关键人生节点:

  一字千金: 吕不韦当了丞相后，召集门客编写了《吕氏春秋》。
            他把书挂在咸阳城门上，悬赏:"能增损一字者，予千金。"
            这就是"一字千金"的典故。没人敢改一个字。
            
  自杀: 嬴政长大后，吕不韦被卷入嫪毐叛乱事件——
        虽然吕不韦未必参与，但嬴政已经不信任他了。
        嬴政写信给吕不韦:"你对秦国有什么功劳?封你十万户。
                           你跟秦国有什么亲?号称仲父。"
        吕不韦知道嬴政要杀他了。他饮鸩自杀。

在游戏中的呈现:
  招募条件: 在邯郸遇到他→共同投资异人(★★★★★)
  可学技能: 奇货可居/一字千金
  专属剧情: 商道路线的终极剧情→"吕不韦之路"
            吕不韦被嬴政逼迫→你能救他吗?
```

## 五、郑国 —— 一个间谍改变了中国农业

```
"我是来当间谍的。但我修了一条渠。这渠比任何间谍活动都更有价值。"
                                   ——郑国(虚构)

基本信息:
  生卒: 不详(约前3世纪)
  国籍: 韩→秦
  最高官职: 秦国水工(工程师)
  历史定位: 郑国渠的设计者/"疲秦计"的执行者变成了"强秦"的功臣

出身与早年:
  郑国是韩国的水利工程师。战国末期的韩国面临秦国巨大威胁，
  韩王想出了一个"疲秦计": 派最优秀的水利工程师去秦国，
  建议秦国修建一条超大规模的水渠，消耗秦国的国力。

  郑国被选中了。他去了秦国。
  秦王嬴政接见了他。郑国详细讲解了从泾河引水灌溉关中的方案——
  嬴政听完后很兴奋:"修!"
  
  工程开始后，秦国投入了大量人力物力。
  然后，韩国的"疲秦计"暴露了。嬴政大怒，要杀郑国。
  
  郑国说了一句话:"始臣为间，然渠成亦秦之利也。
                   臣为王作此渠，不过为韩延数岁之命，
                   而为秦建万世之功。"
  (我当初确实是间谍。但这条渠修好了，对秦国是万世之功。
   我不过是为韩国多续了几年的命，但这条渠能让秦国永远强大。)

  嬴政沉默了。然后说:"继续修。"

性格画像:
  郑国是历史上最奇特的"双重间谍"。
  他确实是被派来消耗秦国的，但他修的渠确实让秦国更加强大。
  郑国渠灌溉了四万顷(约合今天的280万亩)盐碱地，
  关中从此成为"天府之国"——这是秦国统一天下的经济基础。
  
  韩国的"疲秦计"成了史上最大的"乌龙"。
  一个工程师超越了他的政治使命——他选择了"把事做好"。

在游戏中的呈现:
  出现场景: 韩国→秦国(郑国渠修建时期)
  招募条件: 秦国修建郑国渠时→发现他是间谍→
            选择: 杀了他/原谅他/继续修渠(★★★)
  可学技能: 水利工程(建筑效率+30%)
  专属剧情: 郑国渠修建→韩国"疲秦计"暴露→你的选择影响历史
```
