# 秦国人物传记

> 虎狼之国，必有虎狼之臣。秦国的崛起，是一群"偏执狂"用铁与血浇灌出的奇迹。

---

## 一、白起 —— 人屠

```
"我一生杀了近百万人。但我不记得任何一张脸。"
                                   ——白起，临终前

基本信息:
  生卒: ? - 前257年
  国籍: 秦 (郿县人)
  最高官职: 大良造(秦国最高军职)，封武安君
  历史定位: 战国四大名将之首 / "人屠" / 一生70余战未尝败绩

出身与早年:
  白起出身不详。史书没有记载他的父亲是谁。
  这在战国名将中极为罕见——他没有显赫的家世。
  他的一切，都是靠军功一刀一枪拼出来的。
  
  秦国的军功爵制给了他这样的平民子弟一条上升通道:
  斩首一级→爵位一级→斩首越多→地位越高。
  白起是这个制度最完美的产物——也是最极端的产物。

性格画像:
  白起是一个矛盾到极致的人。
  
  在战场上，他是冷静到可怕的杀人机器。
  伊阙之战斩首24万，鄢郢之战水淹楚国军民数十万，
  长平之战坑杀40万赵卒。没有任何将领杀过这么多人。
  
  但在私下里，他沉默寡言。几乎没有任何关于他私生活的记载。
  他不结交权贵，不拉帮结派，不贪财不好色。
  他唯一在做的事就是打仗，打完仗就回营帐，等着下一次出征。
  
  有人说他是冷酷。但换个角度看——
  也许他只是把"杀人"当成了一份工作，仅此而已。
  不享受，也不痛苦。只是"做完了"。

关键人生节点:

  转折1: 伊阙之战 (前293年)
    年仅20多岁的白起，第一次作为主将指挥大战。
    对手是韩魏联军，兵力是秦军的两倍。
    白起看准了韩魏两军互相推诿、都想让对方先上的心理——
    他集中全部兵力先打魏军，魏军一溃，韩军不战自乱。
    斩首24万。一战成名。
    
  转折2: 长平之战 (前260年)
    这是白起人生的巅峰，也是他命运的转折点。
    秦赵两军在长平对峙三年。赵国换下老将廉颇，换上赵括。
    白起率秦军主力秘密接替前线指挥，赵括浑然不知。
    白起佯败，诱赵括全军追击→切断退路→包围46天→
    赵军粮尽→突围失败→赵括战死→全军投降。
    
    然后白起说了一句话: "赵卒反复，非尽杀之，恐为乱。"
    40万降卒，全部坑杀。
    
    从这一刻起，"人屠"之名天下皆知。
    也从这一刻起，白起自己也知道——他已经没有回头路了。

  转折3: 被赐死 (前257年)
    长平之战后，白起主张乘胜灭赵。
    但秦相范雎嫉妒白起的军功，说服秦王接受赵国割地求和。
    白起对此非常愤怒——他失去了灭赵的最佳时机。
    
    第二年，秦王又想起用白起攻赵。白起说"时机已过，不可攻"。
    秦王不听，派其他将领攻赵，果然大败。
    秦王再次请白起出征，白起称病不出。
    
    秦王大怒，加上范雎进谗→白起被贬为士卒→流放。
    在流放途中，秦王派人送来一把剑→"自行了断"。
    
    白起拿着剑，说了一句话:
    "我何罪于天而至此哉?"
    良久，又说:
    "我固当死。长平之战，赵卒降者数十万人，
     我诈而尽坑之，是足以死。"
    
    自刎而死。
    秦国百姓同情白起，乡邑祭祀他。

人际关系网:
  君主: 秦昭襄王 —— 君臣相得→功高震主→赐死
  政敌: 范雎 —— 嫉妒白起军功，进谗言导致白起被赐死
  对手: 廉颇 —— 长平之战前，白起曾说"廉颇在，此战难打"
  无友: 白起没有任何亲密朋友，这是他自己选择的结果

历史功过:
  时人: 秦人尊崇(军功/乡邑祭祀)，六国人恐惧诅咒
  后世: 军事天才(战术/战役层面几乎无懈可击)，
        但"人屠"之名成为暴力的代名词
  客观: 白起是秦统一进程中最重要的军事工具。
        他执行的"歼灭有生力量"战略远超时代。
        但他的残酷也埋下了秦统一后速亡的种子——
        六国遗民对秦的仇恨，很大程度来自白起。

在游戏中的呈现:
  出现场景: 伊阙/长平/鄢郢等历史战役
  可触发事件: 
    - 长平之战后→白起的对话(见318)
    - 白起被赐死时→选择救他(极高难度)
  招募条件: 秦国+统率≥60+完成5场胜利(极难)
  可学技能: 人屠/长平之围
  专属剧情: 白起被赐死→如果你选择救他→白起归隐→
            你可以向他学习全部军事知识→但白起不再上战场

经典语录:
  "我固当死。"
  "长平之战，赵卒降者数十万人，我诈而尽坑之，是足以死。"
```

## 二、王翦 —— 灭国专家

```
"灭国不是杀人，是让一国从地图上消失。"
                                   ——王翦

基本信息:
  生卒: ? - 前214年后 (长寿，活到80+)
  国籍: 秦 (频阳人)
  最高官职: 上将军，封武成侯
  历史定位: 与白起齐名的秦灭六国首席功臣 / 善终的一代名将

出身与早年:
  王翦出身于秦国频阳。少时即好兵法，成年后事秦王政。
  他的成长轨迹与白起完全不同——白起靠个人勇武，王翦靠政治智慧。
  王翦是战国名将中极少数能善终的人。

性格画像:
  如果白起是一把锋利的刀，王翦就是一把带鞘的剑。
  他不仅有军事才能，更有政治智慧。
  
  灭楚之前，秦王政问需要多少兵力。年轻将领李信说"20万足够"。
  王翦说"非60万不可"。
  秦王嘲笑王翦"老矣，怯矣"。——然而李信出兵后大败。
  秦王亲自到王翦家道歉，请他出征。王翦同意，但坚持要60万大军。
  
  在出征路上，王翦做了一件很奇怪的事:
  他连续五次派人回咸阳向秦王请求赏赐——良田美宅。
  部下不解:"将军怎么在打仗路上讨要田宅?"
  王翦说:"秦王多疑。现在全国的兵都在我手里——
        我如果不表现得贪财，秦王会怀疑我有野心。"
  
  这就是王翦的生存智慧: 功高震主时，主动"自污"以保身。

关键人生节点:

  灭赵: 公元前229年，王翦率军攻赵。
        赵国有李牧——王翦知道正面打不过。
        于是他用反间计→赵王中计→杀李牧→赵国不战自溃。
        王翦灭赵，但手段并不光彩。他知道这一点。

  灭楚: 公元前224年，王翦率60万大军伐楚。
        楚国起全国之兵抵抗。
        王翦到达前线后——不进攻。每天让士兵休息、洗澡、吃饭。
        士兵无聊了就在军营里扔石头比赛。
        楚军多次挑战，王翦不应。
        
        这样过了一年多。楚军以为秦军不会进攻，开始松懈，向东撤退。
        此时王翦突然出击——"全军追击!"——大破楚军，杀死项燕。
        灭楚的关键是一个"等"字。
        
        王翦灭楚后，又顺便灭掉了燕国和齐国残部。
        他一个人灭了赵、楚、燕、齐四国。名副其实的"灭国专家"。

  善终: 王翦在灭楚后主动交还兵权，回老家养老。
        秦始皇给了他要的所有良田美宅。他活到了80多岁。
        他的儿子王贲、孙子王离都成为了秦国名将。
        
        王翦证明了: 在一个多疑的君主手下，名将也是可以善终的——
        只要你足够聪明。

人际关系网:
  君主: 秦始皇 —— 君臣互信(王翦用"贪财"维系信任)
  儿子: 王贲 —— 虎父无犬子，继承父业灭魏/燕/齐
  对比: 李信 —— 年轻气盛伐楚大败，反衬王翦的稳健

在游戏中的呈现:
  招募条件: 秦国+秦始皇在位+统率≥70(极难)
  可学技能: 老谋深算/灭国专家
  独特建策: 可以向王翦学习"如何在功高震主时自保"
```

## 三、商鞅 —— 变法者之死

```
"我制定了最完美的法律。但法律保护不了制定它的人。"
                                   ——商鞅，逃亡途中

基本信息:
  生卒: 约前390 - 前338年
  国籍: 卫→魏→秦 (卫国人，在秦变法)
  最高官职: 大良造(秦国)，封商君
  历史定位: 中国历史上最著名的变法者/法家实践第一人

出身与早年:
  商鞅本名公孙鞅，卫国公族后裔(卫国是个极小的国家)。
  年轻时在魏国丞相公孙痤门下做中庶子(低级官员)。
  公孙痤临终前向魏惠王推荐商鞅:"此人可以治国。"
  魏惠王没当回事。公孙痤又说:"如果不用，就杀了他，别让他去别国。"
  魏惠王还是没有在意。
  
  商鞅听说秦孝公在求贤，带着李悝的《法经》去了秦国。
  见了秦孝公三次: 第一次讲"帝道"(尧舜之道)→秦孝公睡着了。
                   第二次讲"王道"(周公之道)→秦孝公还是没兴趣。
                   第三次讲"霸道"(强国之术)→秦孝公听了一整天。
  从此，商鞅在秦国开始了中国历史上最激进的一次变法。

性格画像:
  商鞅是一个极端理性的人。
  他相信"法"高于一切——包括人情、传统、道德。
  他设计的法律细致到令人发指: 连路上倒灰要罚多少钱都规定了。
  
  没有人喜欢商鞅。贵族恨他(变法夺走了他们的世袭特权)，
  平民怕他(法律太严苛了)，同僚躲着他(他太"不近人情")。
  他自己也知道——变法成功那天，就是他自己的死期。
  但他还是做了。
  
  有人问他为什么。"因为秦国不变法，就会灭亡。"
  这是一个用理想主义驱动功利手段的矛盾人物。

关键人生节点:

  徙木立信: 变法令颁布前，商鞅在都城南门立了一根木头。
             "谁能把这根木头搬到北门，赏十金。"
             没人相信。商鞅把赏金提高到五十金。
             终于有人搬了→商鞅当场兑现赏金。
             全城震惊——"商鞅说话算话!"
             徙木立信是历史上最早的成功公关案例之一。

  太子犯法: 变法令生效后，太子(后来的秦惠文王)触犯了法律。
            商鞅说:"太子是未来的君主，不能受刑。"
            "但太子的老师，管教不严——该受刑。"
            于是太子的老师被处以黥刑(脸上刺字)。
            这是对秦国宗室的巨大羞辱。从这天起，商鞅结下了死仇。

  被车裂: 秦孝公死后，太子继位(秦惠文王)。
          旧贵族立即反击→诬告商鞅谋反→商鞅逃亡。
          逃亡途中，他想住店。店家说:"商君之法，无凭证不得住宿。"
          商鞅苦笑: "我制定的法律...反过来困住了我自己。"
          
          最终他被抓住，处以车裂(五马分尸)。
          他的尸体被示众→"这就是变法者的下场!"
          
          但讽刺的是: 秦惠文王虽然杀了商鞅，
          却完整保留了商鞅的所有变法令。
          因为他知道——商鞅是对的。

在游戏中的呈现:
  出现场景: 秦国变法时期
  可触发事件: 
    - 参与商鞅变法(作为支持者/助手)
    - 商鞅被追杀时→选择: 举报/帮助逃亡/视而不见
  招募条件: 秦国+秦孝公在位(特定时间窗口)→无法在其他时期招募
  可学技能: 变法(改革成功+30%)
  专属剧情: 如果你选择帮助商鞅逃亡→获得商鞅遗著→
            解锁隐藏政策"商君之法"
```

## 四、范雎 —— 恩怨分明

```
"一饭之德必偿，睚眦之怨必报。"
                                   ——范雎

基本信息:
  生卒: ? - 约前255年
  国籍: 魏→秦
  最高官职: 秦相，封应侯
  历史定位: 提出"远交近攻"战略的纵横家/因私怨毁掉白起的秦相

出身与早年:
  范雎在魏国时，曾为魏国中大夫须贾的门客。
  一次出使齐国，齐王赏识范雎的才华，送了他礼物。
  须贾嫉妒，回国后诬告范雎通齐卖魏。
  魏国丞相魏齐命人将范雎鞭打至昏死，用席子卷起来扔进厕所。
  宾客们在他身上小便取乐。
  
  范雎在席子中苏醒过来，求看守放了他。看守动了恻隐之心。
  范雎化名张禄，在朋友郑安平的帮助下逃到了秦国。
  
  在秦国，他向秦昭襄王提出了"远交近攻"的战略:
  结好远方的齐国/燕国→吞并邻国韩国/魏国/赵国。
  秦昭襄王大为赞赏，拜范雎为相。

性格画像:
  范雎有两个极端: 对恩人涌泉相报，对仇人加倍奉还。

  当他成为秦相后:
  - 须贾出使秦国(不知道秦相张禄就是范雎)。
    范雎穿着破衣服去见须贾，须贾可怜他，送了他一件绨袍。
    后来须贾发现真相→跪地求饶。
    范雎说:"你送我这件绨袍，说明你还有一点故人之情。
           看在这件袍子的份上，我不杀你。滚!"
    (这就是"绨袍之赠"的典故)

  - 他逼迫魏国交出丞相魏齐(当年侮辱他的人)。
    魏齐逃亡到赵国平原君处→范雎逼迫赵国交出魏齐→
    魏齐最终自杀。范雎把他的头送回魏国。
    睚眦之怨，必报。

  - 他嫉妒白起的军功→在秦王面前进谗言→白起被赐死。
    这是他最大的污点。个人恩怨超越了国家利益。

人际关系网:
  恩人: 郑安平/王稽(救助他逃出魏国的人)→范雎帮他们当了大官
  仇人: 须贾(被羞辱但最终放过)/魏齐(不死不休)
  政敌: 白起(嫉妒军功→进谗言陷害)
  君主: 秦昭襄王→范雎的"远交近攻"深得信任

在游戏中的呈现:
  招募条件: 秦国+昭襄王在位+智略≥60
  可学技能: 远交近攻/恩怨分明(外交关系两极分化)
  专属事件: 须贾出使→选择帮须贾求情或推波助澜
```

## 五、嬴政 —— 始皇帝

```
"朕为始皇帝。后世以计数，二世三世至于万世，传之无穷。"
                                   ——嬴政，公元前221年

基本信息:
  生卒: 前259 - 前210年
  国籍: 秦 (生于赵都邯郸)
  最高官职: 秦王→始皇帝
  历史定位: 中国第一个皇帝/统一六国者

出身与早年:
  嬴政的早年充满危险。父亲异人(子楚)在赵国当人质，
  吕不韦"奇货可居"投资异人，异人回国后继位为王(秦庄襄王)。
  嬴政13岁继位，但朝政掌握在吕不韦和母亲赵姬手中。
  
  少年嬴政亲眼目睹了宫廷的尔虞我诈。
  他的母亲与假宦官嫪毐私通，生了两个孩子。
  嫪毐甚至发动政变试图杀死嬴政。
  嬴政平定叛乱后车裂嫪毐，软禁母亲，流放吕不韦(吕不韦自杀)。
  
  22岁那年，嬴政终于夺回全部权力。
  从这一刻起，没有人能再挑战他的权威。

性格画像:
  嬴政极度多疑。他的一生被多次背叛: 母亲、吕不韦、嫪毐...
  这些经历让他变得冷酷。但他又极度渴望秩序——"天下必须有一个规矩"。
  
  他的统一措施(书同文/车同轨/统一度量衡/郡县制)改变了中国历史。
  但他也焚书坑儒、建造阿房宫、修建长城、寻求不死药...
  他既是伟大的统一者，也是残酷的暴君。
  这两个身份在他身上无法剥离——是同一个人。

在游戏中的呈现:
  出现场景: 秦国后期/统一进程
  不可招募(他是君主，你只能辅佐他或对抗他)
  可触发事件: 统一六国的历史进程/焚书坑儒/求不死药
  如果你辅佐嬴政统一→获得"秦灭六国"成就
  如果你对抗嬴政→这是游戏最高难度的挑战
```
