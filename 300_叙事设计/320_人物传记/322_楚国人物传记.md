# 楚国人物传记

> 楚虽三户，亡秦必楚。这个南方大国的倔强与悲情，刻在每一个楚人的骨子里。

---

## 一、项燕 —— 楚国最后的守护者

```
"楚虽三户，亡秦必楚。"
              ——楚南公(项燕用自己的死践行了这句预言)

基本信息:
  生卒: ? - 前224年(战死)
  国籍: 楚
  最高官职: 楚国上将军
  历史定位: 楚国最后一员名将/项羽祖父

出身与早年:
  项燕出身于楚国的项氏，是楚国世代将门。
  项氏一族在楚国以勇武著称，项燕本人自幼习武，精通兵法。
  他成长的时代是楚国加速衰落的时期——白起已攻破郢都，
  楚国被迫迁都。项燕就是在这样的国难中扛起了楚军的战旗。

性格画像:
  项燕是一个"明知不可为而为之"的悲剧英雄。
  他心里清楚楚国大势已去——秦国太强了。
  但他是将军。将军的职责不是计算胜率，是战斗到最后一刻。
  
  史载项燕爱兵如子。楚军在他的指挥下，虽然国力远不如秦，
  但士气依然高昂。他是用个人魅力在维持一支哀兵。

关键人生节点:

  击败李信: 前225年，秦王派李信率20万大军伐楚。
            项燕率楚军一路跟踪秦军，连续三天三夜急行军——
            在秦军最疲惫的时候发动突袭→大败李信。
            这是秦国灭楚战争中唯一的大败。
            但项燕知道——这只是暂时的。秦国还会来。

  决战王翦: 前224年，王翦率60万大军来了。
            项燕起全国之兵迎战。
            王翦不进攻。每天让士兵休息。
            项燕多次挑战，秦军不应。楚军开始懈怠。
            一年后，楚军向东撤退。
            王翦在此时发动总攻——项燕大败。
            
  战死: 项燕在乱军中被包围。
        他拒绝了投降→"楚人项燕，死于楚地。"→自刎而死。
        
        他的孙子项羽(当时还是孩子)被人藏了起来。
        26年后，项羽率楚军攻入咸阳，火烧秦宫。
        "楚虽三户，亡秦必楚"——项燕的血没有白流。

在游戏中的呈现:
  出现场景: 楚国灭亡战争
  招募条件: 楚国+国家面临灭亡危机(极高难度)
  可学技能: 楚国长城(本土防御+30%)
  专属剧情: 如果你辅佐楚国→在最后一战中力挽狂澜
```

## 二、屈原 —— 投江的诗人

```
"举世皆浊我独清，众人皆醉我独醒。"
                                   ——屈原《渔父》

基本信息:
  生卒: 约前340 - 前278年
  国籍: 楚
  最高官职: 左徒/三闾大夫
  历史定位: 中国第一位伟大诗人/楚辞之祖/端午节纪念的人物

出身与早年:
  屈原出身于楚国宗室(芈姓屈氏)。少年时受过良好教育，
  博闻强志，明于治乱，娴于辞令。
  楚怀王早期很信任屈原，让他参与国政、起草法令、应对诸侯。

性格画像:
  屈原是一个"太干净"的人。在浑浊的楚国政坛，他的"干净"让他孤立。
  他不能容忍妥协，不能容忍腐败，不能容忍君主被小人蒙蔽。
  
  在文学上，这是一个拥有空前想象力的人——
  他写了《离骚》《九歌》《天问》，创造了一个瑰丽奇绝的神话世界。
  在政治上，这是一个完全不适合做官的人——
  他不懂得迂回，不懂得等待，不懂得"看人脸色"。
  
  这两面都是屈原。正因为他在现实中"寸步难行"，
  他才在文学中创造了一个可以自由翱翔的世界。

关键人生节点:

  被流放: 楚怀王听信谗言，疏远屈原。
          屈原被流放到汉北。他在那里写了《离骚》——
          "长太息以掩涕兮，哀民生之多艰"。
          这是中国文学史上最伟大的诗歌之一。

  郢都陷落: 前278年，白起攻破楚国都城郢都。
            屈原正在流放地。听到这个消息——
            他走到汨罗江边，抱着一块大石头，投江而死。
            
            据说当地百姓划船去救他→这就是划龙舟的起源。
            人们往江里扔粽子→希望鱼不要吃他的身体→这就是粽子的起源。
            
            一个诗人的死，变成了一个持续两千多年的民族记忆。

在游戏中的呈现:
  出现场景: 楚国朝廷/汨罗江边
  招募条件: 楚国+智略≥50+尊重楚文化(中高难度)
  不可阻止: 屈原投江(历史事件)→
            但你可以在他投江前与他对话→获得《离骚》遗稿
  遗稿效果: 文化影响力+30%，智略+3(阅读后)
```

## 三、春申君(黄歇) —— 最后的四公子

```
"三千门客，无人能救我于棘门之内。"
                                   ——黄歇，临终

基本信息:
  生卒: ? - 前238年(被刺杀)
  国籍: 楚
  最高官职: 楚国令尹(丞相)，封春申君
  历史定位: 战国四公子之末/结局最惨的四公子

出身与早年:
  黄歇年轻时以博学和善辩闻名。他游学四方，最终回到楚国，
  在楚顷襄王手下做事。秦国计划联合韩魏攻打楚国——
  黄歇出使秦国，说服秦昭襄王放弃这个计划。
  这是他外交生涯的第一个重大胜利。

  后来，他陪同楚国太子(后来的楚考烈王)在秦国做人质。
  楚顷襄王病危时，黄歇设计让太子秘密逃回楚国继位，
  自己留在秦国面对秦王的怒火。秦王本想杀他，
  但在范雎的劝说下放了他——因为"杀了黄歇，楚国会换一个更恨秦的人当令尹"。
  
  黄歇回国后，楚考烈王感激他救了自己→
  封他为春申君，拜为令尹。

性格画像:
  黄歇聪明、有政治手腕，但晚节不保。
  他早年是出色的外交家和忠诚的臣子，
  但晚年他开始贪恋权力——甚至在楚考烈王无子的情况下，
  献上了自己怀孕的妾室(李园的妹妹)，试图让自己的儿子继位。
  
  这个阴谋最终毁了他。
  李园(妾室的哥哥)害怕事情败露，在黄歇入宫时设伏刺杀了他。
  黄歇死在棘门之内，身边三千门客无一人能救他。

在游戏中的呈现:
  招募条件: 楚国+魅力≥60(中高难度)
  可学技能: 门客三千(招募+50%)
  专属剧情: 黄歇晚年阴谋→你可选择: 揭发/参与/置身事外
```

## 四、庄蹻 —— 从将军到滇王

```
"我去了南方，再也没有回来。不是因为忘了楚国——
  而是因为我在那里发现了一个新世界。"
                                   ——庄蹻(虚构)

基本信息:
  生卒: 不详(约前4世纪-前3世纪)
  国籍: 楚
  最高官职: 楚国将军→自立为滇王
  历史定位: 云南滇国的开创者/最早的"殖民地开拓者"

出身与早年:
  庄蹻是楚国将领。楚威王时期，他被派遣率军南征，
  目标是通过巴蜀(后来的四川)向南开拓领土。
  他的军队一路向南，穿越了崇山峻岭，最终到达了今天的云南滇池一带。

  在那里，庄蹻和他的士兵发现了一个完全不同的世界——
  温暖的气候、肥沃的土地、友善的土著。
  然后他听到了一个消息: 秦国攻占了巴蜀，截断了他回楚国的路。

性格画像:
  庄蹻没有试图杀回楚国。他做了一个惊人的决定——留在这里。

  他让士兵们娶当地女子，学习当地语言，融入当地社会。
  他自己也娶了土著首领的女儿。他和他的部下教当地人农业和军事技术，
  当地人教他们如何在滇池边生活。
  
  几十年后，庄蹻建立的滇国成为西南地区最强大的政权。
  当汉武帝时代，滇王向汉朝称臣时——"滇"这个国名已经存在了近200年。
  它始于一个回不了家的楚国将军。

在游戏中的呈现:
  出现场景: 探索滇地(地图最南端，极偏远)
  招募条件: 探索到滇地+接受庄蹻自立+统率≥50(★★★)
  可学技能: 自立为王(脱离原国家→建立独立势力)
  专属剧情: 庄蹻的滇国→你可以加入/征服/贸易
```
