## 一、孙膑 —— 从地狱爬出来的兵圣

```
"围魏救赵，不是救赵国的赵，是救赵括的赵。"
                                   ——孙膑

基本信息:
  生卒: 约前380 - 前316年
  国籍: 齐 (生于阿、鄄之间，孙武后代)
  最高官职: 齐国军师
  历史定位: 《孙膑兵法》作者/围魏救赵和马陵之战的主角

出身与早年:
  孙膑是孙武的后代。年轻时的孙膑和庞涓是同学，一起在鬼谷子门下学习兵法。
  两人曾是好友。庞涓先离开师门，去了魏国当将军。
  庞涓知道自己的才能不如孙膑——于是他设计了一个毒计。
  
  他写信请孙膑来魏国"共图大业"。孙膑欣然前往。
  然后庞涓捏造罪名，将孙膑处以膑刑(挖去膝盖骨)和黥刑(脸上刺字)。
  孙膑的名字"膑"就是刑法的名字——他本名已经无人知晓。
  
  一个被挖掉膝盖的残废，脸上刻着罪犯的标记——
  庞涓认为孙膑从此"废了"。

性格画像:
  但孙膑没有废。他装疯卖傻、吃猪粪以求生存。
  在齐国使臣出使魏国时，孙膑秘密求见→说服使臣带他逃离。
  
  到了齐国后，孙膑在田忌手下做门客。
  田忌赛马的故事就是孙膑的策略: 用下等马对上等马→输一场，
  用上等马对中等马→赢一场，用中等马对下等马→再赢一场。
  三局两胜。田忌第一次见识到孙膑的智慧，将他推荐给了齐威王。
  
  孙膑不恨庞涓吗？他恨。但他报仇的方式不是直接杀庞涓——
  而是用一场完美的战役，让庞涓自己走向死亡。

关键人生节点:

  围魏救赵: 庞涓率魏军包围赵国邯郸。孙膑建议: 
            "我们不直接去邯郸——我们直接打魏国都城大梁，
             庞涓必然回师救援——我们在半路上伏击他。"
            这是"围魏救赵"的经典。庞涓果然中计，在桂陵被齐军击败。

  马陵之战: 十几年后，庞涓又率魏军攻打韩国。
            孙膑用"减灶诱敌"之策: 齐军假装败退，
            第一天造十万灶→第二天五万→第三天三万。
            庞涓判断"齐军已经逃亡大半!"→率轻骑兵猛追。
            
            夜晚，马陵道。一棵大树被剥皮，上面写着字。
            庞涓命人点火看字——"庞涓死于此树之下"。
            箭矢如雨。
            庞涓知道自己逃不掉了。他拔剑自刎，死前说了最后一句:
            "遂成竖子之名!"(就让这小子成名吧!)
            
            孙膑在远处，听到了。他低头看了看自己的残腿——
            十多年的仇恨，在这一刻结束。
            他没有笑，也没有哭。只是沉默。

人际关系网:
  恩师: 鬼谷子
  同门/仇敌: 庞涓(一生之敌)
  搭档: 田忌(军事合作)
  君主: 齐威王

在游戏中的呈现:
  招募条件: 从魏国救出孙膑(帮助他逃离庞涓)+智略≥55
  可学技能: 围魏救赵/减灶诱敌
  专属剧情: 庞涓vs孙膑→马陵之战→孙膑复仇后归隐
```

## 二、田单 —— 火牛阵与复国

```
"即墨城只剩最后一座城门没被围了。
  我站在城门上，看到远处燕军的营火。我知道，今晚要下雨。"
                                   ——田单

基本信息:
  生卒: 不详
  国籍: 齐
  最高官职: 安平君，齐国相国
  历史定位: 以火牛阵复齐的民族英雄

出身与早年:
  田单是齐国王族的远支。在乐毅五国伐齐之前，
  他只是临淄市集上的一个小官吏，默默无闻。
  
  当燕军攻入齐国，齐国主力被摧毁，大片领土沦陷。
  田单带着族人逃到了即墨城——齐国最后的据点之一。
  
  即墨守将战死，城内无人主事。大家推举田单为将。
  一个市集小吏，突然成了齐国最后的希望。

性格画像:
  田单聪明到近乎狡黠。他善用心理战术。
  他知道即墨城的兵力无法正面击败燕军——他需要"奇迹"。
  于是他创造了一个奇迹。

关键人生节点:

  反间计: 燕昭王去世，燕惠王继位。
          田单派间谍去燕国散布谣言: 
          "乐毅不攻即墨，是因为他想在齐国自立为王!"
          燕惠王中计→派骑劫代替乐毅。乐毅被迫逃亡赵国。
          
  火牛阵: 田单收集了城中一千多头牛。
          给牛披上红色绸缎，牛角绑上尖刀，牛尾绑上苇草浇油。
          深夜，城门大开。点燃牛尾——
          
          一千多头燃烧的火牛疯狂冲向燕军营地。
          燕军在睡梦中惊醒，看到无数火牛冲来——
          以为天降神兵，全军大乱，自相践踏。
          
          田单率齐军紧随其后，大破燕军，一鼓作气收复了七十余城。
          齐国复国了。一个市集小吏，用一千头牛，创造了一个神话。

在游戏中的呈现:
  招募条件: 齐国面临灭亡危机+即墨被围(★★★)
  可学技能: 火牛阵/反间计
  专属剧情: 即墨被围→选择帮助田单守城→体验火牛阵
```

## 三、廉颇 —— 老将的尊严

```
"廉颇老矣，尚能饭否?"
    ——赵王派去探望廉颇的使者(被仇人收买后)的回复

基本信息:
  生卒: 约前327 - 前243年
  国籍: 赵→魏→楚(晚年流亡)
  最高官职: 上将军，封信平君
  历史定位: 战国四大名将/负荆请罪/长平之战的最初主将

性格画像:
  廉颇是真正的军人。勇敢、耿直、脾气大，但不记仇。
  他嫉妒蔺相如(一个文官居然官位在他之上)→
  公然说"我见到蔺相如一定要羞辱他"→
  蔺相如听说后处处回避他→廉颇得意→
  后来得知蔺相如是"为了赵国的团结才忍让"→
  廉颇羞愧难当→背着荆条去向蔺相如请罪。
  
  一个老将，脱了上衣，背着荆条，跪在一个文官门前——
  "蔺相如! 我廉颇不是人! 我向你请罪!"
  这就是"负荆请罪"。只有真正有尊严的人，才敢于承认错误。

在游戏中的呈现:
  招募条件: 赵国+统率≥55(★★★★★)
  可学技能: 负荆请罪(关系修复)/老当益壮(年龄>50后统率+5)
  关键事件: 长平之战→如果你劝赵王留住廉颇→可能改变历史
```

## 四、李牧 —— 北疆战神

```
"李牧死，赵国亡。"
              ——赵国百姓在李牧被处死后说的话

基本信息:
  生卒: ? - 前229年(被处死)
  国籍: 赵
  最高官职: 武安君，大将军
  历史定位: 战国四大名将/北逐匈奴、南抗强秦

性格画像:
  李牧是赵国最全面的名将。他能打匈奴(草原骑兵)，
  也能打秦国(重甲步兵)。这在战国非常罕见。
  
  他的防御策略是"坚壁清野": 
  匈奴来了→把人和牲畜撤进堡垒→不跟你打→
  匈奴抢不到东西→无聊→撤退→李牧追击→大破匈奴。
  
  他用这招让匈奴十余年不敢犯赵。省下的军力用于对抗秦国。
  秦国王翦打不过李牧→使用反间计→赵王中计→处死李牧。
  李牧死后三个月，赵国灭亡。

在游戏中的呈现:
  招募条件: 赵国+北疆战事+统率≥60(★★★★★)
  可学技能: 北疆战神/坚壁清野
  关键事件: 王翦反间计→你可以选择提醒赵王(救李牧)或推波助澜
```
