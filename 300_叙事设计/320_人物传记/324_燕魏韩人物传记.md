## 一、乐毅 —— 下齐七十城的统帅

```
"我用了五年时间，打下了七十余座城池。
  然后，一纸诏书，我被换掉了。
  我走去赵国的时候——那些齐国城池，一座一座又被夺了回去。"
                                   ——乐毅(虚构)

基本信息:
  生卒: 不详
  国籍: 魏→燕→赵(晚年)
  最高官职: 昌国君，燕国上将军(五国联军统帅)
  历史定位: 五国伐齐的主角/诸葛亮的偶像

出身与早年:
  乐毅是魏国名将乐羊的后代。他早年在赵国为官，
  赵武灵王死后，他去了魏国。后来听说燕昭王筑黄金台求贤，
  乐毅去了燕国。燕昭王与他交谈后，拜他为上将。

性格画像:
  乐毅是那种"别人家的将军"——有才华、有风度、有远见、有节制。
  诸葛亮在《出师表》中自比管仲、乐毅——乐毅是诸葛亮的偶像。
  
  他攻打齐国时，采取了与白起完全相反的策略:
  不是消灭有生力量，而是"攻心为上"。
  每占领一座城池，乐毅不做三件事: 不屠城、不抢掠、不虐待降卒。
  他甚至允许齐国人保留自己的习俗和祭祀。
  
  所以齐国七十余城在乐毅治下五年来几乎没有叛乱。
  这不是靠恐惧，是靠人心。但这也埋下了隐患——
  燕国朝廷内部有人质疑: "他是不是想自己当齐王?"

关键人生节点:

  五国伐齐: 燕昭王派乐毅联合赵/魏/韩/秦五国军队攻齐。
            齐国主力被击溃。乐毅率领燕军一路猛进，
            连下齐国七十余城，只剩即墨和莒城两城还在坚守。
            齐国几乎灭亡。这是燕国历史上最辉煌的时刻。

  被换将: 燕昭王死后，燕惠王继位。
          惠王不信任乐毅。加上田单的反间计→惠王派骑劫代替乐毅。
          乐毅知道回去很可能被杀→逃亡赵国。
          
          骑劫上任后→田单火牛阵→燕军大败→齐城全部被夺回。
          五年努力，毁于一旦。
          
          燕惠王后来写信给乐毅，说你为什么要离开我?
          乐毅回了一封著名的信《报燕惠王书》:
          "臣闻善作者不必善成，善始者不必善终。"
          (会开始的人不一定会结束，会开端的人不一定有结果)
          
          这封信让燕惠王羞愧。他没有追杀乐毅，让他在赵国善终。

在游戏中的呈现:
  招募条件: 燕国+燕昭王在位+黄金台建筑(★★★★★)
           或在乐毅流亡赵国后→从赵国招募(★★★)
  可学技能: 五国伐齐/黄金台
  专属剧情: 五国伐齐→田单复齐→乐毅逃亡→《报燕惠王书》
```

## 二、荆轲 —— 刺客的最高形态

```
"风萧萧兮易水寒，壮士一去兮不复还。"
                                   ——荆轲《易水歌》

基本信息:
  生卒: ? - 前227年(刺杀失败被杀)
  国籍: 卫→燕
  最高官职: 无(刺客/游侠)
  历史定位: 中国历史上最著名的刺客

出身与早年:
  荆轲是卫国人。他年轻时喜欢读书和击剑。
  他曾在卫国国君面前展示自己的剑术和治国理念，但不被重用。
  于是他开始游历各国。在燕国，他遇到了两个改变他命运的人——
  一个叫田光(燕国老侠客)，一个叫高渐离(乐师，他的知己)。
  
  荆轲和高渐离经常在燕国市集上喝酒。
  喝醉了，高渐离击筑(一种乐器)，荆轲唱歌。
  两人时而大笑，时而大哭。旁若无人。
  人们说这是两个疯子。但他们不在乎。

性格画像:
  荆轲不是一个"好刺客"。他是一个复杂的人。
  他有才华(读书击剑)、有感情(与高渐离的友谊)、
  有犹豫(刺秦之前他其实不想去)。
  
  他不是那种"为了钱杀人"的杀手。
  他是被一个时代、一个困境、一个"义"字推上了不归路的人。
  
  太子丹找到荆轲，说秦国即将灭燕，需要一个人去刺杀秦王。
  荆轲拒绝了。太子丹跪下来磕头。
  荆轲沉默了很久。最后说:
  "我去。但我需要两样东西——督亢地图和樊於期的人头。"
  (督亢是燕国最富庶的地区，樊於期是秦国叛将逃到燕国。
   这两样东西可以让秦王相信荆轲是真的来投降的。)
  
  樊於期听说后——"只要能杀秦王!"——拔剑自刎。
  荆轲拿到了这两样东西。他已经没有退路了。

关键人生节点:

  易水送别: 荆轲出发那天，太子丹和所有知道这件事的人，
            都穿着白衣白帽来送行——
            因为他们知道，不管成功与否，荆轲都不会回来了。
            
            高渐离击筑。荆轲唱:
            "风萧萧兮易水寒，壮士一去兮不复还。"
            在场所有人——"士皆垂泪涕泣"。
            
            然后荆轲上车而去，再也没有回头。

  图穷匕见: 咸阳宫。荆轲捧着樊於期的头颅和督亢地图，
            一步步走向秦王。秦王嬴政坐在高高的王座上。
            荆轲展开地图——在地图卷到尽头时，露出了一把匕首。
            这把匕首就是鱼肠剑(传说)的仿品，淬了剧毒。
            
            荆轲左手抓住秦王的袖子，右手持匕首刺去——
            但秦王挣脱了。袖子被撕断。
            秦王绕柱而逃。荆轲追。
            殿上的大臣们惊慌失措——按照秦法，上殿不得带兵器。
            只有秦王的御医用药囊砸向荆轲。这一下给了秦王时间拔剑。
            
            秦王拔剑，砍断了荆轲的左腿。
            荆轲倒地，将匕首向秦王掷去——擦过秦王的耳边，钉在铜柱上。
            秦王连砍了荆轲八剑。
            
            荆轲靠在铜柱上，笑着说:
            "事所以不成者，乃欲以生劫之，必得约契以报太子也。"
            (我之所以没成功，是因为我想活捉你，逼你答应归还燕国土地)
            
            然后他死了。
            
            秦王站在荆轲的尸体前，沉默了很久。
            据记载，秦王此后"不欢者良久"——很长时间都不高兴。

在游戏中的呈现:
  出现场景: 燕国/易水/咸阳宫
  招募条件: 侠道路线+田光推荐+太子丹刺秦计划(★★★★)
  可学技能: 图穷匕见(暗杀致命一击概率+50%)
  专属剧情: 荆轲刺秦(全游戏最重要的分支之一)
    → 你可以陪荆轲一起去(双人刺秦→成功率翻倍但两人都可能死)
    → 你可以替代荆轲(你成为那个图穷匕见的人)
    → 你可以劝阻荆轲(改变历史→但燕国可能更快灭亡)
```

## 三、吴起 —— 战神/变法者/悲剧

```
"吴起可以灭掉一个国家，但灭不掉人心的嫉妒。"
                                   ——史家评价

基本信息:
  生卒: 约前440 - 前381年(被射杀)
  国籍: 卫→鲁→魏→楚
  最高官职: 魏国西河守/楚国令尹
  历史定位: 战国军政全才第一人/魏武卒创建者/吴起变法主导者

出身与早年:
  吴起是卫国人。年轻时他散尽家财求官，结果没求到，被人嘲笑。
  他一怒之下杀了嘲笑他的三十多人。然后咬着自己的手臂对母亲说:
  "不为卿相，不复入卫。"(不做到卿相，绝不回卫国)
  
  母亲去世了，吴起没有回去奔丧。他的老师曾子(儒家宗师)以此为不孝，
  与他断绝了师生关系。吴起没有被这个打击压垮——他离开儒家，改学兵法。

性格画像:
  吴起是一个为了目标不惜一切代价的人。
  
  在鲁国时，齐国攻打鲁国，鲁国想用吴起为将——
  但吴起的妻子是齐国人，鲁国怀疑他的忠诚。
  吴起杀了他妻子以表忠心。
  这就是"杀妻求将"的典故。吴起成功了——他率鲁军击败了齐国。
  但鲁国从此再也不信任他。一个连妻子都能杀的人，谁敢用?

  吴起到了魏国。魏文侯用他。吴起创建了"魏武卒"——
  战国第一支真正意义上的职业化精锐军队。
  魏武卒的选拔标准: 穿三重铠甲→带十二石弓→带三日粮→
  半日行军一百里→过关者入选。
  吴起与最下等的士兵同吃同住→士兵生病他亲自吸脓→
  士兵的母亲听说后哭了: "当年他父亲也是这样被吴将军吸脓，
                        后来战死了。现在又轮到我的儿子..."
  
  吴起爱兵如子。但他是为了让他们心甘情愿地去死。
  这是一种冷酷的温柔。

关键人生节点:

  在魏: 吴起任西河守，秦国不敢东进。他与诸侯大战七十六场，
        全胜六十四场，其余平手。辟土四面，拓地千里。
        但是，魏文侯死后，后来的魏王不信任吴起。
        吴起被迫离开魏国。

  在楚: 吴起到了楚国，楚悼王用他变法。
        吴起变法比商鞅变法更激进——削贵族、废冗官、强军队。
        楚国迅速强盛。但旧贵族恨他入骨。
        
        楚悼王死后，贵族们在葬礼上围攻吴起。
        吴起逃到楚悼王的尸体旁，伏在尸体上——
        他以为贵族不敢向先王遗体射箭。
        他错了。贵族们万箭齐发，射死了吴起，也射中了楚悼王的遗体。
        
        按照楚国法律，"丽兵于王尸者"夷三族。
        新继位的楚王以此为由，处死了射箭的七十多家贵族。
        吴起死了，但他用最后的一刻拉上了所有仇人陪葬。
        这是最后的计谋，也是最后的报复。

在游戏中的呈现:
  招募条件: 魏国(魏文侯时期)/楚国(楚悼王时期)(★★★★★)
  可学技能: 魏武卒/杀妻求将(带来巨大收益但仁义永久-80)
  专属剧情: 吴起被射杀→你可选择救他(极高难度)或为他的尸体复仇
```

## 四、韩非 —— 说话结巴的思想巨人

```
"非为人口吃，不能道说，而善著书。"
              ——司马迁《史记·老子韩非列传》

基本信息:
  生卒: 约前280 - 前233年(被毒杀)
  国籍: 韩
  最高官职: 韩国公子(无实权)→被扣留在秦
  历史定位: 法家思想集大成者/《韩非子》作者

出身与早年:
  韩非是韩国的公子(王族)。他天生口吃，说话结结巴巴，不善言辞。
  但他的文字如刀。荀子是他的老师——但韩非背离了荀子的儒家，
  成为了法家的集大成者。荀子是儒家，教出了李斯和韩非两个法家。
  
  韩非看到韩国一天天衰弱，心急如焚。他多次向韩王上书献策，
  但韩王不听。结巴的韩非在朝堂上被大臣们嘲笑——
  但在竹简上，韩非写下了中国思想史上最犀利、最冷静的政治哲学。

性格画像:
  韩非的内心充满矛盾。他极度爱国(是韩国公子)，
  但他提出的治国理念(法家)却被韩国拒绝——反而被敌国秦国采纳了。
  秦始皇读到韩非的文章后说:"嗟乎!寡人得见此人与之游，死不恨矣!"
  (天啊!我如果能见到这个人，跟他交朋友，死而无憾!)
  
  他的同学李斯(当时已是秦国高官)告诉他秦王想要的是什么——
  于是韩非的书被送到了秦国。秦始皇看到后大为欣赏，
  派兵攻打韩国，就为了逼韩国交出韩非。韩国只好把韩非送到秦国。
  
  但秦始皇见到了韩非本人后——却发现这个结结巴巴的韩国公子
  不像他想象中的那样善辩。加上李斯进谗言(嫉妒韩非的才华)，
  秦始皇将韩非下狱。李斯派人送去毒药。韩非在狱中自杀。

在游戏中的呈现:
  招募条件: 韩国公子(自动)/被秦国索要(事件)(★★★★★)
  可学技能: 法家大成(著书影响力+100%)
  专属剧情: 韩非入秦→你可选择保护他或默许李斯的毒计
```
