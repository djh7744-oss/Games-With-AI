# 六条人生路线 · 主线任务链

## 路线一：霸道 —— 从士兵到帝王

```
阶段 1：初入军营
├── 任务 1.1「投军」：找到所属国家的征兵官→完成基础训练→正式入伍
├── 任务 1.2「初阵」：随军参加第一场战斗→击杀3名敌人→获得什长提拔
└── 任务 1.3「新兵试炼」：完成3次巡逻/剿匪→获得百夫长提名

阶段 2：崭露头角
├── 任务 2.1「独立指挥」：获得第一支独立部队(50人)→完成一次小规模战斗
├── 任务 2.2「军功积累」：累计击杀500→获得校尉军衔
├── 任务 2.3「大战考验」：参与一场大规模战役→表现优异→获得将军衔
└── 任务 2.4「名将之路」：指挥一场关键战役并获胜→获得封侯提名

阶段 3：封疆大吏
├── 任务 3.1「受封」：获得封地→建设封地(完成3项建设)
├── 任务 3.2「灭国之战」：参与灭国战役→作为主要指挥官
├── 任务 3.3「功高震主」：声望>君主声望时触发→选择: 继续效忠/自立
└── 任务 3.4「称帝」：统一天下→称王→称帝

关键NPC：君主/同僚将领/敌国将领
关键节点：当声望超过君主时触发"功高震主"分支
```

## 路线二：王道 —— 不战而屈人之兵

```
阶段 1：入仕
├── 任务 1.1「拜师」：找到推荐人→拜入某学派门下→完成学业
├── 任务 1.2「献策」：向君主提交一份内政/外交方略→获得采纳
└── 任务 1.3「初试锋芒」：完成一次外交斡旋(调解两国争端)→获得大夫衔

阶段 2：纵横捭阖
├── 任务 2.1「合纵」：联合3个以上国家对抗强国→成功缔结同盟
├── 任务 2.2「和平吞并」：通过外交使某小国主动归附
├── 任务 2.3「化解危机」：在战争一触即发时通过外交手段化解
└── 任务 2.4「万国来朝」：5个以上国家主动与你的势力建交

阶段 3：不战而王
├── 任务 3.1「天下归心」：各国人民主动拥护你→声望达到最高
├── 任务 3.2「和平统一」：在不发动战争的情况下实现统一
└── 任务 3.3「天子」：受周天子禅让→成为天下共主

关键NPC：纵横家/各国君主/学派宗师
关键节点：武力威胁vs外交斡旋的选择决定路线走向
```

## 路线三：侠道 —— 十步杀一人，千里不留行

```
阶段 1：江湖初入
├── 任务 1.1「酒馆初遇」：在酒馆接到第一个赏金任务→完成
├── 任务 1.2「名声渐起」：完成10个赏金任务→解锁更高级任务
└── 任务 1.3「名剑传说」：听到关于某把名剑的传说→开始追寻

阶段 2：侠客行
├── 任务 2.1「行侠仗义」：完成一个保护弱者的重大任务→名声+50
├── 任务 2.2「结识豪杰」：招募3名游侠同伴→组成侠客团
├── 任务 2.3「豪门恩怨」：被某贵族赏识→成为门客→开始接触政治
└── 任务 2.4「荆轲路线」：接受刺杀某国君主的任务→无论成败，千古留名

阶段 3：侠之大者
├── 任务 3.1「民间领袖」：收服各路豪杰→建立民间武装
├── 任务 3.2「起义/被招安」：选择自己建势力或被朝廷招安→成为将军/封君
└── 任务 3.3「侠道霸业」：以侠道为基础建立自己的国家

关键NPC：酒馆老板/贵族/其他游侠/刺客组织
关键节点：荆轲任务——接受后无法回头，无论成败都会改变世界
```

## 交叉路线设计

六条路线不互斥。玩家可以在霸道路线上同时经营商业(商道)，也可以作为将军时拜师学艺(隐道)。交叉路线的触发条件和奖励需要单独设计。

```
示例组合：
  农民出身→入伍(霸道)→立功获爵→获封地→经营封地(商道)→富国强兵
  游侠(侠道)→被赏识→成为门客→入仕→将军(霸道)
  商人(商道)→捐官→入仕→参与朝政(王道)→吕不韦路线
```
