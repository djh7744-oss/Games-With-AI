# 关键NPC对话与分支

## 设计目标

> 对话不是"按A看下一句"的文字瀑布。重要的对话是叙事的分叉口——每一句回答都可能改变NPC对你的态度、触发隐藏任务、或让你走上完全不同的剧情线。本文件设计游戏中最重要的NPC对话场景及其分支后果。

---

## 一、白起 —— 最后的对话

```
场景: 长平之战后，白起站在坑杀赵卒的现场
      你作为他的部下/同僚/上级/敌人在场

对话触发: 长平之战结局选择时

白起: "你看到了吗？四十万人。我杀的。"
      "有人说我是屠夫。可是如果不杀——"
      "十年后，这四十万人会再次拿起武器。"

你的回应(四个选项):
  A. [赞同]"将军说得对。对敌人的仁慈就是对自己的残忍。"
     → 白起好感+20, 仁义-10, 白起教你"人屠"技能的基础版
     
  B. [劝谏]"将军...已经够了。留他们一命吧。"
     → 白起好感-15, 仁义+20, 白起冷笑→"你这个软心肠会害死你"
     → 但被释放的赵卒后代可能在未来回报你
     
  C. [沉默] 什么也不说
     → 白起好感±0, 你内心留下创伤(持续debuff: 士气-2, 持续1年)
     → 后续可能触发"战争创伤"个人剧情线
     
  D. [质问]"你这样滥杀，不怕报应吗？"
     → 白起好感-30, 仁义+30, 但与白起关系永久敌对
     → 白起说"报应？我等着。"→ 触发白起vs你的对立线

后续影响:
  选择A→白起日后被赐死时→你会收到他的遗书(只有选择A的人能收到)
  选择B→被救赵卒的后代在20年后可能成为你的重要NPC
  选择D→白起死亡后→"人屠"技能永久消失(没有传承)
```

## 二、荆轲 —— 易水送别

```
场景: 易水边，荆轲即将出发刺秦
      你是来为他送别的朋友/同行/被送别的对象(如果你选择刺秦)

对话触发: 侠道路线M2.4「荆轲抉择」

(如果荆轲去刺秦，你来送别)
荆轲: (望着易水)"水真冷啊。"
      你: "..."
      荆轲: "别哭。我这是去做一件名垂千古的事。"
      
      高渐离击筑→变徵之声→士皆垂泪涕泣
      
      荆轲: "风萧萧兮易水寒——"(他唱了前半句)
      
你的回应(唱后半句):
  A. "壮士一去兮不复还" (悲壮)
     → 荆轲微笑→"说得好"→出发→刺杀剧情
     → 你获得"易水送别"记忆(永久BUFF: 武勇+3, 武名+10)
     
  B. "壮士此去兮必成功" (激昂)
     → 荆轲大笑→"就冲你这句话!"→出发→士气+20%→刺杀成功概率+5%
     
  C. "壮士...为何一定要去" (挽留)
     → 荆轲沉默→"因为有人必须去"→出发
     → 你内心: 仁义+5(在乎朋友), 但武名-5(不够豪迈)
     
  D. "我跟你一起去" (同行)
     → 荆轲震惊→"你...?"
     → 此选择改变整个剧情→两人同去刺秦
     → 成功概率翻倍(两人配合)→但两人都可能死
     → 如果两人都死→游戏触发双重继承
     → 如果一人死一人活→活下来的人背负巨大的情感重量

(如果你是那个去刺秦的人，荆轲来送你)
荆轲: "兄弟...活着回来。"
      "如果回不来——我每年今日，来这易水边，请你喝酒。"

此场景为全游戏最情感化的时刻之一
几乎所有玩家都会在此停留、思考、截图
```

## 三、商鞅 —— 变法的代价

```
场景: 商鞅变法成功后，他预感到自己的命运
      你是他的门客/学生/朋友

对话触发: 变法完成+商鞅忠诚度事件

商鞅: "你知道我制定了多少条法律吗？"
      "你知道每条法律得罪了多少人吗？"
      商鞅看着窗外→"我没有朋友，只有敌人。"
      "但我没有选择——秦国不变法，就会灭亡。"
      
      "你说——值得吗？"

你的回应:
  A. "值得。秦国的强大证明了你的正确。"
     → 商鞅苦笑→"谢谢。至少有人懂我。"
     
  B. "不值得。你牺牲了太多——包括你自己。"
     → 商鞅沉默→良久→"也许你说得对。但已经回不去了。"
     
  C. "变法的内容是对的，但方式是否可以温和一些？"
     → 商鞅摇头→"温和？温和就是给反对者喘息的机会。"
     → "我们法家不玩温和。"
     
  D. "老师，等变法完成后，你打算怎么办？"
     → 商鞅: "变法完成了，我这样的人...也就不需要了。"
     → 他预见到了自己的死亡

后续事件:
  商鞅最终被车裂→你作为他的学生/朋友
  可以选择: 收尸(仁义+)/躲避(自保)/复仇(对抗秦国王权)
  如果你收尸→商鞅的变法笔记(遗物)传给你→法家影响力+100%
```

## 四、廉颇与蔺相如 —— 将相和

```
场景: 廉颇听闻蔺相如为了国家大局处处忍让自己
      背着荆条来请罪

对话触发: 你在赵国任职+廉颇蔺相如关系修复事件

廉颇: (赤裸上身，背着荆条，跪在蔺相如门前)
      "蔺相如！我廉颇是个粗人！"
      "我嫉妒你，在背后说你坏话！"
      "但你处处为国着想，不跟我计较——"
      "我不是人！我向你请罪！"

蔺相如: (急忙扶起)
        "廉将军！使不得！"
        "你我同朝为臣，和则赵国兴，斗则赵国亡——"
        "我蔺相如算得了什么？赵国才是大事！"

此时你在场，两人看向你:
  廉颇: "小子，你看到了。这就是将相和。记住。"

你的回应:
  A. "两位大人，赵国能有你们，是赵国的福气。"
     → 两人好感+20, 你学到了"将相和"的精神
     
  B. (默默点头，不说话)
     → 你见证了这个历史时刻→获得"将相和见证者"称号
     
  C. "廉将军，蔺大人，我愿意跟随两位学习。"
     → 廉颇: "跟我学打仗!" 蔺相如: "跟我学做人!"
     → 解锁师徒关系→可以向两人学习(统率+魅力双修)

后续:
  你将"将相和"的精神应用到你自己的团队中→
  如果团队中两人关系敌对→你可以尝试调解→成功后获得"和事佬"称号
```

## 五、吕不韦 —— 奇货可居

```
场景: 赵国邯郸，吕不韦第一次见到秦国质子异人
      你在场(作为吕不韦的朋友/商业伙伴)

对话触发: 商道路线+在邯郸遇到吕不韦

吕不韦: (看着远处落魄的异人)"你看到那个人了吗？"
        你: "那是秦国质子吧？看起来...不太走运。"
        
        吕不韦: (眼睛发光)"不走运？他是我的金矿!"
                "此人乃秦国太子之子，现在落魄——"
                "但如果帮他回国继位..."
                "你想想——投资一个未来的秦王——回报是多少？"
                
        他转向你→"我打算把全副身家押在他身上。"
        "你要入股吗？"

你的选择:
  A. 入股→投入50000金→成为吕不韦的商业伙伴
     → 如果异人成功继位→你获得秦国官职+巨额回报
     → 如果失败→你血本无归(高风险投资)
     
  B. 不投→"吕兄，这风险太大了"
     → 吕不韦: "富贵险中求。你不来，我自己来。"
     → 吕不韦独自投资→成功后你失去参与机会
     
  C. 观望→"让我想想"
     → 可以在30天内随时决定→但越晚投入→股份越少

后续:
  这就是历史上真实的"奇货可居"故事
  如果你入股→这是游戏中最赚钱的一笔投资(回报率500-1000%)
  但也是最高风险→异人可能在回国前被杀/被扣留/不被立为太子
```

## 六、孙膑与庞涓 —— 师兄弟对决

```
场景: 马陵道，孙膑设伏，庞涓中计
      你在场(作为任何一方的部下/旁观者)

对话触发: 马陵之战(魏齐之战)触发时

夜幕降临，马陵道。
一棵大树被剥了皮，上面写着字。
庞涓命人点火看字→"庞涓死于此树之下"

庞涓: (看到火光中的字→瞳孔收缩)
      "孙膑...果然是你..."
      
      箭矢如雨。
      庞涓中箭→他知道自己输了。
      
      庞涓: "遂成竖子之名!"(就让这小子成名吧!)
      拔剑自刎。

孙膑: (在远处，听到庞涓死了)
      "师兄...你何必走到这一步。"
      孙膑低头→他看着自己的残腿(庞涓害的)
      没有说话。

你在场→你的反应:
  A. [孙膑一方] 安慰孙膑→"军师，战争结束了。"
     → 孙膑: "不。我心里的战争永远不会结束。"
     
  B. [庞涓一方] 为庞涓收尸→信义+20(义气)
     → 孙膑知道后→对你的敬意+10→"你是值得尊敬的对手"
     
  C. [旁观者] 目睹这一切→沉默
     → 获得"马陵之战见证者"记忆→智略+3
     → 你学到了: 嫉妒足以毁掉天才

此场景为全游戏最有悲剧色彩的时刻
兄弟反目→天才相残→没有赢家
```

## 对话分支设计原则

```
1. 每个对话至少有2个选项，重要对话3-4个
2. 选项之间不是"好/坏"而是"不同价值观"
3. 对话结果持续影响后续剧情(跨章节)
4. 某些对话选项会锁死/解锁隐藏内容
5. 对话选择会被NPC记住(影响关系/信任)
6. 重要对话可以跳过但不能"重来"(不可逆)
7. 对话语气匹配角色性格(玩家选了什么出身/学派→影响可选语气)
8. "沉默"永远是一个有效选项

对话的情感层次:
  Lv1: 功能性对话(商人: "要买什么?")
  Lv2: 信息性对话(NPC: "听说秦军要来了")
  Lv3: 关系性对话(战友: "打完这场仗，一起去喝酒")
  Lv4: 道德性对话(将军: "你选择杀还是不杀?")
  Lv5: 终极性对话(荆轲: "风萧萧兮易水寒...")
  
  每段Lv4+对话都值得单独设计
```
