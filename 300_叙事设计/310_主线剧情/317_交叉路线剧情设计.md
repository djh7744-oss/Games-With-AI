# 交叉路线剧情设计

## 设计目标

> 六条路线不互斥，玩家可同时推进多条路线。交叉路线是游戏最深度的叙事体验——当两条路线都在阶段2以上时，触发独特的"交叉里程碑"事件。这些事件不是你"选择"的，而是你的人生"自然发生"的。

---

## 交叉里程碑触发机制

```
交叉条件:
  两条路线都达到阶段2(里程碑≥4)
  玩家在该两条路线对应的属性都≥50
  触发一个"交叉事件"(Crossroad Event)

交叉事件类型:
  Type A: 融合型 → 两条路线合二为一，解锁独特身份
  Type B: 冲突型 → 两条路线的价值观冲突，必须做出选择
  Type C: 互补型 → 两条路线互相增强，获得双向加成
```

## 15组交叉里程碑

### 霸道 × 王道 = 霸王之道

```
触发条件: 霸道阶段2 + 王道阶段2 + 统率≥60 + 魅力≥55

交叉事件「上兵伐谋」:
  场景: 你同时收到两份命令→
        将军命令你:"明日攻城，由你打头阵"
        使臣请求你:"请给我两天时间，我可以让守将投降"
  
  选择:
    A. 听将军的→立即攻城→霸道推进→奖励: 统率+5
    B. 给使臣两天→等待外交结果→王道推进→奖励: 魅力+5
    C. 两边都做→白天攻城施压+夜晚遣使劝降→(最优但最难)
       成功→守将在攻城第一天就投降→奖励: 统率+3/魅力+3/称号"文武双全"

  失败: C选项如果攻城伤亡太大→守将拒绝投降→两面都失败

解锁身份: "霸王"(文武双全的最高评价)
效果: 军事威胁+外交谈判可同时进行(普通只能二选一)
```

### 霸道 × 商道 = 以战养商

```
触发条件: 霸道阶段2 + 商道阶段2 + 统率≥55 + 经营≥55

交叉事件「战争财」:
  场景: 一场大战即将爆发→你发现了一个巨大的商机
        敌军需要粮草→你的商队可以卖给他们
        友军需要兵器→你的工坊可以供应
        "战争是地狱，但地狱里也有生意"
  
  选择:
    A. 只供应友军→正常盈利→仁义+，但利润有限
    B. 两边都供应→"战争没有立场，商人只有利益"
       →利润×3但仁义-40, 信义-20
    C. 囤积居奇→等价格涨到最高再出手→利润×5但可能被征用

解锁身份: "军商"(以战养商，以商养战)
效果: 战争期间军费消耗-30%+战利品价值+50%
```

### 霸道 × 侠道 = 义军之将

```
触发条件: 霸道阶段2 + 侠道阶段2 + 武勇≥65 + 仁义≥45

交叉事件「军心所向」:
  场景: 你的部队被命令去镇压一群"暴民"→
        但你去了才发现→他们只是活不下去的农民
        军官命令你"杀!"→你的良知说"不!"
  
  选择:
    A. 执行命令→击杀暴民→武名+但仁义-30
    B. 抗命→保护农民→仁义+30但被降职/处罚
    C. 折中→驱散农民但不上报→"我什么都没看见"

解锁身份: "义将"(士兵发自内心追随你)
效果: 麾下士兵士气+20%, 仁义声望影响军队战斗力
```

### 王道 × 商道 = 吕不韦之路

```
触发条件: 王道阶段2 + 商道阶段2 + 魅力≥60 + 经营≥65

交叉事件「奇货可居」:
  场景: 你在赵国发现了一个落魄的秦国质子→
        他叫异人，是秦国太子之子，但被送来当人质
        吕不韦(如果他在场)微笑着说→"此奇货可居也"
  
  选择:
    A. 投资异人→花重金帮他改善生活→结交秦国权贵→
       几年后如果他继位→你将成为秦国最有权势的人
       (这就是历史上吕不韦的操作)
    B. 不投资→"政治风险太高，我还是安心做生意"

解锁身份: "国商"(用财富影响国家命运的人)
效果: 解锁"政治投资"特权→用金换取政治影响力
特殊结局: 投资成功→"吕不韦结局"
```

### 王道 × 隐道 = 圣贤治国

```
触发条件: 王道阶段2 + 隐道阶段2 + 魅力≥60 + 智略≥65

交叉事件「经世致用」:
  场景: 你的学说已经完成了理论→现在你想让它改变世界
        你向君主上书→"请以臣之学说，治理天下"
        但老臣们反对→"书生之见，不切实际"
  
  任务: 在你的封地/管辖区域试行你的学说→用实际成效反驳反对者
  成功: 试行成功→君主采纳→你的学说成为国策→"从学者到国师"
  失败: 试行失败→承认理论需要修正→"我还是回去继续做学问"

解锁身份: "儒相/法家国师"(根据你的学派)
效果: 国家政策可包含学派理念→政策效果+25%
```

### 侠道 × 隐道 = 侠儒双修

```
触发条件: 侠道阶段2 + 隐道阶段2 + 武勇≥55 + 智略≥60

交叉事件「文武之辩」:
  场景: 你的侠客朋友说"书生无用，关键时刻还得靠拳头"
        你的学者朋友说"匹夫之勇，不足为谋"
        两人争吵→看向你→"你说!"
  
  选择:
    A. 赞同侠客→武勇+3, 智略-2(暂时)
    B. 赞同学者→智略+3, 武勇-2(暂时)
    C. 调解→"文能安邦，武能定国，缺一不可"→魅力+5

解锁身份: "侠儒"(文武双全的游侠学者)
效果: 武勇和智略互相给予10%加成
```

### 商道 × 匠道 = 工艺大师企业家

```
触发条件: 商道阶段2 + 匠道阶段2 + 经营≥60 + 工艺≥60

交叉事件「从作坊到产业」:
  场景: 你的工坊生产出了高品质的武器→
        但单件精品 vs 批量生产→你面临选择
  
  选择:
    A. 坚持精品路线→每件都是艺术品→高利润但低产量
    B. 转向批量生产→品质降低但产量翻倍→"让好武器惠及更多人"
    C. 两条线并行→精品定制线+标准量产线→(需要极高管理能力)

解锁身份: "工业先驱"(战国时代的"企业家")
效果: 工坊可同时运行精品线和量产线
```

### 霸道 × 匠道 = 兵器革新者

```
触发条件: 霸道阶段2 + 匠道阶段2 + 统率≥55 + 工艺≥60

交叉事件「军备革新」:
  场景: 连续的战争中你发现→武器质量决定伤亡率
        "如果我的士兵拿着更好的武器..."
        你开始亲自设计/监督军队装备的锻造
  
  效果: 麾下军队的装备品质自动+1级(上限紫色)
        武器维护成本-30%

解锁身份: "兵器总监"(军队装备的最高负责人)
```

### 商道 × 隐道 = 知识商人

```
触发条件: 商道阶段2 + 隐道阶段2 + 经营≥55 + 智略≥60

交叉事件「知识就是财富」:
  场景: 你发现竹简(书籍)在不同国家之间有巨大差价
        "知识可以被买卖"→你开创了"书籍贸易"
        但学者们反对→"学问岂可买卖?"
  
  选择:
    A. 尊重学者→放弃书籍贸易→仁义+
    B. 坚持商业→"知识传播本身就是好事"→智略+
    C. 折中→只卖给学宫和官员→"高雅的知识传播"

解锁身份: "书商/文化传播者"
效果: 科技传播速度+50%→你成为知识流动的中间人
```

### 霸道 × 王道 × 侠道 = 天下三分

```
触发条件: 三条路线同时达到阶段3
  (极其罕见，需要极高的属性+极长的游戏时间)

交叉事件「三条路」:
  场景: 你站在人生的十字路口→
        军事上你可以统一天下(霸道)
        外交上你可以联合天下(王道)
        道义上你可以感召天下(侠道)
        
        你选择哪一条?

  三条路线同时完成的结局:
    "天命之子结局"→无论选择哪种方式→
    你都将统一天下→但方式决定了后世对你的评价
```

## 交叉路线叙事设计原则

```
1. 自然涌现
   交叉事件不应是"弹窗提醒"
   而应通过NPC、事件、世界状态自然引导
   例: 霸道×商道的"战争财"→在战争前夕自动触发商人NPC来找你

2. 道德灰度
   没有"正确"选择→只有"不同"选择
   每个选择都有合理的理由和真实的代价
   玩家应该纠结→这正是好叙事的标志

3. 持久影响
   交叉选择的结果持续影响后续游戏
   选择"吕不韦路线"→之后的外交/政治选项永久改变
   选择"义军之将"→你的军队永远有独特的士气加成

4. 不可逆
   大多数交叉选择是单向的
   一旦走上"以战养商"→回头做"纯粹商人"会损失惨重
   这增加了决策的重量感

5. 历史回响
   某些交叉路线会触发"历史评价"机制
   统一后→后世史官会对你的选择做出评价
   "XX以武立国，以商富国，以儒治国，后世称其为..."
```
