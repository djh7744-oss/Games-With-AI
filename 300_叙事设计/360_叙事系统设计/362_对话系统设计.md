# 对话系统设计

## 一、对话架构

```
对话分为五个层级:

Lv1 系统消息 —— "收到500金。" (无对话UI)
Lv2 功能性对话 —— "要买什么?"→商品列表 (简单选项)
Lv3 信息性对话 —— 打听情报/询问意见 (分支选项2-3个)
Lv4 关系性对话 —— 影响好感/信任/忠诚 (分支选项3-4个,有后果)
Lv5 叙事性对话 —— 关键剧情场景 (全屏对话UI,逐句演出,4+选项)
```

## 二、对话UI设计

```
Lv3-4 对话UI (常规对话):
  ┌─────────────────────────────────────┐
  │ [NPC头像]  [NPC名字]  [好感度条]     │
  │                                     │
  │ "客官，要买点什么?"                  │
  │                                     │
  │ ┌─────────────────────────────┐     │
  │ │ A. 看看货物                  │     │
  │ │ B. 打听附近有什么消息        │     │
  │ │ C. 闲聊 (提升好感)           │     │
  │ │ D. 离开                      │     │
  │ └─────────────────────────────┘     │
  └─────────────────────────────────────┘

Lv5 对话UI (叙事对话——关键剧情场景):
  全屏半透明背景 + 角色立绘/3D模型
  对话逐句显示(可快进不可跳过首次观看)
  当前说话者高亮
  
  选项区在对话结束后出现:
  ┌─────────────────────────────────────┐
  │                                     │
  │  [角色A立绘]    [角色B立绘]          │
```

## 三、对话选项设计原则

### 选项不是"好/坏"——是"不同的价值观"

```
❌ 错误设计:
  A. [善良] 帮助他 (+仁义)  ← 显然是"好"选项
  B. [邪恶] 杀了他 (+武名)  ← 显然是"坏"选项

✅ 正确设计:
  A. "我帮你。但我要你记住——你欠我一个人情。"
     → 实用主义: 帮了，但有条件
  B. "不需要回报。我只是做我认为对的事。"
     → 理想主义: 帮了，不求回报
  C. "这是你自己的问题。自己解决。"
     → 个人主义: 不帮，但也不伤害
  D. [假装没听见，走开]
     → 逃避主义: 不介入

  每个选项都有合理的理由。
  不同玩家会选择不同的选项——没有"正确答案"。
```

### 选项数量规则

```
Lv2: 2-3个选项 (简单交互)
Lv3: 3-4个选项 (信息/意见)
Lv4: 3-4个选项 + 至少1个有"持久后果"
Lv5: 4-5个选项 + 至少2个有"重大后果" + "沉默"永远是可选选项

"沉默"选项:
  当玩家不想表达立场/不知道该怎么选时
  → 选择沉默本身就是一个"表态"
  → NPC会注意到你的沉默——并对此做出反应
```

## 四、对话语气与角色一致性

### 语气受因素影响

```
NPC说话的语气取决于:
  1. NPC性格参数 (傲慢/平和/热情/冷漠...)
  2. NPC与玩家的关系 (好感度)
  3. 玩家的声望 (高声望→尊敬，低声望→轻蔑)
  4. 当前世界状态 (和平→轻松, 战争→紧张)
  5. 玩家的出身 (贵族→客气，农民→可能轻蔑)

示例: 同一个NPC(商人)对"买东西"的对话

  好感80+高声望:
    "哎呀，是您来了! 请坐请坐。最近新到了一批好东西，
     别人我都不给看——您先挑。价钱好商量!"

  好感50中声望:
    "客官要买什么? 本店货真价实，童叟无欺。"

  好感10低声望:
    "看看可以，别乱碰。碰坏了你赔不起。"

  好感-30(讨厌你):
    (看到你走进来，假装没看见，继续算账)
```

### 特殊对话标记

```
对话中的特殊标记(不显示给玩家，但影响演出):

  [停顿] → NPC在对话中短暂停顿(0.5-1.5秒)
  [低声] → 音量降低，可能伴随镜头拉近
  [颤抖] → 声音颤抖(恐惧/愤怒/寒冷)
  [笑] → 伴随笑声(真笑/假笑/苦笑，基于上下文判断)
  [哭] → 哭泣或哽咽
  [独白] → 这是NPC的内心想法，其他角色听不到
  [旁白] → 叙事者的注释，打破第四面墙(极少使用)

示例:
  白起: "我一生杀了近百万人。[停顿]但我不记得任何一张脸。"
  荆轲: "风萧萧兮易水寒——[低声]壮士一去兮不复还。"
  商鞅: "我制定的法律...困住了我自己。[苦笑]"
```

## 五、对话记忆系统

```
NPC会"记住"你说过的话:

记忆类型:
  承诺记忆: "你答应过帮我找儿子的。找到了吗?"
            → NPC会在后续对话中追问
  侮辱记忆: "上次你说我是'唯利是图的小人'——我没忘。"
            → 好感永久-20
  秘密记忆: "那件事——我不会说出去。但你也别忘了我替你保密。"
            → NPC可能在未来用这个秘密做筹码
  情感记忆: "那天在易水边...你哭了。我看到你哭了。"
            → 情感绑定的对话

记忆时限:
  重要记忆(承诺/侮辱): 永久
  一般记忆(闲聊): 30-90天后衰减
  情感记忆: 永久(但可能被更强烈的情感覆盖)
```

## 六、对话与游戏性的"咬合"

```
对话不应"暂停游戏"——对话应该是游戏的一部分:

✅ 正确: 
  对话中得知"秦军三天后进攻"→对话结束→你只有三天准备
  (对话直接影响了游戏性决策)

❌ 错误:
  对话→"秦军要来了"→大量信息→对话结束→但游戏性没有任何变化
  (对话与游戏性脱节)

✅ 正确:
  在讨价还价对话中，你的经营属性影响可选选项
  (角色属性→对话选项→游戏性结果)

✅ 正确:
  在劝降对话中，你的声望影响对方的反应
  (游戏性积累→影响叙事→再影响游戏性)
```
