# 长平之战 ——《四十万》

## 剧本概述

| 项目 | 内容 |
|------|------|
| 时间范围 | 前262年-前260年 |
| 篇章数 | 8章 |
| 视角 | 双主角：白起(秦) / 廉颇→赵括(赵) |
| 核心机制 | 饥饿系统、堑壕对垒、反间计、包围与突围 |
| 结局数 | 8种 |
| 预计时长 | 8-12小时 |
| CG级别 | A级(2分30秒) |
| 核心情感 | 绝望——"当粮食吃完，当援军不来，你还能握住手中的剑吗？" |

## 角色设定

### 可扮演角色

**白起 (秦国视角)**
- 年龄: 约38岁
- 身份: 秦国大良造(战前为左庶长)
- 配音风格: 低沉、缓慢、几乎不带感情
- 性格关键词: 冷酷、精确、沉默、内心深处有裂缝
- 角色弧线: 从"秦国最强的武器"到"意识到自己也是受害者"

**廉颇 (赵国视角·前期)**
- 年龄: 约55岁
- 身份: 赵国上将军
- 配音风格: 沙哑、稳重、带着老将的疲惫
- 性格关键词: 坚韧、务实、不图虚名、知天命
- 角色弧线: 从"坚持正确的战术"到"被剥夺指挥权的无力"

**赵括 (赵国视角·后期)**
- 年龄: 约25岁
- 身份: 赵国将军(马服子)
- 配音风格: 年轻、自信→逐渐崩塌→最后的决绝
- 性格关键词: 才华横溢但缺乏经验、自负→幻灭
- 角色弧线: 从"纸上谈兵的年轻人"到"被历史碾碎的悲剧"

## 八章详细剧本

### 第一章「上党之争」——命运的导火索

```
时长: 45-60分钟
视角: 赵括(引入)

开篇: 公元前262年，韩国上党郡。
      秦军攻占韩国野王，切断了上党与韩国本土的联系。
      上党太守冯亭站在城墙上，看着远方的秦军营地。
      
冯亭: "上党...保不住了。与其降秦，不如献赵。"

切入赵国朝堂:
  平原君: "天赐上党! 不费一兵一卒，领土扩十七城!"
  蔺相如: "这是祸根! 秦国会因为上党与我们开战!"
  赵王: (犹豫) "赵括，你是年轻人，你怎么看?"
  
  (玩家首次做出选择)
  选项A: "上党是战略要地。控制了上党，就控制了太行山。必须接受。"
  选项B: "蔺大人说得对。接受上党就是引火烧身。"
  选项C: "接受上党，但同时派人去秦国谈判。两手准备。"

  历史路线: 赵国接受上党 → 秦国震怒 → 战争爆发
  架空路线: 如果拒绝 → 上党归秦 → 暂时和平(但秦国更强了)

过渡: 上党被赵国接收。秦军开始向长平集结。
      战争机器已经开动。没有人能阻止了。
```

### 第二章「廉颇守势」——老将的壁垒

```
时长: 90-120分钟
视角: 廉颇(赵国)

开篇: 长平前线。廉颇在巡视三道防线。
      他刚刚完成了防线的部署。
      
廉颇: (对手下将领)"秦军远来，粮草补给线长。我们以逸待劳。"
      "谁也不准主动出击。违令者斩。"

游戏性核心循环:
  每日管理: 
    早晨: 巡视防线 → 检查士兵状态 → 分配当天任务
    白天: 秦军挑战 → 选择"应战"或"不应"
          应战 → 小规模冲突(战斗教学)
          不应 → 士兵士气微降，但保存了实力
    傍晚: 统计伤亡/粮草/士气 → 写军报发回邯郸
    深夜: 和士兵聊天 → 了解他们的故事 → 建立情感连接

  关键事件(第15天): 秦军大规模进攻。
    廉颇亲自坐镇第一道防线。
    血战一天。防线守住了——但伤亡惨重。
    一个年轻的赵国士兵死在廉颇怀中。
    廉颇合上他的眼睛。没有说话。

  关键事件(第30天): 粮草开始紧张。
    赵国的补给线被秦军骚扰。
    廉颇要求增加补给 → 邯郸回复:"知道了"但什么都没送来。
    
  关键事件(第60天): 赵王派使者来前线。
    使者: "陛下问你——什么时候进攻?"
    廉颇: "现在不能进攻。请陛下再给我半年。"
    使者: "...陛下可能等不了半年。"
    
  伏笔: 使者的眼神闪烁 → 朝中有人想换掉廉颇

章节结尾: 廉颇站在防线高处。远方的秦军营地灯火通明。
          他自言自语: "能撑多久...就看天了。"
          
          (画面切换到咸阳)
          范雎对秦王说: "臣有一计，可破廉颇。"
```

### 第三章「范雎反间」——看不见的敌人

```
时长: 60-90分钟
视角: 白起(秦国·咸阳) + 赵括(赵国·邯郸)

秦线——白起视角:
  秦昭襄王召见白起: "范雎说可以用反间计除掉廉颇。你怎么看?"
  白起: "廉颇是良将。除掉他，赵国无人能敌我。"
  秦王: "那你去办。"
  
  白起被任命为秘密总指挥。但他不在前线——
  他在咸阳策划这场看不见的战争。
  
  游戏性: 
    - 派遣间谍进入赵国(选择间谍/选择目标官员/选择谣言内容)
    - 收买赵国朝臣(选择贿赂对象/金额/条件)
    - 散布不同的谣言:
      谣言A: "廉颇老了，只会死守，浪费国家钱粮"
      谣言B: "赵括精通兵法，是赵国的希望"
      谣言C: "廉颇与秦军有秘密往来"
    - 监测谣言在赵国朝堂的传播效果

赵线——赵括视角:
  赵国朝堂。议论纷纷。
  有大臣上书请求换将。赵王动摇。
  
  赵括被召入宫中。
  赵王: "你父亲是名将。你从小就学兵法。你觉得——"
        "你能打赢秦军吗?"
  
  赵括的选择(玩家):
    A. (谦虚)"臣不如廉颇将军。但臣愿去前线学习。"
       → 赵王不悦但不一定换将(魅力判定)
    B. (自信)"臣熟读兵法，只需一次机会证明自己。"
       → 赵王大喜→换将
    C. (诚实)"此战风险极大。如果换将是秦国的诡计——"
       → 赵王脸色变→"你是在教训本王吗?"(高风险)
  
  无论选择什么，朝堂气氛已经变了。
  赵括离开时，听到身后有大臣小声说:"马服子...纸上谈兵。"
  赵括握紧了拳头。

章节结尾: 
  白起收到密报:"赵王已下令换将。廉颇被召回，赵括接任。"
  白起放下竹简。他的手指在"赵括"这个名字上停了一秒。
  "年轻人。祝你好运。"
  但他知道——赵括不会有好运。
```

### 第四章「纸上谈兵」——年轻的将军

```
时长: 90-120分钟
视角: 赵括(赵国·长平前线)

开篇: 赵括到达长平前线。
      廉颇在交接。两人面对面站着。
      
廉颇: "这四十万人，交给你了。"
       (沉默片刻)
       "赵括...不要太自信。战场不是书简。"
赵括: "廉将军，我会记住的。"
       但赵括的眼神里——廉颇看到了年轻时的自己。
       那份自信。那份"我一定能行"。
       廉颇叹了口气。"年轻人..."

游戏性:
  赵括接掌指挥权。
  他的第一个命令: "全线出击，渡河决战!"
  
  这是一个重大的军事决策。但游戏给玩家"时机选择":
    - 立即出击(历史路线)
    - 先侦察再出击(偏谨慎)
    - 派小部队试探(折中)
  
  如果选择"立即出击"(历史):
    赵军全线渡河。气势如虹。
    秦军"败退"。赵军追击。
    追击中，赵括副将提醒:"将军，秦军败退得太整齐了..."
    赵括:(可以选择听或不听)
    
  如果赵括选择追击到底 → 中计 → 被切断后路 → 被包围。
  
  被包围的瞬间:
    赵括站在高处，看到四面八方涌来的秦军。
    他明白了——这是陷阱。
    他的副将惊慌:"将军! 我们被包围了!"
    赵括:(沉默五秒)"...我知道。"
    
  这个停顿是赵括角色弧线的转折点。
  从"自信"到"意识到自己的错误"——只用了一瞬间。

章节结尾: 夕阳下，四十万赵军被围在长平谷地。
          赵括坐在地上，手里握着父亲留给他的剑。
          远方传来秦军的号角。
```

### 第五章「白起奇袭」——猎人与猎物

```
时长: 60-90分钟
视角: 白起(秦国·前线)

开篇: 白起秘密到达前线。
      他走进军帐。所有秦军将领起立。
      白起: "有敢泄露武安君在此者，斩。"
      
      然后他开始部署:
      "二万五千人，切断赵军退路。"
      "五千骑兵，插入赵军营垒之间，分割他们。"
      "主力在正面——但不要进攻。围而不攻。"
      
      一个将领问:"将军，为什么不直接进攻?"
      白起: "因为他们还有粮食。等他们吃完。"
      
游戏性:
  白起部署包围圈。
  玩家需要: 
    - 在关键位置布置兵力
    - 预测赵军可能的突围方向
    - 管理包围圈的"松紧度"(太紧→赵军拼死突围/太松→可能逃走)
    - 处理降卒(零星投降的赵兵)
  
  白起的内心独白(选择触发):
    看着被围的赵军营地——
    "赵括...二十五岁。比我当年还年轻。"
    "如果给他十年——不，五年——他可能成为一代名将。"
    "但战争不会等人成长。"
    
  关键事件: 赵军第一次突围尝试。
    白起站在高处观战。
    赵军骑兵冲向秦军——战斗激烈——但最终被击退。
    白起看到了赵括的身影——在乱军中，年轻的将军亲自带队冲锋。
    白起的副将: "将军，要放箭吗?"
    白起: "不。让他回去。"
    副将: "为什么?"
    白起: "...让他多活几天。"
    
  这可能是白起少有的"不忍"。
  但也只是"多活几天"。

章节结尾: 白起在营帐中写信给秦王:
          "赵军已困。不出两月，粮尽力竭。"
          写完。他看着信。加了一句:
          "赵括此人，年少有才。若降，望王赦之。"
          然后他把这一句划掉了。
          夜深。白起一个人坐在黑暗中。
```

### 第六章「饥饿围困」——四十六天

```
时长: 120-180分钟(DLC最长章节)
视角: 赵括(赵国·被围中)

开篇: 第一天。粮官报告:"存粮还能支撑三十天。"
      赵括:"节粮。每人减半。"
      第三十天。粮官报告:"...粮尽了。"
      赵括:"杀马。"
      第四十天。马也吃完了。
      第四十六天。赵括下令:"全军突围。"

游戏性: 这是整个DLC最独特的章节。
  玩家管理一个"正在崩溃的军队"。
  
  每日决策:
    粮食分配: 
      第一天: "每人减半"→士兵抱怨但还行
      第十五天: "每人一碗粥"→士兵开始虚弱
      第三十天: "已经没有粥了"→吃草根/树皮/皮带
      第四十天: "...有些人在吃死人。"
    
    士气管理:
      安抚 → "援军会来的! 再撑几天!"
      镇压 → "有敢言降者，斩!"
      诚实 → "我不知道援军会不会来。但我们必须试。"
      
    医疗:
      伤兵营人满为患。没有药。
      每天有人死去。
      
    突围尝试:
      第10天: 小规模试探→被击退
      第20天: 集中兵力冲击一点→几乎成功→但秦军援军赶到
      第40天: 最后一次绝望的尝试→失败
  
  情感事件(程序化选择):
    - 一个老兵把最后一口干粮给了年轻士兵
    - 一个母亲的信被送到前线(通过信鸽)→收信人已经死了
    - 赵括的副将郑安平:"将军...如果我死了...告诉我娘，我没有投降"
    - 夜间，赵括听到士兵在唱赵国的民歌
    - 赵括在月光下擦拭父亲的剑——这把剑从未上过战场
  
  第46天——最后的突围:
    赵括召集所有还能站起来的士兵。
    他站在高处，看着这些瘦骨嶙峋、但仍握着武器的赵国人。
    
    赵括:"我知道——你们恨我。"
         "是我把你们带到这里。是我的错。"
         沉默。
         "...但我们现在只有一件事可以做。"
         "走出去。走出去!"
         
    五千骑兵——最后的力量——冲向秦军阵线。
    赵括在最前面。
    箭如雨下。
    
    赵括中箭。从马上摔落。
    他倒在地上，看着长平的天空。
    和父亲教他兵法时看到的天空——一样蓝。
    
    他的副将郑安平跪在他身边，抱着他。
    郑安平:(哭着)"将军...将军..."
    
    赵括的遗言(取决于玩家在游戏中的表现):
      如果打得好(虽败犹荣): "我尽力了。告诉父王——对不起。"
      如果打得不好: "纸上谈兵...我终究是...纸上谈兵..."
      如果玩家选择投降(可选): "让士兵们活下去。这是我最后的命令。"

章节结尾: 郑安平抱着赵括的尸体，走向白起的营地。
          他的身后，四十万饥饿的赵卒，放下了武器。
```

### 第七章「最后突围」——赵括的终末

(本章为赵括视角的最后一章，衔接第六章结尾)

```
如果赵括战死(历史线):
  赵括战死后 → 赵军失去指挥 → 投降
  白起接收降卒。他看着赵括的尸体。
  "找个地方，好好葬了他。他是值得尊敬的对手。"
  但接下来，白起面对的是——如何处置四十万降卒。

如果赵括突围成功(架空线):
  赵括带着残部(约3000-5000人)杀回赵国。
  他回到邯郸——满身是血，跪在赵王面前。
  "臣...输了。愿以死谢罪。"
  赵王的反应取决于赵括在朝中的人际关系。
  可能被赦免(极低概率) / 被贬为庶人 / 被处死。
  
  如果赵括活下来:
    他余生都在为长平之战的失败赎罪。
    但他从失败中学到了真正的战争。
    十年后——如果赵国还没灭亡——他会成为比父亲更出色的将领。

如果赵括被俘(中间线):
  白起见到了这个年轻的对手。
  两人面对面站着。
  
  白起:"你打得不错。如果粮食再多十天——你可能突围成功。"
  赵括:"...输了就是输了。"
  白起: "我可以给你一个机会。为秦国效力。"
  
  选择:
    A. 拒绝 → "我是赵人。" → 白起尊重他 → 被关押(但不会被杀)
    B. 接受 → 被赵国唾弃 → 但可能在未来影响秦国军事决策
    C. 自杀 → "我赵括，生是赵人，死是赵鬼。"
```

### 第八章「坑杀抉择」——历史的重量

```
时长: 30-45分钟(短但极其沉重)
视角: 白起

开篇: 四十万赵卒跪在长平谷地。
      白起站在高处。秦军将领等着他的命令。
      
      白起的副将:"将军，他们投降了。"
      白起: "...我知道。"
      
      副将: "四十万人。我们怎么养?"
      白起没回答。
      
      另一个将领: "放了他们...十年后，这四十万人会再次拿起武器。"
      
      白起闭上眼睛。

游戏性: 这是纯粹的叙事章节。没有战斗。
  白起在做出这个决定前——可以在营地中走一走。
  
  他可以:
    - 和降卒说话: 一个赵国老兵说"我儿子还在等我回家"
    - 和秦军将领讨论: 副将说"将军，这就是战争"
    - 看看粮草账本: "如果收编降卒，秦军粮草撑不过十天"
    - 读秦王的最新命令: (信中没有明说，但暗示"处理掉")
    
  白起的内心独白——全DLC最沉重的一段:
    "四十万人。每一个人都有名字。"
    "有父母。有妻子。有孩子。"
    "如果放了他们——十年后，我可能要再杀四十万人。"
    "如果杀了他们——我这辈子，杀的人超过了任何将领。"
    "...后世会怎么称呼我?"
    "屠夫。大概。"

  最终决定(四个选项):
    A. 全部坑杀 (历史)
       → 白起传令。秦军开始行动。
       → 白起转过身，不看了。
       → 当晚，他坐在黑暗中。没有吃饭。没有说话。
       → "人屠"之名，永留史册。
    
    B. 坑杀一半
       → 杀二十万，放二十万。
       → 白起说:"让活着的人回去——告诉赵王，不要再来送死。"
       → 效果: 秦国朝堂不满(觉得你不够狠)，赵国极度恐惧
    
    C. 全部收编
       → 白起向上报告:"降卒愿为秦效力。"
       → 但降卒中随时可能叛乱。你需要布置大量警戒。
       → 后期可能出现"降卒叛乱"事件
    
    D. 全部释放
       → 白起说:"让他们回去。"
       → 副将震惊:"将军! 他们回去后会——"
       → 白起: "会再次成为敌人。我知道。但——四十万人。太多了。"
       → 秦国朝堂震怒→白起被召回问罪→可能被贬/被赐死
       
  无论选择什么——这件事会永远改变白起。
  
终章CG:
  长平谷地。四十年后。
  一个老人带着孙子来到这里。
  孙子:"爷爷，这里发生过什么?"
  老人看着空荡荡的山谷。
  风吹过。远处似乎有四十万人的回声。
  老人:"...风太大了。听不见。"
```

## DLC专属机制

### 饥饿系统
```
围困期间(第六章):
- 每日粮食自动减少
- 士兵体力/士气与粮食挂钩
- 极端饥饿 → 可能出现"人相食"事件(概率触发，极为沉重)
- 粮草分配是核心玩法 → 影响士兵士气和你自己的体力
```

### 堑壕对垒
```
廉颇章节(第二章):
- 修建/维护三道防线
- 防御工事的"磨损度"→需要持续投入资源修复
- 秦军的进攻强度随时间递增
```

### 反间计机制
```
范雎章节(第三章):
- 选择谣言内容/目标/传播方式
- 监测谣言在赵国朝堂的影响力
- 时机选择: 太早→效果不够/太晚→可能来不及
```
