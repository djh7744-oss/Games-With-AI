# 历史事件与DLC剧本

## 设计目标

> 8个历史战役DLC不是沙盒本体的"附加任务"——它们是独立的线性叙事体验，每个DLC是一段8-10小时的浓缩战国史诗。玩家扮演特定历史角色，在已知的历史框架中探索"如果当时做了不同选择会怎样"。本体沙盒是"你的人生由你定义"，DLC是"这段历史由你改写"。

## 剧本设计框架

### 篇章结构(以DLC1长平之战为例)

```
DLC 1: 长平之战 ——《四十万》

时间范围: 前262年(上党争端) → 前260年(坑杀)
视角: 双主角 → 白起(秦) / 廉颇→赵括(赵)

第一章「上党之争」—— 序幕: 命运的导火索
  场景: 韩国的上党郡守冯亭不愿降秦，将上党献给赵国。
        赵国朝廷激烈争论——平原君说"这是天赐"，蔺相如说"这是祸根"。
        玩家(赵括视角)在朝会上第一次目睹了国家大事的争论。
        
  关键选择: 接受上党 vs 拒绝上党
    → 接受: 触发秦赵战争(历史路线)
    → 拒绝: 上党归秦→秦国进一步强大→赵国获得暂时的和平
           但战争的阴云仍然逼近 (进入架空路线)

第二章「廉颇守势」—— 坚守: 老将的壁垒
  场景: 秦军越过太行山，赵军在长平迎击。
        廉颇修筑三道防线。秦军每天挑战——廉颇不应。
        他在等的不是战机——是秦军粮尽的那一天。
        
  玩家体验: 廉颇视角→管理堑壕防御、调配粮草、安抚士兵
            日复一日，防线在消耗中磨损。士气在等待中下降。
            但廉颇知道：等下去是对的。

第三章「范雎反间」—— 阴谋: 看不见的敌人
  场景: 秦军久攻不下，白起还未被派到前线。
        范雎在咸阳策划了一场反间计——他派间谍去赵国散布谣言:
        "秦军不怕廉颇，廉颇只会死守。秦军最怕赵括。"
        "赵括是名将赵奢的儿子，精通兵法——他才是秦军的克星。"
        
  赵国朝堂开始动摇。赵王召见赵括。
  
  玩家体验: 双线并行——
    秦国线: 指挥范雎的间谍网络，策划谣言的具体内容
    赵国线(赵括视角): 被召入宫中→赵王问你"能打赢吗?"
      → 赵括的回答决定了他的命运
      
  关键选择(赵括): 
    A. 谦虚:"臣不如廉颇将军"→赵王不悦但可能留下廉颇
    B. 自信:"臣能胜秦，只需一个机会"→赵王大喜→替换廉颇
    C. 推辞(极高魅力要求):"请王三思"→赵王犹豫→给廉颇更多时间

第四章「纸上谈兵」—— 宿命: 年轻的将军
  场景: 赵括到达前线，替换廉颇。廉颇交接时沉默。
        赵括看到四十万大军——这是他从未指挥过的规模。
        他的第一个命令: 全线出击。
        
  玩家体验: 赵括视角→指挥40万大军渡河出击。
            如果按历史: 全军追击→被白起切断后路→被包围。
            
  本DLC的核心架空点: 如果你(赵括)做出了不同的战术选择呢?
    如果你没有全军渡河? 
    如果你在追击时保持了警惕?
    如果你识破了白起的佯败?
  
  (游戏的8种结局从这里开始分支)

第五章「白起奇袭」—— 杀神: 猎人与猎物
  场景: 白起秘密到达前线。他做的第一件事: 传令——
        "有敢泄露武安君在此者，斩。"
        赵括完全不知道自己的对手已经换了。
        
  白起部署:
    25000人切断赵军后路
    5000骑兵插入赵军营垒之间
    主力佯败→引诱赵括追击→包围圈合拢
  
  玩家体验: 白起视角→部署包围圈→
            每一步都像是在棋盘上落子→计算赵括的每一步反应。
            但白起的内心独白中透露出: 
            "这个年轻人...让我想起了当年的自己。"
            但手上不停。犹豫一秒，可能就错过战机。

第六章「饥饿围困」—— 绝望: 四十六天
  场景: 赵军被围，粮道断绝。第一天→第五天→第十五天→第三十天。
        粮食从"省着吃"→"一天一顿"→"杀马"→"吃草根"→"..."
        
  玩家体验: 赵括视角(如果你是赵括)
            每天面对: 
              - 士兵饥饿: 分配剩余的粮食
              - 士气崩溃: 安抚/镇压/说谎
              - 突围: 尝试→失败→再尝试→再失败
              - 等待援军: 他们真的在路上了吗?
              
            第46天。赵括最后一次集结还能站起来的士兵。
            他说:"援军不会来了。但我们可以自己冲出去。"
            五千精锐骑兵——最后的希望——冲向秦军阵线。
            
  如果玩家在白起视角: 
    看着包围圈中越来越微弱的赵军营火。
    部下问:"将军，他们还能撑多久?"
    白起看着:"...快了。"

第七章「最后突围」—— 结局: 战神的末路
  场景: 赵括亲率最后的精锐骑兵突围。
        箭如雨下。赵括中箭落马。
        他倒在地上，看着天空——长平的天空。
        
  赵括的最终命运(分支):
    A. 战死: (历史)赵括中箭阵亡，赵军投降
    B. 突围成功: 率残部杀出一条血路→回到赵国→面对: 
       "我输了四十万人...我该以死谢罪还是活下去赎罪?"
    C. 被俘: 被白起俘虏→白起看着这个年轻人→
       "你很有才华。但你太年轻了。"→选择: 杀/留/用
    D. 与白起谈判: (如果赵括在包围中表现出了真正的军事才能)→
       白起提出: "投降。我保证你的士兵不死。"
       赵括:"...(历史上你骗了四十万人)"
       白起沉默:"...这次不同。"

第八章「坑杀抉择」—— 尾声: 历史的重量
  场景: 赵军投降。四十万降卒跪在长平谷地。
        白起站在高处。秦军将领们等着他的命令。
        
  白起的内心独白(全游戏最沉重的一幕):
    "四十万人。放了他们，十年后他们会再次拿起武器。"
    "杀了他们...我被后世咒骂千年。"
    
  关键选择(白起视角):
    A. 全部坑杀: (历史)白起"人屠"之名永留史册
       → 白起日后被赐死时的临终独白改变
    B. 坑杀一半: 杀二十万，放二十万→
       既威慑赵国，又不至于"人屠"
    C. 全部收编: 将降卒编入秦军→
       秦军实力暴增→但降卒随时可能叛乱
    D. 全部释放: 白起说"让他们回去告诉赵王——不要再来送死"
       → 仁义之举→但秦国朝堂震怒→白起被召回问罪
       
  每个选择对应一个DLC结局CG。
  
  终章CG: 长平谷地。四十年后。一个老人(当年的赵卒)带着孙子来到这里。
          孙子问:"爷爷，这里发生过什么?"
          老人看着空荡荡的山谷:"...风太大了。听不见。"
```

## DLC与本体沙盒的关系

```
本体沙盒:
  ├── 玩家在开放世界中自由发展
  ├── DLC历史事件在沙盒时间线中"可能发生"
  ├── 玩家可以参与/干预/旁观
  └── 沙盒中的历史事件 = 简化版(非完整剧本)

DLC剧本:
  ├── 玩家选择特定角色 → 体验完整剧本
  ├── 独立的存档系统 (DLC存档 ≠ 本体存档)
  ├── 更高质量的过场动画/CG/配音
  ├── 聚焦在8-10小时的浓缩叙事体验
  └── 多结局 → 鼓励重玩

联动:
  ├── 完成DLC → 在本体中解锁专属内容
  │   例: DLC1通关 → 本体中白起可招募(原本不可招募)
  ├── DLC中选择的结局 → 影响本体的"默认历史"
  └── 本体沙盒中遇到DLC事件 → "似曾相识但结局可能不同"
```
