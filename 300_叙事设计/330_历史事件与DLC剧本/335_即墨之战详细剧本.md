# 即墨之战 ——《火牛》

## 剧本概述

| 项目 | 内容 |
|------|------|
| 时间 | 前284-279年 |
| 篇章 | 6章 |
| 视角 | 田单(齐) |
| 核心机制 | 火牛阵、反间计、心理战、绝地反击 |
| 结局 | 6种 |
| 时长 | 6-8小时 |
| CG | B级(90秒) |
| 情感 | 逆袭——"五年的绝望，一夜的奇迹。" |

## 剧本核心

齐国被五国联军攻破，七十余城沦陷。只剩即墨和莒城还在坚守。
田单——一个市集小吏，被推举为守将。

这不是一个"名将守城"的故事。这是一个"普通人创造奇迹"的故事。

### 六章结构

**第一章「齐国沦陷」**
乐毅率五国联军攻齐。齐王被杀。齐国主力被摧毁。
田单带着族人逃向即墨——路上全是难民。
"我们能去哪里?" "即墨。齐国的最后一座城门。"

**第二章「即墨守将」**
即墨守将战死。田单被推选为将。
田单: "我只是个管市场的小吏...我不懂打仗。"
众人: "你是唯一一个站出来的人。"
田单看着即墨城墙外的燕军营火。"...那好吧。"

**第三章「反间乐毅」**
田单的第一个计策: 不是军事的，是心理的。
他派间谍去燕国散布: "乐毅不攻即墨，是想在齐国自立为王!"
燕惠王中计。乐毅被换掉。骑劫接任。
乐毅离开齐国时回头看了一眼——他五年的努力，从这一刻开始瓦解。

**第四章「制造奇迹」**
田单需要奇迹。他知道即墨的兵力无法正面击败燕军。
于是他开始"制造"奇迹:
- 让城中百姓吃饭前先在院子里祭祀祖先→飞鸟聚集在城上空
  →燕军以为"天降神鸟保佑齐人"
- 散布谣言: "齐人最怕被割鼻子，割了鼻子就不敢打仗了"
  →骑劫把俘虏的鼻子割了→即墨守军看到→愤怒至极
- 散布谣言: "齐人在城外有祖坟"
  →骑劫派人挖开齐人的祖坟→即墨人发誓和燕军拼到底

这些都是田单的心理战——他要把"困兽"变成"猛兽"。

**第五章「火牛阵」**
田单收集城中一千多头牛。
牛角绑尖刀，牛尾绑苇草浇油，牛身披红色绸缎。
深夜。城门大开。点燃牛尾——

一千多头燃烧的火牛疯狂冲向燕军营地。
蹄声如雷，火光冲天。
燕军从梦中惊醒 → 以为天降神兵 → 营中大乱 → 自相践踏。
田单率即墨军民紧随其后。
一夜之间。燕军溃败。

**第六章「复国」**
即墨大胜。齐人看到了希望。
被燕国占领的七十余城，一城接一城起义响应田单。
齐国复国。
田单——市集小吏，用五年绝望的坚守和一个疯狂的想法(火牛阵)——重建了一个国家。

最后。田单回到临淄。这座城市曾被燕军洗劫，满目疮痍。
他站在废墟中。但身边站满了齐国人。
"我们回来了。"

### 六种结局

| 结局 | 火牛阵结果 | 复国范围 | 田单结局 |
|------|-----------|---------|---------|
| A(历史) | 大成功 | 七十余城全部收复 | 封安平君 |
| B | 部分成功 | 收复半数 | 继续战斗 |
| C | 反间失败(乐毅未走) | 无法复国 | 即墨最终失守 |
| D | 火牛阵失败 | — | 即墨城破 |
| E | 骑劫醒悟 | 战争延长 | 被燕国新的进攻击败 |
| F | 全奇迹 | 复国且反攻燕国 | 以少胜多的传奇 |

## 核心情感

这不是一个名将的故事。这是一个普通人创造了奇迹。
每一个绝望的选择——反间计(不够光明正大)/心理战(利用对方的残忍)/火牛阵(疯狂的想法)——都透露出: 当你被逼到绝境时，你会做出什么?
