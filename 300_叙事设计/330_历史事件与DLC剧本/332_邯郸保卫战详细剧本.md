# 邯郸保卫战 ——《孤城》

## 剧本概述

| 项目 | 内容 |
|------|------|
| 时间范围 | 前259年-前257年 |
| 篇章数 | 6章 |
| 视角 | 双主角：平原君(赵·内政外交)/廉颇(赵·军事防守) |
| 核心机制 | 围城消耗、外交求援、民众动员、窃符救赵 |
| 结局数 | 6种 |
| 预计时长 | 6-8小时 |
| CG级别 | A级(2分钟) |
| 核心情感 | 坚守——"城在人在，城亡人亡。" |

---

## 角色设定

**平原君(赵胜)** — 赵国公子，战国四公子之一。门客三千。邯郸保卫战期间的外交核心——他必须亲自去楚国和魏国求援。他的妻子是信陵君的姐姐——这个关系将决定赵国的存亡。

**廉颇(赵)** — 刚从长平被召回的老将。他知道自己是对的(不该出击)，但赵国已经输掉了长平。现在他要守的是邯郸——这座城不能再输了。

**信陵君(魏无忌)** — 魏国公子，平原君的小舅子。他的国家不想出兵救赵——但他的良心在燃烧。关键选择: 窃符救赵。

## 六章剧本

### 第一章「长平之后」

```
时长: 45-60分钟

长平惨败的消息传到邯郸。
全城家家戴孝——四十万赵军，几乎每家都有亲人。

平原君站在城墙上，看着城中升起的白幡。
他身后是赵王——赵王在哭。不是因为悲痛，是因为恐惧。

平原君: "秦军下一个目标，就是邯郸。"
         "陛下——我们必须求援。"

游戏性: 平原君开始在邯郸城内组织防御。
  - 统计城中可用资源(粮草/兵力/士气/城墙状况)
  - 决定"动员谁"——征召所有成年男子? 甚至妇女和老幼?
  - 第一次体验"围城前的最后平静"

夜晚。平原君回到府中。
他的妻子(信陵君的姐姐)对他说:"去找我弟弟。魏国会出兵。"
平原君:"如果魏国不出兵呢?"
他的妻子沉默。
"...那我就跪在他门前，直到他出兵。"
```

### 第二章「四面楚歌」

```
时长: 60-90分钟
视角: 廉颇

秦军围城。邯郸被困。

廉颇接管城防。他巡视城墙——这座城曾经是他守卫过的。
他知道: 邯郸的城墙不够厚，粮草不够多，士兵不够用。
但他说:"够了。"

游戏性: 围城防守战。
  - 每天秦军进攻。守城战——箭塔/滚木/金汁(煮沸的粪水)
  - 资源极度紧张 → 分配箭矢/药品/食物
  - 处理城内的恐慌和不满
  - 处理投降派: "有人想开城投降。怎么处置?"

关键场景: 廉颇在城墙上和一个老兵的对话。
  老兵: "将军，我们能守住吗?"
  廉颇: "...我不知道。但我会在这里——直到最后。"
  老兵: "那就够了。"
```

### 第三章「平原君求援」

```
时长: 90-120分钟
视角: 平原君

平原君带领二十个门客出发——去楚国求援。
出发前，他需要选出门客中最优秀的人。

毛遂自荐: "公子，带我去。"
平原君: "你在我门下三年，我从未听说过你有什么才能。"
毛遂: "那是因为你从未把我放进囊中。如果早放进——
        我早就脱颖而出了，不仅仅是锥尖!"

游戏性: 外交任务。
  - 出使楚国: 说服楚王出兵
  - 出使魏国: 说服魏王(或信陵君)
  - 谈判中的博弈: 平原君使用外交技巧
  - 关键: 如果楚国不出兵 → 赵国可能灭亡

楚国朝堂辩论(剧本高潮):
  平原君从日出说到日中，楚王仍然犹豫。
  毛遂按剑上前: "合纵的利害，两句话就说完了。为什么说到中午?"
  楚王怒: "我和你主人说话，你插什么嘴!"
  毛遂按剑再上前一步: "大王之所以敢呵斥我，是因为楚国人多。
                        但十步之内，大王的人多也没用——
                        大王的命在我手里。"
  (全场死寂)
  楚王: "...你说得对。楚国出兵。"
```

### 第四章「窃符救赵」

```
时长: 60-90分钟
视角: 平原君(远程) + 信陵君(魏国·切镜)

魏国答应出兵——但只是表面上的。
魏军驻扎在边境，观望。秦军威胁魏王: "谁敢救赵，下一个灭谁。"
魏王害怕了，命令军队不准前进。

信陵君知道姐姐在邯郸——邯郸被围，姐姐随时可能死。
他决定: 窃符救赵。

游戏性: 信陵君视角。
  - 说服魏王宠妃如姬偷取虎符(如姬的父亲被仇人所杀，信陵君帮她报了仇)
  - 带着虎符到军中 → 假传王命
  - 将军晋鄙怀疑 → 信陵君的门客朱亥锤杀晋鄙
  - 夺取兵权 → 率魏军向邯郸进发

关键场景: 朱亥锤杀晋鄙。
  晋鄙: "公子，这虎符是真的...但此事蹊跷，容我核实..."
  朱亥从袖中取出四十斤的铁锤。
  一击。晋鄙倒地。
  信陵君闭上眼睛。"...对不起。"
```

### 第五章「城下之战」

```
时长: 90-120分钟
视角: 廉颇 + 平原君

魏楚联军到达邯郸城外。城内的赵军看到援军——
城墙上的士兵欢呼——但廉颇没有欢呼。

廉颇: "援军来了，但仗还没打完。开城门——内外夹击!"

最终决战:
  - 邯郸城门大开 → 赵军杀出
  - 魏楚联军从外围进攻
  - 秦军腹背受敌 → 大败 → 撤退

平原君和信陵君在战场上相见。
信陵君: "姐姐还好吗?"
平原君: "她一直在等。每天都在城墙上望着西边。"
信陵君点了点头。两人并肩站在战场上。
```

### 第六章「邯郸命运」

```
时长: 30-45分钟

战争结束。秦军撤退。邯郸守住了。

但代价是什么?
  - 邯郸城内饿死了多少人?
  - 信陵君窃符救赵 → 他还能回魏国吗?(在历史上他留在了赵国十年)
  - 赵国活下来了 → 但能活多久?

终章: 平原君和廉颇站在邯郸城墙上。
      太阳升起来了。城中的白幡还在飘。
      平原君: "我们赢了。"
      廉颇: "赢了...但赵国还剩多少人?"
      平原君没有回答。
```

### 六种结局
