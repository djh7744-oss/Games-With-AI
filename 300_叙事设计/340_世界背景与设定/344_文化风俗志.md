# 文化风俗志

## 一、日常生活

### 衣——看衣识人

```
等级森严的服饰制度:

贵族:
  丝绸长袍 → 广袖深衣
  玉佩组佩 → 身份越高，佩玉越多
  冠 → 成年男子的标志。不同的冠代表不同等级

士人:
  长衫 → 材质多为麻布
  佩剑 → 士人佩剑是身份的象征(虽然不一定能用)

平民:
  短褐 → 方便劳动的短衣
  麻布 → 最便宜的衣料
  赤脚或草鞋

商人:
  虽然有钱，但法律限制穿衣(某些朝代规定商人不得穿丝绸)
  战国商业发达 → 这种限制逐渐松动

胡服骑射(赵):
  赵武灵王推行胡服 → 短衣/长裤/皮靴 → 方便骑马
  最初遭到贵族们的强烈反对("衣冠文物，国之大事")
  但军事效果显著 → 最终被接受 → 影响后世的服饰
```

### 食——五谷与肉食

```
主食: 粟(小米)、黍(黄米)、稻(南方)、麦、菽(豆类)
      "五谷"是战国人的基本食物

肉食:
  贵族: 牛羊猪鱼都有
  平民: 很少吃肉。七十岁以上才可以吃肉(孟子说的理想标准)
  士兵: 战争中可能有肉食配给(军粮)

饮品:
  酒 → 粮食酿造，贵族日常饮用
  茶 → 尚未普及(茶文化在唐代以后才兴盛)
  浆 → 米汤/酸味饮料

调味:
  盐 → 最重要的调味品(齐国靠海盐致富)
  梅 → 酸味来源(当时没有醋)
  酱 → 发酵豆制品的前身
  姜/花椒 → 有限的香料

围城中的食物(游戏机制参考):
  初期: 正常饮食
  中期: 粥水+马肉
  后期: 草根/树皮/皮带
  极端: ...(长平之战的饥饿系统参考)
```

### 住——从宫殿到茅屋

```
宫殿:
  高台建筑(夯土台基+木结构)
  咸阳宫/阿房宫 → 秦始皇的超级工程
  朝堂/寝殿/苑囿 → 前朝后寝的传统格局

贵族府邸:
  多进院落
  前堂(会客/办公) + 后室(居住)
  门客的住所(四公子门客三千)

平民住宅:
  夯土墙+茅草顶
  一室或两室
  院子小或没有

城市布局:
  宫城(政治中心)
  市(商业区) → "前朝后市"
  坊(居民区) → 里坊制度的雏形
  城墙+城门+护城河
```

### 行——车马与步行

```
贵族: 马车(双马或四马)
      身份越高，马的匹数越多
      天子六马

士人: 一匹马或步行
      毛驴/骡子(更便宜)

平民: 步行
      牛车(运输用，不是代步)

远程旅行:
  驿站 → 官方传递信息/官员出差
  客栈 → 商人/旅人住宿
  道路 → 秦国驰道(统一后) → "车同轨"的基础设施

交通效率:
  步行: 30-50里/天
  骑马: 100-200里/天
  驿站: 300-500里/天(换马不换人)
  大军: 20-30里/天
```

## 二、礼仪与节庆

```
冠礼(成年礼):
  男子二十岁行冠礼 → 戴冠 → 表示成年
  父亲/师长主持
  此后可以使用"字"(成年后的名字)

婚礼:
  六礼: 纳采→问名→纳吉→纳征→请期→亲迎
  贵族政治联姻 → 结婚=外交
  平民 → 简单的仪式

丧礼:
  三年之丧(儒家提倡，但只有贵族能做到)
  殉葬制度 → 战国逐渐废除(用陶俑代替活人)
  秦始皇兵马俑 → 殉葬制度最后的辉煌(以俑代人)

祭祀:
  祭天地 → 君主主持
  祭祖先 → 宗庙祭祀
  祭山河 → 地方祭祀
  社祭(春社/秋社) → 民间节日

节庆(游戏中的特别时段):
  岁首(正月): 新年 → 祭祀/宴饮/走亲
  上巳(三月): 踏青/河边沐浴
  端午(五月): 赛龙舟(楚地)/挂艾草
  七夕(七月): 乞巧
  中秋(八月): 赏月
  重阳(九月): 登高
  腊日(年终): 祭祀祖先/大扫除
```

## 三、社会风俗

### 尚武之风

```
战国是一个"尚武"的时代:
  士人佩剑(不一定会用，但必须有)
  各国以军功为荣
  即使是商人，遭遇盗贼也要能自保
  女性也会学习一些防身技巧(少见但不奇怪)

但各国尚武风格不同:
  秦: 全民皆兵 → 闻战则喜
  赵: 骑兵文化 → 马上英雄
  楚: 勇猛剽悍 → "楚人一诺，重若千金"
  齐: 技击文化 → 剑术流派众多
  韩: 精密尚武 → 劲弩天下第一
  魏: 重装步兵 → 魏武卒的铁血纪律
  燕: 苦寒尚武 → 耐寒坚韧
```

### 重义轻利 vs 重利轻义

```
这是战国时代最大的价值观冲突之一:

儒家(重义轻利):
  "君子喻于义，小人喻于利"
  仁义比财富更重要

法家(看似重义，实则重利):
  "人性好利" → 利用人的"利己心"来设计制度
  军功爵制的基础 = 人都是趋利的

商人(重利):
  吕不韦以商人身份成为秦相 → 打破了"商人地位低"的观念
  但儒家/法家都排斥纯粹的商业主义

墨家(节用, 反对奢侈):
  兼爱 + 非攻 + 节用
  反对战争，但擅长守城(技术派)
```

### 门客文化

```
四公子(孟尝/平原/信陵/春申)门客三千:
  门客 = 依附于权贵的士人/游侠/奇人异士
  鸡鸣狗盗、屠夫、刺客、学者...什么人都有
  权贵提供食宿 → 门客在需要时出力
  
  毛遂自荐(平原君门客)/冯谖弹铗(孟尝君门客)
  朱亥锤杀晋鄙(信陵君门客)/李园刺杀春申君
  
  门客文化是战国土人阶层的特殊现象:
    反映了"社会流动"的可能性
    但也反映了"人身依附"的现实
```
