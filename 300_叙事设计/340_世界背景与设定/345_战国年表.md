# 战国年表

## 前453-前350: 战国初期——格局形成

```
前453年  三家分晋 → 赵/魏/韩三国建立
前403年  周天子正式承认赵/魏/韩为诸侯
前386年  田氏代齐 → 田和取代姜姓成为齐侯
前381年  吴起在楚国被射杀 → 楚国变法中途夭折
前361年  秦孝公继位 → 发布"求贤令" → 商鞅入秦
前356年  商鞅第一次变法
前353年  孙膑围魏救赵 → 桂陵之战
前350年  商鞅第二次变法 → 秦迁都咸阳
```

## 前341-前260: 战国中期——合纵连横

```
前341年  马陵之战 → 孙膑杀庞涓 → 魏国衰落
前334年  苏秦合纵六国 → 佩六国相印
前318年  公孙衍发起五国伐秦
前307年  赵武灵王胡服骑射
前293年  伊阙之战 → 白起斩首24万
前284年  乐毅五国伐齐 → 下齐七十余城
前279年  田单火牛阵复齐
前278年  白起攻破郢都 → 屈原投江
前270年  范雎入秦 → 提出"远交近攻"
前262年  上党争端 → 秦赵长平之战的导火索
前260年  长平之战 → 白起坑杀40万赵卒
```

## 前259-前230: 战国后期——秦国独大

```
前259年  邯郸保卫战开始
前257年  信陵君窃符救赵 → 秦军解围
前256年  秦灭周 → 周朝终结(前1046-前256，791年)
前251年  秦昭襄王去世 → 在位56年
前247年  嬴政(13岁)继位为秦王
前238年  嬴政亲政 → 平定嫪毐叛乱
前233年  韩非入秦 → 被李斯害死在狱中
前230年  秦灭韩 → 六国第一个灭亡
```

## 前228-前221: 统一战争

```
前228年  秦灭赵 → 邯郸陷落
前225年  秦灭魏 → 水淹大梁
前223年  秦灭楚 → 王翦60万大军
前222年  秦灭燕 → 燕王被杀
前221年  秦灭齐 → 齐国不战而降
          秦始皇统一天下 → 战国时代结束
```

## 前221-前206: 秦朝(短暂统一)

```
前221年  称始皇帝 → 书同文/车同轨/统一度量衡
前220年  修建驰道 → 以咸阳为中心辐射全国
前215年  蒙恬北逐匈奴 → 修筑万里长城
前213年  焚书 → 焚烧六国史书和百家著作
前212年  坑儒 → 坑杀方士和儒生460余人
前210年  秦始皇驾崩 → 赵高/李斯矫诏立胡亥
前209年  陈胜吴广起义 → "王侯将相宁有种乎"
前207年  秦二世被杀 → 秦朝名存实亡
前206年  刘邦入咸阳 → 秦朝灭亡
```

## 游戏中的时间标记

```
重要: 游戏时间是"流动的"——
      玩家角色的一生约60-80年
      继承系统可以让玩家跨越整个战国时代

玩家可能经历的历史时刻(取决于出生时间):
  早生(前360年左右出生):
    见证商鞅变法 → 马陵之战 → 合纵连横 → 长平之战
    
  中生(前300年左右出生):
    见证长平之战 → 秦灭六国 → 统一天下
    
  晚生(前260年左右出生):
    出生在长平之战后 → 成长在秦统一过程中
    
  跨时代(通过继承):
    第一代: 参与长平之战
    第二代: 参与秦灭六国
    第三代: 见证秦的速亡
```
