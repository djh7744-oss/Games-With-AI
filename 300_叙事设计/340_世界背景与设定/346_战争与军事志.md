# 战争与军事志

## 一、战争形态的演变

```
春秋战争:
  规模: 几百到几千人
  时间: 几天到几周
  性质: "争霸"——要对方承认自己的霸主地位，不灭国
  礼仪: 有一定的"战争礼仪"(不重伤/不擒二毛)
  战车: 战车是主要作战单位

战国战争:
  规模: 几万到几十万人
  时间: 几个月到几年(长平对峙三年)
  性质: "灭国"——消灭对方的国家和宗庙
  礼仪: 几乎没有礼仪("兵不厌诈")
  多兵种: 步兵/骑兵/弓弩兵/水军协同
  职业化: 常备军取代临时征兵
```

## 二、各国军事特色

### 秦军 —— 战争机器

```
核心: 军功爵制驱动的嗜血军队
特点: 
  标准化装备(兵器统一规格)
  严酷纪律(连坐法: 一人逃跑，全队受罚)
  高效后勤(粮草补给体系)
  
经典战术: 歼灭战(白起)
  不追求击溃，追求"让对方再也无法组织军队"
  斩首数字异常巨大 → 消灭有生力量

为什么秦军这么强?
  制度激励: 斩首=爵位=土地 → 真金白银的激励
  标准化装备: 兵马俑的兵器展现了高度标准化
  残酷军法: 畏惧军法甚于畏惧敌人
  改革红利: 商鞅变法的成果
```

### 赵军 —— 骑兵为王

```
核心: 胡服骑射后的骑兵优势
特点:
  战国最强骑兵
  马上机动性
  北方草原马种

缺陷:
  骑兵对后勤要求极高(马要吃粮食/草料)
  赵国国力不足以支撑长期大规模骑兵作战
  长平之战 → 骑兵被围在谷地 → 失去机动性 → 全军覆没
```

### 楚军 —— 庞大但松散

```
核心: 水军优势 + 数量优势
特点:
  兵力最多(领土最大，人口基数大)
  水军最强(长江流域天然优势)
  战车传统(楚国贵族善用战车)

弱点:
  内部不统一(各贵族各自为政)
  地广人稀，反应慢
  吴起变法失败 → 军队改革不彻底
```

### 齐军 —— 技击之士

```
核心: 个人武艺 + 不擅团队作战
特点:
  剑术精良(齐国技击之术)
  装备精良(商业采购)
  但团队协作不够

荀子评价: "齐之技击，不可以遇魏之武卒;
          魏之武卒，不可以遇秦之锐士。"
```

### 魏军 —— 武卒的兴衰

```
魏武卒(前5世纪-前4世纪使用):
  中国最早的职业化精锐步兵
  严格选拔 → 专业化训练 → 高薪酬 → 可世袭
  魏国在战国初期依靠魏武卒成为霸主

衰落:
  精锐成本太高 → 魏国经济无法长期支撑
  孙膑两次击败魏军 → 武卒损失惨重
  后继乏人 → 魏国转为防守
```

### 韩军 —— 劲弩防线

```
核心: 弩兵防御体系
特点:
  天下劲弩出韩国
  守城弩阵极其致命
  宜阳冶铁支撑的武器质量

缺陷:
  只能防守(进攻能力极弱)
  宜阳铁矿被秦攻占后 → 武器优势丧失
  国力太小 → 无法抗衡秦国
```

### 燕军 —— 北疆坚韧

```
核心: 耐寒 + 辽东骑兵
特点:
  适应严寒作战
  辽东马场提供战马
  边军经验丰富(对匈奴)

弱点:
  国力最弱
  远离中原 → 无法参与中原争霸
  乐毅伐齐是唯一的高光时刻
```

## 三、战争的经济账

```
战国战争成本:

征兵:
  一个壮丁离开农田去打仗 → 农业减产
  十万大军 → 需要二十万人运粮(民夫)
  
粮草:
  一个士兵每天消耗约2-3斤粮食
  十万大军 → 每天消耗20-30万斤粮食
  长平之战对赵国的粮草压力 → 巨大的经济负担

兵器:
  一把铁剑 → 需要数斤铁矿石 → 冶铁 → 锻造
  十万大军的装备 → 天文数字的铁矿石和工时
  韩国的宜阳/楚国的宛城 → 战国最大的兵工厂

战争掠夺 vs 战争消耗:
  胜仗 → 掠夺战利品(可以覆盖部分成本)
  败仗 → 血本无归
  围城/持久战 → 消耗战 → 看谁先撑不住
```

## 四、围城与守城

```
围城困局(以邯郸保卫战为例):
  围城方(秦):
    优势: 可以切断补给
    劣势: 自己的粮草也在消耗
          长期围城 → 国内不满 → 政治压力
    
  守城方(赵):
    优势: 有城墙保护
          有存粮(如果准备充分)
    劣势: 被切断外部联系
          粮食一天比一天少
          士气一天比一天低
          
围城的三个阶段:
  第一阶段(1-3个月): 双方都充满信心
  第二阶段(3-12个月): 开始出现粮荒
  第三阶段(12个月+): 城内的生存已经成了问题
  
破了城之后:
  安抚 → 仁义+，城市完好
  纵兵劫掠 → 士兵士气+，但城市受损
  屠城 → 威慑，但仁义永损
```
