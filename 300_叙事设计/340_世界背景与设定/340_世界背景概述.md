# 世界背景与设定

## 设计目标

> 战国不是"欧洲中世纪的东方翻版"。它有自己独特的政治逻辑、社会结构、文化氛围和地理格局。本文档建立游戏世界的"地基"——让设计师、美术、程序在同一个战国世界里工作。每一座城池的位置、每一个国家的性格、每一种社会制度的运作，都有历史和逻辑的支撑。

---

## 一、时代背景

```
公元前475年 —— 公元前221年
254年的战国时代。

开端: 晋国分裂为韩赵魏三家(三家分晋，前453年)
      田氏代齐(前386年)
      七雄并立的格局形成

核心矛盾:
  周天子名存实亡 → 谁来统一天下?
  秦国崛起 → 六国联合对抗 → 合纵连横
  旧制度崩溃 → 变法与改革 → 郡县 vs 分封
  百家争鸣 → 思想的大爆发

终结: 公元前221年 秦灭六国 统一天下
      
游戏时间跨度: 约前350年-前221年 (约130年)
玩家角色自然寿命: 60-80年 (覆盖其中一段)
继承系统: 可延续到统一完成
```

## 二、地理设定

### 九大地理区域

```
1. 关中 (秦国核心)
   位置: 今陕西中部
   特征: 四塞之地(东函谷关/南武关/西散关/北萧关)
         易守难攻的天然堡垒
         "据崤函之固，拥雍州之地"
   主要城池: 咸阳(都城)/雍城/栎阳
   战略价值: 秦国统一天下的根据地

2. 中原 (魏国/韩国/周室)
   位置: 今河南中西部
   特征: 天下之中，四方交汇
         四通八达 → 也是四面受敌
         文化中心 → 学宫/百家聚集
   主要城池: 大梁(魏)/新郑(韩)/洛阳(周)
   战略价值: 控制中原 = 控制天下交通枢纽

3. 河北 (赵国/燕国南部)
   位置: 今河北中南部
   特征: 广阔平原 + 太行山屏障
         骑兵的理想战场
   主要城池: 邯郸(赵)/中山/邢城
   战略价值: 赵国骑兵的根据地

4. 山东 (齐国)
   位置: 今山东大部
   特征: 东临大海，渔盐之利
         商业最发达的区域
         远离秦国 → 最后被灭
   主要城池: 临淄(齐)/即墨/莒城
   战略价值: 经济中心，富甲天下

5. 荆楚 (楚国核心)
   位置: 今湖北/湖南
   特征: 长江横贯 + 云梦大泽
         水网密布 → 水军天然优势
         地广人稀 → 扩张空间大
   主要城池: 郢都/宛城/寿春(后期迁都)
   战略价值: 最大的领土，最多的人口潜力

6. 巴蜀 (秦国南部)
   位置: 今四川/重庆
   特征: 盆地地形，天府之国
         秦灭巴蜀后 → 成为秦国的粮仓
   主要城池: 成都/江州(重庆)
   战略价值: 粮食基地 + 攻楚跳板

7. 山西 (赵国西部/韩国北部)
   位置: 今山西
   特征: 黄土高原，铁矿丰富
         韩国宜阳 = 天下冶铁中心
   主要城池: 宜阳(韩)/晋阳(赵)
   战略价值: 战略资源(铁矿石)

8. 燕北 (燕国核心)
   位置: 今河北北部/北京/辽宁
   特征: 寒冷偏远，与游牧接壤
         燕国最弱但最安全
   主要城池: 蓟城(燕)/辽东
   战略价值: 北疆屏障 + 辽东马场

9. 吴越 (楚国东部)
   位置: 今江苏/浙江/上海
   特征: 水乡泽国，后来居上
         楚国吞并越国后纳入版图
   主要城池: 会稽/姑苏/金陵
   战略价值: 沿海贸易 + 南方开发潜力
```

## 三、社会结构

```
战国社会阶层 (自上而下):

1. 君主/王室 (0.1%)
   一国之主 → 绝对权力但绝对责任

2. 贵族/封君 (1%)
   世袭特权阶级 → 变法的头号敌人
   拥有封地/私兵/司法权
   代表: 四公子(孟尝/平原/信陵/春申)

3. 官僚/士人 (3%)
   新兴阶层 → 取代贵族管理国家
   通过学问/军功/推荐入仕
   郡县制的执行者、变法的受益者

4. 士兵/军官 (5%)
   职业军人的出现(魏武卒/秦锐士)
   军功爵制 → "斩首一级，赐爵一级"
   从"临时征兵"到"常备军"

5. 商人/工匠 (8%)
   战国商业的空前繁荣
   但社会地位仍然不高
   代表: 吕不韦(超越阶层限制的例外)

6. 农民 (70%)
   人口主体
   缴纳赋税、服兵役和徭役
   变法的终极受益者 or 受害者(取决于变法内容)

7. 奴隶/刑徒 (13%)
   战俘/罪犯/债务奴隶
   大规模公共工程的劳动力
   
社会流动:
  战国是社会流动性最大的时代之一
  农民 → 士兵 → 军功 → 爵位 (秦国军功爵制)
  商人 → 捐官 → 入仕 (极少但可能)
  士人 → 献策 → 被重用 → 卿相 (苏秦/张仪/商鞅)
  
  这种"流动的可能性"是自由人生支柱的历史基础
```

## 四、七国性格速写

```
秦国: "虎狼之国"
  关键词: 法治/效率/冷酷/尚武
  国民性格: 务实、坚韧、服从纪律
  一句话: "秦国的法律就是秦国的灵魂。"
  
楚国: "南方巨兽"
  关键词: 辽阔/神秘/骄傲/保守
  国民性格: 尚武而浪漫、排外而自尊
  一句话: "楚人不是被征服的——楚人是被自己打败的。"
  
齐国: "东方富商"
  关键词: 富裕/文化/中立/享乐
  国民性格: 重利、重学、轻战
  一句话: "齐国人想的不是征服天下，而是怎么赚天下人的钱。"
  
赵国: "北方硬汉"
  关键词: 骑兵/坚韧/多变/悲壮
  国民性格: 勇猛、善变、不服输
  一句话: "赵国人每一次倒下，都会重新站起来。直到最后。"
  
燕国: "最北的孤岛"
  关键词: 偏远/自卑/坚韧/隐忍
  国民性格: 保守、隐忍、重视尊严
  一句话: "燕国太小了。但燕国人从不承认自己小。"

魏国: "中原之脑"
  关键词: 人才/务实/四面楚歌
  国民性格: 精明、善变、重视人才
  一句话: "魏国最大的悲哀——人才太多了，但留不住。"

韩国: "夹缝中的工匠"
  关键词: 精密/恐惧/智慧/悲剧
  国民性格: 机巧、敏感、求生欲强
  一句话: "天下劲弩出韩国，天下悲剧也出韩国。"
```

## 六、技术与社会发展阶段

```
战国时代的技术水平:

军事: 铁兵器取代铜兵器 (正在进行中)
      弩机成为步兵标配
      骑兵开始取代战车 (胡服骑射)
      攻城器械日趋成熟
      
农业: 铁犁牛耕推广
      大型水利工程 (都江堰/郑国渠)
      轮作休耕制度
      
手工业: 冶铁技术飞跃
        纺织业繁荣
        原始瓷器出现
        
商业: 金属货币普及
      跨国贸易网络形成
      高利贷和钱庄萌芽
      
文化: 百家争鸣 (思想大爆发)
      竹简作为主要书写材料
      学宫/私学兴起
      
社会: 世袭贵族衰落
      官僚制度兴起
      军功爵制的革命性
```
