# 七国志

## 秦国 —— 虎狼之国

```
国都: 咸阳
君主世系(游戏时间范围内):
  秦孝公(前361-338) → 商鞅变法
  秦惠文王(前337-311) → 杀商鞅，保留变法
  秦武王(前310-307) → 举鼎而死
  秦昭襄王(前306-251) → 白起/范雎时代，秦国大扩张
  秦孝文王(前250) → 在位仅三天
  秦庄襄王(前249-247) → 吕不韦"奇货可居"的对象
  秦始皇(前246-210) → 统一六国

变法历程:
  前356年 商鞅第一次变法
  前350年 商鞅第二次变法(迁都咸阳)
  变法核心: 废井田/开阡陌/军功爵/连坐法/重农抑商/郡县制
  结果: 秦国从"西陲弱国"变成"虎狼之国"

为什么是秦国统一了天下?
  1. 地理: 四塞之地，后顾无忧
  2. 制度: 商鞅变法 → 最高效的战争机器
  3. 人才: 吸引六国人才(商鞅/张仪/范雎/李斯都是"外国人")
  4. 战略: 远交近攻 → 逐一击破
  5. 运气: 六国互斗，给了秦国逐个击破的机会
```

## 楚国 —— 南方巨兽

```
国都: 郢都 → 陈 → 寿春(屡次迁都)
君主: 楚怀王(前328-299) → 被张仪欺骗/被秦扣留/死在秦国

楚国的问题:
  太大了 → 难以有效管理
  贵族太强 → 吴起变法失败，旧势力反扑
  内部蛮族 → 需要大量兵力维护内部稳定
  政治腐败 → 屈原被流放是最典型的例子

楚国灭亡的原因:
  不是被打败的——是被自己拖垮的。
  白起只是"加速了这个过程"。
  郢都被攻破 → 楚国迁都 → 但底蕴还在 →
  王翦带60万人 → 最终灭楚。
```

## 齐国 —— 东方富商

```
国都: 临淄
特色: 最富裕、最文化、最不爱打仗

为什么齐国"不出兵"?
  齐国太富了。齐国人不想打仗——打仗会影响做生意。
  每次合纵，齐国都是"出钱不出兵"。
  齐国以为可以靠金钱和外交永保太平。
  结果: 公元前221年，秦军到来时，齐国不战而降。
  
  齐王建被流放，饿死在流放地。
  "齐国富甲天下三百年，最后一个国君连一顿饱饭都吃不上。"
```

## 赵国 —— 北方硬汉

```
国都: 邯郸
转折: 前307年 赵武灵王胡服骑射 → 赵国骑兵成为战国最强

赵国的悲剧:
  长平之战前 → 赵国是唯一能正面抗衡秦国的国家
  长平之战 → 四十万主力被歼灭 → 元气大伤
  邯郸保卫战 → 守住了 → 但只是"延命"
  李牧被反间计害死 → 赵国最后的希望破灭
  前228年 → 邯郸陷落 → 赵国灭亡

赵国的故事是所有国家中最悲壮的:
  每一次倒下都爬起来。直到最后再也爬不动了。
```

## 燕国 —— 最北的孤岛

```
国都: 蓟城(今北京)
特色: 最偏远/最弱小/但最安全(秦国打过来要先灭赵/魏/齐)

燕国的辉煌时刻:
  燕昭王筑黄金台招贤 → 乐毅率五国联军伐齐 → 下齐七十余城
  这是燕国历史上唯一的高光时刻。
  然后燕昭王死 → 乐毅被换 → 田单复齐 → 燕国回到原点。

荆轲刺秦:
  太子丹的绝望之举。
  失败。秦国以此为借口加速灭燕。
```

## 魏国 —— 中原之脑

```
国都: 安邑 → 大梁
鼎盛: 魏文侯时期(前445-396) → 战国第一个霸主
      吴起/李悝/西门豹/乐羊...人才济济

衰落原因:
  1. 四面受敌(谁都能打它)
  2. 人才流失(商鞅/吴起/孙膑/范雎...都曾在魏国，都被排挤走了)
  3. 战略失误(放走了商鞅 → 成就了秦国)

"魏国最大的人才输出国"——输出的人才成了敌人。
```

## 韩国 —— 夹缝中的工匠

```
国都: 新郑
特色: 最小/最弱/但冶铁和弩机技术天下第一

韩国的悲剧:
  位置最差——夹在秦/魏/楚之间，谁都能打它。
  申不害变法 → 暂时中兴 → 申不害死后 → 迅速衰落。
  宜阳铁矿被秦攻占 → 失去最重要的经济来源。
  前230年，秦国攻韩。韩王安投降。韩国第一个灭亡。

  "天下劲弩出韩国，天下悲剧也出韩国。"
```

## 周室 —— 名义上的天子

```
国都: 洛阳
地位: 名义上的天下共主，实际上连一个小国都不如。

公元前256年，秦昭襄王派兵攻占洛阳。
周赧王献上土地和人口簿，周朝灭亡。
八百年的周朝，以这样安静的方式终结了。
```
