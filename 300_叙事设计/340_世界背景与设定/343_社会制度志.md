# 社会制度志

> "秦之变法，非变秦也，变天下也。"
> 战国时代是社会制度剧烈变革的时代。旧制度(世袭、分封、井田)在崩溃，新制度(郡县、军功、私有土地)在诞生。理解这些制度变革，才能理解战国。

---

## 一、政治制度

### 从分封到郡县

```
分封制(周制):
  天子 → 诸侯(公侯伯子男) → 大夫 → 士
  每一层都有自己的封地、人民、军队
  层层分封 → "我的附庸的附庸不是我的附庸"
  
  问题: 诸侯坐大 → 天子无权 → 战争不断

郡县制(秦制):
  中央直接任命郡守/县令
  官员没有封地，只有俸禄
  官员可以被调离、罢免、升迁
  "普天之下，莫非王土"真正实现
  
  商鞅变法在秦国推行郡县制 → 秦国成为第一个"中央集权"的国家
  统一后，秦始皇在全国推行郡县制

分封 vs 郡县的博弈(贯穿整个战国):
  旧贵族支持分封(他们靠分封吃饭)
  变法者支持郡县(削弱贵族，加强君权)
  商鞅被车裂 → 表面上是"变法者没好下场"
              实质上是旧贵族对郡县制的最后反击
```

### 世卿世禄 vs 选贤任能

```
世卿世禄(周制):
  你父亲是将军 → 你也是将军(世袭)
  你父亲是大夫 → 你也是大夫(世袭)
  能力不重要，血统才重要

军功爵制(秦制):
  "斩首一级，赐爵一级"
  不管你父亲是谁，你自己去砍敌人的头
  砍得越多，爵位越高
  一个农民可以通过军功获得爵位和土地
  这在世卿世禄时代是不可想象的

选贤任能:
  墨家最早提出"尚贤"
  法家实践: 商鞅/李斯都不是秦国人，但被秦国重用
  这是战国相比春秋最大的变化之一
```

## 二、经济制度

### 井田制 → 私有土地制

```
井田制(周制):
  土地被划分为"井"字形 → 中间一块是"公田"，周围八块是"私田"
  农民先种公田，再种私田
  农民被固定在土地上 → 相当于领主的农奴

私有土地制:
  商鞅变法"废井田，开阡陌"
  土地可以买卖
  农民可以离开土地
  "耕者有其田" → 生产积极性大增
  
  影响: 秦国的农业产出暴增 → 能养活更多军队
```

### 货币制度

```
华夏各国使用青铜货币，主要有四种形式：
  布币(铲形) —— 韩/赵/魏
  刀币(刀形) —— 齐/燕
  圜钱(圆形) —— 秦/周
  蚁鼻钱(贝形) —— 楚
  郢爰(金版) —— 楚(大宗交易)

秦国统一后: 废六国货币，统一使用秦半两(圆形方孔钱)
"圆形方孔"成为此后两千年中国古代货币的标准形制
```

### 盐铁

```
盐:
  齐国垄断海盐
  魏国河东有池盐
  盐是古代最重要的调味品和防腐剂
  控制盐=控制民生=控制经济

铁:
  韩国的宜阳/楚国的宛城是冶铁中心
  铁兵器取代铜兵器(正在进行中)
  冶铁技术=军事技术
  "谁控制了冶铁，谁就控制了战争"
```

## 三、军事制度

### 征兵制 → 常备军

```
春秋征兵:
  战时临时征召，农闲时训练
  打完仗回家种田
  军队素质参差不齐

战国常备军:
  魏武卒(中国最早的职业化军队)
    → 严格选拔 → 专业化训练 → 高薪酬 → 可世袭
  秦锐士
    → 军功爵制 → 斩首=爵位=土地 → 秦国士兵"闻战则喜"
  
  职业化军队的出现是战国军事革命的核心
```

### 军功爵制(秦国)

```
二十等爵位(从低到高):
  1-4级: 士(公士/上造/簪袅/不更)
  5-8级: 大夫(大夫/官大夫/公大夫/公乘)
  9-12级: 卿(五大夫/左庶长/右庶长/左更)
  13-16级: 中高级(中更/右更/少上造/大上造)
  17-20级: 最高级(驷车庶长/大庶长/关内侯/彻侯)

  斩杀一个带甲敌军 → 升一级
  斩杀一个敌军官 → 升两级+
  集体斩杀 → 全队有赏

  这就是为什么秦军在战场上"闻战则喜"——
  因为战争=升迁=爵位=土地=财富=社会地位
  秦军是战国唯一"喜欢打仗"的军队
```

## 四、法律制度

```
郑国子产"铸刑书"(前536年) → 中国最早公布的成文法
魏国李悝《法经》 → 中国第一部系统的法典
秦国商鞅"徙木立信" → 树立法律的权威
韩非"法治"理论 → 法家思想的集大成

法家的核心:
  1. 法律面前人人平等(除了君主)
  2. 严刑峻法 → "以刑去刑"
  3. 法律必须公开、明确、稳定
  4. 赏罚分明 → "有功必赏，有过必罚"

法家的矛盾:
  理论上是"法治"，实际上是"君主意志的体现"
  商鞅自己也被自己制定的法律所困(逃亡时无凭证不能住宿)
  韩非的理论被秦始皇采用，但韩非本人被李斯害死
```
