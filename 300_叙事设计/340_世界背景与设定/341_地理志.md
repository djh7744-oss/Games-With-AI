# 战国地理志

## 九大区域详考

### 一、关中——秦国的"四塞之地"

```
"秦地被山带河，四塞以为固。"
                      ——《史记》

地理特征:
  东有函谷关 → 一夫当关万夫莫开
  南有武关 → 通往楚国的唯一大道
  西有散关 → 通往西域的咽喉
  北有萧关 → 防御匈奴的屏障
  
  渭河横贯关中 → 灌溉了八百里秦川
  这是中国最早的"天府之国"(比成都平原更早获得此名)

战略意义:
  你打不进来 → 函谷关太险
  你能打出去 → 从函谷关出兵，直插中原
  这是攻守兼备的完美地理格局
  秦国可以"躲在关中发展，时机到了就东出"

城池:
  咸阳(都城): 渭河北岸。公元前350年秦孝公迁都于此
             不是最大的城，但是最重要的城
  雍城: 秦国旧都。宗庙所在
  栎阳: 商鞅变法的起点
  蓝田: 玉矿所在地。蓝田玉是秦国的重要资源

关隘:
  函谷关: 秦国的"东大门"。战国时期从未被攻破过。
          苏秦合纵六国，最东也只到函谷关前。
          函谷关=秦国安全的终极保障。
  武关: 通往楚国的要道。白起伐楚经此。
  大散关: 通往巴蜀的咽喉。
```

### 二、中原——天下之中，四战之地

```
"中原者，天下之大本也。"
                      ——《荀子》

地理特征:
  黄河中下游平原 → 广阔平坦，农业发达
  但——无险可守。四面都是敌人。
  魏国在战国初期最强，后来衰落了——
  很大程度上因为"谁都可以打它"。

城池:
  大梁(魏): 中原最大的城池之一。后因水淹而毁
  新郑(韩): 韩国都城。被秦军攻破，韩国第一个灭亡
  洛阳(周): 周天子所在地。象征意义大于实际价值
            公元前256年，秦灭周。周的时代正式终结

纵横交错:
  中原是交通枢纽 → 控制了中原就控制了交通
  但也意味着 → 四面受敌，永无宁日
  魏国从最强国衰落为二流国家 → 地理是重要因素
```

### 三、河北——骑兵的摇篮

```
"燕赵多慷慨悲歌之士。"
                      ——韩愈

地理特征:
  华北平原北部 + 太行山
  广阔平原 → 骑兵驰骋的理想战场
  赵国骑兵 → 战国最强骑兵
  胡服骑射在这里诞生

城池:
  邯郸(赵): 战国最大的都城之一。人口超过三十万
            邯郸保卫战——这座城被秦军围了两年。
            最终陷落。但它的名字成了"坚守"的代名词
  代郡: 赵国北疆。马场所在。战国最好的战马产地之一
  中山: 一个小国，被赵国吞并。白狄建立的国家

北疆:
  赵国和燕国与匈奴接壤
  李牧在赵国北疆筑长城，逐匈奴
  这是中国历史上最早的"长城防线"
```

### 四、山东——东方富庶之地

```
"齐地方二千里，带甲数十万，粟如丘山。"
                      ——苏秦游说齐王

地理特征:
  东临大海 → 渔盐之利(古代最重要的两种资源)
  泰山在境内 → 齐鲁文化的发源地
  远离秦国 → "最后一个被灭的"

城池:
  临淄(齐): 战国最繁华的城市。人口超过三十五万
            "临淄之途，车毂击，人肩摩"
            街上人多到车轴碰车轴，人挤人
  即墨: 田单复齐的地方。一座被围了五年的城
  莒城: 齐国最后的据点之一

经济:
  海盐 → 齐国垄断了食盐供应
  丝绸 → "齐纨鲁缟"天下闻名
  渔业 → 沿海城镇富庶
  商业 → 齐国是战国最商业化的国家
```

### 五、荆楚——南方巨兽

```
"楚虽三户，亡秦必楚。"
                      ——楚南公

地理特征:
  长江横贯 → 水网密布
  云梦大泽 → 楚国腹地的巨大湖泊和沼泽
  领土面积最大 → 从河南南部到湖南/江西
  
  但——地广人稀，"大而不强"
  内部蛮族众多 → 需要大量兵力维持秩序
  白起攻破郢都后 → 楚国被迫迁都，但领土仍然广大

城池:
  郢都(早期都城): 白起攻破后焚毁
  宛城: 冶铁中心。楚国的兵工厂
  寿春(后期都城): 迁都到这里，但已无力回天
  会稽: 越国故地。楚国吞并越国后纳入版图

特色:
  楚国是"神秘"的代名词
  屈原的诗歌→楚地神话体系→瑰丽奇绝
  楚人"信巫鬼，重淫祀"
  中原各国眼中的楚国→"半开化的南方大国"
```

### 六-九：巴蜀/山西/燕北/吴越 略

```
巴蜀(秦): 
  四川盆地。"天府之国"。
  被秦国吞并后成为秦国的粮仓。
  都江堰建成后，"水旱从人，不知饥馑"。

山西(赵韩):
  黄土高原。
  韩国的宜阳铁矿 → 天下冶铁中心 → "劲弩出韩国"的物质基础

燕北(燕):
  最偏远的国家。
  与匈奴/东胡接壤。秦开曾在此拓地千里。
  辽东马场 → 燕国的战略资源

吴越(楚):
  水乡泽国。
  越国被楚国吞并。吴国已被越国灭亡。
  这里曾经是吴王夫差和越王勾践的故事发生地。
  现在，只剩下楚国的统治和江南的烟雨。
```

## 战略要地

```
关隘:
  函谷关(秦): 从未被攻破
  武关(秦楚之间): 秦楚战争的关键
  井陉(赵): 太行山中的重要通道
  居庸塞(燕): 北方屏障

渡口:
  孟津(黄河): 周武王伐纣在此渡河
  蒲坂(黄河): 秦晋之间的重要渡口
  采石(长江): 楚国南方的要津

山脉:
  太行山 → 分割山西与河北
  秦岭 → 关中的南侧屏障
  崤山 → 函谷关所在

河流:
  黄河 → "母亲河"，也是军事分界线
  长江 → 楚国的大动脉
  淮河 → 南北分界线
  汉水 → 秦楚走廊
```
