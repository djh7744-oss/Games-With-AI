# 数据表模板说明

## 命名规范

文件命名: `[类别]_[内容]_[版本].csv`

类别码:
  - UNIT = 兵种数据
  - NPC = NPC/将领数据
  - ITEM = 物品/装备数据
  - CITY = 城池数据
  - TECH = 科技数据
  - QUEST = 任务数据
  - DIP = 外交数据
  - ECON = 经济数据

示例:
  - `UNIT_七国兵种_v01.csv`
  - `NPC_秦国将领_v01.csv`

## 格式规范

- 编码: UTF-8 (无BOM)
- 分隔符: 逗号
- 第一行: 列标题(中文描述)
- 第二行起: 数据
- 注释行: 以 # 开头

## 数据验证

每张表提交前必须检查:
- [ ] 所有ID唯一
- [ ] 引用完整性(外键存在)
- [ ] 数值在合法范围内
- [ ] 无中文乱码
- [ ] 无重复行
