# 模板目录说明

本目录包含项目所有文档和数据表的模板文件。创建新文档时请使用对应模板。

## 目录结构

```
Templates/
├── README.md              # 本文件
├── 系统设计模板.md         # 系统设计文档模板(已有)
├── 数值表模板.csv          # CSV数值表模板(已有)
├── 设计文档/              # 设计类模板
│   ├── 叙事设计模板.md     # 叙事/剧情/人物设计
│   ├── 关卡设计模板.md     # 地图/关卡设计
│   └── 美术规范模板.md     # 美术/视觉规范
├── 数据表/                # 数据表模板
│   └── 数据表模板说明.md   # CSV命名/格式/验证规范
├── 传记/                  # 人物传记模板
│   └── 人物传记模板.md     # 历史人物传记
```

## 使用方式

1. 复制对应模板到目标目录
2. 按模板格式填写内容
3. 将 `[占位符]` 替换为实际值
4. 更新 frontmatter (id/title/author/date)
5. 提交前检查所有必填项

## Frontmatter 标准字段

```yaml
---
id: [三位数编号]
title: [文档标题]
version: [版本号]
status: draft | review | approved | implemented | obsolete
author: [作者]
last_modified: [YYYY-MM-DD]
reviewers: [评审人列表]
related: [关联文档编号]
game_version: [对应游戏版本]
---
```
