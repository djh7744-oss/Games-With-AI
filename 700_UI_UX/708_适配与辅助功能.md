# 适配与辅助功能

## 一、分辨率适配

```
最低: 1280×720 (720p)
推荐: 1920×1080 (1080p)
理想: 2560×1440 (1440p)
超宽: 3440×1440 (21:9) / 5120×1440 (32:9)

适配策略:
  16:9: 标准布局
  21:9+: HUD居中(保持16:9区域)，两侧扩展显示游戏世界
  4:3: 垂直适配(缩小UI)
  
UI缩放: 
  独立于分辨率
  0.8x(小) / 1.0x(标准) / 1.2x(大) / 1.5x(特大)
  快捷键: Ctrl+滚轮调整
```

## 二、色盲模式

```
三种预设:
  红色弱(Protanopia): 红→蓝偏移
  绿色弱(Deuteranopia): 绿→蓝偏移  
  蓝黄色弱(Tritanopia): 蓝→黄偏移

受影响的关键UI:
  友方/敌方颜色(纯红绿→红+圆形标记/蓝+菱形标记)
  血条(红→橙红)
  成功/失败提示(纯绿红→绿+✓/红+✗)

实现:
  后处理颜色校正LUT(查找表)
  关键信息叠加图标+纹理(不纯依赖颜色)
```

## 三、辅助功能

```
视觉辅助:
  字幕: 语音转文字(可开关/可调节大小/可调节背景透明度)
  字幕颜色: 按说话者颜色编码(方便识别谁在说话)
  过场动画: 音频描述轨道(可选)
  高对比度模式: 提高UI对比度(白底黑字)

听觉辅助:
  声音可视化: 战斗中的攻击方向指示器(屏幕边缘)
  重要音效的视觉提示: 字幕中标注[兵器碰撞][马蹄声]
  对话历史: 可回看最近20条对话文本

操作辅助:
  按键保持/连发: 可配置(减轻重复按键)
  鼠标灵敏度: 0.5x-3.0x
  手柄死区调节: 摇杆/扳机
  完全键盘操作: 支持(无鼠标模式)
  眼动追踪: 可选支持(Tobii)

认知辅助:
  教程强度: 详细/标准/最少/关闭
  提示频率: 频繁/标准/稀少/关闭
  复杂操作引导: 分步提示(可跳过)
  术语解释: 悬停关键词→弹出简短解释
```

## 四、本地化适配

```
支持语言:
  简体中文(默认)/繁体中文/English/日本語/한국어

UI适配注意事项:
  英文文本长度: 比中文长30-50% → 按钮/面板预留30%宽度
  日语: 混用汉字+假名 → 字体需包含日文字符集
  韩语: 谚文字符 → 字体需包含韩文字符集
  阿拉伯语(未来): RTL布局支持 → 镜像UI

字体回退:
  主字体(篆书)无对应字符时 → 回退到仿宋→系统默认
  确保无"豆腐块"(缺失字符显示为□)

日期/时间格式:
  中文: 秦王政二十年·秋·九月
  English: September, 20th Year of King Zheng of Qin
  日本語: 秦王政二十年·秋·九月
  한국어: 진왕 정 20년·가을·9월

数字格式:
  中文/日/韩: 12,500金
  English: 12,500 Gold
```

## 五、无障碍测试

```
测试范围:
  □ 纯键盘可完成所有操作
  □ 屏幕阅读器可读取关键信息(Windows Narrator)
  □ 色盲模式下信息可辨识
  □ 高对比度模式下UI清晰可见
  □ 最小分辨率(1280×720)下信息不丢失
  □ 所有可操作元素有明确的焦点状态
  □ 过场动画有字幕(可开关)

标准参考:
  WCAG 2.1 AA级(Web内容无障碍指南，适用于游戏)
  Xbox Accessibility Guidelines
```
