## 三、内政指令界面

```

┌────────────────────────────────────┐
│ 可用指令 (本月剩余行动: 3/5)       │
│                                    │
│ [开垦农田] 500金+200劳力 → 农业+5  │
│ [发展商业] 800金 → 商业+4          │
│ [修筑城墙] 1000金+300石 → 城防+8   │
│ [安抚民心] 300金+50粮 → 民忠+5     │
│ [搜索人才] 200金 → 概率发现在野    │
│ ...                                │
│                                    │
│ 执行官员: 王绾(经营72)影响效率     │
│ 预计效果: 农业+5(基础)+3(官员)=+8  │
│ [执行]                              │
└────────────────────────────────────┘

绿色=推荐(数值缺口大)/黄色=可选/灰色=条件不足
```
