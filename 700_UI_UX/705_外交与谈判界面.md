# 外交与谈判界面设计

## 一、外交总览 (D键)

```
┌─────────────────────────────────────────────────┐
│ [外交总览] [派遣使者] [会盟] [间谍网络]         │
├─────────────────────────────────────────────────┤
│                                                 │
│         ┌─关系网可视化(中央)────────┐           │
│         │         ·燕                │           │
│         │        /                   │           │
│         │   赵──你(秦)──齐           │           │
│         │   /    |    \              │           │
│         │  魏   韩    楚             │           │
│         │              |             │           │
│         │             周             │           │
│         └────────────────────────────┘           │
│                                                 │
│ 选中: 楚国  关系: 敌对(-45)                     │
│ 条约: 无  战争状态: 否  (战争一触即发!)         │
│                                                 │
│ [提议条约] [送礼] [宣战] [间谍] [详情]          │
└─────────────────────────────────────────────────┘

图例:
  绿色连线=盟友 / 红色=战争 / 黄色虚线=条约
  连线粗细=关系紧密程度
  节点大小=国力强弱
```

## 二、谈判界面

```
选择"提议条约" → 进入谈判:

┌─────────────────────────────────────────────────┐
│ 秦国(你)                       楚国(对方)       │
│                                                 │
│ 提议: 互不侵犯条约 (24月)                       │
│                                                 │
│ 你的条件:                     对方要求:         │
│ [每年进贡1000金]              [你方割让武关]    │
│ [联姻:嫁女]                   [你方质子:送子]   │
│ [+添加条件]                   [+讨价还价]       │
│                                                 │
│ 预计对方接受率: 62%                             │
│ ████████████░░░░░░░░                            │
│                                                 │
│ 使者: 张仪(魅力95)           预计耗时: 20天     │
│ [派遣使者]  [亲自出使]  [取消]                  │
└─────────────────────────────────────────────────┘

交互:
  点击添加条件 → 从条件池中选择
  拖动条件 → 调整优先级
  对方反应: 实时显示接受率变化
  接受率颜色: 绿(>70%)/黄(40-70%)/红(<40%)
```
