## 三、军团指令面板(F键菜单)

```
按F键后 → 底部展开指令菜单:

┌─────────────────────────────────────────────────┐
│ F1移动  F2阵型  F3射击  F4姿态  F5战术  F6委任 │
├─────────────────────────────────────────────────┤
│ [前往位置] [急行军] [推进]  [跟随]  [撤退]      │
└─────────────────────────────────────────────────┘

选中编队高亮 → 指令对该编队生效
不选编队(全选) → 指令对全军生效
```

## 四、伤害与反馈

```
伤害数字飘字:
  普通: 白色数字，向上飘+淡出
  暴击: 金色大字+震动+特效
  治疗: 绿色数字
  
受击指示器:
  屏幕边缘红色方向指示(哪个方向被攻击)
  生命<30%: 屏幕边缘持续泛红+心跳音

击杀提示:
  中央上方: "击杀 [敌将名]" (金色)
  连续击杀: "三连斩!" (叠加提示)
```

## 五、小地图

```
圆形小地图(右上角):
  显示: 周围200m地形+己方(绿色)/敌方(红色)/中立(灰色)
  玩家位置: 中心金色三角
  任务目标: 金色标记(+距离)
  敌人方向: 红色扇形(仅显示可视范围内的)
  
缩放: 滚轮在小地图区域 → 缩放小地图
点击: 打开完整地图
```
