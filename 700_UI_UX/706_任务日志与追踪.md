## 四、任务地图标记

```
大地图上的任务图标:

🟡 金色感叹号 = 主线任务位置
🔵 蓝色标记 = 追踪任务位置
⚪ 白色问号 = 可接任务来源(酒馆/官府)
🔴 红色闪烁 = 紧急事件位置
🟣 紫色标记 = 隐藏任务线索
✅ 绿色勾 = 已完成可在此交付
❌ 灰色叉 = 失败/过期
```
