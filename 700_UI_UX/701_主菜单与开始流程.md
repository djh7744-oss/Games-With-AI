# 主菜单与开始流程

## 一、主菜单

```
背景: 动态战国地图(水墨风格渲染)
  镜头缓慢横移 → 展示七国领土
  竹简碎片粒子从屏幕上方飘落
  编钟+战鼓背景音乐《乱世风云》

主菜单按钮(竖排居中):
  ┌─────────────┐
  │  新 游 戏   │ (篆书大字, 金色悬停)
  │  继 续     │
  │  加载存档   │
  │  设 置     │
  │  DLC管理   │
  │  制 作 组   │
  │  退 出     │
  └─────────────┘

版本号/版权信息: 左下角小字
```

## 二、新游戏流程

```
Step 1: 选择性别
  男/女 → 影响社会待遇(战国真实存在性别差异)
  提示: "在战国，男女的社会角色差异显著。但你的命运由你决定。"
  选择后→下一步

Step 2: 选择出身 (6张出身卡)
  视觉: 6张竹简卡片平铺
  每张卡: 立绘+出身名+难度星级+一句话简介
  悬停: 卡片浮起+属性修正预览
  点击: 卡片翻转→展示详细出身信息
  [贵族 ★☆☆☆☆] [士人 ★★★☆☆] [商人 ★★★☆☆]
  [游侠 ★★★★☆] [工匠 ★★★☆☆] [农民 ★★★★★]
  
  详见 [121_出身系统]

Step 3: 分配属性点
  可分配点数: 15 + 3技能专精点
  六维雷达图: 中央显示，拖动或点击调整
  出身基础值: 已锁定(出身决定)
  推荐分配: 可点击"推荐"按钮→系统给出建议

Step 4: 选择起始国家
  地图展示: 七国版图高亮
  点击国家→显示该国简介+推荐出身
  选择后→该国边境高亮
  提示: "你可以加入其他国家，但起始位置在这里。"

Step 5: 命名与面容
  姓名: 输入框(可随机生成符合时代的名字)
  性别对应的称谓: (男: 氏+名 / 女: 姓+名)
  面容定制: 
    预设脸型×8 + 微调滑块(发型/胡须/肤色/伤疤/体型)
  预览: 3D角色旋转展示

Step 6: 确认
  展示全部选择的汇总
  出身/国家/属性/面容
  确认→开场CG→游戏开始
```

## 三、加载/存档界面

```
存档列表:
  每个存档显示:
    角色名+出身+当前日期(游戏时间)
    当前位置/当前爵位/游戏时长
    缩略图(自动截图的游戏画面)
  最多10个存档 + 1个自动存档

存档操作:
  点击→加载(确认弹窗)
  右键→删除(需确认)
  云存档图标(Steam Cloud同步状态)

新游戏: 覆盖空存档位或新建
```

## 四、设置界面

```
分类Tab:
  游戏/画面/音频/控制/辅助功能

游戏设置:
  难度: 简单/普通/困难/极难(不可在游戏中更改)
  战斗规模: 小型(100人)/中型(500)/大型(1000)/史诗(3000)
  时间流速: 0.5x/1x/2x
  暂停模式: 菜单时暂停/从不暂停
  自动存档: 每15/30/60分钟/关闭
  提示频率: 详细/标准/最少/关闭

画面设置:
  分辨率/窗口模式/垂直同步/FPS上限
  画质预设: 低/中/高/极高/自定义
  视野距离/阴影质量/纹理质量/特效质量
  亮度/对比度

音频设置:
  主音量/音乐/音效/语音/环境
  动态范围: 完整/夜间模式(压缩)
  扬声器配置: 立体声/5.1/7.1/耳机虚拟环绕

控制设置:
  键鼠/手柄/自定义按键映射
  鼠标灵敏度/反转Y轴
  手柄振动: 开/关/强度调节

辅助功能:
  见 [708_适配与辅助功能]
```
