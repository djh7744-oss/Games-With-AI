# UI组件库与设计系统

## 一、设计令牌 (Design Tokens)

```
色彩:
  主色: 竹简棕 #2a1a0a / 绢帛米 #f5e6c8
  强调: 朱砂红 #c43a2a / 青铜金 #8B7500
  功能: 成功绿 #3a8a3a / 警告橙 #c47a2a / 危险红 #8B0000

间距 (基于4px网格):
  xs: 4px / sm: 8px / md: 16px / lg: 24px / xl: 32px / xxl: 48px

圆角:
  sm: 2px (按钮) / md: 4px (面板) / lg: 8px (弹窗)

阴影:
  sm: 0 2px 4px rgba(0,0,0,0.3) (卡片浮起)
  md: 0 4px 8px rgba(0,0,0,0.4) (面板)
  lg: 0 8px 16px rgba(0,0,0,0.5) (弹窗遮罩)

字体:
  标题: 篆书风格 / 正文: 仿宋 / 数字: 等宽
  字号: h1=48px / h2=32px / h3=24px / body=18px / caption=14px
  行高: 1.5(正文) / 1.2(标题)
```

## 二、组件库

```
按钮(Button):
  变体: 主按钮(CTA)/次按钮/文字按钮/危险按钮/图标按钮
  状态: 默认/悬停(金色描边)/按下(缩进2px)/禁用(灰色)
  尺寸: sm(24px)/md(32px)/lg(48px)
  带图标: 图标+文字 水平排列

面板(Panel):
  变体: 标准面板/弹出面板/侧滑面板/全屏面板
  标题栏: 深色木纹理+篆书标题+关闭按钮(右上)
  内容区: 可滚动
  拖拽: 标题栏可拖拽移动(非模态面板)

输入框(Input):
  变体: 单行文本/多行文本/数字/搜索
  状态: 默认/聚焦(金色边框)/错误(红色边框+提示)
  带前缀图标: 搜索框前的🔍

下拉菜单(Dropdown):
  展开方向: 向下/向上(根据屏幕空间自适应)
  选项: 文字+可选图标
  分组: 分隔线+组标题

滑块(Slider):
  水平滑块+数值显示
  范围: 显示最小值/当前值/最大值

进度条(Progress Bar):
  变体: 竹简展开式/填充式
  颜色: 绿色(正向)/红色(危险)/蓝色(进度)
  带标签: 百分比+文字描述

标签(Tab):
  水平排列(上)/垂直排列(左)
  选中态: 下划线(金)+文字高亮

工具提示(Tooltip):
  延迟0.5s出现，跟随鼠标
  背景: 深色半透明+文字

弹窗(Modal):
  居中遮罩(70%黑)
  标题+内容+按钮(确认/取消)
  Esc关闭/点击遮罩关闭(可配置)

通知(Toast):
  位置: 右上角(默认)/顶部居中
  类型: 信息/成功/警告/错误
  自动消失: 3-5秒
  可手动关闭
```

## 三、动画规范

```
页面过渡:
  打开: 从右滑入(300ms, ease-out)
  关闭: 向右滑出(200ms, ease-in)
  弹窗: 缩放弹出(200ms, spring)

微交互:
  按钮悬停: 缩放1.05+金色描边(150ms)
  按钮按下: 缩放0.95(50ms)
  数值变化: 数字滚动动画(500ms)
  进度填充: 平滑增长(300ms)

加载:
  加载指示器: 编钟旋转(3D)+"加载中..."
  骨架屏: 灰色占位块→内容淡入(500ms)
```

## 四、组件使用规范

```
按钮层级:
  一个面板中: 最多1个CTA按钮(金色)+最多2个次按钮+最多1个危险按钮
  
表单验证:
  实时验证: 输入时即检查
  错误提示: 红色边框+下方文字说明
  提交前: 汇总所有错误→焦点第一个错误字段
  
空状态:
  列表为空: 插画+"暂无数据"+"去XXX"引导按钮
  搜索无结果: "未找到'XXX'的相关结果"
  
加载状态:
  首次加载: 骨架屏
  刷新: 下拉刷新(移动端)/顶部加载条
  操作中: 按钮显示loading(禁用+旋转图标)
```
