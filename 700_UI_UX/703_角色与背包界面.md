# 角色与背包界面设计

## 一、角色面板 (C键)

```
┌─────────────────────────────────────────────────┐
│ [装备] [属性] [技能] [声望] [关系] [家族]  ←Tab │
├─────────────────────────────────────────────────┤
│                                                 │
│  ┌──────┐   六维雷达图        装备槽位           │
│  │      │     统率 85        头盔 [精铁盔]      │
│  │ 3D   │   武勇╱╲72        身甲 [鱼鳞甲]      │
│  │ 角色 │  智略╱  ╲65       武器 [精铁剑]      │
│  │ 预览 │ 魅力╱    ╲58      盾牌 [铜盾]        │
│  │      │ 经营╲    ╱45      护臂 [铁护臂]      │
│  │      │ 工艺╲  ╱38        护腿 [铁护腿]      │
│  └──────┘      ╲╱           坐骑 [战马]        │
│                                马甲 [无]         │
│  姓名: 赵括                                     │
│  出身: 贵族  爵位: 大夫                         │
│  等级: 24 (45,200/50,000)                       │
│  生命: 680  体力: 120                           │
│  负重: 18.5/35 (石)                             │
│                                                 │
│  [装备对比] [属性历史] [换装方案]               │
└─────────────────────────────────────────────────┘

装备Tab:
  双击装备槽→打开装备选择(显示背包中可装备的物品)
  右键装备→卸下/装备对比/丢弃
  
属性Tab:
  详细属性列表(含加成来源)
  每项属性旁: 成长进度条+冷却指示器
  近期属性变化曲线图(小)

技能Tab:
  六条技能树→点击展开
  已学节点=亮色/可学=闪烁/锁定=灰色
  剩余技能点: 显示在顶部
  重置按钮(需要金+冷却)

声望Tab:
  仁义/信义/武名/霸业 四维条
  每项显示: 当前值+最近变化+特殊事件阈值

关系Tab:
  NPC关系列表(按好感排序)
  关系网络可视化(节点+连线)
  点击NPC→查看详细关系+历史互动

家族Tab:
  家族树(配偶+子女+父母)
  继承人标记
  家族资产总值
```

## 二、背包界面 (I键)

```
┌─────────────────────────────────────────────────┐
│ [全部] [武器] [防具] [消耗品] [材料] [任务]     │
├─────────────────────────────────────────────────┤
│ 格状背包 (6×8=48格)                             │
│ ┌──┬──┬──┬──┬──┬──┐     重量: 18.5/35          │
│ │剑│盾│药│药│..│..│     价值: 45,200金         │
│ ├──┼──┼──┼──┼──┼──┤                           │
│ │..│..│..│..│..│..│    [整理] [筛选] [搜索]    │
│ └──┴──┴──┴──┴──┴──┘                           │
│                                                 │
│ ┌─选中物品详情─────────────────────┐            │
│ │ [精铁剑] 品质: 绿色              │            │
│ │ 伤害: 42  速度: 0.35  破甲: 12% │            │
│ │ 耐久: 280/300  重量: 0.9石      │            │
│ │ 特殊: 武勇+2                     │            │
│ │                                  │            │
│ │ [装备] [丢弃] [修理] [对比]      │            │
│ └──────────────────────────────────┘            │
│                                                 │
│ 快捷栏: [1]剑 [2]盾 [3]弓 [4]药 [5]空 [6]空    │
└─────────────────────────────────────────────────┘

排序: 品质/类型/名称/重量/最近获得
筛选: 点击Tab(全部/武器/防具/...) + 品质筛选(白/绿/蓝/紫/金)
搜索: 输入文字过滤
整理: 自动堆叠同类物品+排列
```

## 三、装备对比

```
选中背包中武器 → 点击"对比" →

┌─────────────────────────────────────────────┐
│         当前装备              候选装备       │
│         [精铁剑]             [百炼钢剑]     │
│         伤害: 42   ←  →     伤害: 58       │
│         速度: 0.35          速度: 0.32      │
│         破甲: 12%   ↗       破甲: 18%       │
│         耐久: 280/300        耐久: 420/450   │
│                                             │
│  变化: 伤害+16 速度+0.03 破甲+6%            │
│  [装备此物品]  [取消]                       │
└─────────────────────────────────────────────┘

绿色=提升 / 红色=下降 / 白色=不变
```
