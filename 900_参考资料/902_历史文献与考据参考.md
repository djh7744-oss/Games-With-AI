## 二、考古与实物参考

### 兵器

```
秦始皇兵马俑 (陕西临潼):
  出土: 青铜剑/戟/矛/弩机/箭镞/盾
  参考: 
    - 兵器标准化程度(箭镞几乎一模一样→秦标准化生产)
    - 青铜剑表面镀铬(2000年不锈→工艺先进)
    - 弩机结构(精密机关→机械设计)

曾侯乙墓 (湖北随州):
  出土: 编钟/兵器/漆器
  参考: 
    - 战国早期青铜工艺巅峰
    - 编钟音律(跨五个八度→音乐系统参考)

江陵楚墓 (湖北荆州):
  出土: 越王勾践剑/楚国兵器/漆器
  参考:
    - 越王剑千年不锈(金属工艺)
    - 楚国漆器花纹(UI纹样参考)
```

### 城池与建筑

```
燕下都遗址 (河北易县):
  城墙遗存: 夯土城墙底宽40m，残高6-8m
  参考: 战国都城规模感

赵邯郸故城 (河北邯郸):
  丛台遗存: 赵武灵王观看军事操演的高台
  参考: 军事化城池布局

齐临淄故城 (山东淄博):
  城址面积: 约15km²(战国最大城市之一)
  参考: 商业城市布局/稷下学宫位置

楚郢都纪南城 (湖北荆州):
  城址: 长方形，面积约16km²
  参考: 水门/码头/南方城池特色

秦咸阳遗址 (陕西咸阳):
  宫殿区: 高台建筑，规模宏大
  参考: 秦国建筑风格(厚重/实用/标准化)
```

### 货币与器物

```
货币实物参考:
  秦半两(圆形方孔) → 标准化程度最高
  齐刀币(刀形) → 铸造精美
  楚蚁鼻钱(贝形) → 南方特色
  楚郢爰(金版) → 最早的黄金货币之一
  赵/魏/韩布币(铲形) → 三晋通用

器物参考:
  青铜器: 礼器/兵器/生活用具
  漆器: 楚国最为精美(彩绘/镶嵌)
  玉器: 礼器/佩饰/葬玉
  陶器: 灰陶/印纹硬陶/原始瓷器
  丝织品: 绢/锦/刺绣(参考马王堆汉墓)
```

## 三、参考书目推荐(团队)

```
必读:
  1. 司马迁《史记》(战国部分) — 了解人物和事件
  2. 《战国策》 — 了解外交辞令和纵横术
  3. 杨宽《战国史》 — 现代学者最全面的战国通史

选读:
  4. 李学勤《东周与秦代文明》 — 考古视角
  5. 杜正胜《编户齐民》 — 社会制度
  6. 许倬云《中国古代社会史论》 — 社会结构
  7. 孙机《汉代物质文化资料图说》 — 物质生活参考

影视辅助:
  - 纪录片《复活的军团》(兵马俑)
  - 纪录片《楚国八百年》
  - 电视剧《大秦帝国》系列(注意区分历史与演绎)

博物馆推荐:
  - 秦始皇兵马俑博物馆(西安)
  - 湖北省博物馆(编钟/越王剑)
  - 荆州博物馆(楚文化)
  - 河北省博物馆(中山国)
  - 国家博物馆(中国古代陈列)
```
